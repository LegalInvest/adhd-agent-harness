# ADHD 极致研究 — 执行计划

## 目标
对 ADHD 进行穷尽性深度分析，覆盖四个维度：
1. ADHD 是否是“超能力”？—— 重新定义与能力取向视角
2. “超聚焦”（Hyperfocus）推动人类突破的历史与科学证据
3. “正常人方法论”对 ADHD 为何有害？穷尽清单与机制分析
4. 所有论述的科学依据与文献支撑

## Stage 1 — 深度研究（并行多维度调查）
**加载技能**: `deep-research-swarm`

**研究子任务**（并行）：
- **研究员A — 重新定义视角**: ADHD as neurodiversity / superpower debate, giftedness research, cognitive strengths literature (creativity, divergent thinking, resilience)
- **研究员B — 超聚焦与历史突破**: 达芬奇、马斯克等案例的史实核查；hyperfocus 的神经科学机制；ADHD 与创业/创新/科技突破的实证研究
- **研究员C — 有害方法论穷尽**: 所有“正常人方法论”对 ADHD 有害的清单 — 时间管理、自律系统、传统教育、番茄钟、冥想正念、CBT适应性、药物争议等
- **研究员D — 科学依据与机制**: ADHD 的神经生物学基础（多巴胺、去甲肾上腺素、前额叶皮质）、双重路径模型、reward deficiency theory、基因研究

**输出**: 4份研究报告，每份带引用来源

## Stage 2 — 交叉验证与整合
- 检查各研究员发现的一致性与矛盾
- 补充缺失证据
- 形成统一研究简报

## Stage 3 — 报告撰写
**加载技能**: `report-writing`
- 基于 Stage 2 简报，撰写完整长篇分析报告
- 结构：引言 → 重新定义 → 超聚焦机制与历史证据 → 有害方法论穷尽清单 → 神经科学基础 → 整合视角 → 结论

## Stage 4 — 格式化输出
**加载技能**: `docx`
- 将最终报告转为 .docx 格式交付
