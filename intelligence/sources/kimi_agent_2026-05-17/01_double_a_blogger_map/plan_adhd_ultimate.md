# Plan.md — ADHD全史文献计量学终极考古

从ADHD概念诞生(1902)到2026年，穷尽全球学术数据库、名人数据库、隐藏规律，进行纵向历时性+横向共时性的极致分析。

## Stage 1: 纵向历时性考古 — ADHD概念史全梳理
- 1902 Still → 1957 Laufer → 1968 DSM-II → 1980 DSM-III → 1987 DSM-III-R → 1994 DSM-IV → 2013 DSM-5 → 2022 DSM-5-TR
- 关键论文、关键人物、关键争议
- 使用scholar/arXiv搜索里程碑论文
- 文献计量学：citation burst, keyword evolution

## Stage 2: 横向共时性考古 — ADHD×AI/AIGC交叉文献
- 所有ADHD+artificial intelligence论文
- 所有ADHD+machine learning论文
- 所有ADHD+large language model论文
- 所有neurodiversity+AI论文
- 使用scholar搜索，获取真实论文数据

## Stage 3: ADHD名人数据库穷尽
- 政治领袖、科学家、艺术家、企业家、运动员、创作者
- 确诊ADHD vs 疑似/回顾性诊断
- 使用web_search穷尽全网名单

## Stage 4: 隐藏规律与交叉分析
- ADHDer的创造性爆发模式
- ADHD与创业/创新/冒险行为的量化关系
- ADHD在不同历史时期的社会认知变迁
- ADHD与AI革命的结构性共振
