## 391. 为什么 ADHD 是注意力经济的工伤，不是病？

主标题：ADHD 是注意力经济的工伤，不是病

备选标题：

- 当平台经济靠打断你赚钱，无法专注到底是谁的缺陷？
- 你的注意力不是坏了，是被反复征用了
- 为什么 AI 时代的 ADHD 不该只被放进医院，也该被放进劳动法

事实弹药：

- 知识工作者平均每天切换 App 约 1200 次。
- 小红书 ADHD 话题浏览量巨大，“自测确诊、社媒治疗”现象泛滥，但网络量表可靠性存疑。
- Tangle 等缓解型产品被部分 ADHD 打工人追捧，但对成人 ADHD 的科学证据边界仍需谨慎。

因果链：

平台设计依赖打断和刷新 -> 注意力碎片化成为商业模式 -> 正常人出现 ADHD 样症状 -> 自我诊断潮扩散 -> 焦虑被消费品接住 -> 产业闭环完成。

## 392. 为什么 ADHD 流行标签最容易伤到真正需要帮助的人？

主标题：当 ADHD 成为流行标签，真正患病的人反而更难被看见

备选标题：

- “自我提升”产业最大的受害者，是被错误脚本困住的人
- ADHD 知识越流行，误诊和漏诊为什么可能同时增加？
- 为什么女性 ADHD 的晚诊，不是一个医学小误差

事实弹药：

- 女性 ADHD 平均确诊年龄比男性更晚，晚诊造成青春期、自我认同、职业与关系层面的长期损害。
- 童年期男孩诊断率显著高于女孩，成年后男女比例更接近。
- 社交媒体普及 ADHD 知识的同时，也会把复杂症状压缩成流行标签。

因果链：

社交媒体普及 ADHD -> 大量自诊与标签化 -> 女性/高功能/代偿型人群继续被忽视 -> 晚诊累积损害 -> 确诊时已从预防变成修复 -> 误诊和漏诊同时扩大。

入口资产：ADHD 误诊/漏诊边界问题清单。

发布优先级：A。

正文状态：待写。

ADHD 启动困难 -> AI 接管任务拆解 -> 完成率提升 -> 大脑不再经历“拖延到启动”的完整过程 -> 时间体感弱化 -> 功能性依赖形成 -> 需要重新设计手动入口。

- 为什么企业买了包容性培训，却没有改变招聘结构？

- 神经多样性产业规模快速增长。

因果链：

神经多样性意识提升 -> 产业规模膨胀 -> 企业购买培训和工具 -> 招聘结构未变 -> 支出变成品牌公关 -> 真正需要工作的人没得到工作 -> 产业继续增长 -> 问题被无限续杯。
