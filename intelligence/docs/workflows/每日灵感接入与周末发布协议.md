# 每日灵感接入与周末发布协议

更新时间：2026-05-21

## 一句话判断

作者每天都会有新的 ADHD / AI / Agent 思考，这是项目的燃料，不应该被压制。Agent 的职责不是劝作者少想，而是把持续放电接入一个不会冲垮发布系统的节律：平日接入、编号、去重、成稿、模板化；周末统一进入微信公众号草稿箱并设置下一周定时发布。

## 平日：灵感接入规则

作者可以随时发送：

- 一句话灵感
- 一个比喻
- 一段自我观察
- 一个评论区反馈
- 一个产品念头
- 一个法律/职业/投资/AI 的跨域连接
- 一个情绪爆点或失败现场

Agent 收到后默认做 6 件事：

1. 判断它属于：新选题、旧题补强、模板入口、发布包素材、产品验证、反馈回流。
2. 与现有 `001-260` 去重，避免同义反复。
3. 如果是新题，给出下一个编号，并补齐真实痛点、产品入口、验证动作。
4. 如果是旧题补强，写入对应文章或发布包素材。
5. 如果高信号，直接写成正文或模板，不只停在标题。
6. 更新索引、质检、项目状态或周末发布队列。

## 编号规则

- 新题继续从 `261` 往后编号。
- 只有满足至少一个条件，才进入新编号：
  - 有新痛点
  - 有新比喻
  - 有新产品入口
  - 有新目标人群
  - 有新发布角度
  - 有真实评论/反馈支撑
- 如果只是旧题换说法，默认并入旧题，不新开编号。

## 周末：公众号发布规则

周末由作者上传到微信公众号草稿箱，并设置下一周定时发布。

Agent 负责提前准备：

- 公众号长文版
- 公众号标题/副标题
- 导语
- 结尾评论区问题
- 小红书痛点版
- 模板/产品入口
- 48 小时回流记录表

## 周发布节律建议

一周建议 5 篇：

- 周一：高共鸣主炮文
- 周二：模板/工具入口文
- 周三：旧粉迁移或高压职业桥接文
- 周四：AI/Agent 工作流文
- 周五：产品化/复盘/回流文

周末不追求临时写新文，周末只做排期、检查入口、设置回流表。

## 防失控规则

- 允许每天新增灵感，但每周至少推进 1 个可复用入口。
- 允许扩题，但新题必须带产品入口或验证动作。
- 允许自嗨，但每次高能扩展后至少完成一个 15 分钟现实动作。
- 不用“不要扩题”压制作者；用“扩题后自动接入发布和产品化”驯化发散。

Done：每日灵感与周末发布节律已固化。
