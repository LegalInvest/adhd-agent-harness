## 安全边界

脚本主动跳过：

- auth / token / password / secret / API key 相关文件；
- SQLite / 数据库文件；
- 浏览器用户数据目录；
- 微信认证/数据目录；
- 缓存、插件包、临时目录和工具内部运行噪声。

页面只展示脱敏后的文件名、标题和摘要，不输出凭据或认证内容。

## 复跑命令

```powershell

```

## 用法

这不是严格审计报告，而是“成就回放系统”：用于帮 ADHD 大脑把过去一年已经完成、生成、交付、沉淀的资产重新登记成可见证据。

## 关于空白日期

页面中的“无本地完成证据”不等于当天没有工作、没有思考、没有阅读或没有聊天。它只表示：在当前这台电脑、当前可读取且未被安全规则排除的数据里，没有找到足够明确的本地文件产出或可解析会话完成线索。

常见原因：

- 当天主要活动发生在手机、微信、网页、云端或外部平台，没有落到本地文件。
- 聊天记录不在本机，或本机只保留了 2026-04-30 之后的一部分 Claude / Codex / Cursor 会话。
- 文件经过复制、解压、下载、同步后，原始完成日期被覆盖成复制日期。
- 为了安全，脚本跳过了浏览器用户数据、微信数据、数据库、认证文件、缓存和插件目录。
- 读书、思考、沟通、研究、决策等真实进展，未必会形成可被自动识别的“完成项”。
