# Style Guide

## 文风

- 中文为主。
- 强判断、强比喻、强机制解释。
- 读者要感觉“这不是鸡汤，是我大脑的说明书”。
- 避免模板化、空泛励志和泛心理学安慰。

## 常用结构

1. 先给一个具体场景。
2. 再给反常识判断。
3. 展开机制。
4. 给铁证。
5. 落到一个小动作或 Agent 接口。

## 适合保留的表达

- 你不是懒，是系统接口错了。
- ADHD 不需要更努力，需要外部执行系统。
- Agent 不是工具，是外部前额叶。
- 没有反馈回流的想法只是爽感，不是现金流。

## 需要警惕

- 不要把每篇文章都写成同一套“你不是 X，你是 Y”。
- 不要只用 AI、Agent、前额叶这些词堆概念。
- 不要把痛苦过度浪漫化。
- 不要给医疗诊断或药物建议。
