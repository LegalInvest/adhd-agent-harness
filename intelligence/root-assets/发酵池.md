# 发酵池 v1.0

> 用途：跟踪从"倾倒"到"成文"的素材发酵状态。
> 规则：每次对话结束后，把有价值的素材按模板录入。不要筛选——让未来的你决定要不要写。
> 存放位置：此文件常驻桌面，每次打开即可追加。

---

## 一、成熟度定义

| 成熟度 | 判断标准 | 下一步动作 |
|---|---|---|
| **生** | 只有一个钩子/金句/疑问，无机制，无铁证 | 继续倾倒，等待关联素材出现 |
| **半熟** | 有钩子 + 初步机制，但缺铁证或个人故事 | 标记关联选题，等待补充素材 |
| **熟** | 有收敛型钩子 + 穿透式机制 + 至少1个铁证 + 个人故事 | 可成文，按需变压到目标接口 |
| **过熟** | 已写成文章并发布 | 移入"已归档"区，只保留索引 |

---

## 二、新素材录入模板

复制以下模板，在"待发酵素材"区追加：

```
### 素材编号：[字母+数字，如 F021]

- **来源**：[对话/阅读/观察/经历，具体到session或日期]
- **原始倾倒**：[粘贴原始想法，不要整理]
- **成熟度**：[生/半熟/熟]
- **关联选题**：[象限编号+题号，如 007、131–140]
- **可成文方向**：
  - 《问题XXX：标题？》
  - 或：变压到小红书/知乎/播客/短视频
- **情绪安全检查**：[如涉及抑郁/创伤/争议话题，标注风险等级：绿/黄/红]
```

## 六、反模板化检查清单（每次变压前必过）

> 来源：009 的核心教训。每次把素材变成文章前，逐条核对。

- [ ] **语流检查**：文章里有没有 3 个以上"AI 不会这样写"的句子？
- [ ] **段落呼吸**：长段落和短段落是否交替出现，还是均匀得像装配线？
- [ ] **数字分隔**：是否过度使用 01/02/03 机械分隔？显式标题是否足够？
- [ ] **怪念头存活**：最初倾倒时最跳跃、最不像"正确文章"的想法，还在不在正文里？
