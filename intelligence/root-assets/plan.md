# 全球AI博主情报蒸馏作战计划

## 目标
1. 建立【全球前100 + 中国前50】全网社交媒体平台表格
2. 逐平台定位大厂AI博主/KOL（非公众号的中文平台 + 海外平台）
3. 基于公开行为特征分析疑似ADHD博主比例与特征
4. 输出【ADHD特质爆火AI内容发布策略】

## Stage 1 — 全球社交媒体平台梳理（并行搜集）
- **Agent 全球平台研究员**: 搜集全球主流社交媒体平台Top100（按MAU/用户量），覆盖Twitter/X、YouTube、LinkedIn、Instagram、TikTok、Reddit、Substack、Medium、Discord、Telegram、Threads、Bluesky、Mastodon等
- **Agent 中国平台研究员**: 搜集中国主流社交媒体平台Top50（按MAU），覆盖微信（公众号/视频号）、抖音、小红书、B站、微博、知乎、快手、即刻、豆瓣、斗鱼、虎牙、头条、百家号、网易号、搜狐号、雪球等
- **输出**: 两张表格，含平台名称、MAU、核心用户画像、AI内容生态成熟度、是否适合AI博主发展

## Stage 2 — 逐平台AI大厂博主定位（并行搜集）
基于Stage 1的平台列表，分组并行搜索：
- **Agent 海外平台搜捕队A**: Twitter/X + YouTube + LinkedIn + Substack上的头部AI博主
- **Agent 海外平台搜捕队B**: Reddit + Discord + Medium + Bluesky + Threads + Mastodon上的AI社区/博主
- **Agent 中文平台搜捕队A**: 抖音 + 小红书 + B站 + 知乎上的AI大厂博主
- **Agent 中文平台搜捕队B**: 微博 + 快手 + 即刻 + 头条上的AI博主/KOL
- **Agent 大厂员工博主专项组**: 专门搜索各大厂（OpenAI、Google DeepMind、Meta AI、Anthropic、字节、阿里、百度、腾讯、华为、小米等）员工在社交媒体上活跃的AI博主
- **输出**: 每个平台10-20个头部AI博主，含账号名、平台、粉丝量、大厂背景、内容风格、更新频率

## Stage 3 — ADHD特质行为分析（并行分析）
- **Agent ADHD行为分析师A**: 分析Twitter/X上AI博主的ADHD疑似特征（发文频率/时段分布、话题跳跃度、回复模式、内容碎片化程度、情绪波动、多线程输出等）
- **Agent ADHD行为分析师B**: 分析中文平台AI博主（数字生命卡兹克、赛博禅心等已知账号）的ADHD行为特征
- **Agent ADHD产业研究**: 搜集科技行业ADHD研究、著名ADHD科技创业者/博主案例（如Elon Musk公开讨论过ADHD倾向等）
- **输出**: ADHD疑似博主清单 + 行为特征模式总结 + 科技行业ADHD比例估算

## Stage 4 — ADHD爆火内容策略输出
- **Agent 策略设计师**: 基于Stage 2-3的分析，设计"ADHD特质每日10篇爆火AI文章"发布策略
- **输出**: 完整策略文档，包含内容类型矩阵、时间节奏、平台适配、标题公式、互动策略

## Stage 5 — 整合输出（主Agent执行）
- 汇总所有表格、分析、策略
- 生成最终 .md + .xlsx + .docx 输出
- 交付：平台地图 + 博主情报网 + ADHD分析 + 爆火策略

## 执行纪律
- 所有Agent使用浏览器+搜索工具进行实时数据采集
- 允许浏览器访问社交媒体公开页面
- 每轮搜索需交叉验证至少2个来源
- 输出格式标准化：Markdown表格为主，便于整合
