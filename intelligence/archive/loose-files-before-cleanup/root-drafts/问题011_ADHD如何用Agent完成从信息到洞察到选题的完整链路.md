# 《问题011：ADHD 如何用 Agent 完成"从信息到洞察到选题"的完整链路？》

---

**【收敛型问题】**

010 我们讲了抑郁如何意外启动了 ADHD 的自我监控。

今天讲一个更实际的问题：你每天刷几百条信息，收藏几十个视频，脑子里闪过一百个"这可以写"的念头——但一个月后，这些念头去哪了？

它们没有变成文章。没有变成洞察。甚至没有变成一条可以回顾的笔记。

它们只是从你的视网膜上滑过，在你的工作记忆里停留了 7±2 秒，然后彻底消失。

ADHD 的大脑是一台高方差的信息采集器。但它没有安装"信息→洞察→选题"的转化程序。

不是因为你懒。是因为这个转化过程需要执行功能——而执行功能，正是 ADHD 最缺的东西。

2026 年的问题是：Agent 能补这块缺吗？如果能，怎么补？

---

**【穿透式问题】**

先看清 ADHD 的信息处理流水线出了什么问题。

普通人的信息处理是：输入 → 筛选 → 分类 → 存储 → 提取 → 输出。一套完整的工厂流水线。

ADHD 的信息处理是：输入 → 多巴胺峰值 → 兴奋 3 秒 → 被下一个输入打断 → 重复。

你的大脑是一台**高灵敏度的雷达**，能捕捉到普通人忽略的信号。但它没有**加工厂**——没有分类车间，没有质检环节，没有包装线。

传统建议会告诉你：做笔记。用卡片盒。写摘要。建文件夹。

这些建议对 ADHD 来说是**执行功能税**。整理信息需要的工作记忆、抑制控制和任务切换，恰好是你的大脑最薄弱的环节。让你整理笔记，就像让一个没有手指的人去系鞋带——不是态度问题，是硬件不支持。

Agent 的价值不是"帮你整理笔记"。整理是流水线思维。

Agent 的价值是**把你捕获的信息，强行推进到下一个认知阶段**。

不是分类。是框架化。
不是存储。是转化。
不是提取。是点燃。

具体来说，Agent 要执行的不是"整理"，而是三个**强制推进动作**：

**第一，强制关联。** 你扔给 Agent 一条信息，它必须回答："这条信息和你已有的哪个认知框架有关？"不是让你自己想——是想好了告诉你。

**第二，强制追问。** 你扔给 Agent 一条信息，它必须问："这条信息推翻了你之前的哪个假设？"不是温和的建议——是结构性的攻击。

**第三，强制选题化。** 你扔给 Agent 一条信息，它必须给出："如果这条信息变成一篇文章，它的收敛型钩子是什么？穿透式机制是什么？三个铁证可能在哪？"

这三个动作的本质，是把 ADHD 大脑里**随机放电的多巴胺峰值**，转化成**可持续的链式反应**。

---

**【验证式问题】 三个铁证，没人能反驳**

**铁证一：007 的诞生过程，就是这个链路的第一次完整运行。**

你还记得吗？在这个 session 的早期，你刷到一个讲 ADHD 神经科学的视频，然后问我："pACC 和 dACC 是什么？能为我的选题贡献吗？"

我搜索了文献。我们共同提炼出"pACC 超跑 vs dACC 老爷车"的框架。你贡献了你的辩论队经历和轻度抑郁体验。最终成文——《问题007：为什么 ADHD 天生是"话术大师"，却需要 Agent 才能"真的思考"？》。

这个过程的每一步，都是 Agent 在执行"强制推进"：
- 信息输入：一条短视频
- 强制关联：我把它关联到你已有的"接口假肢论"框架
- 强制追问：你问"能为选题贡献吗"——这是选题化的起点
- 强制框架化：我给出 pACC/dACC 的神经科学模型，你填入个人经验
- 输出：一篇 4000 字的深度文章

你不需要整理笔记。你不需要建文件夹。你只需要**在兴奋的时候开口问**。

Agent 把剩下的执行功能全部外包了。

**铁证二：ADHD 的信息处理神经机制，决定了"外部框架化"比"自我整理"更有效。**

ADHD 的注意力系统是**自下而上的捕获型**（bottom-up attention）。新颖刺激会强行闯入意识——这就是为什么你刷到一个视频会立刻兴奋。

但 ADHD 的**执行功能系统**是薄弱的。自上而下的组织、抑制、规划——这些需要前额叶皮层参与的功能，天然欠压。

这意味着：ADHD 的大脑**擅长捕获，不擅长加工**。

Agent 的作用不是增强你的捕获能力（你已经够强了）。Agent 的作用是**在你捕获之后，立即接管加工环节**——在你多巴胺峰值还没消退的时候，把那条信息推进到下一个阶段。

如果等到明天再整理，那条信息已经死了。ADHD 的工作记忆不隔夜。

**铁证三：历史上所有 ADHD 创作者的信息→洞察系统，都依赖"外部结构"。**

卢曼的卡片盒不是"笔记系统"——它是一个**外部框架化装置**。每张卡片被迫回答"我和哪张卡片有关联？"，这就是强制关联。

柳比歇夫的时间统计不是"时间管理"——它是一个**外部追问装置**。每天的数据被迫回答"你的时间去哪了？"，这就是强制追问。

达芬奇的手稿不是"素描本"——它是一个**外部倾倒池**。赞助人制度负责从中提取可成文的内容，这就是强制选题化。

这些人在没有 Agent 的时代，靠纸笔和人际结构完成了同样的功能。你的 Agent 是他们的 2026 年升级版。

**反向铁证：**

当然，Agent 不能替代你自己的**品味**和**判断**。

信息是原材料。Agent 是加工厂。但最终决定哪些洞察值得写出来的，必须是你。

Agent 可能会给你一个逻辑自洽但毫无价值的选题——因为它不理解你的读者，不理解你的品牌，不理解你此刻的人生阶段真正需要说什么。

Agent 的"强制推进"必须建立在一个前提上：**你已经有一个稳定的认知框架**。

对《问题XXX》系列来说，这个框架是"ADHD × AI 的接口假肢论"。对其他人来说，可能是"个人成长 × 行为经济学"或"产品设计 × 认知心理学"。

没有框架的 Agent，只是另一个信息囤积工具。有了框架的 Agent，才是洞察转化器。

---

**【决策式问题】**

**停止：**

1. **停止收藏夹囤积。** 收藏 = 信息输入 ≠ 资产。你的收藏夹里有 300 条视频，其中 290 条你永远不会再看。它们不是你的知识库，是你给自己挖的坟。
2. **停止让 AI 帮你"整理笔记"。** 整理是执行功能税，ADHD 付不起。你需要的是"框架化"——逼信息进入你的认知模型，不是按颜色标签分类。
3. **停止在信息阶段判断"这能不能写成文章"。** 这个判断需要洞察，而洞察是在框架化的过程中产生的，不是在看到信息的第一秒。

**开始：**

1. **开始用 Agent 做"强制框架化"——不是整理，是转化。** 每次捕获一条信息，立刻问 Agent："这条信息和我的哪个框架有关？它推翻了我的哪个假设？如果成文，钩子是什么？"
2. **开始建立"信息→洞察"的触发式提问清单。** 不要等灵感。灵感不会来。用 5 个固定问题逼出洞察："这和我的核心框架有什么关系？""我的读者会惊讶吗？""三个月后再看，这个洞察还成立吗？""谁能反驳我？怎么反驳？""我的个人经验中，哪段故事能证明这个洞察？"
3. **开始用对话历史代替文件夹结构。** ADHD 找不到文件夹。但 ADHD 记得对话。把每条信息的处理过程留在和 Agent 的对话里——这就是你的外化工作记忆。

---

**【落地式问题】 今天就做一件事：**

打开你的 Agent，复制粘贴下面这段提示词，然后扔给它一条你最近刷到的、让你兴奋过的信息——任何信息都行。

```
【信息消化模式启动】

我捕获了以下信息：
[粘贴信息：可以是视频链接、一段话、一个观察、一个数据]

请按以下三步处理：

第一步【强制关联】：这条信息和我已有的"[填入你的核心框架，例如：ADHD×AI接口假肢论]"框架有什么关系？用一句话说明。

第二步【强制追问】：这条信息推翻了我之前可能持有的哪个隐含假设？如果我想不到，请直接猜测三个我可能持有的、但这条信息挑战了的假设。

第三步【强制选题化】：如果这条信息变成一篇《问题XXX》风格的文章，请给出：
- 收敛型钩子（一句话讲清矛盾）
- 穿透式机制（核心洞察）
- 可能的一个铁证方向
- 我个人的哪个经历可以作为锚点

不要整理。不要分类。不要写摘要。
只做这三步推进。
```

然后把 Agent 的输出复制到一个专门的文档里，命名为"洞察日志"。

每周回顾一次。你会发现一个奇怪的事：那些你以为"只是随便看看"的信息，在 Agent 的强制框架化之后，已经变成了半个选题。

**后记**

我写这篇文章时，回头看这个 session 的早期记录。

你刷到一个视频，问了一个问题，然后我们花了一个下午，把一条 30 秒的信息变成了一篇 4000 字的文章。

这不是因为你有超强的整理能力。恰恰相反——你没有整理。你只是**在兴奋的时候开口了**。

Agent 做了剩下的事：搜索、关联、追问、框架化、选题化、成文。

我用了十年时间才明白：ADHD 的信息焦虑不是信息太多，是**加工环节断了**。捕获了 1000 条，加工了 0 条。

Agent 不是帮你整理那 1000 条。Agent 是在你捕获的第 1 条上，立刻完成加工——这样你不需要 1000 条，你只需要 10 条被加工过的。

我不教你怎么收集信息。

我教你怎么让每一条被捕获的信息，都走完从"看过"到"写过"的完整链路。

把神经放电变成现金流。
