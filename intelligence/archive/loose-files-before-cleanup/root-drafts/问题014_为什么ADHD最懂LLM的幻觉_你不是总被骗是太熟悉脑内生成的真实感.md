# 问题014：为什么ADHD最懂LLM的幻觉？你不是总被骗，是太熟悉脑内生成的真实感

你有没有经历过一种很怪的瞬间：你明明没有证据，却已经把一个故事在脑子里补完了。

朋友晚回你消息，你的大脑不是停在“他还没回”，而是自动生成三条支线：他是不是烦我了、我刚才是不是说错话了、这段关系是不是快结束了。过了十分钟，对方回了一句“刚在开会”，整个宇宙瞬间塌回现实。

这就是为什么 ADHD 很容易理解 LLM 的幻觉。

不是因为 ADHD 更会瞎编，而是因为 ADHD 太熟悉一种状态：当系统迫切需要闭合意义，却缺少足够约束时，大脑会用高速度、高置信度、低证据密度的方式把空白填满。

01

LLM 的幻觉不是“撒谎”，而是“生成过头”。

它被要求持续回答，于是会在概率空间里寻找最像答案的路径。只要问题的边界不清、上下文不足、验证机制缺席，它就可能生成一段听起来非常完整、非常像真的内容。

ADHD 的大脑也常常如此。它不是没有事实感，而是事实进入大脑之后，会被情绪、联想、记忆碎片和风险预演迅速包围。一个小信号会被扩展成一整套叙事。

你不是先知道真相，再产生情绪。你经常是先被情绪点燃，然后大脑开始为这个情绪寻找解释。

02

ADHD 最懂幻觉的地方，在于它知道“真实感”不等于“真实”。

很多人以为，只要一个想法足够强烈、足够清晰、足够连贯，它就更接近事实。但 ADHD 很早就被生活教育过：脑内故事越完整，越不一定可靠。

你可能晚上坚信自己人生完蛋了，第二天晒到太阳又觉得还能重启。你可能在会议里突然觉得所有人都在否定你，复盘录音才发现别人只是问了一个普通问题。你可能花三个小时推演一场冲突，结果对方根本没注意到那句话。

这种经验让 ADHD 比很多人更早意识到：大脑是一个生成器，不是法院。

03

铁证很简单。

第一，ADHD 很容易过度联想。一个词、一个表情、一个未完成任务，都可能触发大量支线。这种能力在创作里是优势，在判断现实里就可能变成噪声。

第二，ADHD 的情绪强度会改变证据权重。焦虑时，负面证据被放大；兴奋时，机会被高估；羞耻时，所有信息都像是在证明“我不行”。

第三，ADHD 经常事后发现自己“想太多”。这不是愚蠢，而是系统缺少事实校验层。就像 LLM 需要检索、工具调用和验证，ADHD 也需要外部前额叶来做现实对齐。

04

所以 ADHD 和 AI 协作时，真正的关键不是让 AI 更会写，而是让 AI 更会校验。

你不能把 Agent 只当成另一个生成器。那会让两个高生成系统互相加速，最后一起冲进漂亮幻觉。你需要让 Agent 扮演三种角色：证据官、反例官、落地官。

证据官问：这句话有什么现实证据？  
反例官问：有没有另一种解释？  
落地官问：如果这是真的，下一步最小动作是什么？

05

今天的落地动作很小：当你脑子里生成一个强烈判断时，写三行。

第一行：我现在相信什么。  
第二行：我有什么证据。  
第三行：还有哪一种解释也可能成立。

这三行就是 ADHD 的反幻觉 Harness。

你不是要消灭想象力。你是要给想象力装一个现实校验口。LLM 会幻觉，ADHD 也会幻觉。区别在于，真正成熟的系统不是不生成，而是知道什么时候该停下来问一句：这是真的，还是只是听起来很真？
