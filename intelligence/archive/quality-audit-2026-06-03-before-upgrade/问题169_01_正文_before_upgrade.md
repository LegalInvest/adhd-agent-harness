# 问题169：为什么 ADHD 需要 done list，而不是更多待办？你的大脑缺的不是任务，是已完成证据

很多 ADHD 的一天，是从负债开始的。

睁眼第一秒，不是“今天我要做什么”，而是“昨天又有多少没做”。手机还没拿起来，脑子已经开始滚动播放未完成事项。待办清单越完整，你越像站在一面债务墙前面。

于是你继续加待办，继续换软件，继续买模板，继续告诉自己这次要规划好。可真正缺的东西不是任务。任务你一直都有，甚至多到过载。

你缺的是已完成证据。

## 01 收敛

一句话打穿：ADHD 不该只用 todo list 管理人生，还要用 done list 修复现实感。

todo list 负责告诉你世界还欠多少动作。done list 负责告诉你你已经把哪些东西带进了现实。

如果只有待办，没有已完成，你的大脑会形成一个很扭曲的世界观：人生永远是欠账，自己永远是落后，努力永远不够。

这不是高标准，这是单边记账。

## 02 穿透

待办软件默认假设一个人能稳定地看见未来、规划路径、按时执行、从完成中获得奖励。这个假设对 ADHD 很奢侈。

ADHD 的时间感和奖励感都更容易断裂。未来任务会在脑中变成一团压力，而已完成成果又很快失去刺激强度。结果就是：未完成的东西越来越大，已完成的东西越来越透明。

done list 的作用，是把已经发生的现实重新钉回墙上。

它不是给你做心理按摩，而是建立证据链：今天推进了什么，留下了什么资产，产生了什么反馈，下一步能从哪里接上。

对 ADHD 来说，done list 其实是一种外部现实校准器。

## 03 铁证

生活场景证据：你可能一天里回了三条重要消息，修了一个文件，整理了一段资料，和 AI 跑通一个流程。但晚上只要有一个大任务没完成，你就会把整天判成失败。

个人观察：很多高强度创作者最容易低估“中间推进”。他们只承认发布、成交、最终交付，却不承认素材收集、框架搭建、问题澄清、脚本修复、看板更新。可真实项目就是靠这些中间推进活下来的。

反事实：如果你每天只看待办，你会越来越焦虑；如果你每天能看见三条完成证据，你会开始恢复连续性。连续性对 ADHD 很关键，因为它告诉你：我不是每天从零开始，我是在接续昨天。

## 04 决策

你要把 done list 当成生产系统，不要当成情绪日记。

它至少要记录四类东西：

- 完成：今天真的落地了什么。
- 推进：虽然没完成，但哪个项目往前移动了。
- 证据：文件、链接、截图、评论、对话、交付物在哪里。
- 回流：外部世界给了什么反馈。

这四类加起来，才叫现实资产。

如果只有“我今天很努力”，低谷时大脑不会相信。如果有文件、有日期、有链接、有反馈，低谷就很难彻底篡改历史。

## 05 落地

今天的落地动作：写一个 3+1+1 done list。

格式：

- 今天完成的 3 件小事：
- 今天推进的 1 个项目：
- 今天收到或制造的 1 个外部反馈：

注意，完成的小事可以非常小：创建文件、修复链接、写完开头、回复关键消息、跑通脚本。ADHD 最需要承认的，恰恰是这些容易被自己抹掉的推进。

如果接 AI，让 Agent 每天晚上问你一句：今天有什么东西从脑子里进入了现实？

这句话比“明天要做什么”更能救你。
