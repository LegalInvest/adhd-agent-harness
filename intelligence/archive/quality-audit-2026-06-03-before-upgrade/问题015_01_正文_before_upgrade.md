# 问题015：为什么ADHD和LLM都需要Harness？不是限制想象力，而是给高能系统装轨道

你最烦别人跟你说“你要自律一点”。

因为你不是没有能量。很多时候，你的问题恰恰是能量太多、入口太多、联想太快。你坐下来本来只想写一篇文章，十分钟后已经打开了三个资料、两个工具、一个新项目，还顺手想重构整个人生。

这就是 ADHD 和 LLM 的共同命运：它们都不是缺生成能力，而是缺 Harness。

01

Harness 不是束缚，是牵引系统。

LLM 如果没有提示词、上下文、工具边界和输出格式，它会变成一个流畅但不稳定的语言喷泉。ADHD 如果没有外部结构、任务边界、启动协议和反馈回路，也会变成一个高能但难落地的认知喷泉。

很多人误会 ADHD，以为它需要的是更强的意志力。实际上 ADHD 更需要的是把意志力从“持续压制自己”改成“提前设计轨道”。

轨道不是为了让火车失去自由。轨道是为了让高速系统不用每一秒都重新决定自己往哪走。

02

ADHD 的痛苦，经常来自没有 Harness 的自由。

看起来你拥有无数选择：今天可以写内容、做产品、研究市场、整理资料、搭自动化、回消息、学新工具。但没有 Harness 时，这些选择不是自由，而是并发请求。

每一个可能性都在喊你。每一个方向都像有道理。你不是没有目标，你是目标之间没有路由协议。

LLM 也是这样。你问得越空泛，它越容易泛化；你不给格式，它就自己发明格式；你不给验证，它就用流畅感冒充可靠性。

ADHD 和 LLM 都需要一句话：请在这个边界里输出。

03

铁证有三个。

第一，你最能产出的时刻，往往不是最自由的时刻，而是有人给了你明确题目、截止时间和交付格式的时候。外部约束没有降低你，反而释放了你。

第二，你最容易崩溃的时刻，往往不是没事做，而是所有事都能做。选项过多会让 ADHD 的执行系统过载。

第三，优秀的 Agent 工作流从来不靠一句“帮我变好”。它靠具体接口：输入是什么、输出是什么、判断标准是什么、失败时怎么降级。

04

所以 ADHD 的自救，不是把自己改造成神经典型者，而是给自己写一套 Harness。

早上的 Harness 可以是“今日唯一动作”。  
写作的 Harness 可以是“五步框架”。  
发布的 Harness 可以是“正文到公众号、小红书、知乎、短视频四接口”。  
情绪的 Harness 可以是“先倾倒，再判断”。  
项目的 Harness 可以是“每次会话必须留下状态文件”。

这些东西看起来机械，其实很温柔。它们替你承担了最消耗前额叶的部分：决定边界。

05

今天就做一个最小 Harness。

选一个你最常失控的场景，写下四句话：

输入是什么？  
输出是什么？  
最多做多久？  
失败时降级到什么动作？

比如写作：

输入：一个选题。  
输出：前三段正文。  
时长：25 分钟。  
降级：只写标题和开头场景。

这就是 Harness。

你不是要把自己锁住。你是要给高能系统装轨道。没有轨道，你的发散会变成噪声；有了轨道，你的发散才会变成穿透现实的速度。
