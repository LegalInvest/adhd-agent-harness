# 3 个 LLM 工程概念，彻底重写了我对 ADHD 任务启动困难、不会拆解的理解

> agent 用 planner-executor 循环分解长任务 如何反向照亮 ADHD 的任务启动困难、不会拆解。

你有没有过这样的时刻：坐在电脑前，任务清单摊开，明明知道该做什么，但就是动不了。不是懒，不是不想做，而是大脑像卡住了一样——启动一个任务需要巨大的心理能量。同时，你或许也听说过AI Agent：它们能自主规划、执行长任务，但经常在第一步就“愣住”，或者分解任务时偏离方向。

这两件事，看似无关，实则共享同一个底层结构。

## 1. 同构：ADHD大脑与LLM都是“高产但缺调度”的生成核心

ADHD大脑与大型语言模型（LLM）在架构上惊人相似：两者都是高产的生成核心——ADHD大脑擅长联想、创意、高兴趣驱动，LLM擅长语言生成、模式匹配——但都缺乏可靠的内置执行调度层（来源：[[ADHD 大脑与 LLM 的同构]]、[[生成核心与调度层]]）。ADHD在执行功能（计划、组织、工作记忆、冲动抑制）上存在缺陷，LLM则在状态保持、真实规划、上下文控制上存在缺陷（来源：[[执行功能]]、[[工作记忆]]）。

这意味着，ADHD的“任务启动困难”和“不会拆解”，本质上与LLM Agent的“规划失败”是同一个问题：生成核心产出了大量可能性，但调度层无法将其转化为有序的执行序列。

## 2. Agent的解法：planner-executor循环

在LLM Agent工程中，解决这一问题的标准模式是**planner-executor循环**：Agent先由planner将长任务分解为子步骤，然后由executor逐步执行，每步完成后反馈给planner调整下一步计划。这种循环结构相当于给LLM套了一个外部harness，补偿其缺失的调度能力。

工具如[[Motion]]和[[Reclaim.ai]]正是这一思路的ADHD版本：Motion自动排程并动态调整，Reclaim.ai通过智能时间块保护深度工作，它们本质上都是“外部调度器”，把“现在该做什么、做多久、何时切换”的决策从大脑中卸载（来源：[[Motion]]、[[Reclaim.ai]]）。[[Tiimo]]则将时间视觉化，直接应对[[时间盲]]——相当于把时间感知外化为可见的进度条。

## 3. 历史人物的harness：孔子与毛泽东的“外部调度层”

孔子和毛泽东，两位看似截然不同的人物，却都构建了精密的个人harness系统来管理自己的ADHD特质。

孔子身高1米9，精力旺盛，周游列国14年坐不住；冲动爱骂人，见南子急得对天发誓，骂宰予“朽木不可雕”；对音乐超专注到“三月不知肉味”，对种地等俗事零耐心。他的专属harness是“克己复礼”——用外在的“礼”作为行为边界，每日反省（“吾日三省吾身”），70岁才做到“从心所欲不逾矩”。这套系统本质上是一个**外部行为调度器**：用固定的仪式（礼）和定时反思（日课）来约束冲动的生成核心。

毛泽东同样如此：小时候是问题儿童，一辈子读书不辍，永远在动，冲动敢赌（四渡赤水、抗美援朝），思维极度跳跃。他的harness是“批评与自我批评”（民主生活会）、“调查研究”（没有调查就没有发言权）、“民主集中制”（用集体决策制衡个人冲动）、“党指挥枪”（用组织系统管理军队）。这些机制构成了一个**外部决策与执行监控层**：通过他人反馈、实地调研、集体投票来校准自己的判断和行动。

这两套harness与LLM Agent的planner-executor循环同构：孔子的“日课”相当于定时re-grounding（重新锚定），毛泽东的“调查研究”相当于获取外部环境反馈。它们都不是消除生成核心的创造力，而是给它套上可靠的调度层。

## 4. 脚手架 vs 拐杖：关键边界

这里有一个重要的争议：AI工具是“外部执行功能层”还是“永久拐杖”？（来源：[[拐杖与脚手架]]）

我的判断是：关键在于**是否保留用户的主动选择权**。脚手架是临时搭建、可拆卸的辅助结构，目的是让用户最终独立完成；拐杖则是永久替代，用户一旦离开就无法行走。

好的AI工具应该像脚手架——比如[[Goblin Tools]]的“魔法待办”将任务拆解为小步骤（来源：[[任务启动]]），[[Saner.AI]]提供启动策略如“两分钟法则”（来源：[[Saner.AI]]），这些功能降低启动门槛，但用户仍然可以自己调整步骤、选择是否使用。相反，如果工具完全接管了决策（比如自动安排一切且不允许用户修改），那就可能成为拐杖。

## 5. 局限与争议

必须诚实指出：ADHD大脑与LLM同构的论点目前主要是类比推理，缺乏神经科学或计算机科学的直接证据（来源：[[矛盾与存疑]]）。AI工具对任务启动的长期效果也未知——短期有效，但依赖可能导致能力退化（来源：[[矛盾与存疑]]）。此外，超聚焦是否应被引导而非打断，个体差异极大，尚无定论（来源：[[超聚焦]]、[[矛盾与存疑]]）。

## 今天就能试的行动

1. **微任务分解**：明天早上，选一个你拖延已久的任务，用[[Goblin Tools]]的“魔法待办”或手动拆成5步以内、每步不超过10分钟的小块。完成第一步就停下来。
2. **设置外部提醒**：在手机上设一个定时闹钟（比如每45分钟），问自己“我现在应该做什么？”——这相当于LLM Agent的re-grounding步骤。
3. **视觉化时间**：用[[Tiimo]]或纸笔画出今天的时间块，把“自由时间”也标出来。视觉化能缓解[[时间盲]]。
4. **建立反馈循环**：像毛泽东的“民主生活会”一样，找一位朋友或同事，每天花2分钟告诉他你完成了什么，让他给你一个简单的反馈。这相当于Agent的executor向planner报告。

ADHD不是缺陷，而是一种不同的架构。与其对抗它，不如像Agent工程师一样，给它搭一个可靠的harness。
