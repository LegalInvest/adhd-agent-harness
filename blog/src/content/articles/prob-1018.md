# Reflect 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 Reflect：它到底补上了哪一层执行功能？

## 问题：为什么 Reflect 用错了？

ADHD 患者和 LLM agent 工程师面临同一个深层困境：**生成能力强，但执行调度弱**。ADHD 大脑的创意发散与超聚焦常导致目标漂移——分心走神或钻入细节无法自拔；LLM agent 在长程任务中同样会因上下文膨胀、注意力分散而偏离初始目标。两者都需要一个外部机制来“重锚定”注意力。Reflect（一款 AI 笔记工具）常被宣传为 ADHD 的救星，但许多人用错了——他们把它当作被动记录仪，而非主动调度层。真正的价值在于：**Reflect 应该像 agent harness 中的重锚定 system prompt 一样，在关键时刻主动将注意力拉回主线**。

## 同构脊柱：生成核心 + 外部调度层

ADHD 大脑与 LLM 在结构上同构：两者都是**高产但缺乏可靠执行调度层的生成核心**（来源：AI 与 ADHD 的专注力）。ADHD 的执行功能缺陷——工作记忆差、时间盲、任务启动困难——对应 LLM 的无状态性、上下文控制薄弱、目标漂移。解决方案也同构：为生成核心套上 harness（外部执行功能层）。

在 LLM/agent 侧，harness 通过 human-in-the-loop 监督、护栏（token 预算、工具调用上限）、定时检查点来提供重锚定（来源：人在回路与身体在场）。例如，agent 每执行 5 步就暂停并询问用户是否继续，这相当于 system prompt 的“重锚定”。

在 ADHD 侧，最有效的策略之一就是**身体在场效应**：有另一个人在场时，执行功能障碍被绕过，任务得以启动和维持（来源：人在回路与身体在场）。Focusmate 正是利用 AI 调度实现虚拟身体在场，提供实时问责（来源：Focusmate）。这相当于 agent harness 中的人类检查点——一个外部实体在关键节点介入，防止漂移。

## 人物案例：曾国藩的 harness 系统

曾国藩是 ADHD 特质鲜明的历史人物：30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”；读书慢，“他人目下二三行，余或疾读不能终一行”；一辈子严重银屑病，情绪不稳定，打败仗就跳水自杀。他的专属 harness 是**日课十二条**：黎明即起、读书不二、谨言、写日记反省等。其中“读书不二”要求一本书不读完绝不碰下一本，这相当于为注意力设置硬边界；“写日记反省”则是每日重锚定——如同 agent 每步结束后记录日志并比对初始目标。

曾国藩的 harness 与 LLM agent harness 同构：**日课↔定时 re-grounding**（如每 25 分钟检查一次目标），**日记↔外部记忆系统**（记录状态以便回顾调整），**“结硬寨打呆仗”↔护栏**（用最笨最稳的方法抵消冲动）。他的成功证明：外部执行功能层可以补偿内在缺陷，但前提是**主动且结构化**，而非被动记录。

## 工具实测：Reflect 补上了哪一层？

Reflect 作为 AI 笔记工具，核心功能是自动记录、回顾和关联信息。这确实补上了**工作记忆**这一层——ADHD 患者和 LLM 一样“无状态”，需要外部记忆维持连续性（来源：AI 与 ADHD 的专注力）。但仅此不够。许多用户把 Reflect 当作被动储存库，事后回顾却无法在当下阻止漂移。真正的脚手架应像 RescueTime 那样**主动揭示时间去向**并阻断分心（来源：RescueTime），或像 Focusmate 那样**在任务启动时提供即时问责**。

Brain.fm 通过神经锁相技术影响大脑状态，帮助进入专注（来源：Brain.fm），但它缺乏主动调度——它创造环境，却不锚定目标。Reflect 若只记录不干预，就和 Brain.fm 一样是“被动背景”，而非“主动重锚定”。

## 核心观点：脚手架 vs 拐杖的边界

我的判断是：**Reflect 作为外部执行功能层的价值，取决于它是否在“注意力即将漂移”的时刻主动介入**。如果它只是事后记录，那就是拐杖——依赖它反而让内在执行功能退化。如果它能像 agent harness 的 system prompt 重锚定一样，在检测到目标偏离时弹出提醒或引导回顾，它才是脚手架。

证据显示，AI 工具长期使用可能导致依赖风险（来源：矛盾与存疑），但尚无纵向研究。个体差异也很大：有些 ADHD 用户需要强力打断（如番茄钟），有些则需柔性引导（来源：矛盾与存疑）。因此，Reflect 的“锚定强度”应可调节。

## 局限与争议

本文的类比存在局限：ADHD 大脑与 LLM 的同构是隐喻而非严格等价，LLM 的“注意力机制”与人类注意力有本质差异（来源：AI 与 ADHD 的专注力）。此外，Reflect 等工具的临床证据仍薄弱，多为用户报告。曾国藩的案例虽生动，但个体经验不能直接推广。

## 今天就能试的行动

1. **设置每日“重锚定”提醒**：在 Reflect 中创建一条笔记，列出当日核心目标，每 25 分钟（一个番茄钟）弹出并自问：“我现在做的事是否与目标一致？” 这模仿了 agent 的定时检查点。
2. **用 Focusmate 替代被动记录**：如果 Reflect 让你只是回顾而非行动，尝试每周 3 次 Focusmate 时段，在他人面前工作，体验身体在场效应（来源：Focusmate）。
3. **结合 RescueTime 的主动阻断**：将 RescueTime 的分心阻断功能与 Reflect 结合——先阻断分心网站，再用 Reflect 记录专注内容，形成“环境+记录”的双重锚定（来源：RescueTime）。
4. **模仿曾国藩的“日课”**：每天固定时间用 Reflect 写 5 分钟“反省笔记”，对照目标评估当日注意力漂移点，并规划明日调整。这相当于 agent 的日志回放与重规划。
