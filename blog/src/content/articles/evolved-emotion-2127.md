# 为什么用 Claude 治 ADHD 的情绪失调，和给 agent 套 会褪去的脚手架 是一回事？

> Claude 实测：同一套 harness 思路，ADHD 与 LLM 两边都成立。

## 问题：为什么你的情绪像失控的Agent？

你打开Claude，想写一封邮件，结果它开始编造一个不存在的会议纪要。你生气地关掉窗口，却发现自己已经刷了半小时手机——原本只想查个日程。

这就是ADHD情绪失调的日常：大脑像一台没有调度层的生成模型，不断产出焦虑、挫败、兴奋，却无法有效抑制或重定向。而Claude的“幻觉”问题，本质上是一样的——一个高产生成核心，缺乏可靠的任务分解与验证机制。

## 同构：高产核心与缺失的调度层

ADHD大脑与LLM的结构同构，不是比喻，而是工程层面的真实映射。ADHD大脑的“生成核心”（默认模式网络、情绪中枢）不断产生想法和情绪，但前额叶（调度层）工作记忆不稳定、时间感知模糊，导致任务启动困难、情绪波动（来源：AI与ADHD的情绪调节）。LLM同样：它能生成流畅文本，但需要外部harness提供规划循环、任务分解和验证，否则就会跑偏（来源：Agent Scaffolding: Architecture and Design Patterns for Agentic AI）。

两边共享同一个瓶颈：**生成能力强，调度能力弱**。因此，解决方案必然同构——用外部脚手架补偿调度缺陷。

## 证据：两边都成立的Harness方案

### ADHD侧：结构化脚手架稳定情绪

情绪调节的核心不是“安抚情绪”，而是修复执行功能链条。Inflow基于CBT，将情绪调节分解为每日技能培养，通过结构化练习减少情绪触发源（来源：Inflow）。Goblin Tools的Magic ToDo把模糊任务分解为可执行步骤，降低启动门槛（来源：Goblin Tools）。Saner.AI用本地记忆减少搜索循环和标签切换，直接缓解因遗忘引发的焦虑（来源：Saner.AI）。这些工具的共同点：**作为外部执行功能层，补偿工作记忆和规划缺陷**（来源：AI与ADHD的情绪调节）。

### LLM侧：Agentic Harness约束输出

LLM agent同样需要外部脚手架。Motion和Reclaim.ai自动创建和优先排序任务，动态调整日程（来源：规划循环与任务分解）。AgentGen通过环境和任务生成增强LLM的规划能力（来源：规划循环与任务分解）。这些harness本质上就是“会褪去的脚手架”——初期必须依赖，但最终目标是将规划能力内化到模型本身。

### 历史案例：释迦牟尼的Harness

释迦牟尼29岁放弃王位，不断试错求道，49年游行说法，一字未写。他的ADHD特质显而易见：高产出（说法49年）、低调度（无法安定于一地）。他的harness是“八正道”和“持戒”——一套外部行为准则，相当于今天ADHD的日课系统和LLM的prompt模板。他说“自依止法依止”，不依赖权威，自己证悟——这正是脚手架的内化目标：最终撤掉外部依赖，形成内在调度能力。

## 核心观点：脚手架不是拐杖，但必须会褪去

拐杖是永久替代，脚手架是临时结构，最终要拆除。ADHD工具和LLM harness如果设计成永久依赖，就变成了拐杖——用户无法内化技能，模型无法自主规划。关键在于：**脚手架必须设计为可褪去的**。

Inflow教用户建立技能，而非永远依赖应用（来源：Inflow）。Goblin Tools的“辣度”调节允许用户逐步减少分解粒度（来源：Goblin Tools）。同样，LLM agent的harness应逐步减少外部约束，让模型学会自主规划。否则，用户或模型永远无法独立。

## 争议与局限

这一同构假设仍有边界。ADHD大脑的生物复杂性远超LLM，情绪调节涉及神经递质、环境交互等（来源：矛盾与存疑）。工具的证据也不足：Inflow缺乏严格随机对照试验（来源：Inflow），Motion缺乏独立临床验证（来源：矛盾与存疑）。过度依赖可能削弱内在执行功能（来源：矛盾与存疑）。因此，本文的“同构”是工程类比，而非科学定论。

## 今天就能试的行动

1. **用Goblin Tools分解一个让你焦虑的任务**：输入“整理房间”或“写报告”，调节辣度到适中，按子步骤执行。感受外部调度如何降低启动门槛。
2. **设置一个“褪去提醒”**：在用Inflow或Saner.AI时，每周问自己一次：“如果今天没有这个工具，我能完成这个任务吗？”逐步减少依赖。
3. **为Claude写一个“会褪去的prompt”**：例如“请先列出步骤再回答，但下次回复时只给结果。”训练模型（和你的大脑）逐步减少外部提示。
4. **尝试Focusmate一次**：虚拟身体在场通过他人存在提升专注，间接稳定情绪（来源：身体在场效应）。注意：它不能替代真实人际互动，但可作为临时脚手架。

同构不是巧合，是工程与神经科学的交汇。当你用Claude治情绪失调时，你其实在给大脑套一个会褪去的脚手架——正如agent需要harness才能可靠执行。区别只在于：你的大脑最终要学会自己调度。
