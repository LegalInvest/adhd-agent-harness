# 用 Perplexity 给 ADHD 补上重锚定与目标漂移，是脚手架还是拐杖？

> Perplexity 接进 ADHD 工作流：哪些交给它、哪些必须自己留着。

你正埋头搜索一个关键信息，半小时后却发现自己刷完了十条无关网页，而最初的问题早已模糊。这种「目标漂移」——分心走神或超聚焦跑偏、丢失初始意图——是 ADHD 人群的日常，也是长程 AI agent 工程中最棘手的难题。当 agent 在复杂任务链中逐渐偏离原始目标时，工程师会为其加上重锚定（re-grounding）机制，如定期重读 system prompt 或引入外部验证节点。那么，Perplexity 这类 AI 搜索工具，能否为 ADHD 大脑也装上类似的「重锚定」模块？

## 同构：高产但缺调度层的生成核心

ADHD 大脑与 LLM 存在深层结构同构：两者都是强大的生成核心，但缺乏可靠的内置执行调度层。ADHD 的工作记忆缺陷与 LLM 的无跨会话状态同构——两者都依赖外部记忆系统来补偿内部状态管理不足（来源：主题综述『AI 与 ADHD 的专注力』）。曾国藩在日记中痛斥自己“无恒”“浮躁”，用“日课十二条”如“黎明即起”“读书不二”来强制重锚定；王阳明则用“致良知”作为唯一决策标准，在龙场悟道后以静坐反省来校准行动。这些历史人物的 harness 系统，本质上就是一套外部调度层：曾国藩的“日课”对应定时 re-grounding，王阳明的“事上练”对应在行动中持续验证意图。

在 LLM 侧，单个 LLM 不足以可靠完成多步骤任务，需要外部 scaffold 提供任务分解、规划循环和验证机制（来源：概念页『规划循环与任务分解』）。AgentGen 等框架通过环境与任务生成增强 agent 的规划能力，而 AMAP 等系统则引入显式的规划循环来防止目标漂移。ADHD 大脑与 LLM 的解决方案结构同构：都需要一个外部调度器来定期把注意力拉回目标。

## Perplexity 作为重锚定工具：证据与机制

Perplexity 的核心价值并非信息检索，而是「带上下文的目标回溯」。当 ADHD 用户在搜索中漂移时，Perplexity 的对话式搜索能保持上下文连贯，用追问和摘要帮助用户回溯原始问题。这类似于 RescueTime 的自动时间追踪揭示真实时间去向（来源：RescueTime 页面），但 Perplexity 更主动：它通过生成回答而非罗列链接，减少了用户在工作记忆中的额外负担。

然而，Perplexity 的重锚定效果取决于用户是否主动使用它来校准目标。若用户只是被动接收答案，它可能沦为另一个信息消费的入口——这正是「拐杖 vs 脚手架」的边界。Focusmate 通过虚拟身体在场提供外部结构（来源：Focusmate 页面），Brain.fm 通过神经锁相技术帮助进入专注状态（来源：Brain.fm 页面），这些工具都在补偿执行功能缺陷，但若用户从不内化其机制，则可能永久依赖。

## 争议与局限：脚手架还是拐杖？

现有证据存在明显局限。Brain.fm 在 2026 年一项 44 款 ADHD 应用测试中位列最佳 9 款，但缺乏 ADHD 人群的独立临床研究（来源：Brain.fm 页面）。RescueTime 的自动分类可能不够精准，对执行功能较弱的用户仍是挑战（来源：RescueTime 页面）。更重要的是，「同构」目前仅是功能类比，缺乏神经机制层面的实证（来源：矛盾与存疑页面）。

「拐杖与脚手架」概念强调需警惕过度依赖（来源：主题综述）。Perplexity 若被用来替代思考——让 AI 直接给出答案而不经自身验证——就是拐杖；若被用来辅助思考——先自己尝试，再用 Perplexity 验证或扩展——就是脚手架。曾国藩的“日课”并非天生内化，而是写了三十年日记骂自己才逐渐成为习惯。王阳明的“致良知”也是在无数次失败后沉淀的。AI 工具同样需要这样的内化过程。

## 今天就能试的行动

1. **用 Perplexity 做「目标回溯」实验**：每次搜索前，在对话开头写下原始问题。漂移后，让 Perplexity 总结当前对话与原始问题的关联，强制自己校准。
2. **设置「re-grounding 闹钟」**：每 25 分钟（一个番茄钟），用 RescueTime 或手机闹钟打断自己，问「我现在的行动是否还在原始目标上？」。这与 LLM agent 的定期重读 system prompt 同构。
3. **尝试 Focusmate 的「身体加倍」**：预约一次虚拟 coworking 时段，在他人注视下完成一项任务。这种外部问责直接补偿内在调度缺陷。

Perplexity 能否成为 ADHD 的重锚定工具，答案不在工具本身，而在使用模式。当你主动用它来校准目标时，它是脚手架；当你被动接受信息时，它是拐杖。真正的调度层，最终仍需长在自己的大脑里。
