# 用 ChatGPT 给 ADHD 补上采样温度与表现波动，真的能像 agent 那样工作吗？

> 把 ChatGPT 当作 ADHD 的外部执行层来实测：它补的是哪一块短板？

## 问题：状态波动，是 ADHD 的“采样温度”太高了

你有没有过这样的体验：早上醒来，大脑像一台刚启动的生成模型，思路奔涌、创意爆棚，但一坐到工位上，注意力就像随机采样一样飘忽不定——上一秒还在写报告，下一秒已经刷了半小时短视频。到了下午，能量骤降，连“打开文档”这个动作都需要巨大的意志力。这种状态日内的大幅波动，是 ADHD 最折磨人的特征之一。

而如果你是一名做 Agentic Harness 的工程师，你一定熟悉另一个版本的“波动”：同一个 LLM，温度设成 0.8 时输出天马行空，降到 0.2 时又死板重复；不加结构约束，模型会在上下文膨胀后推理退化，甚至忘记自己的角色。

这两个场景，看似风马牛不相及，实则共享同一个底层结构：**一个高产但缺乏调度层的生成核心**。ADHD 大脑的默认模式网络（DMN）高活跃，类似 LLM 的生成能力，但前额叶执行功能薄弱，类似 LLM 缺乏内置调度（来源：ADHD 大脑与 LLM 的同构）。那么，用 ChatGPT 来给 ADHD 补上这个调度层，真的能像 agent 那样稳定工作吗？

## 同构：ADHD 大脑与 LLM 共享的“缺调度”困境

先看 ADHD 侧的真实证据。执行功能缺陷模型指出，ADHD 的核心问题在于任务启动困难、工作记忆差、时间感知弱（来源：执行功能）。这导致大脑像一台“温度过高”的生成器：想法多但无法排序，冲动强但难以抑制。历史人物孔子就是典型：他精力旺盛、思维跳跃，《论语》全是场景化语录，无系统著作；他通过“克己复礼”作为外部行为边界，每日反省，70 岁才做到从心所欲不逾矩。这套“礼”就是他的 harness——用外在结构约束内在波动。

再看 LLM 侧。Transformer 注意力机制在短上下文中表现正常，但随序列变长（认知负荷增加），模型在冲突试次上骤降到随机水平——无法抑制优势反应、无法解决认知冲突（来源：Deficient Executive Control in Transformer Attention）。这与 ADHD 执行功能缺陷的核心标志一一对应：注意控制不足、干扰抑制缺陷、随认知负荷增加而崩溃。

两边都是“高产但缺调度”：ADHD 需要外部执行功能层来启动、排序、提醒；LLM 需要 agent harness 来分解任务、管理上下文、约束输出。同构不是比喻，而是功能上的映射。

## 实验：ChatGPT 能当这个 harness 吗？

现在回到核心问题：用 ChatGPT 给 ADHD 补上采样温度与表现波动，真的能像 agent 那样工作吗？

答案是：**可以，但前提是你把它当作一个 harness 来用，而不是一个聊天对象。**

先看工具层面的证据。Goblin Tools 的 Magic ToDo 功能能自动将模糊任务（如“清理厨房”）分解为具体子步骤，用户可调节“辣度”控制粒度（来源：Goblin Tools）。这本质上就是 agent 中的 task decomposition——把一个大任务拆成小步骤，降低启动门槛。Routinery 通过可视化倒计时和过渡提示引导用户逐步完成流程，直接针对时间盲和任务切换的认知负荷（来源：Routinery）。Habitica 的游戏化设计提供即时多巴胺奖励，弥补内在动机不足（来源：Habitica）。这些工具的共同点是：**它们都在扮演外部调度层，而不是替代大脑本身。**

但 ChatGPT 的特殊性在于，它本身就是一个生成核心，自带“采样温度”。如果你直接问它“我今天该做什么”，它会随机生成一堆建议，反而加重认知负荷。真正的用法是：**用结构化的 prompt 把它锁在一个低温度的调度模式里**。比如，你可以让它每天固定时间问你“现在要启动哪个任务？”，然后它只输出一个选项，不提供多个选择——这就是 agent 里的“greedy decoding”。或者，你可以让它帮你分解任务后，把步骤写进 Routinery 的流程里，形成闭环。

孔子如果活在今天，他的“日三省吾身”就是每天固定三次 re-grounding prompt；左宗棠的“结硬寨打呆仗”则是用固定流程对抗冲动。这些历史人物已经证明：**harness 的核心不是智能，而是结构。**

## 边界：脚手架，不是拐杖

但这里有一条红线。同构类比虽然吸引人，但缺乏大规模临床验证（来源：矛盾与存疑）。ChatGPT 不会真正“理解”你的状态波动，它只是根据上下文生成文本。如果你过度依赖它来替你做所有决策，你的内在执行功能可能进一步萎缩——这就是“拐杖 vs 脚手架”的边界。脚手架是你自己搭的、可拆卸的结构；拐杖是你离开就无法站立的依赖。

另外，ADHD 症状异质性很大，并非所有人都受益于同一类工具（来源：矛盾与存疑）。有些人需要视觉化的时间锚点（如 Routinery），有些人需要游戏化的多巴胺刺激（如 Habitica），有些人只需要一个安静的“外部秘书”来帮自己启动。ChatGPT 可以扮演这些角色，但前提是你知道自己缺什么。

## 今天就能试的行动

1. **固定一个“低温度” prompt**：每天早晨用同一句话让 ChatGPT 列出今天最重要的 1 个任务，并只输出那一项。例如：“基于我昨天的进度，今天唯一必须完成的任务是什么？只输出任务名称。”
2. **用 Goblin Tools 分解一个拖延任务**：把一件你拖了很久的事（比如“整理书桌”）输入 Magic ToDo，调节辣度到你觉得“立刻能动手”的程度，然后按步骤执行。
3. **建立“日课”模板**：模仿孔子的“三省”，在 ChatGPT 里设三个固定问题（如“现在该做什么？”“刚才分心了吗？”“下一步是什么？”），每天定时问自己。
4. **警惕过度依赖**：每两周检查一次：如果没有 ChatGPT，我还能独立完成这些任务吗？如果答案是否定的，说明你在用拐杖，需要逐步减少依赖。

## 结语

ADHD 大脑和 LLM 都不是坏掉的机器，它们只是缺了一个调度层。ChatGPT 可以成为这个调度层的一部分，但前提是你把它当作一个结构化的 harness 来使用，而不是一个万能的对话伙伴。真正的稳定，不是消除波动，而是学会在波动中给自己搭一个脚手架。
