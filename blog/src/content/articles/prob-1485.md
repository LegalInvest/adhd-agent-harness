# 用 ChatGPT 给 ADHD 补上重锚定与目标漂移，真的能像 agent 那样工作吗？

> 把 ChatGPT 当作 ADHD 的外部执行层来实测：它补的是哪一块短板？

你打开 ChatGPT，想写一份周报，结果半小时后你在问它“如何用柠檬做电池”。这不是你的错——ADHD 大脑和 LLM 一样，都是高产但缺执行调度层的生成核心。分心走神或超聚焦跑偏，本质是“目标漂移”：大脑的上下文窗口被无关信息劫持，就像 agent 在长程任务中偏离 system prompt。那么，用 ChatGPT 来给自己补上重锚定，真的能像 agent harness 那样工作吗？

## 同构：ADHD 大脑与 LLM 的共享困境

ADHD 的工作记忆缺陷与 LLM 的无跨会话状态同构：两者都记不住刚才在做什么，需要外部记忆系统来补偿。RescueTime 自动追踪时间，揭示“时间到底去了哪里”（来源：Revolutionizing ADHD Management with AI Assistants），就像 agent 的日志系统。Focusmate 通过虚拟身体在场提供外部问责，将目标达成可能性从 25% 提升至 95%（来源：ADHD Task Managers That Work: Top AI Tools 2025），这相当于 agent 的 human-in-the-loop 监督。

但最核心的痛点是目标漂移。ADHD 者常因环境刺激或内在冲动偏离原计划，LLM agent 则因上下文膨胀或 prompt 遗忘而跑偏。两者的解法结构同构：都需要一个外部调度层，定期执行“重锚定”——把注意力拉回原始目标。曾国藩的“日课十二条”就是一套手动 harness：黎明即起、读书不二、写日记反省，用最笨最稳的方法抵消冲动（来源：人物案例）。这相当于 agent 的定时 re-grounding：每完成一个子任务，检查是否仍在主路径上。王阳明的“事上练”则是在行动中不断校准良知，相当于 agent 的在线 learning 与 prompt 修正。

## ChatGPT 作为外部执行层：它补的是哪一块？

ChatGPT 不能替代执行功能，但它可以成为你的“外部调度器”。具体来说，它补偿的是任务启动和重锚定这两个环节。

**任务启动**：ADHD 者常因多巴胺不足而无法迈出第一步。ChatGPT 的微任务分解功能——比如用 prompt “把‘写论文’拆成 5 个可执行的步骤”——直接对抗启动瘫痪（来源：ADHD Task Managers That Work: Top AI Tools 2025）。Goblin Tools 的“魔法待办”也做同样的事，但 ChatGPT 更灵活。

**重锚定**：当你发现自己正在研究柠檬电池时，对 ChatGPT 说“提醒我原本的目标是什么”，它能立即回溯对话，输出原始任务。这相当于 agent 的 system prompt 重载。更高级的用法是设置定时 prompt：“每 30 分钟问我一次‘当前任务是否偏离主目标？’”——这就是一个手动 re-grounding 循环。

但这里有一个关键区别：agent 的 harness 是自动的，而 ADHD 者的重锚定需要主动发起。如果你忘记设置提醒，或者无视提醒，ChatGPT 就无能为力。

## 证据与局限：同构不是万能药

现有工具的证据支持了外部执行层的有效性，但局限明显。Brain.fm 利用神经锁相技术生成专注音乐，在 2026 年一项 44 款 ADHD 应用测试中位列最佳 9 款之一，但缺乏 ADHD 人群的独立临床证据（来源：ADHD Apps: We tested 44 Apps and Here're the Best 9 in 2026）。Focusmate 的效果与身体在场效应一致，但其核心是真人匹配，AI 仅优化调度。RescueTime 的自动分类可能不精准，对执行功能较弱的用户仍是挑战。

更根本的争议是：AI 工具是脚手架还是拐杖？脚手架帮助内化技能，拐杖则导致永久依赖（来源：拐杖与脚手架）。目前没有长期追踪研究能回答这个问题。此外，同构性本身只是类比，缺乏实证基础（来源：矛盾与存疑）。ADHD 大脑与 LLM 的功能相似性，不意味着神经机制对应。

## 今天就能试的 3 个行动

1. **设置重锚定 prompt**：在 ChatGPT 中新建一个会话，第一句话写“我的目标是 [具体任务]，每完成一步请问我‘这一步是否朝向目标？’”。
2. **用 Focusmate 替代自律**：预约一个 50 分钟的身体加倍时段，让外部问责帮你启动。
3. **微任务分解**：下次面对复杂任务时，对 ChatGPT 说“把 [任务] 拆成 10 个可执行的步骤，每一步不超过 5 分钟”。

ChatGPT 不是你的执行功能，但它可以是你的调度器。前提是——你愿意为自己设计 harness。
