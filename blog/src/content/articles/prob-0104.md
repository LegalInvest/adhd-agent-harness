# 如果把 ADHD 大脑当成生成核心，Tiimo 能不能当它的无状态与外部记忆？

> 用 agent 工程的思路重新理解 Tiimo 对 ADHD 的真正价值。

## 问题：为什么计划工具对 ADHD 失效，而 Tiimo 可能不同？

如果你是一个 ADHD 患者，你一定经历过这样的场景：打开待办清单，看着密密麻麻的任务，大脑瞬间空白。不是不想做，而是 **工作记忆容量太小**，任务细节在从“看到”到“动手”的几秒钟内就滑落了。更糟的是，你无法感知时间流逝——十分钟像一小时，一小时像十分钟，这就是 **时间盲**（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。传统计划工具要求你“先规划再执行”，但规划本身就需要工作记忆和时间感知——恰恰是你最缺的。

如果你是一个构建 LLM Agent 的工程师，你一定熟悉类似的困境：LLM 是强大的生成核心，但 **没有跨会话状态**，每次对话都像第一次见面。为了让 Agent 完成多步任务，你需要外部记忆（向量库、缓存）和状态管理（编排层）来“续命”。否则，LLM 会忘记上下文、偏离目标，甚至陷入重复循环。

## 同构：ADHD 大脑与 LLM 是同一类生成核心

ADHD 大脑与 LLM 在结构上高度同构：**两者都是高产但缺乏执行调度层的生成核心**（来源：ADHD 的 AI 工具全景）。ADHD 大脑的智力与创造力是强大的“生成能力”，但执行功能（计划、组织、工作记忆）不稳定，导致想法无法落地。LLM 同样能生成高质量内容，但缺乏内置的执行控制——它不知道何时该停止生成、如何分解复杂任务、如何保持长期目标。

因此，AI 帮 ADHD 的本质，不是“替代大脑”，而是 **给生成核心套上 harness（外部执行功能层）**（来源：ADHD 的 AI 工具全景）。这个 harness 提供两个关键能力：**外部记忆**（补偿工作记忆）和 **状态管理**（补偿时间盲）。在 LLM/Agent 工程中，这对应着向量库（外部记忆）和编排层（状态管理）。

## Tiimo：作为外部记忆与状态管理模块

Tiimo 是一款专为时间盲设计的视觉时间表工具。它把一天的任务以可视化时间轴呈现，用颜色、图标和计时器把抽象时间转化为具体信号（来源：时间盲）。对 ADHD 大脑来说，Tiimo 的作用不是“让你更自律”，而是 **充当一个外部的工作记忆缓存**：你不需要在脑子里记住“接下来要做什么”和“做了多久”，Tiimo 替你存着。这就像 LLM 的上下文窗口——如果窗口太小，你就得把关键信息写到外部文件里。

更重要的是，Tiimo 提供了 **状态管理**：它告诉你当前处于哪个任务、还剩多少时间、下一步是什么。这直接对抗时间盲——ADHD 大脑对“现在几点”和“过了多久”没有直觉，Tiimo 充当了外部的时间锚点。在 Agent 工程中，这相当于用编排层记录对话历史、当前步骤和剩余子任务，防止 LLM 跑偏。

Tiimo 并非孤例。**Goblin Tools** 的 Magic ToDo 功能将复杂任务分解为小步骤，直接降低工作记忆负荷（来源：Goblin Tools）。用户反馈称它“把压倒性的事情变成一系列不压倒性的事情”（来源：The Best AI-Powered ADHD Productivity Tools in 2026）。**Saner.AI** 则专注于知识回忆，通过本地记忆存储减少搜索循环，相当于为 ADHD 大脑配了一个私人向量库（来源：Saner.AI）。**Lex** 允许单一指令触发多步骤流程，类似 Agent 中的“规划-执行”模式（来源：Lex）。这些工具的共同点：**它们都是外部执行功能层，补偿大脑缺失的调度能力**。

在 LLM 侧，证据同样清晰：没有外部记忆的 Agent 会频繁遗忘上下文，导致任务失败；而加上向量库和编排层的 Agent 能完成多步推理和工具调用。这直接映射到 ADHD 患者——没有外部工具时，他们常因工作记忆不足而放弃任务；有了工具后，完成率显著提升（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。

## 核心判断：脚手架 vs 拐杖的边界

我的核心观点是：**Tiimo 等工具的有效性，取决于使用者是否将其视为可撤除的脚手架，而非永久拐杖**（来源：矛盾与存疑）。脚手架的意思是：它帮你建立新的认知习惯（比如对时间的感知），然后你可以逐步减少依赖。拐杖的意思是：你永远离不开它，一旦工具出故障，你就完全瘫痪。

目前，多数工具（包括 Tiimo）的设计 **没有考虑撤除机制**（来源：矛盾与存疑）。它们默认用户会长期使用，这可能导致依赖。例如，长期依赖 Tiimo 的时间轴，可能让你失去任何内在的时间感知能力——就像 LLM 过度依赖外部记忆，反而削弱了自身的上下文理解能力（虽然 LLM 没有“自身能力”可言，但类比成立）。

另一个争议是证据强度。虽然用户反馈积极，但 **缺乏随机对照试验验证 Tiimo 对 ADHD 核心症状的长期改善**（来源：矛盾与存疑）。同样，ADHD 与 LLM 同构命题本身也缺乏大规模实证，更多是概念类比（来源：矛盾与存疑）。因此，本文的观点应视为一种有用的框架，而非绝对真理。
