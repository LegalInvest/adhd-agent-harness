# Focusmate 之于 ADHD，就像 function calling 工具调用 之于 LLM——但有人用错了

> 从同构视角实测 Focusmate：它到底补上了哪一层执行功能？

## 问题：为什么 Focusmate 对有些人有用，对另一些人却是“付费版番茄钟”？

Focusmate 的卖点很直接：你预约一个 50 分钟的视频时段，和一个陌生人同时在线工作，互相监督。对 ADHD 用户来说，它的效果两极分化——有人靠它完成了论文、写完了代码，有人却觉得“不过是花钱找人盯着自己，该分心还是分心”。

问题的根源在于：**Focusmate 补上的不是“他人在场”这个表面功能，而是执行功能中缺失的“调度层”**。如果只把它当作“陪学工具”，就忽略了它真正的同构价值——就像给 LLM 加上 function calling 一样，它是在生成核心（你的大脑）外面搭了一个外部执行层。

## 同构视角：ADHD 大脑 = 没有 function calling 的 LLM

ADHD 大脑与 LLM 在结构上同构：两者都是强大的生成核心，但缺乏可靠的内置执行调度层（来源：外部执行功能层）。LLM 本身是无状态、仅生成文本的核心，需要外部“脚手架”和“harness”来提供工具接口、上下文管理、安全执行等编排能力（来源：外部执行功能层）。同样，ADHD 大脑的核心困难在于执行功能的不稳定，尤其是工作记忆的“掉落”和任务启动困难（来源：外部执行功能层）。

最新研究进一步揭示了其神经机制：Transformer 自注意力机制在长上下文下冲突解决能力崩溃至随机水平，这与 ADHD 执行功能障碍的核心神经机制一致（来源：外部执行功能层）。ADHD 患者呈现“强记忆、弱控制”的认知剖面：工作记忆容量可能正常甚至超常，但认知灵活性和注意控制存在核心缺陷（来源：外部执行功能层）。

**Focusmate 补上的，正是这个“调度层”**。它通过以下机制实现：

- **任务启动**：预约时段相当于一个“外部启动信号”，解决了 ADHD 最难的“从0到1”问题。
- **工作记忆**：他人在场提供了一个“外部记忆锚点”，防止你在任务中忘记初衷。
- **上下文管理**：50 分钟的固定时段相当于一个“上下文窗口”，强制你维持注意力焦点。

这就像 LLM 的 agent scaffolding——“一个由提示、记忆、代码、工具和编排逻辑组成的模块化框架，围绕 LLM 构建，将其转化为可用的、目标驱动的系统”（来源：生成核心与调度层）。Focusmate 就是这个框架中的人肉“编排器”。

## 人物案例：孔子与王阳明的“外部调度层”

孔子和王阳明都是典型的 ADHD 大脑（高产生成 + 低调度可靠性），但他们各自建立了自己的“harness 系统”。

**孔子**的 ADHD 特质很明显：身高 1 米 9，精力旺盛，周游列国 14 年坐不住；冲动爱骂人，见南子急得对天发誓，骂宰予“朽木不可雕”；对音乐超专注到“三月不知肉味”，对种地等俗事零耐心；思维跳跃，《论语》全是场景化语录，无系统著作。他的专属 harness 是“克己复礼”——用外在的“礼”作为行为边界，每日反省，“吾日三省吾身”。这个“礼”系统，本质上就是一个外部调度层：它提供了行为规范、时间节奏和决策标准，替代了内在执行功能的缺失。

**王阳明**同样有 ADHD 特质：5 岁不会说话（神经发育延迟），12 岁挑战老师说读书为做圣贤，15 岁私出塞考察；兴趣爆发式转移：格物→兵法→道→佛→龙场悟道→军事→讲学；格竹子 7 天 7 夜格到吐血。他的 harness 是“致良知”和“知行合一”——用良知作为决策的唯一标准，静坐反省，军旅中也坚持讲学，“事上练”。这个系统同样是一个外部调度层：它提供了价值优先级和行动验证机制。

**这两个系统与 LLM harness 的结构同构**：孔子的“日课”对应 LLM 的“定时 re-grounding”，王阳明的“事上练”对应 agent 的“验证循环”。而 Focusmate 就是现代版的“日课”和“事上练”——通过外部他人在场，强制你进入执行状态。

## 但有人用错了：脚手架 vs 拐杖

Focusmate 被用错的情况很常见：用户只是开着视频，但并没有真正进入工作状态，反而因为需要“表演专注”而消耗额外认知资源。这就是**把脚手架当成了拐杖**。

脚手架（scaffolding）与拐杖（crutch）的区别在于：脚手架在补偿缺陷的同时，最终目标是让你学会独立行走；而拐杖是永久替代，让你越来越依赖。AI 工具辅助 ADHD 的本质是“给生成核心套 harness”——通过外部脚手架补偿执行功能缺陷（来源：主题综述）。但若过度依赖，AI 工具可能从脚手架变为永久拐杖，削弱内在执行功能（来源：矛盾与存疑）。

**Focusmate 的正确用法**：把它当作“外部调度器”，而不是“陪学工具”。你的目标是利用它来训练自己的执行功能，就像 LLM 通过 function calling 训练工具使用能力一样。具体来说：

- **使用前**：明确本时段的目标（如“完成论文引言”），并写在可见位置。
- **使用中**：如果分心，利用对方的“身体在场效应”重新锚定（来源：身体在场效应）。
- **使用后**：反思本时段的效率，并记录“下次如何更好”。

## 其他工具的协同：Goblin Tools、Saner.AI 与 Lex

Focusmate 解决的是“调度层”问题，但任务分解、工作记忆和时间感知还需要其他工具配合：

- **Goblin Tools**：它的 Magic ToDo 功能能将复杂任务分解为可管理的小步骤（来源：Goblin Tools），降低启动门槛。你可以把 Focusmate 时段的目标用 Goblin Tools 分解，然后逐项执行。
- **Saner.AI**：它通过增强知识回忆，减少搜索循环和标签切换（来源：Saner.AI）。在 Focusmate 时段中，你可以用 Saner.AI 快速找回所需信息，避免因工作记忆不足而中断。
- **Lex**：它允许通过单一指令触发多步骤任务序列（来源：Lex），适合在 Focusmate 时段中快速启动复杂任务。

这些工具共同构成了一个“ADHD harness 系统”，就像 LLM 的 agent scaffolding 一样——Focusmate 提供调度层，Goblin Tools 提供任务分解层，Saner.AI 提供记忆层，Lex 提供快速启动层。

## 诚实指出局限

必须承认，ADHD 与 LLM 的同构性目前仅是一种理论类比，并非经过验证的科学事实（来源：矛盾与存疑）。Focusmate 的效果也缺乏严格的临床验证，多数证据来自用户报告（来源：矛盾与存疑）。此外，部分用户可能因社交焦虑而抗拒 Focusmate，反而增加认知负担。
