# 如果把 ADHD 大脑当成生成核心，Inflow 能不能当它的无状态与外部记忆？

> 用 agent 工程的思路重新理解 Inflow 对 ADHD 的真正价值。

你打开 Inflow，看到“今日课程”和“打卡”，然后关掉。五分钟后，你想不起刚才要做什么。这不是你的错——你的工作记忆像 LLM 的上下文窗口，容量不小，但控制弱，容易被干扰挤占。明尼苏达大学的研究发现，LLM 的工作记忆容量甚至超过常人，但认知灵活性和注意控制存在核心缺陷（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。这几乎就是 ADHD 的神经心理剖面。耶鲁大学进一步揭示，自注意力机制本身会导致工作记忆容量限制：随着 N-back 任务难度增加，注意力分数熵增、注意力弥散（来源：Self-Attention Limits Working Memory Capacity of Transformer-Based Models）。你和 LLM 共享同一个计算瓶颈：不是记不住，是管不住该记什么。

## 问题：生成核心缺一个调度层

ADHD 大脑和 LLM 本质上都是“高产但缺执行调度层的生成核心”。LLM 能写诗、编程、推理，但如果没有外部记忆（向量库）和上下文管理（prompt 工程），它会丢失会话状态、重复回答、被无关信息带偏。ADHD 大脑能产生海量创意和洞察，但工作记忆容量小、边做边忘、任务启动困难。两者的解法同构：为生成核心套上一个 harness（脚手架），提供无状态与外部记忆的补偿。

## 同构：Inflow 作为外部记忆与启动脚本

Inflow 的设计恰好对应这个 harness 结构。它的“今日课程”和“打卡”功能，本质上是为 ADHD 大脑提供：
- **外部记忆**：课程内容持久化，不依赖你的工作记忆。这类似 LLM agent 的向量数据库——每次会话开始时检索相关记忆，而不是让模型自己记住。Saner.AI 的“知识回忆”功能也是同一思路：通过本地记忆减少搜索循环和标签切换（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。
- **启动脚本**：每日任务列表降低“冷启动”难度，相当于 LLM agent 的 system prompt 和 few-shot 示例。Goblin Tools 的 Magic ToDo 将复杂任务分解为小步骤，Lex 允许单一指令触发多步骤流程（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar），都是为生成核心提供低门槛的启动脚本。

从 agent 工程角度看，Inflow 就是你的“harness”：它管理上下文（只显示今天的任务），提供外部状态（打卡进度），并给出即时反馈（完成✅）。这完全对应 LLM agent 的“无状态 + 外部记忆”架构。

## 脚手架 vs 拐杖：边界在哪？

但这里有一个关键争议：Inflow 是脚手架还是拐杖？wiki 资料明确指出，AI 工具应作为脚手架促进能力发展，而非永久拐杖，过度依赖可能削弱内在能力（来源：拐杖与脚手架）。Inflow 的课程试图教你策略，但如果你每天只是机械打卡而不内化，它就变成了拐杖。同样，LLM agent 如果完全依赖外部记忆而不优化 prompt，也会退化。

我的判断是：Inflow 的有效性取决于你是否把它当作“训练工具”而非“替代品”。它提供的外部记忆和启动脚本，应该像健身房的辅助器械——帮你完成当前动作，但最终目标是让你肌肉更强。如果 Inflow 让你永远依赖它的提醒，那它就是拐杖。

## 证据与局限

现有证据主要来自用户报告。Inflow 本身在 wiki 资料中被列为工具，但缺乏独立随机对照试验（来源：矛盾与存疑）。Goblin Tools 的用户反馈称其“将压倒性的事情变成一系列不压倒性的事情”（来源：The Best AI-Powered ADHD Productivity Tools in 2026），但同样缺乏严格临床验证。同构视角虽然有力，但本质是概念类比，不能替代疗效证据。

## 今天就能试的行动

1. **把 Inflow 当作你的“向量数据库”**：每天早晨打开 Inflow，把今日课程要点抄到便签上，然后关掉 app。这训练你主动提取记忆，而非被动接收。
2. **用 Goblin Tools 分解一个让你拖延的任务**：输入“整理桌面”，得到步骤列表，每完成一步就划掉。这模拟了 agent 的 step-by-step 执行。
3. **设置一个“冷启动”仪式**：在开始工作前，用 Saner.AI 或 Mem 快速检索昨天留下的笔记，作为外部上下文注入。这相当于给 LLM 喂历史对话。
4. **每周反思一次“脚手架 vs 拐杖”**：问自己：今天 Inflow 是帮我学会了新策略，还是只是让我完成了打卡？如果答案是后者，调整使用方式。

## 结论

把 ADHD 大脑当成生成核心，Inflow 就是它的无状态与外部记忆——不是治愈，而是补偿。它和 LLM agent 的 harness 同构：都是为高产但缺调度层的系统提供外部执行功能。但记住，最好的脚手架是让你最终不再需要它。
