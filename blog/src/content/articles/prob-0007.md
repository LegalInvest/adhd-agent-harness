# 给 ADHD 加一道「核对清单/验证步骤」，和给 agent 加 self-critique 循环，是同一种工程吗？

> ADHD 侧：冲动下结论、想当然；LLM/harness 侧：LLM 自信地编造（幻觉），靠验证与自我批判兜底。用双域证据作答。

## 问题：ADHD 的冲动下结论，和 LLM 的自信编造，是同一个bug？

你是否曾脱口而出一个结论，事后才发现自己漏掉了关键信息？你是否见过 ChatGPT 一本正经地编造一个不存在的引用，语气却无比笃定？如果你既是 ADHD 患者又是 AI 工程师，这种“似曾相识”可能会让你后背发凉：ADHD 大脑的冲动下结论、想当然，和 LLM 的幻觉（hallucination），会不会是同一个底层问题的不同表现？

答案藏在“工程”里。ADHD 患者靠核对清单、验证步骤来兜底；LLM agent 靠 self-critique 循环、验证器来纠偏。这两种做法，本质上是同一类工程——给一个“高产但缺执行调度层的生成核心”套上外部脚手架。

## 同构：两个生成核心，一个调度缺陷

ADHD 大脑与 LLM 在架构上惊人相似：两者都是强大的生成器，但缺乏可靠的内置执行调度层（来源：ADHD 大脑与 LLM 的同构）。ADHD 大脑的发散思维、联想能力，与 LLM 的文本生成能力类似，但都容易偏离轨道、被干扰带偏。LLM 的“无状态”对应 ADHD 的工作记忆缺陷——无法可靠保留跨会话上下文（来源：无状态与外部记忆）。两者都需要外部系统来补偿任务启动困难、注意力分散和认知负荷超载（来源：外部执行功能层）。

更硬的证据来自一项研究：用经典 Stroop 冲突任务测试 Transformer 注意力，发现短上下文中表现正常，但随序列变长（认知负荷增加），模型在冲突试次上骤降到随机水平——无法抑制优势反应、无法解决认知冲突。这与 ADHD 执行功能缺陷的核心标志一一对应：注意控制不足、干扰抑制缺陷、随认知负荷增加而崩溃（来源：Deficient Executive Control in Transformer Attention）。

## 历史中的 harness：曾国藩的“日课”与民主集中制

晚清名臣曾国藩是典型的 ADHD 患者：30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”；读书慢，“他人目下二三行，余或疾读不能终一行”；情绪不稳定，打败仗就跳水自杀。他的自我管理方案——日课十二条（黎明即起、读书不二、谨言、写日记反省等）和“结硬寨打呆仗”——本质上就是一套外部执行调度层。日课相当于定时 re-grounding 机制，写日记相当于自我 critique 循环。这套 harness 让他从浮躁青年变成“半个圣人”，平定太平天国，发起洋务运动。

毛泽东同样有 ADHD 特质：小时候是问题儿童，爱读闲书；一辈子读书不辍，行军路上也带着书；思维跳跃，讲话充满比喻；冲动敢赌，四渡赤水、抗美援朝都是险棋。他的 harness 系统是“批评与自我批评”“调查研究”“民主集中制”——用集体决策制衡自己的冲动，用组织系统管理军队和国家。这些与 LLM agent 的“多 agent 辩论”“验证器”“外部知识库”在结构上同构：都是引入外部视角来纠偏。

## 今天的工具：从 Goblin Tools 到 Motion

当代 AI 工具正在实践这种同构工程。Goblin Tools 的 Magic ToDo 功能将模糊任务分解为具体步骤，用户可调节“辣度”控制粒度（来源：Goblin Tools）。这相当于为 ADHD 大脑加了一道“任务分解验证”：不再凭直觉跳进任务，而是先核对步骤。Saner.AI 通过本地记忆和快速检索减少搜索循环，相当于外部工作记忆（来源：Saner.AI）。Motion 自动排程并动态调整，消除了“下一步该做什么”的决策负担（来源：Motion）。

这些工具的共同逻辑是：在外层加一个调度器，在生成冲动和实际行动之间插入一道验证步骤。这与 LLM agent 的 self-critique 循环完全对应：模型生成输出后，由另一个 prompt 或验证器检查逻辑、事实、一致性，再决定是否采纳。

## 争议与边界：脚手架还是拐杖？

但同构不等于万能。当前证据多为概念类比和初步实验，缺乏大规模随机对照试验证明同构假设或 AI 工具的临床有效性（来源：矛盾与存疑）。最大的争议是：AI 工具是“外部执行功能层”还是“永久拐杖”？过度依赖可能导致能力退化（来源：拐杖与脚手架）。曾国藩的日课和毛泽东的批评与自我批评，最终内化为习惯；而今天的 AI 工具如果只做“自动代劳”，反而可能削弱用户自身的执行功能。

此外，个体差异巨大：超聚焦是否应被引导而非打断？部分用户反馈强行打断（如番茄钟）有效，而柔性引导可能不够强力（来源：矛盾与存疑）。AI 工具的学习使用成本本身也可能增加认知负荷（来源：矛盾与存疑）。

## 今天就能试的行动

1. **用 Goblin Tools 分解一个让你拖延的任务**：输入“写报告”，看 AI 拆成 5-10 步，每完成一步打勾——这就是你的 self-critique 循环。
2. **给常用 AI 工具加一道“验证 prompt”**：在提问后追加“请检查你刚才的回答，指出可能不准确的地方”——模拟多 agent 辩论。
3. **设置定时 re-grounding 闹钟**：每 25 分钟响铃，问自己“我现在在做的，是我计划要做的吗？”——曾国藩的“日课”现代版。

同构工程不是要消除 ADHD 的创造力或 LLM 的生成能力，而是给它们配上可靠的调度层。正如曾国藩用日记骂了自己一辈子，才换来“结硬寨打呆仗”的定力——真正的 harness，是在冲动与行动之间，永远留一道验证的缝隙。
