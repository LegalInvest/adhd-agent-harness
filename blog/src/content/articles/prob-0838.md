# Focusmate 之于 ADHD，就像 planner-executor 任务分解 之于 LLM——但有人用错了

> 从同构视角实测 Focusmate：它到底补上了哪一层执行功能？

## 问题：为什么「启动」这么难？

ADHD 群体常被一种矛盾折磨：大脑里想法翻涌、创造力爆棚，但面对一个具体任务时，却像被钉在原地。这不是懒——而是执行功能缺陷导致的任务启动困难（来源：任务启动）。多巴胺系统失衡让动机难以激发，工作记忆容量有限让任务步骤无法清晰呈现，时间盲让紧迫感失真（来源：执行功能障碍）。

与此同时，工程师们在构建 AI Agent 时也遇到类似困境：LLM 能生成高质量文本、推理复杂问题，但让它执行一个多步骤任务（比如“写一篇论文并提交”）时，它常常迷失方向、忘记上下文、无法拆解子目标。于是出现了 planner-executor 循环——让一个规划模块将长任务分解为可执行的小步骤，再交给执行模块依次完成。

这两种困境惊人地同构：ADHD 大脑和 LLM 都是高产的生成核心，但缺乏可靠的执行调度层（来源：AI 与 ADHD 的时间管理）。Focusmate——一个提供虚拟共在（body doubling）的服务——恰好补上了这一层。

## Focusmate 补上了哪一层？

Focusmate 的核心机制很简单：你预约一个时段，与另一位用户视频连线，各自安静工作。它不提供任务分解、不自动排程，只做一件事——制造一种「被看见」的在场感。

这看似朴素，却直击 ADHD 执行功能缺陷的核心：任务启动。身体在场效应（body doubling）表明，物理环境中他人的存在能降低启动阻力（来源：任务启动）。Focusmate 将这种效应数字化：当你看到屏幕另一端有人也在工作，你的大脑会进入一种「共同注意」状态，减少犹豫和分心。

类比到 LLM 领域，Focusmate 相当于一个外部 scheduler——它不替你思考，但替你维持执行窗口。就像 planner-executor 循环中的 planner，它负责「现在该做什么」的决策，而不是具体的生成内容。

## 有人用错了：把它当拐杖而非脚手架

问题在于，许多 ADHD 用户依赖 Focusmate 作为唯一启动手段，没有它就无法工作。这就像 LLM 的 planner 模块如果只依赖外部提示词而不内化分解能力，一旦提示词消失，agent 就会瘫痪。

拐杖与脚手架的区别在于：脚手架设计为可逐步撤除的临时支撑，而拐杖是永久依赖（来源：矛盾与存疑）。Focusmate 本身没有撤除机制——它鼓励长期订阅，而非训练用户内在的执行功能。同样，Motion 和 Reclaim.ai 自动排程，减少了决策负担，但长期使用可能削弱用户自己规划的能力（来源：Motion；Reclaim.ai）。

证据显示，多数 AI 工具设计为长期使用，未提及撤除机制（来源：矛盾与存疑）。这并非说工具无效——而是提醒我们，任何外部调度层都应有「撤离计划」。

## 真实证据：两边的同构验证

ADHD 侧：研究指出，任务启动困难是 AI 干预的重点领域，Goblin Tools 的「魔法待办」通过分解任务降低启动门槛（来源：任务启动）。Tiimo 通过视觉化时间直接应对时间盲（来源：Tiimo）。这些工具的共同点：它们都提供了缺失的调度层。

LLM/Agent 侧：Planner-executor 架构已被广泛采用，例如 ReAct 模式、Tree-of-Thoughts。这些方法本质上是将生成核心（LLM）与外部规划器耦合。证据表明，没有规划器的 LLM 在长任务上表现急剧下降。

但两边的证据都有局限：ADHD 工具缺乏长期随机对照试验，同构命题本身是理论模型，需更多实证（来源：矛盾与存疑）。

## 今天就能试的行动

1. **用 Focusmate 做「启动仪式」**：每天选一个最难启动的任务，预约 25 分钟 Focusmate 时段。完成后记录：没有它你是否能启动？逐步缩短使用时间，尝试用计时器代替。
2. **把任务分解外包给 AI**：用 Goblin Tools 的「魔法待办」将复杂任务拆成 3-5 小步，然后手动执行每一步。这就像 agent 的 planner 生成子目标，但执行仍由你控制。
3. **设置「调度层检查点」**：使用 Motion 或 Reclaim.ai 自动排程后，每天花 2 分钟审视日程，问自己：「如果明天没有这个工具，我能自己排吗？」记录哪些决策可以内化。
4. **对比实验**：一周用 Focusmate 启动，一周用普通计时器。比较两种方法的完成率和情绪。这能帮你识别工具是脚手架还是拐杖。

## 诚实面对局限

同构视角是强大的分析框架，但并非事实。现有证据多来自概念类比和用户反馈，缺乏严格临床研究（来源：矛盾与存疑）。多巴胺干预效果存在争议，部分工具声称的神经科学原理缺乏独立验证（来源：矛盾与存疑）。

更重要的是，个体差异巨大：对某些人，Focusmate 是救命稻草；对另一些人，它加剧社交焦虑。工具的有效性高度依赖个人反应（来源：任务启动）。

最终，Focusmate 和 planner-executor 循环都指向同一个真理：生成核心需要调度层。但最好的调度层，是最终能被内化的那一个。
