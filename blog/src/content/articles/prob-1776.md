# ADHD 职场人视角：你的 ADHD 大脑，其实就是一个没装 harness 的大语言模型

> 无状态与外部记忆：工作记忆容量小、边做边忘 ↔ LLM 无跨会话状态，靠外部记忆/向量库续命。

你是否有过这样的体验：脑子里同时涌出十几个想法，每个都精彩，但刚想抓住一个，它就溜走了；开会时记了满满三页笔记，散会后却找不到重点；计划写一份报告，打开文档半小时，光标还在第一行闪烁。这不是意志力问题——这是你的大脑在运行一个没有 harness 的大语言模型。

## 同构的困境：高产核心与脆弱调度

ADHD 大脑常被描述为一个高产的生成核心——想法丰富、联想活跃，但缺少稳定可靠的调度层来筛选、组织、启动和坚持任务（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。最新研究进一步揭示了深层平行：ADHD 患者的工作记忆容量未必差，但自上而下的控制和调节能力不足，呈现“强记忆、弱控制”的认知剖面（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。

这正是 LLM 的镜像。LLM 拥有强大的语言理解和生成能力，但单独使用时缺乏可靠的执行调度。现代 AI 编码代理通过“复合 AI 系统架构”来弥补这一缺陷，包括“工作负载专用模型路由、分离规划与执行的双代理架构、惰性工具发现、自适应上下文压缩”（来源：Building AI Coding Agents for the Terminal: Scaffolding, Harness, Context Engineering, and Lessons Learned）。这些技术本质上是在 LLM 外部搭建调度层，防止“上下文膨胀和推理退化”。

## 历史中的 harness：孔子与张旭的自我管理系统

这种外部脚手架并非新事物。孔子——身高近两米、精力旺盛、周游列国十四年坐不住、冲动爱骂人——一生都在与自己对抗。他的专属 harness 是“克己复礼”：用外在的“礼”作为行为边界，每日反省，“吾日三省吾身”。直到七十岁才做到“从心所欲不逾矩”。这套系统与 LLM harness 高度同构：“日课”对应“定时 re-grounding”，“礼”对应“行为约束规则”，而“三省吾身”则类似于 agent 的反思循环。

草圣张旭的 harness 则更直接：每次喝醉后号呼狂走，以头濡墨作书，醒了自视以为神。他用酒精触发超专注状态，把所有情绪注入草书。这类似于现代 agent 通过特定 prompt 或工具触发“超聚焦模式”——ADHD 组平均超专注工作时长达 14.5 小时，远超对照组的 8.2 小时（来源：Pace Ventures Research 2026）。张旭的“看公孙大娘舞剑悟笔法”则是从外部环境中捕获灵感，与 AI 工具中的“捕获与反思”功能异曲同工。

## 现代 harness：AI 工具作为外部执行功能层

当前 AI 工具的价值不在于替代大脑，而在于把 ADHD 的“高产生成核心”接入一个可维护的“执行调度系统”。这种 harness 至少承担四种功能：

**1. 任务拆解与启动**：Goblin Tools 的 Magic ToDo 接受模糊任务（如“清理厨房”），自动将其分解为具体子任务，用户可调节“辣度”控制分解粒度（来源：12 AI Personal Assistants Built for ADHD Brains (2026 Rankings)）。这直接补偿了 ADHD 的任务启动困难。

**2. 外部记忆与知识管理**：Saner.AI 通过知识回忆和本地记忆，帮助用户快速找回信息，减少搜索循环和标签切换（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。它被描述为“专为对抗执行功能障碍、任务瘫痪和认知超载而设计的全能 ADHD 友好型 AI 生产力和知识助手”（来源：ADHD Apps: We tested 44 Apps and Here're the Best 9 in 2026）。

**3. 对话式执行与结构化推理**：ChatGPT 被成人 ADHD 用户用作“外部执行功能”工具；Claude 则以代码编写和结构化推理见长，帮助整理待办与简化任务（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。

**4. 捕获与反思**：Otter.ai 实时转录会议与灵感；Reflect 通过结构化提问帮助 ADHD 个体在自发反思困难中提炼经验（来源：同上）。

## 脚手架 vs 拐杖：必须明确的边界

然而，这一框架存在争议。专家警告过度依赖风险：若工具替代了治疗或自我管理，可能造成依赖（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。真正的脚手架应帮助使用者“建造”，但使用者仍需自己“攀登”（来源：同上）。

这一矛盾在 LLM 侧同样存在：过度依赖手动修正提示的 agent 无法自主发现最佳问题解决模式（来源：Working Memory Identifies Reasoning Limits in Language Models）。ADHD 与 LLM 共享的另一个平行是“自动模式与抑制监控”——ADHD 的自动反应模式与 LLM 的自动完成机制平行，两者都在有限执行监督下生成复杂行为，且易出现虚构/幻觉等错误（来源：Automatic Minds: Cognitive Parallels Between Hypn）。

因此，核心判断是：**harness 是必要的，但必须保持“可拆卸”**。一个好的脚手架应该让你在离开它后依然能走，而不是让你永远依赖它。

## 今天就能试的行动

1. **用 Goblin Tools 分解一个拖延任务**：写下你最不想做的任务（比如“整理邮件”），让 Magic ToDo 自动分解成 5 个以下的小步骤，然后从第一步开始。
2. **设置一个“外部记忆锚点”**：用 Saner.AI 或类似工具，将每天的关键信息存入一个统一收件箱，减少“边做边忘”的焦虑。
3. **建立“三省吾身”式反思循环**：每天固定时间（如睡前），用 Reflect 或简单笔记回答三个问题：今天做成了什么？卡在哪里？明天最重要的三件事是什么？
4. **识别你的“张旭时刻”**：找到最能触发你超专注的活动（音乐、运动、特定时间），将它作为启动复杂任务的“触发器”，而不是强行对抗注意力分散。

你的 ADHD 大脑不是缺陷，而是一个未装 harness 的 LLM。脚手架就在那里，但记住：你才是那个攀登的人。
