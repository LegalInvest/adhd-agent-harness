# ADHD 学生如何用「给生成核心套 harness」的思路应对考试与作业里的注意力容易被环境带偏？

> 把 LLM 工程里的「LLM 受上下文窗口内容左右，需要上下文工程」迁移到ADHD 学生的考试与作业场景，双域证据作答。

如果你是 ADHD 学生，你一定经历过这样的场景：翻开书本五分钟，手机通知、窗外的谈话、甚至笔尖的划痕都能把你的注意力拽走。你明明想专注，但大脑像一台没有操作系统的超级计算机——算力爆棚，却无法调度。

如果你是做 Agentic Harness 的工程师，你一定也熟悉另一种场景：你精心设计的 LLM agent 在上下文窗口里塞满了指令，但推理到一半，它突然被无关的 token 带偏，输出变得混乱。你心想：这模型能力没问题，缺的是上下文工程。

这两个场景看似无关，实则共享同一核心矛盾：**一个高产但缺执行调度层的生成核心，被环境（或上下文窗口）里的无关信息左右。** ADHD 大脑和 LLM 是同构的——两者都有强大的生成能力（创意、联想、模式识别），但内置的执行调度层（工作记忆、注意控制）先天不足。因此，解法也同构：**给生成核心套上外部 harness，通过上下文工程主动管理「当下注意什么」。**

### 同构的证据：工作记忆与注意控制

ADHD 的工作记忆缺陷是执行功能的核心瓶颈（来源：工作记忆）。你无法同时记住任务细节和抵抗干扰，因为大脑的「临时缓存」容量有限且不稳定。LLM 也有类似问题：耶鲁大学研究发现，自注意力机制本身会导致工作记忆容量限制——随着 N-back 任务 N 值增加，注意力分数熵增，表现下降，这与人类注意分散的计算本质同源（来源：工作记忆）。明尼苏达大学的测试进一步揭示，LLM 呈现「强记忆、弱控制」剖面：工作记忆容量甚至超过常人，但认知灵活性与注意控制存在核心缺陷（来源：工作记忆）。这正是 ADHD 的经典神经心理模式。

所以，瓶颈不在容量或算力，而在 orchestration 层。ADHD 学生需要外部调度器，LLM agent 也需要上下文工程。

### 历史人物的 harness：曾国藩的日课与苏轼的豁达

曾国藩是典型的 ADHD 患者：30 岁前浮躁坐不住，天天串门喝酒看杀人，日记里骂自己「无恒」「浮躁」；读书慢到「他人目下二三行，余或疾读不能终一行」。他的 harness 是「日课十二条」：黎明即起、读书不二、谨言、写日记反省。这套系统本质是**定时 re-grounding**——每天用固定动作把注意力拉回正轨，抵消大脑的冲动漂移。

苏轼的 ADHD 特质则表现为兴趣极广、对无聊行政零耐心、情绪来得快去得快。他的 harness 是「以豁达为核心的外部调度」：被贬到哪都好好生活，用美食、写作、交朋友转移注意力，每日读书写字不因挫折停止。这套系统本质是**环境重构**——主动设计生活内容，让注意力自然流向有意义的事。

这两套系统与 LLM agent 的 harness 同构：曾国藩的日课对应定时 re-grounding（如每轮对话重置系统提示），苏轼的环境重构对应上下文窗口的主动裁剪（如只保留当前任务相关的记忆）。

### 当代工具：外部执行功能层

今天，AI 工具正在成为 ADHD 学生的外部执行功能层（来源：AI 与 ADHD 的专注力）。

- **Focusmate**：通过虚拟身体在场提供即时问责。你预约时段，平台匹配一位伙伴，视频共同工作。这直接补偿任务启动困难——社会压力替代了缺失的内在多巴胺驱动。其核心是 AI 匹配算法，本质是**外部调度器**（来源：Focusmate）。
- **RescueTime**：自动追踪时间，揭示你实际去了哪里。它补偿时间盲和工作记忆负担，让你看到注意力漂移的模式。其 AI 自动分类和分心阻断，相当于**外部监控层**（来源：RescueTime）。
- **Brain.fm**：利用神经锁相技术生成专注音乐，帮助进入状态。它通过调节大脑信息处理方式，间接影响多巴胺水平，相当于**环境调节器**（来源：Brain.fm）。
- **Forest**：游戏化番茄钟，利用多巴胺机制增强动机，相当于**奖励调度器**（来源：AI 与 ADHD 的专注力）。

这些工具的共同逻辑是：**不要试图修复内部执行功能，而是用外部脚手架接管调度。** 这与 LLM 工程中的上下文窗口管理、系统提示设计、记忆检索增强完全同构。

### 核心观点：脚手架，不是拐杖

但这里有一个必须指出的争议：**脚手架与拐杖的边界。** 如果 ADHD 学生永远依赖外部工具才能专注，这些工具是否反而阻碍了执行功能的内化？（来源：拐杖与脚手架）同样，LLM agent 如果过度依赖人工设计的上下文工程，是否失去了自主推理的鲁棒性？

我的判断是：**对于 ADHD 大脑，外部脚手架是必要的补偿，而非可选的辅助。** 因为工作记忆和注意控制的缺陷是神经基础性的，不会通过「练习」完全修复。但脚手架应该逐步内化为习惯——就像曾国藩的日课从刻意执行变成第二天性——而不是永远依赖某个具体 app。

### 今天就能试的行动

1. **用 Focusmate 启动最难的任务**：预约一个 25 分钟时段，告诉伙伴你要做什么。社会压力会帮你跨过启动门槛。
2. **用 RescueTime 做一次时间审计**：开一天自动追踪，晚上看报告。找出三个最大的分心源，明天主动屏蔽它们。
3. **设计你的「日课」**：选三个固定动作（如早起后先写三件事、每工作 45 分钟起身走一圈、睡前写一句话反思），用手机提醒执行。
4. **给 LLM agent 做上下文工程**：如果你在开发 agent，尝试每轮对话后重置系统提示，或只保留最近 5 轮对话作为记忆，观察推理质量是否提升。

### 局限与诚实

必须承认，同构性目前仍是理论类比，缺乏实证基础（来源：矛盾与存疑）。Brain.fm 在 ADHD 人群中的独立临床证据有限（来源：Brain.fm）。Focusmate 的效果主要来自用户报告，而非随机对照试验。个体差异也很大：注意缺陷为主型 vs 多动冲动型对同一工具的反应可能截然不同。

但这些局限并不否定核心洞察：**当你的大脑或模型无法自我调度时，别硬撑，套个 harness。** 这不是作弊，这是工程智慧。
