# Perplexity 之于 ADHD，就像 外部记忆/向量库 之于 LLM——但有人用错了

> 从同构视角实测 Perplexity：它到底补上了哪一层执行功能？

## 你被 Perplexity 骗了——它本不该是“搜索工具”

如果你有 ADHD，你可能已经试过 Perplexity：打开它，问一个问题，得到带引用的回答，然后关掉标签页。十分钟后，你又打开它问同一个问题——因为你忘了答案。如果你在做 LLM agent 工程，你可能也干过类似的事：每次对话都从头开始，让模型重复思考，因为你没给它挂上外部记忆。

这不是你的错。这是工具被用错了。

Perplexity 之于 ADHD，就像向量数据库之于 LLM——但绝大多数人只把它当成一个更漂亮的搜索引擎。它真正的能力不是“找到答案”，而是“记住你刚刚在干什么”。

## 无状态的大脑，无状态的模型

ADHD 大脑与 LLM 共享一个致命弱点：它们都是“高产但缺执行调度层的生成核心”。你的大脑可以产生无数创意，但工作记忆容量小，边做边忘（来源：[[工作记忆]]）。LLM 可以生成流畅文本，但每次对话都从零开始，没有跨会话状态。

这正是“无状态”问题。LLM 靠外部记忆（向量库、检索增强生成）来续命；ADHD 大脑也需要外部记忆系统来补偿工作记忆缺陷（来源：[[无状态与外部记忆]]）。Perplexity 恰恰提供了这种外部记忆——但前提是你用它来“保持上下文”，而不是“搜索答案”。

## Perplexity 补上了哪一层执行功能？

执行功能包括任务启动、工作记忆、时间管理等（来源：[[执行功能障碍]]）。Perplexity 主要补上的是**工作记忆的“持续访问”层**。

想象你在写一篇报告，突然想起一个需要查证的数据。普通搜索引擎会让你跳转、阅读、关闭，然后回到文档——此时你已经忘了刚才写到哪。Perplexity 的“连续对话”功能让你在一个线程里不断追问，所有上下文都保留。你不需要在多个标签页间切换，不需要记住你查过什么——它替你记住了。

这正是 Saner.AI 和 Mem 的设计逻辑：通过本地记忆和快速检索，减少“搜索循环”和“标签切换”（来源：[[Saner.AI]]）。Perplexity 本质上在做同一件事：它把一次性的问答变成了持续的对话，把“查资料”变成了“与外部记忆交谈”。

## 为什么说“有人用错了”？

很多人把 Perplexity 当作“更好的 Google”——问一个问题，得到一个答案，然后结束。这不是外部记忆，这是外部拐杖。你依赖它找到信息，但你没有内化任何东西，也没有构建一个可复用的上下文。

真正的脚手架用法是：**把 Perplexity 当作你的“第二工作记忆”**。在开始一个任务前，先开一个 Perplexity 线程，把任务相关的背景、问题、中间结论都丢进去。然后在整个任务过程中，反复回到这个线程，而不是新建搜索。这样，Perplexity 就成了你的“持久化上下文”。

这和 LLM agent 工程中的向量数据库完全同构：你不应该每次 query 都从头检索，而应该维护一个会话记忆，让 agent 知道“我们已经讨论过什么”。

## 证据在哪里？矛盾在哪里？

现有证据主要来自用户报告和工具设计逻辑。Goblin Tools 的任务分解被用户评价为“将压倒性的事情变成一系列不压倒性的事情”（来源：[[Goblin Tools]]）。Lex 的单一指令触发多步骤执行，直接针对任务启动困难（来源：[[Lex]]）。这些工具的有效性逻辑与 Perplexity 的“持续上下文”逻辑一致：它们都在为生成核心提供外部执行功能层。

但矛盾同样明显：**缺乏大规模对照实验**，证据以定性为主（来源：[[矛盾与存疑]]）。过度依赖可能削弱内在能力——如果你总让 Perplexity 替你记住，你的工作记忆可能更不锻炼（来源：[[矛盾与存疑]]）。个体差异也很大：有人觉得 Perplexity 的连续对话是救星，有人觉得它只是另一个分心源。

## 今天就能试的 3 件事

1. **把 Perplexity 当作“任务伴侣”**：每次开始一个复杂任务（写文章、做计划、研究课题），先开一个 Perplexity 线程，把任务目标、已有信息、问题都写进去。然后在整个任务中，只在这个线程里搜索，不新建标签页。

2. **利用“收藏”功能构建外部记忆库**：把有用的回答收藏到 Perplexity 的收藏夹，定期回顾。这相当于你的个人向量数据库。

3. **对比使用前后的大脑负荷**：用 Perplexity 连续对话完成一个任务，再用传统搜索引擎完成另一个类似任务。感受哪个让你更少“我刚刚要干什么”的瞬间。

Perplexity 不是搜索工具，它是你的外部工作记忆。别再用它问完就忘——让它替你记住，你才能腾出大脑去生成。
