# ADHD 家长如何用「给生成核心套 harness」的思路应对陪伴 ADHD 孩子里的任务启动困难、不会拆解？

> 把 LLM 工程里的「agent 用 planner-executor 循环分解长任务」迁移到ADHD 家长的陪伴 ADHD 孩子场景，双域证据作答。

## 问题：启动与拆解，ADHD 孩子的“死循环”

ADHD 孩子面对作业或家务时，常常卡在“开始”这一步：不是不想做，而是不知道从哪里下手。大脑里塞满了想法，却像一团乱麻——任务启动困难与不会拆解互为因果。家长反复催促“快去做呀”，孩子却更焦虑、更拖延。这不是态度问题，而是执行功能缺陷：工作记忆容量有限且不稳定，时间感知扭曲（时间盲），导致孩子无法把“写作文”分解成“找题目→列提纲→写开头”等可执行的子任务（来源：AI 与 ADHD 的时间管理）。

## 同构：ADHD 大脑与 LLM agent 共享同一个“核心缺陷”

如果你熟悉 LLM agent 工程，会发现一个惊人的相似：一个没有 harness 的 LLM 只是一个无状态的 API 调用（来源：无状态与外部记忆）。它每次对话都从零开始，没有跨会话记忆，容易上下文膨胀、推理退化（来源：上下文工程）。而生产级 agent 通过 planner-executor 循环来分解长任务：planner 把“写一篇报告”拆成“搜索资料→分析数据→生成初稿→润色”，executor 依次执行，每步结果反馈给 planner 调整下一步（来源：The Anatomy of an AI Agent: Memory, Tools, Planning, and Execution Explained）。

ADHD 大脑恰恰也是这样一个“高产但缺乏可靠执行调度层的生成核心”（来源：AI 与 ADHD 的时间管理）。孩子能产生无数创意，但缺少一个内置的 planner 来把“大任务”拆成“小步骤”，也缺少一个 executor 来启动每一步。家长的角色，就是充当那个“外部 harness”——帮孩子搭建规划循环与任务分解的脚手架。

## 解法：给“生成核心”套 harness——planner-executor 循环的亲子版

### 第一步：家长做 planner，帮孩子分解任务

就像 Motion 这类工具通过自动排程消除“下一步做什么”的决策负担（来源：Motion），家长可以先替孩子把任务可视化地拆开。例如，用 Tiimo 的视觉化时间线将“整理书包”变成“检查课表→放课本→放作业本→放文具”的序列（来源：Tiimo）。分解的关键是颗粒度要细到孩子能直接启动——就像 LLM agent 中的 planner 把“写文章”拆成“生成大纲→写第一段→写第二段”这样的原子步骤。

### 第二步：孩子做 executor，家长做进度提示

孩子执行每个子任务时，家长提供类似 Reclaim.ai 的“保护性时间块”——设定一个明确的起止时间，并在此间不干扰（来源：Reclaim.ai）。同时，家长可以像 Todoist 的智能提醒一样，在每一步结束时给予外部提示：“完成了检查课表，现在该放课本了”（来源：上下文工程）。这种“过渡提示”能防止孩子被环境带偏，相当于 agent 中的上下文管理。

### 第三步：反馈循环，动态调整

完成一轮后，家长和孩子一起复盘：哪个步骤卡住了？需要更细的分解吗？就像 Motion 的动态日程调整，根据实际完成情况重新规划下一轮（来源：Motion）。这种循环让任务分解能力逐渐内化。

## 核心观点：脚手架 vs 拐杖——关键在“逐步撤除”

本文的核心判断是：**ADHD 家长用 planner-executor 循环搭建的 harness，本质是“外部执行功能层”，其终极目标是让孩子自己长出内部调度层，而非永远依赖家长。** 这与 AI 工具作为“补偿”而非“替代”的理念一致（来源：AI 与 ADHD 的时间管理）。但 wiki 资料也警告：过度依赖可能削弱内在时间感知能力，工具可能从“脚手架”沦为“永久拐杖”（来源：矛盾与存疑）。因此，家长应像 agent 工程中的“逐步减少上下文注入”那样，随着孩子能力提升，逐渐减少分解粒度，从“家长全拆”过渡到“孩子自己拆、家长检查”，最终让孩子独立使用 planner-executor 模式。

## 诚实局限：证据与个体差异

必须承认，上述同构目前仍属理论类比，缺乏神经科学与计算机科学的交叉验证（来源：AI 与 ADHD 的时间管理）。现有工具如 Motion、Reclaim.ai 的有效性多基于用户报告，缺乏对照研究（来源：Motion、Reclaim.ai）。此外，ADHD 亚型（注意力缺陷 vs. 多动冲动）对工具的响应不同，现有工具多面向注意力缺陷型（来源：AI 与 ADHD 的时间管理）。家长需根据孩子的具体表现调整策略，避免“一刀切”。

## 今天就能试的行动

1. **用“番茄钟+视觉清单”模拟 planner-executor**：选一个简单任务（如做数学作业），家长用纸笔帮孩子拆成 3-5 个步骤，每个步骤设定 10 分钟番茄钟，完成后打勾。这相当于给 LLM 提供 prompt 模板和任务分解（来源：上下文工程）。
2. **引入 Tiimo 或类似视觉时间线**：把每天放学后的时间块可视化，让孩子看到时间流逝（来源：Tiimo）。如果孩子对数字工具敏感，可尝试 Structured 的视觉日程（来源：矛盾与存疑）。
3. **设置“启动仪式”**：在开始任务前，家长和孩子一起做一个简短的口头规划：“第一步做什么？做完后第二步做什么？”这相当于 agent 中的 planner 初始化，能降低启动门槛（来源：任务启动困难）。
4. **每周一次“撤除实验”**：选择孩子已熟练的任务，家长只提供分解框架，让孩子自己填步骤。逐步增加孩子自主规划的比例，避免依赖（来源：矛盾与存疑）。
