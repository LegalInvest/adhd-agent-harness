# Mem 之于 ADHD，就像 function calling 工具调用 之于 LLM——但有人用错了

> 从同构视角实测 Mem：它到底补上了哪一层执行功能？

你是否有过这样的体验：打开 Mem 想记一个灵感，结果半小时后还在整理标签、调整格式，最初的念头早已消散？或者，你给 agent 配了十个 function calling 工具，它却在简单问题上反复调用、陷入循环？

这两个场景背后是同一个问题：工具被当成了大脑的替代品，而非执行层的补充。

## 同构的困境：高产核心 vs 缺失的调度层

ADHD 大脑与 LLM 在结构上惊人相似：底层都是一个高产的生成核心——想法喷涌、联想活跃，但都缺乏可靠的内置执行调度层。ADHD 者难以启动任务、维持注意力、管理时间；LLM 则容易在长上下文里迷失目标、重复输出、忘记工具调用结果。

神经科学证据表明，ADHD 的多巴胺功能低下导致中脑皮质分支失调，引起执行功能缺陷（来源：多巴胺）。而 LLM 的架构缺陷同样明显：它没有内置的“任务管理器”来保持目标一致性。

两者的解法也同构：外部脚手架。对 ADHD 来说，是“克己复礼”式的行为边界；对 LLM 来说，是 agent scaffolding——用 function calling 将精确计算外包给工具，用结构化提示约束输出。

## Mem 补上了哪一层？

Mem 被定位为“AI 第二大脑”，核心卖点是外部记忆与知识管理。它试图补偿 ADHD 的工作记忆不稳定（来源：ADHD 的 AI 工具全景）。但问题在于：多数人把 Mem 当作一个更智能的笔记 app，不断往里塞信息，却忘了它本应是“执行调度系统”的一部分。

真正的价值在于 Mem 的“回忆”功能：当你需要某个信息时，它能快速检索，而不是让你在标签和文件夹中迷失。这相当于给 ADHD 大脑装了一个外部搜索索引——就像 LLM 的 function calling 把数学计算交给计算器，而不是让模型硬算。

但 Mem 的局限也很明显：它擅长“存储与检索”，却不擅长“提醒与启动”。它不能像 Goblin Tools 那样自动分解任务（来源：Goblin Tools），也不能像 Lex 那样用单一指令触发多步骤序列（来源：Lex）。如果你指望 Mem 帮你从“想”过渡到“做”，它大概率会让你失望。

## 孔子的 harness：日课与外部调度器

孔子是 ADHD 特质的典型：身高 1 米 9，精力旺盛，周游列国 14 年坐不住；冲动爱骂人，思维跳跃，《论语》全是场景化语录，无系统著作。他的 harness 系统是“克己复礼”——用外在的“礼”作为行为边界，每日反省（“吾日三省吾身”）。

这套系统与 LLM 的 agent scaffolding 高度同构：
- **日课 ↔ 定时 re-grounding**：孔子每天反省，相当于 agent 每轮对话重置上下文，防止目标漂移。
- **礼 ↔ 外部调度器**：礼是一套外部规则，约束冲动行为；LLM 的 function calling 同样是用外部工具约束输出，避免自由生成带来的混乱。

孔子 70 岁才做到“从心所欲不逾矩”，说明 harness 需要长期训练才能内化。同样，AI 工具也需要持续使用才能成为直觉的一部分，而非随时可能丢弃的拐杖。

## 脚手架 vs 拐杖：边界在哪？

这是最核心的争议。wiki 资料指出，AI 工具作为“外部执行功能层”补偿缺陷，但过度依赖可能导致能力退化（来源：矛盾与存疑）。

我的判断是：**当工具帮你完成你本可以学会但不愿做的任务时，它是拐杖；当工具帮你完成你永远无法稳定执行的任务时，它是脚手架。**

例如，Goblin Tools 的任务分解适合“我不擅长拆解任务”的人——这是脚手架，因为拆解能力可以通过练习提升。但 Mem 的记忆检索适合“我工作记忆容量有限”的人——这是脚手架，因为工作记忆容量受神经基础限制，很难改变。

关键在于：工具是否在帮你“做”而非“替代”。如果你用 Mem 记住所有灵感，却从不回顾、组织、输出，那你只是在囤积；如果你用它快速回忆上次的思路，然后继续写作，那它才是真正的执行层。

## 今天就能试的行动

1. **给 Mem 设一个“输出规则”**：每次记录灵感后，必须用一句话写下“下一步行动”。这相当于给 Mem 加了一个 function calling 的返回格式。

2. **用 Goblin Tools 分解一个你拖延的任务**：比如“整理书桌”，让 AI 拆成 5 步，然后只做第一步。这模拟了 LLM 的 step-by-step planning。

3. **设置一个“日课”提醒**：每天固定时间回顾 Mem 中的笔记，问自己“今天哪个想法值得推进？”——就像孔子的“三省吾身”。

4. **警惕“工具囤积症”**：如果你同时用 Mem、Saner.AI、Obsidian 等三个以上知识管理工具，很可能你陷入了“用收集代替行动”的陷阱。选一个，用透它。

## 局限与争议

必须诚实指出：ADHD 与 LLM 的同构目前仍是理论框架，缺乏神经科学的直接证据（来源：矛盾与存疑）。此外，超聚焦是否应被引导而非打断，尚无定论（来源：矛盾与存疑）。本文的观点基于现有工具和案例，但个体差异极大——对一个人是脚手架的，对另一个人可能是拐杖。

最终，工具只是放大器。Mem 不会让你变成执行大师，就像 function calling 不会让 LLM 变成真正的 agent。真正的改变，来自你愿意用脚手架搭建自己的调度系统，哪怕它一开始很笨拙。
