# ChatGPT 之于 ADHD，就像 调低采样温度 之于 LLM——但有人用错了

> 从同构视角实测 ChatGPT：它到底补上了哪一层执行功能？

你打开 ChatGPT，输入“帮我列一下今天要做的事”，它立刻给出一个漂亮清单。你感觉头脑清晰了。但三小时后，你又瘫在沙发上，手机屏幕上还挂着那个清单，一个都没动。

这不是你的错。这也不是 ChatGPT 的错。这是你把它当成了“调低采样温度”的按钮，而问题根本不在温度——在于你缺了一个 **调度层**。

## 同构：ADHD 大脑与 LLM 共享同一个核心困境

ADHD 大脑的默认模式网络（DMN）高度活跃，像一台永不停止的生成器，随时能冒出几十个创意、计划、担忧。但前额叶执行功能薄弱，就像 LLM 虽然能生成流畅文本，却没有内置的调度器来控制输出的顺序、时机和可靠性。

LLM 的采样温度控制着输出的随机性：温度越高，输出越散乱；温度越低，输出越确定。ADHD 个体的表现日内波动，注意力时好时坏，本质上就是“生成核心”缺乏稳定的温度调节（来源：采样温度与表现波动）。工程师面对 LLM 的非确定性，会用确定性工作流、状态机、计划-执行分离来约束输出（来源：AI Agent Systems: Architectures, Applications, and Evaluation）。

ADHD 患者需要的，不是另一个“调低温度”的工具，而是一套 **外部 harness**——就像给 LLM 套上 agent scaffolding，强制它按步骤执行、定期回检、用外部记忆弥补无状态缺陷。

## 人物案例：孔子与张居正的“外部调度层”

孔子身高近1米9，精力旺盛，周游列国14年坐不住；冲动爱骂人，见南子急得对天发誓，骂宰予“朽木不可雕”。但他对音乐超专注到“三月不知肉味”，对种地等俗事零耐心。他的 harness 是“克己复礼”——用外在的“礼”作为行为边界，每日反省，“吾日三省吾身”。这本质上是一个 **定时 re-grounding 机制**，类似 LLM agent 中每隔 N 步就重置上下文或执行一次状态检查。

张居正12岁中秀才，16岁中举人，少年天才。当首辅后推行考成法，严格考核官员，用制度管别人也管自己。他每天办公到深夜，58岁累死在任上。考成法就是他的 **外部调度器**——把庞大的改革目标分解为可量化的步骤，定期检查进度，类似 agent 中“计划-执行”分离模式：先由 planner 制定计划，再由 executor 按确定性流程执行。

两人的共同点：不是靠“意志力”硬扛，而是主动设计外部结构来约束自己。这正是 ADHD 与 LLM 同构的解法——**用 harness 代替内置调度**。

## 工具实测：ChatGPT 到底补上了哪一层？

ChatGPT 能做什么？它能把模糊任务分解成步骤（类似 Goblin Tools 的 Magic ToDo，来源：Harnessing Artificial Intelligence to Live Better with ADHD - CHADD），能写邮件、做研究、分析数据（来源：The 12 Best AI Tools for 2026 (That People Actually Use)）。但它 **不负责执行**。

ADHD 用户反馈，Goblin Tools 在任务分解方面“很棒”（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT），但分解之后呢？如果没有一个外部调度层来提醒“现在该做第一步”，清单就只是另一堆文字。

Routinery 用可视化倒计时和过渡提示，把步骤串成流程，直接针对任务启动困难和时间盲（来源：11 Best ADHD Productivity Apps for Fluctuating Energy - rivva blog）。Habitica 用游戏化积分提供即时多巴胺奖励，弥补内在动机不足（来源：ADHD科技行业深度研究）。这些工具不是“调低温度”，而是 **在生成核心外面套了一个确定性控制流**。

所以，ChatGPT 补上的其实是 **prompt engineering** 那一层——帮你把模糊意图转化为清晰指令。但 agent scaffolding（外部调度层）需要由 Routinery、Habitica、甚至一张纸质清单来提供。

## 脚手架 vs 拐杖：边界在哪里？

同构类比虽然吸引人，但需诚实面对局限。目前缺乏实证支持 ADHD 大脑与 LLM 在神经结构上真正同构，这仅是一种理论类比（来源：矛盾与存疑）。更关键的是，工具可能从临时脚手架变成永久拐杖。

研究显示，ADHD 组在超专注状态下的工作时长可达 14.5 小时，远超对照组的 8.2 小时（来源：Pace Ventures Research 2026）。这说明 ADHD 大脑并非“不能专注”，而是 **无法可靠地进入并维持专注**。如果工具只是帮你“调低温度”让你暂时稳定，却没有训练你识别波动、主动调整，那么一旦工具失效，你会比之前更无助。

真正的脚手架应该：1）不增加额外认知开销（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）；2）允许你逐渐内化其结构；3）在你不需要时可以撤掉。

## 今天就能试的 3 件事

1. **用 Goblin Tools 分解一个你拖延已久的任务**，然后把每个子任务写在一张便利贴上，按顺序贴在墙上。这相当于给 LLM 一个确定性工作流。
2. **设置 25 分钟倒计时**（Routinery 或任何计时器），配合一个“过渡提示”——比如铃响后站起来走两步。这模拟了 agent 中的“状态重置”。
3. **用 ChatGPT 生成每日计划时，加一句指令**：“请把每个步骤拆成 5 分钟内能完成的动作，并告诉我第一步具体是什么。” 这相当于调低采样温度 + 强制输出格式。

记住：ChatGPT 不是你的执行功能，它只是你的 prompt engineer。真正的调度层，需要你自己搭——或者像孔子和张居正那样，用一套外部规则把自己绑住。
