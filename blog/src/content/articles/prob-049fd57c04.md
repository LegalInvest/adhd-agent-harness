# 为什么执行功能假肢最怕的不是失效，而是你已经忘了自己怎么走路？

> 《问题302》人工精修选题，双域证据作答。

*本文为《问题302》人工精修选题，双域证据作答。*

## 问题：为什么假肢越顺手，人越慌张？

如果你是一个 ADHD 成年人，最深的折磨可能不是“做不到”，而是“我明明有想法，却搞不懂自己的大脑到底在什么时候、为什么卡住”。如果你是一名 Agentic Harness 工程师，最深的焦虑可能不是 LLM 生成质量差，而是它“什么都写得出来，却永远不自己收尾、不检查、不计划、不拒绝”。

这两类焦虑共享一个结构：本体（生成核心）很强，但执行调度层很薄。ADHD 大脑的前额叶执行功能网络激活不足，表现为工作记忆不稳定、任务启动困难、时间感知失真；LLM 则无状态、无内置调度层，会上下文膨胀、任务漂移、无限生成（来源：ADHD × AI 的科学与研究前沿）。所以我们都在做同一件事：给这个高产但缺调度的核心，套上一个外部 harness。

但这里有一个悖论。当 harness 太好用，你会开始把它当成“自己的一部分”。直到某天它失效、迁移或被关掉，你才发现：自己早已忘了没有它时怎么走路。

## 01 同构：缺的不是智商，是调度层

fMRI 研究显示，ADHD 患者的前额叶执行功能网络激活不足，导致难以把抽象意图转化为有序行动（来源：ADHD × AI 的科学与研究前沿）。这并不意味着“生成能力”差——很多 ADHD 个体有高度发散、直觉、大画面思维和创造性跳跃。他们缺的是把工作记忆里的目标，稳定地转译为“现在做这一步、然后做那一步”的机制。

LLM 的处境几乎相同。它是个强大的语言生成核心，但本质上无跨会话状态，没有稳定的内部调度器，不会自己管理优先级、检查进度、终止循环。因此工程师必须给它套上 ReAct、Chain-of-Thought、Tool-use 等外部脚手架，本质上就是在为生成核心补一个「执行功能层」（来源：ADHD × AI 的科学与研究前沿）。

这种同构还体现在几个具体层面：

- **工作记忆与外部记忆**：ADHD 的“眼不见，心不烦”与 LLM 的无状态性类似，都需要外部记忆系统（笔记、知识库、RAG）维持连续性（来源：ADHD × AI 的科学与研究前沿）。
- **时间盲**：ADHD 患者常“真的感觉不到 45 分钟已经过去”；LLM 没有内在时间感知，必须靠外部时序控制（来源：时间盲）。
- **上下文工程**：ADHD 容易被环境带偏，LLM 会因上下文膨胀退化，两者都需要主动设计“当下注意什么”（来源：ADHD × AI 的科学与研究前沿）。
- **身体在场**：他人在场能提升 ADHD 的启动与专注，AI 也能通过虚拟“身体加倍”降低寻找伙伴的成本（来源：身体在场效应）。

所以，ADHD 侧的 harness 与 LLM 侧的 harness 不是简单类比，而是同一类架构补丁。

## 02 从工具到结构：两边 harness 如何互相映射

现在来看真实的工具。ADHD 侧的辅助工具，几乎都能在 LLM 工程侧找到对应物。

**Goblin Tools** 的 Magic ToDo 可以把模糊任务（如“清理厨房”）拆成具体、可执行的小步骤，并调节“辣度”控制粒度（来源：Goblin Tools）。这对应 LLM 侧的 **Chain-of-Thought / ReAct**：把一个大目标拆成可验证的中间步骤，降低推理的启动阈值。

**Motion** 自动根据任务、会议、截止日期动态排程，并在任务可能延期前几周预警（来源：Motion）。它对应 LLM 侧的 **planning agent / scheduler**：不是让模型自己决定“现在该干嘛”，而是把调度层外置，让模型始终只面对当前子任务。

**Saner.AI** 强调“本地记忆”和“知识回忆”，减少 ADHD 用户的搜索循环和标签切换（来源：Saner.AI）。这对应 LLM 侧的 **RAG 与长期记忆库**：把上下文外置，避免模型在每次生成时重新“想起”关键信息。

**Focusmate / Tiimo** 则用虚拟身体加倍和视觉时间表来对抗时间盲（来源：身体在场效应；时间盲）。对应 LLM 侧的 **human-in-the-loop / critic**：让外部存在提供隐性问责和检查点，防止系统独自滑向无限循环。

这套映射的结论是：好的 ADHD 工具，本质上是在为人类大脑做 Agentic Harness；好的 Agentic Harness，本质上是在为 LLM 做 ADHD 式的外部执行功能补偿。

## 03 一个历史 harness：毛泽东的外部调度系统

在众多名人 harness 中，毛泽东的系统最能说明“外部调度层”如何驯服一个高产但冲动的生成核心。

他的 ADHD 特质非常典型：童年是问题儿童，思维极度跳跃，讲话充满场景化比喻，一辈子在读书、调查、移动；决策上冲动敢赌，四渡赤水、抗美援朝都是险棋（来源：名人 harness / 毛泽东）。但他并没有依靠“意志力”来压制这些特质，而是为自己建了一套外部 harness：

- **批评与自我批评、民主生活会**：让别人定期提意见，相当于给系统加了外部“critic / red team”。
- **调查研究，没有调查就没有发言权**：强制 grounding，防止思维在空中打转。
- **民主集中制、党指挥枪**：用集体决策和组织系统约束个人冲动，相当于把调度层外置给机构。

映射到 LLM 工程：这就是 **外部监督器 + 检索增强 grounding + 集体决策/多智能体制衡**。对个人 ADHD 而言：这就是 **外部问责伙伴 + 事实核查流程 + 规则化启动机制**。两者都承认生成核心不可替代，但绝不把调度权交给核心本身。

## 04 脚手架 vs 拐杖：最怕的不是失效，而是“它变成我”

同构框架里有一个被低估的陷阱：harness 一旦成功，就会模糊“我”和“工具”的边界。

ADHD 用户用了半年 Motion 或 Goblin Tools，可能会突然忘记自己原来是如何启动任务的。工程师的 Agent 一旦稳定运行，团队也可能失去手动 debug 和手动拆解问题的能力。更隐蔽的是：当外部工具替你做了“下一步该做什么”的决策，你的前额叶训练机会被替换，而不是被支持。

wiki 资料中的“矛盾与存疑”部分也指出了这一点：拐杖与脚手架的界限尚不明确，长期使用 AI 脚手架可能导致执行功能进一步退化；身体在场效应是否真能被 AI 替代，也存在争议；类似 Motion 这样的工具缺乏独立临床验证（来源：矛盾与存疑）。

所以我的判断是：**harness 必须保持“可见性”**。它应该像脚手架，而不是像器官。一个器官坏了你会恐慌，但那是因为你失去了一部分自己；而脚手架倒塌时，你应该仍然知道没有它怎么爬楼。

区分两者的关键不是“用了多久”，而是：**没有它时，你是否还能描述自己的执行步骤、识别自己的卡点和重启流程？** 如果答案是“不知道”，它就已经从脚手架变成拐杖。

## 05 局限：同构仍是假说，工具不是药

必须诚实地说，同构框架仍然是一种理论类比，而非被完全验证的神经科学事实（来源：矛盾与存疑）。不同 ADHD 亚型、不同任务、不同文化环境，是否需要不同 harness，目前证据还很有限。工具如 Motion、Brain.fm 等缺乏独立临床验证；AI 身体加倍是否真能替代真实人际互动，也尚无定论（来源：矛盾与存疑）。

此外，没有任何工具可以替代对 ADHD 的医学、心理和社会支持。AI 辅助应该被视为“降低认知负荷的环境设计”，而不是“治愈执行功能障碍”的疗法。对工程师来说，同样不要把 Agentic Harness 当成让 LLM 真正“拥有”执行功能——它仍然只是一个被外置的规则系统。

## 06 今天就能试的四条

1. **给工具加一个“显式边界”**：无论你用 Goblin Tools、Motion 还是 Saner.AI，每周留一个没有该工具的小任务，用手写清单完成，并记录自己是如何启动的。这样你保留“裸足行走”的地图。

2. **建立“元日志”**：每天花 3 分钟写下：今天我靠工具做了什么决策？我自己的情绪、注意力、卡点在哪个环节出现？工具是补偿了执行功能，还是替我做了一个我本可以做的选择？

3. **如果你在做 Agentic Harness**：不要只给 LLM 加任务分解（CoT）和工具调用，还要加一个“停止与自检”循环——让它必须周期性地回答：我现在偏离目标了吗？我可以停止了吗？这相当于给 LLM 装一个民主生活会。
