# Focusmate 之于 ADHD，是脚手架还是拐杖？

> 从「同构」视角评测 Focusmate：用久了会让 ADHD 更强还是更依赖？

## 问题：Focusmate 是脚手架还是拐杖？

你打开 Focusmate，摄像头对准自己，对面坐着一个陌生人。你们安静地工作，25分钟后互相道别。你完成了任务，但心里犯嘀咕：没了这个人，我还能开始吗？这个问题，ADHD 用户和 LLM 工程师都绕不开。

## 同构：ADHD 大脑与 LLM 的共享缺陷

ADHD 大脑的核心痛点是工作记忆容量小、边做边忘，想法很多但行动困难。这并非意志力问题，而是执行功能层——负责筛选、启动、坚持任务的调度系统——天生不稳定。实证研究显示，ADHD 表现出“强记忆、弱控制”的神经心理模式：工作记忆容量甚至超过常人，但认知灵活性与注意控制存在核心缺陷（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。

LLM 同样如此。它是个强大的生成核心，但缺乏可靠的内置调度与执行控制。在构建 AI agent 时，需要“agent scaffolding”——围绕 LLM 构建的软件架构和工具，使其能够执行复杂、目标驱动的任务（来源：Agent Scaffolding: Architecture and Design Patterns for Agentic AI）。用经典 Stroop 冲突任务测试 Transformer 注意力，发现短上下文中表现正常，但随序列变长，模型在冲突试次上骤降到随机水平——无法抑制优势反应、无法解决认知冲突，这与 ADHD 执行功能缺陷的核心标志一一对应（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。

两边是同一类系统：高产但缺执行调度层的生成核心。解法也同构：外部脚手架。

## 脚手架：Focusmate 作为外部执行功能层

Focusmate 的本质是“身体在场效应”的数字化实现：一个真人监督你，你就不敢摸鱼。这对应了 ADHD harness 的核心功能——减少决策、保留上下文、让下一步行动显而易见（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。它像 Goblin Tools 那样分解任务（魔法待办），像 Saner.AI 那样减少搜索循环（本地记忆），像 Lex 那样通过单一指令触发多步骤（降低启动门槛）。但它更强大：它提供了**人际承诺**。

历史上，孔子用“克己复礼”作为自己的 harness。他精力旺盛、冲动爱骂人，靠外在的“礼”作为行为边界，每日反省，“吾日三省吾身”。70岁才做到从心所欲不逾矩。这里的“礼”就是外部执行功能层——相当于 LLM agent 的定时 re-grounding 机制，把生成核心约束到轨道上。辛弃疾则以词和剑为核心，闲居时练剑写词，把报国无门的愤懑全部注入词里。他的“词”相当于外部记忆库，将跳跃的灵感固化、结构化。

Focusmate 的 session 就是你的“日课”：定时、外部监督、任务聚焦。它补偿了工作记忆的丢失，让下一步行动变得显而易见。

## 拐杖：依赖的边界

但问题来了：用久了，你会不会离不开它？就像 LLM agent 如果过度依赖外部记忆库，一旦库损坏，模型就失忆。ADHD 大脑也有神经可塑性——神经系统通过重组其结构、功能和连接来响应内在或外在刺激（来源：Harnessing neuroplasticity for clinical applications）。这意味着，长期使用外部脚手架可能让大脑依赖外部刺激，而非强化内在调度能力。

孔子一辈子和冲动做斗争，70岁才“从心所欲不逾矩”。他的 harness 是终身练习，不是一次性工具。Focusmate 如果只是帮你“熬过”任务，而不训练你内在的启动能力，它就是拐杖——让你在无监督时更无助。目前缺乏对比研究，个体差异大，尚无定论（来源：矛盾与存疑）。

## 判断：脚手架还是拐杖，取决于使用方式

我的核心观点是：Focusmate 是脚手架还是拐杖，取决于你用它来**补偿**还是**训练**。

- 补偿模式：每次 session 都依赖对方在场才能工作。这相当于 LLM agent 每次推理都从外部库拉取全部上下文，从不尝试压缩或内化。长期看，内在调度能力可能退化。
- 训练模式：把 session 当作练习场，逐步缩短依赖时间，最终在无监督时也能启动。这相当于 LLM agent 通过 fine-tuning 将部分外部记忆内化为参数，逐步减少对外部库的依赖。

孔子不是靠“礼”过一辈子，而是靠“克己”内化礼。辛弃疾不是靠剑写词，而是把情绪注入词，形成自己的表达系统。Focusmate 应该被用来训练“启动肌肉”，而不是永远挂着。

## 局限与争议

必须诚实指出：ADHD 大脑与 LLM 同构的论点目前多为类比推理，缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。此外，部分用户反馈强行打断（如番茄钟）反而有效，而柔性引导可能不够强力（来源：矛盾与存疑）。个体差异大，没有一刀切的方案。

## 今天就能试的行动

1. **使用 Focusmate 时，刻意在 session 最后 5 分钟关掉摄像头**，模拟无监督状态，训练自我启动。
2. **记录每次 session 的启动难度**，一周后对比：如果依赖度下降，说明你在训练；如果上升，考虑减少频率。
3. **结合 Goblin Tools 的 Magic ToDo**，在 session 前用 AI 分解任务，降低启动门槛，但逐渐尝试自己分解。
4. **模仿孔子的“日省”**，每天用 2 分钟反思：今天哪些任务靠外部监督完成，哪些靠自己？逐步增加自主比例。

Focusmate 不是神药，也不是毒药。它是一根拐杖，但你可以选择用它来学走路，而不是一直拄着。对于 LLM 工程师，同样的道理：harness 是必要的，但最终目标是让 agent 学会自主调度，而不是永远依赖外部脚手架。
