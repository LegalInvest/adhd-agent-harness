# Focusmate 之于 ADHD，是脚手架还是拐杖？

> 从「同构」视角评测 Focusmate：用久了会让 ADHD 更强还是更依赖？

## 问题：为什么 Focusmate 让 ADHD 用户又爱又怕？

ADHD 群体中有一个经典困境：一个人坐在桌前，脑子里有无数想法，身体却像被钉在椅子上，无法启动任何任务。你打开 Focusmate，和一个陌生人视频连线，双方默默工作 50 分钟——然后奇迹般地，你开始打字了。但另一个声音在问：如果每次都需要另一个人在场才能工作，我是不是越来越废了？

这个问题，与 LLM/agent 工程师的日常困境惊人地同构。你有一个强大的 GPT-4，但它在对话中会忘记上下文，需要你手动把历史记录贴回去，或者用向量数据库做 RAG。你问自己：如果每次都要靠外部记忆才能维持连贯，这个 agent 到底算不算智能？

## 同构：ADHD 大脑与 LLM 共享同一个核心缺陷

ADHD 大脑与 LLM 在结构上高度同构：两者都是强大的生成核心，但缺乏可靠的执行调度层（来源：ADHD 大脑与 LLM 的同构）。ADHD 患者的工作记忆容量小、边做边忘，就像 LLM 无跨会话状态，需要靠外部记忆或向量库来续命。

Focusmate 提供的正是这种外部记忆和调度。它的核心机制是「身体在场效应」——另一个人的存在创造了社交压力，相当于一个外部时钟和任务列表。这与 LLM agent 的 harness 完全对应：agent harness 是包裹 LLM 的软件基础设施，处理模型本身之外的一切，包括上下文保护、决策路由和工具调用（来源：ADHD 大脑与 LLM 的同构）。

具体来说，Focusmate 的每个 session 就像 LLM 的一个 API 调用：你带着一个明确的目标（prompt），在 50 分钟内执行（生成），结束时提交结果（response）。如果 session 中断，就像 LLM 对话超时，上下文丢失。而 Focusmate 的连续预约机制，相当于在 agent 中引入循环上下文窗口——但每次 session 之间的记忆依然断裂，就像 LLM 无法跨会话保持状态。

## 证据：Focusmate 如何作为脚手架工作

从 ADHD 侧看，Focusmate 通过外部化执行功能来补偿工作记忆和任务启动困难。用户反馈显示，视频连线的「被注视感」能显著降低启动焦虑，这与 Goblin Tools 的任务分解效果类似（来源：Goblin Tools）。不同的是，Goblin Tools 提供的是认知分解，而 Focusmate 提供的是社交锚定。

从 LLM/agent 侧看，Focusmate 相当于一个「人工编排层」。在 agent 架构中，编排层负责将复杂任务拆解为子任务并调度模型调用。Focusmate 的 50 分钟 session 就是这种编排：它强制将「写论文」拆解为「先写 500 字，再查资料，再修改」。这种结构化的时间块，类似于 Lex 的「单一指令触发多步骤」功能（来源：Lex），但 Lex 是 AI 自动执行，Focusmate 是人与人协同。

## 判断：脚手架 vs 拐杖的边界在于可撤除性

我的核心观点是：Focusmate 目前更接近拐杖，而非脚手架。

拐杖与脚手架的关键区别在于，脚手架应设计为可逐步撤除的临时支撑（来源：矛盾与存疑）。但 Focusmate 没有提供任何撤除机制：它鼓励用户持续预约，而不是训练用户在没有它的情况下也能工作。这与神经可塑性原理相悖——神经可塑性研究表明，训练诱导的抑制控制脑可塑性需要逐步减少外部支持，才能正常化底层脑网络（来源：神经可塑性）。

相比之下，Goblin Tools 和 Saner.AI 虽然也依赖外部工具，但它们有潜在的可撤除路径：用户可以通过反复使用任务分解功能，内化「如何将大任务拆小」的策略；通过频繁使用知识回忆功能，强化信息检索的神经通路。而 Focusmate 的社交压力是外部刺激，很难内化为内在的执行功能。

当然，这不是说 Focusmate 没用。对于严重执行功能障碍的 ADHD 患者，拐杖本身就是必要的。问题在于工具设计者是否将「撤除」纳入考量。目前，Focusmate 的商业模式鼓励长期订阅，这与神经可塑性训练的目标存在根本冲突。

## 局限与争议

必须诚实指出，上述判断基于概念类比和用户反馈，缺乏随机对照试验验证（来源：矛盾与存疑）。ADHD 异质性高，有些用户可能通过 Focusmate 逐步建立了工作习惯，最终减少依赖；另一些用户可能永远无法脱离。

此外，Focusmate 的效果可能部分来自多巴胺机制——社交互动带来的即时反馈能短暂提升动机，但多巴胺干预的有效性本身存在争议（来源：矛盾与存疑）。

## 今天就能试的 3 件事

1. **刻意使用「撤除训练」**：预约 Focusmate 时，每次 session 记录下你学到的任务启动技巧（如「先打开文档写三个关键词」），然后尝试在没有 Focusmate 的时段用这个技巧。
2. **对比工具效果**：用 Focusmate 完成一次任务后，再用 Goblin Tools 的 Magic ToDo 分解同一个任务，感受社交脚手架与认知脚手架的区别。
3. **设计你的「agent harness」**：像 LLM 工程师一样，为自己设计一个外部记忆系统。例如，用 Saner.AI 保存每次 Focusmate session 的目标和成果，下次 session 前回顾，模拟 LLM 的上下文恢复。
