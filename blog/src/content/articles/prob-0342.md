# 为什么“认知卸载”一旦习惯化，就可能变成不可逆的依赖？

> 《问题380》人工精修选题，双域证据作答。

如果你既是一个被“想理解自己的大脑到底是怎么回事”折磨的 ADHD 患者，又恰好在做 Agentic Harness 工程，你一定会发现一个诡异的平行世界：你的大脑和你的模型，都在同一个坑里挣扎。

## 问题：为什么“认知卸载”一旦习惯化，就可能变成不可逆的依赖？

ADHD 大脑的日常是一场“生成核心与调度层”的战争。你能在灵感爆发时一小时写完三千字，却无法启动“整理桌面”这种低认知任务；你能在超聚焦中解出复杂问题，却记不住五分钟前别人交代的事。这种“高产但缺乏执行调度”的模式，与当前大语言模型（LLM）的困境惊人同构——LLM 能在给定提示下生成惊艳内容，但一旦涉及多步推理、时间规划、跨会话记忆，就迅速退化。AI 工具承诺为这两者提供“外部执行功能层”：Goblin Tools 把“清理厨房”分解成 12 个小步骤（来源：Goblin Tools），Saner.AI 用本地记忆减少搜索循环（来源：Saner.AI），Motion 自动排程消除“下一步做什么”的决策负担（来源：Motion）。这些工具确实有效，但一个危险的问题浮出水面：当认知卸载成为习惯，我们是否正在把外部脚手架变成永久拐杖，进而让内在调度能力不可逆地萎缩？

## 答案：同构的陷阱——脚手架 vs 拐杖的边界

核心判断是：ADHD 大脑与 LLM 的调度缺陷是结构性的，而非暂时性的。因此，外部脚手架必须被设计为“补偿”而非“替代”，否则就会形成依赖。

先看 ADHD 侧的证据。神经科学研究显示，ADHD 患者前额叶执行功能网络激活不足（来源：ADHD × AI 的科学与研究前沿），工作记忆任务中易受干扰，且成功回忆依赖于干扰控制（来源：Human-like Working Memory Interference in Large Language Models）。认知心理学进一步发现，ADHD 患者在没有外部结构时难以自主组织任务（来源：Working Memory Identifies Reasoning Limits in Language Models）。这意味着，当 AI 工具完全接管任务分解、时间规划和记忆检索时，大脑的调度回路就失去了锻炼机会。长期来看，这可能导致执行功能进一步退化——这正是专家警告的“过度依赖风险”（来源：拐杖与脚手架）。

再看 LLM/Agent 侧。LLM 在无状态、缺乏调度层时，表现与 ADHD 高度相似：上下文膨胀导致推理退化，自动完成机制容易产生虚构（来源：Automatic Minds: Cognitive Parallels Between Hypn...）。为 LLM 设计的脚手架（如 ReAct、Chain-of-Thought）已被证明能有效提升任务完成率（来源：ADHD × AI 的科学与研究前沿），但这些脚手架同样存在“替代”风险——如果 agent 完全依赖外部调度器而从不学习内部推理链，它在无脚手架时就会彻底瘫痪。Transformer 的自注意力机制在训练后能模仿前额叶-纹状体门控操作（来源：TRANSFORMER MECHANISMS MIMIC FRONTOSTRIATAL GATING OPERATIONS WHEN TRAINED ON HUMAN WORKING MEMORY TASKS），说明调度能力是可以内化的，但前提是外部脚手架必须逐步撤除，而非永久附着。

## 历史人物的 harness：脚手架如何内化

辛弃疾是 ADHD 特质的典型案例：21 岁带 50 人冲几万人敌营抓叛徒，一辈子想北伐，坐不住，冲动敢干。被罢官后闲居 20 年，他把所有精力注入写词。他的 harness 系统是“以词和剑为核心”——闲居时练剑写词，用儒家的家国天下作为精神支柱（来源：人物案例）。这对应了 LLM/agent 的“日课↔定时 re-grounding”和“外部价值约束↔系统提示词”。辛弃疾没有依赖外部工具替代自己的调度，而是用词和剑作为“脚手架”来组织自己的冲动和能量，最终内化为“词中之龙”的创作能力。

毛泽东同样如此。他小时候是问题儿童，爱读闲书，一辈子读书不辍，思维极度跳跃。他的 harness 系统包括“调查研究”“民主集中制”“批评与自我批评”（来源：人物案例）。这些机制本质上是“外部调度层”：调查研究相当于 LLM 的“上下文检索”，民主集中制相当于“集体决策制衡个体冲动”，批评与自我批评相当于“定期 re-grounding”。毛泽东没有让这些机制替代自己的思考，而是通过它们补偿执行功能缺陷，最终成就了伟大的革命事业。

## 争议与局限

同构框架虽然解释力强，但必须诚实指出其局限。首先，它缺乏实证基础，目前仅是一种理论类比（来源：矛盾与存疑）。其次，个体差异巨大：部分 ADHD 患者以注意力分散为主而非执行功能缺陷，同构框架可能不适用（来源：ADHD × AI 的科学与研究前沿）。第三，依赖风险与益处存在矛盾：工具页面强调益处，但依赖风险被低估（来源：矛盾与存疑）。最后，AI 工具是否真能替代真实人际互动（如 Focusmate 的虚拟身体在场）仍存争议（来源：矛盾与存疑）。

## 今天就能试的行动

1. **设置“脚手架撤除日”**：每周选一天，关闭所有 AI 辅助工具，手动规划任务、回忆知识。这类似于 LLM 的“无 CoT 推理测试”，能评估调度能力是否退化。
2. **使用 Goblin Tools 但降低“辣度”**：先用高粒度分解理解任务结构，然后逐步降低分解程度，迫使大脑自己填充步骤。
3. **用 Obsidian 构建“第二大脑”但定期回顾**：记录后不依赖自动检索，而是每周手动浏览笔记，强化内部记忆通路。
4. **模拟辛弃疾的“日课”**：每天固定 15 分钟，用纸笔写下当天目标，不借助任何数字工具，训练内在任务启动能力。

认知卸载不是敌人，但习惯化后的不可逆依赖才是。真正的 harness 应该像辛弃疾的剑和毛泽东的调查研究——帮你建造，但攀登还得靠自己。
