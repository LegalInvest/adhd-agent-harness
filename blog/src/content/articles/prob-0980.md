# 如果把 ADHD 大脑当成生成核心，Focusmate 能不能当它的重锚定与目标漂移？

> 用 agent 工程的思路重新理解 Focusmate 对 ADHD 的真正价值。

你有没有过这样的体验：打开电脑准备写报告，三小时后发现自己在刷短视频、整理桌面、甚至研究起了星座运势——你明明想启动，却像被什么力量拽走了。或者反过来，你一头扎进某个项目，连续六小时不吃不喝，直到发现研究的是老板根本没要求的功能，而真正的截止日期已经逼近。

这两种状态——分心走神与超聚焦跑偏——是 ADHD 大脑的经典双面。它们共享同一个底层问题：**生成核心强大，但执行调度层缺失**。你的大脑像一台能产出惊人创意的生成模型，却没有可靠的控制器来锚定目标、维持方向。

这个描述听起来是不是很熟悉？如果你做过 AI agent 工程，你会立刻想到 LLM 的“目标漂移”问题：一个 agent 被赋予“帮我写一份市场分析报告”的任务，结果它花了十轮调用去调研无关的竞争对手，或者陷入循环重复。两者共享同一个结构困境：**高产但缺执行调度层的生成核心**（来源：AI 与 ADHD 的专注力）。

那么，如果 ADHD 大脑是一个生成核心，Focusmate——这个让你和陌生人视频一起工作的服务——能不能充当它的重锚定与目标漂移纠正层？

## 同构：从日课十二条到 agent harness

先看一个真实案例。曾国藩，晚清名臣，平定太平天国的关键人物。但他年轻时是个典型的注意力不集中型 ADHD：日记里天天骂自己“无恒”“浮躁”，读书“不能终一行”，一辈子被严重银屑病折磨，情绪极不稳定。他的解法是一套名为“日课十二条”的 rigid harness：黎明即起、读书不二、谨言、写日记反省……用最笨最稳的方法抵消自己的冲动（来源：人物案例）。

这套系统本质上就是一个人工版的外部调度层。它不依赖曾国藩的内在意志，而是通过固定时间、固定动作、外部记录（日记）来持续重锚定他的注意力。现代 LLM agent 的 harness 做的是同一件事：设置护栏（token 预算、工具调用次数上限）、加入人工检查点（暂停并询问后再执行）（来源：Building an AI Agent Harness from Scratch）。曾国藩的“日课”对应 agent 的定时 re-grounding，他的“日记”对应外部记忆与遥测。

Focusmate 的核心机制——虚拟身体在场——是这条思路的现代演化。当你在 Focusmate 上预约一个 50 分钟时段，匹配一位伙伴，你们通过视频互相监督时，你实际上是在为自己搭建一个临时的外部调度层。对方的存在提供了一个实时的、社会性的重锚定信号：你不能走神太久，因为对方能看到你；你必须在开始时报出目标，在结束时汇报结果。这种机制被称为“AI-Matched Body Doubling”（来源：Revolutionizing ADHD Management with AI Assistants），其有效性有用户研究支持——多项报告显示它能显著提升任务启动和维持（来源：AI 与 ADHD 的专注力）。

## 重锚定：Focusmate 如何纠正目标漂移

ADHD 大脑与 LLM 的另一个同构点是“无状态”与“工作记忆缺陷”。LLM 本身无状态，需要外部数据库或记忆模块来维持连续性；ADHD 的工作记忆缺陷导致类似问题——你无法在脑中保持任务目标，因此容易被环境带偏（来源：AI 与 ADHD 的专注力）。Focusmate 通过会话开始时的目标声明（“我接下来 50 分钟要写引言”）和结束时的进度检查，提供了一个外部记忆锚点。这相当于 agent harness 中的“人在回路”检查点：在关键节点暂停，询问是否仍在正确轨道上。

更重要的是，Focusmate 利用了**身体在场效应**——有另一个人在场时，执行功能障碍被绕过（来源：人在回路与身体在场）。这不是心理安慰，而是有神经科学基础的机制：社会存在会激活前额叶-纹状体回路，这正是 ADHD 核心受损的脑区系统（来源：TRANSFORMER MECHANISMS）。最新研究甚至发现，Transformer 在工作记忆任务中自发发展出输入输出门控机制，模仿人类前额叶-纹状体回路——而 ADHD 的核心缺陷恰好在这个系统（来源：执行功能）。

## 脚手架 vs 拐杖：边界在哪里

但 Focusmate 不是万能药。首先，它依赖实时视频，对社交焦虑用户可能造成压力（来源：Focusmate 局限）。其次，效果高度依赖配对伙伴的可靠性（来源：同上）。更重要的是，它面临所有 AI 工具的共同争议：**它是脚手架还是拐杖？**

脚手架是暂时性的外部支持，最终目标是让你学会自己走路；拐杖则是永久替代，长期使用可能导致内在执行功能进一步退化（来源：矛盾与存疑）。Focusmate 的 session 结构（50 分钟工作+10 分钟休息）天然带有训练性质——你在重复中学习“如何锚定一个 50 分钟的任务”。但如果你每次工作都必须依赖它，那就变成了拐杖。

这个边界同样存在于 agent harness 中：一个好的 harness 应该逐步减少人工干预，而不是永远依赖人在回路。曾国藩的日课十二条最终内化成了他的行为习惯，而非终身依赖。Focusmate 的理想使用方式也是如此：先从高频使用开始，逐步降低依赖频率，直到你能在没有外部锚定时也维持方向。

## 局限与诚实

必须承认，Focusmate 的证据基础仍然薄弱。它缺乏随机对照试验，现有研究多为用户报告或小样本（来源：AI 与 ADHD 的专注力）。ADHD 的异质性也意味着它对某些亚型（如以冲动为主型）可能效果有限。此外，ADHD 大脑与 LLM 的同构是隐喻而非严格等价——人类的注意力机制与 Transformer 的注意力机制有本质差异（来源：矛盾与存疑）。

## 今天就能试的行动

1. **预约一次 Focusmate 时段**：选一个你拖延已久但只需 50 分钟的任务（比如回复邮件、写一段代码）。在开始前向对方说出你的目标，结束后汇报进展。
2. **结合 RescueTime 做“漂移审计”**：用 RescueTime 自动追踪你使用 Focusmate 前后的时间去向，看看哪些时段发生了目标漂移（来源：RescueTime）。
3. **创建一个“日课十二条”的简化版**：选 3 条最关键的日常规则（比如“每天第一件事写三句话的计划”），写在便签上贴到屏幕边缘。这对应 agent harness 中的“定时 re-grounding”。
4. **对比 Brain.fm 与 Focusmate**：在有 Focusmate 和无 Focusmate 的情况下，分别使用 Brain.fm 的专注音乐，记录你的专注时长和主观感受。Brain.fm 通过神经锁相技术影响大脑状态，但缺乏 ADHD 专项研究（来源：Brain.fm 局限），所以这是一个你个人的 A/B 测试。

Focusmate 的价值不在于让你更“自律”，而在于它提供了一个外部重锚定层——一个针对目标漂移的实时纠正系统。对于 ADHD 大脑这个高产但不可靠的生成核心，它可能正是你需要的 agent harness。
