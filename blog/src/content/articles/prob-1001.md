# 用 ChatGPT 给 ADHD 补上重锚定与目标漂移，是脚手架还是拐杖？

> ChatGPT 接进 ADHD 工作流：哪些交给它、哪些必须自己留着。

## 同一个问题，两种视角

ADHD 人群常陷入两种困境：要么分心走神，任务启动后迅速被环境干扰淹没；要么超聚焦跑偏，深陷细节而忘记原始目标。这两种状态本质都是**目标漂移**——大脑的高产生成能力缺乏可靠的执行调度层，导致输出失控。

同样的问题，在 AI agent 工程中有一个专业名称：**agentic harness 中的重锚定**。当你让一个 LLM 执行长程任务（比如写一份 50 页报告），它很容易在中间步骤“忘记”原始指令，产生偏离目标的内容。工程师为此设计了编排层（harness），通过外部上下文保存、阶段性提示注入等方式，反复将 agent 拉回正轨。

ADHD 大脑与 LLM 在结构上同构：两者都是高产的生成核心，但缺乏内置的执行调度层（来源：ADHD 大脑与 LLM 的同构）。因此，用 ChatGPT 来“重锚定”注意力，本质上是在给大脑套一个 AI harness——问题在于，这个 harness 是临时脚手架，还是永久拐杖？

## 脚手架的证据：同构解法在两边都有效

从 ADHD 侧看，现有工具已在三个关键维度上弥补执行功能缺陷：

- **上下文保持**：ADHD 的工作记忆易丢失上下文，就像 LLM 的无状态性。RescueTime 通过自动记录时间使用，充当外部记忆，提升对时间盲的觉察（来源：RescueTime）。用户无需手动追踪，AI 自动分类活动模式，减轻工作记忆负担。
- **任务启动**：ADHD 的启动困难源于多巴胺调节异常。Focusmate 利用 AI 匹配虚拟同伴，提供实时问责，弥补任务启动的调度缺失（来源：Focusmate）。其核心机制是“AI-Matched Body Doubling”，通过智能调度提高配对效率。
- **注意力维持**：Brain.fm 通过神经锁相技术生成音乐，旨在影响大脑处理信息的方式，帮助用户进入专注状态（来源：Brain.fm）。虽然缺乏针对 ADHD 的临床研究，但其原理与外部环境调度注意力状态一致。

从 LLM/agent 侧看，同样的解法被工程化验证。在构建 AI agent 时，需要“agent harness”——即“包裹 LLM 或 AI agent 的软件基础设施，处理模型本身之外的一切”（来源：ADHD 大脑与 LLM 的同构）。这种架构通过上下文工程、决策路由、外部记忆管理，确保生成核心不偏离目标。例如，工程师会在每个工作流中绑定不同 LLM，并依赖模型连接器管理工具调用（来源同上）。

两边用的都是同一套逻辑：**用外部结构补偿内部调度缺失**。ADHD 用户通过 AI 工具减少决策、保留上下文、让下一步行动显而易见（来源：ADHD 大脑与 LLM 的同构），就像 agent harness 通过编排层确保 LLM 输出可靠。

## 拐杖的隐忧：脚手架能否撤除？

然而，关键争议在于：**这种外部补偿是否阻碍内在能力发展？**

神经可塑性研究表明，训练可以诱导抑制控制脑可塑性，帮助正常化 ADHD 的执行功能（来源：神经可塑性）。如果 AI 工具长期替代执行功能，用户可能失去锻炼这些能力的机会。当前多数工具（如 Goblin Tools、Saner.AI）设计为长期使用，未提及撤除机制（来源：矛盾与存疑）。这引发了一个严肃问题：**脚手架是否应该设计为可逐步撤除的临时支撑？**

另一个争议是证据强度。Brain.fm 的神经锁相技术宣称有效，但缺乏针对 ADHD 的独立临床研究（来源：Brain.fm）。Focusmate 的效果归因于身体在场而非 AI 本身（来源：矛盾与存疑）。多巴胺干预的有效性也存在分歧（来源：矛盾与存疑）。这些意味着，我们目前看到的“效果”可能部分来自安慰剂或用户主观报告，而非客观改善。

## 核心判断：脚手架，但需有意识撤除

我的观点是：**ChatGPT 对 ADHD 是脚手架，不是拐杖——前提是你把它当作训练工具，而非永久替代。**

同构视角的启示是：harness 是临时编排层，最终目标是让 agent 学会自主调度。同样，AI 工具应帮助 ADHD 用户逐步内化执行功能。例如，使用 RescueTime 一段时间后，用户应能更准确估计时间；使用 Focusmate 后，应能逐渐独立启动任务。如果用户永远离不开这些工具，它们就成了拐杖。

工程上的对应是：好的 agent harness 设计会逐步减少外部提示，让 LLM 学会在上下文中自我纠正。ADHD 工具也应遵循类似原则——设计“可撤除”的脚手架。

## 今天就能试的行动

1. **用 ChatGPT 做“任务分解+重锚定”**：在开始一项长任务前，让 ChatGPT 生成 3-5 个步骤，每个步骤完成后要求它提醒你“回到原始目标”。这模仿了 agent harness 中的阶段性提示注入。
2. **试用 RescueTime 一周**：开启自动追踪，周末查看报告，识别你的注意力漂移模式。注意：不要过度关注数据，而是用它训练时间感知。
3. **尝试 Focusmate 一次**：预约一个 25 分钟时段，体验虚拟身体在场效应。观察它是否降低了任务启动的阻力。
4. **设置“撤除提醒”**：在日历上每月标记一次，问自己：“如果今天没有这些工具，我能完成同样的事吗？”这有助于避免过度依赖。

## 诚实面对局限

本文的论证基于概念对齐和工具机制，而非大规模临床研究。ADHD 亚型（注意力缺陷 vs 多动冲动）对 AI 工具的反应可能不同（来源：矛盾与存疑）。此外，神经可塑性干预的既定措施仍然很少（来源：神经可塑性）。因此，请将以上建议视为实验性尝试，而非医疗建议。

最终，脚手架与拐杖的边界在于使用者的意图。如果你用 AI 来**学习**如何专注，它就是脚手架；如果你用它来**逃避**培养专注力，它就是拐杖。
