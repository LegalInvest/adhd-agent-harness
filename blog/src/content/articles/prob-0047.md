# 3 个 LLM 工程概念，彻底重写了我对 ADHD 冲动下结论、想当然的理解

> LLM 自信地编造（幻觉），靠验证与自我批判兜底 如何反向照亮 ADHD 的冲动下结论、想当然。

“他肯定在针对我。”“这个项目我闭着眼都能做完。”——在 ADHD 大脑里，这类结论常常不是证据先到场，而是多巴胺驱动的生成核心已经抢先交稿。与此同时，做 Agentic Harness 的工程师对另一个场景同样熟悉：LLM 用同样自信的语气，输出一段完全编造的引用或数据。两边的痛点惊人地一致：一个高产但缺少执行调度层的生成核心，在缺少 harness 时，会把“流畅”当成“正确”。

ADHD 大脑与 LLM 在结构上同构——两者都是“高产但缺执行调度层的生成核心”（来源：ADHD × AI 的科学与研究前沿）。这个同构视角不是修辞，它彻底改写了冲动下结论和想当然：它们不是性格缺陷，而是系统架构问题——生成核心跑在了验证与执行层前面。治疗或工程上的真正任务，不是“让生成核心闭嘴”，而是给它套上可拆卸的 harness。

## 概念一：幻觉与验证循环——“先输出，后核实”

LLM 最让工程师警醒的，不是它不会，而是它太会“自信地编造”。幻觉不是偶发 bug，而是生成机制在没有外部事实锚点时的默认行为。工程上的解法从来不是改模型人格，而是加一层验证循环：检索增强、工具调用、自我批判、多轮检查。一句话，把“生成”和“验证”拆成两个环节。

ADHD 的冲动下结论正是同构现象。多巴胺系统，尤其是中脑皮质分支功能低下，会导致注意反应缺陷与行为计划不良（来源：多巴胺）；执行功能和工作记忆的不足，让人难以在心里“跑一遍”反事实和反证。于是大脑像一台没有 critic 的 LLM，把最先冒出来的叙事当成答案。

毛泽东的 harness 系统与此同构。他的“没有调查，就没有发言权”相当于 LLM 的检索与 grounding；民主生活会和“批评与自我批评”则是一个外部 critic 循环，让个人判断回到集体证据里重新校准。他一辈子行军也带着书、走到哪里调查到哪里，不是勤奋表演，而是把“验证循环”制度化。对 ADHD 而言，这意味着：任何强烈结论，必须先经过外部证据或他人视角的第二遍检查，而不是让生成核心直接对外发布。

## 概念二：外部执行层与任务调度——把“下一步”外包出去

LLM 是无状态的，它缺乏一个稳定的执行调度层。所以做 Agentic Harness 时，工程师必须外挂规划器、记忆库、工具接口和向量数据库——也就是给模型搭脚手架，让它不只是生成，而是能按步骤执行（来源：外部执行功能层；来源：无状态与外部记忆）。

ADHD 的困境同样不是“不想做”，而是“启动不了”。多巴胺不足削弱行动动机，工作记忆容量有限让任务步骤无法清晰排列，时间盲让人低估紧迫性，最终卡在任务启动前（来源：任务启动）。工程上的对应解法是外部执行层：把任务、时间和优先级从大脑里搬出来。

Goblin Tools 的 Magic ToDo 把“清理厨房”这类模糊任务拆解成可执行的子任务，通过降低启动门槛对抗任务瘫痪（来源：Goblin Tools）；Motion 自动根据任务、截止日期和可用时间动态排程，减少“下一步该做什么”的决策负担（来源：Motion）；Tiimo 用时间轴可视化帮助用户感知时间流逝，缓解时间盲（来源：任务启动）。这些工具不是替你思考，而是替你调度。

释迦牟尼的 harness 提供了另一组同构映射。他以“八正道”和“持戒”作为外部行为规则，用戒律管理行动，用正念把乱跑的念头拴住；又以“自依止、法依止”拒绝依赖权威，自己验证。这很像 LLM agent 的“约束 + 自校验”：先给系统加上 guardrails 和计划器，再让它在执行中自我修正。对 ADHD 来说，日程表、微任务清单、自动化排程工具，就是可拆卸的“戒”与“定”。

## 概念三：上下文工程——防止“注意跑偏”

LLM 的上下文窗口有限，一旦上下文膨胀，注意力会被无关 token 稀释，推理质量退化。工程上需要上下文工程：结构化提示、剪枝无关信息、把核心目标反复写进 prompt。否则模型会被自己的输出带偏（来源：上下文工程）。

ADHD 的注意力机制也存在类似的“跑偏”：工作记忆不稳定，难以在脑中持续保留跨会话的目标；环境里的新刺激很容易把注意拽走；而超聚焦又可能把全部资源锁进单一任务，忘了原本要解决的问题。身体在场效应说明，外部“他者在场”能提升专注度；ChatGPT 也被一些 ADHD 成年人用作执行功能外挂，帮助任务启动和维持方向（来源：任务启动）。

Saner.AI 的做法是同构的上下文工程：通过本地记忆和知识回忆，减少在多个标签页和应用之间切换的搜索循环，降低认知负荷（来源：Saner.AI）。把目标、证据、待办事项外化到同一个可靠的外部记忆里，相当于给 ADHD 大脑也做了一个“系统提示词”——在上下文膨胀时，重新把核心目标读出来。
