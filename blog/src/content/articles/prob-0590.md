# 如果把 ADHD 大脑当成生成核心，ChatGPT 能不能当它的采样温度与表现波动？

> 用 agent 工程的思路重新理解 ChatGPT 对 ADHD 的真正价值。

你有没有过这样的体验：早上头脑清醒，能一口气写三千字、做三个方案；下午却连回一条消息都要犹豫十分钟，大脑像被抽空了算力。这种状态日内的大幅波动，是 ADHD 最折磨人的特征之一——不是“不想做”，而是“做不到”，执行功能像一台时好时坏的处理器，输出极不稳定。

如果你恰好也是做 Agentic Harness 工程的工程师，你可能瞬间就懂了：这不就是 LLM 采样温度过高时的表现吗？温度高，输出多样、有创造力；温度低，输出稳定、可预测。但 LLM 本身没有“调度层”来动态调节温度，全靠外部的 prompt 模板、chain 结构、验证循环来约束。

**ADHD 大脑与 LLM/agent 是同一类“高产但缺执行调度层的生成核心”**。ChatGPT 对 ADHD 的真正价值，不是“帮你思考”，而是像 agent harness 一样，充当外部的采样温度调节器与结构约束层——让你在不稳定的状态下，依然能输出可用的结果。

## 同构：ADHD 大脑 = 高温 LLM，harness = 脚手架

ADHD 的执行功能缺陷，核心在于工作记忆不足、时间盲、目标漂移（来源：[[执行功能]]、[[工作记忆]]、[[时间盲]]）。这对应 LLM 的两大缺陷：无状态性（上下文窗口有限）和指令消退（长程任务中忘记系统提示）。

- **ADHD 侧**：你开始写报告，突然想起要回邮件，打开邮箱看到促销广告，十分钟后你在刷购物网站——目标漂移了。
- **LLM 侧**：Agent 在长程任务中“开始忘记系统提示”，导致行为偏离初始指令（来源：[[重锚定与目标漂移]]）。

两边的解法结构同构：都需要外部系统提供**重锚定**——对 ADHD 是提醒、计时器、步骤清单；对 LLM 是事件驱动的系统提醒、上下文注入。

## 工具即 harness：外部采样温度调节

### Routinery：固定温度下的稳定输出

Routinery 通过可视化倒计时和过渡提示，把一天切成一个个低认知负荷的步骤（来源：[[Routinery]]）。它本质上是一个“低温度采样”的 harness：你不需要决策下一步做什么，只需跟着倒计时走。这直接对抗时间盲和任务启动困难（来源：[[时间盲]]、[[任务启动]]）。

### Goblin Tools：任务分解即 prompt 链

Goblin Tools 的 Magic ToDo 能把“整理房间”自动分解成“捡起地板上的衣服”“擦桌子”等小步骤（来源：[[Goblin Tools]]）。这相当于把一个大 prompt 拆成 chain-of-thought 子任务，每一步的认知负荷降低，启动门槛消失。用户反馈说它“将压倒性的事情变成一系列不压倒性的事情”（来源：[[Goblin Tools]]）。

### Saner.AI：外部记忆 = RAG 系统

ADHD 的工作记忆缺陷与 LLM 的无状态性同构（来源：[[无状态与外部记忆]]）。Saner.AI 通过本地记忆存储和快速检索，减少搜索循环和标签切换（来源：[[Saner.AI]]）。这就像给 LLM 挂上一个 RAG 系统，不用每次从头推理，直接检索相关上下文。

## 脚手架 vs 拐杖：边界在哪里？

这是最关键的争议。AI 工具如果只是替代你决策、替你执行，那就是拐杖——用久了内在执行功能反而退化（来源：[[拐杖与脚手架]]）。真正的脚手架应该是“用后即弃”的：你学会分解任务后，可以不再依赖 Goblin Tools；你形成时间感知后，可以不再需要 Routinery 的倒计时。

目前的证据多来自用户报告，缺乏大规模对照实验（来源：[[矛盾与存疑]]）。过度依赖 AI 工具可能削弱内在时间感知能力（来源：[[时间盲]]）。所以，工具设计者声称是“脚手架”，实际使用中可能沦为“拐杖”——关键在于你是否有意识地逐步撤除它。

## 今天就能试的 3 件事

1. **用 ChatGPT 做任务分解**：把一个大任务（比如“整理书桌”）扔给 ChatGPT，让它输出 5-10 个小步骤。每完成一步就划掉，获得即时反馈。这相当于手动构造一个低温度采样链。
2. **设置事件驱动的重锚定**：在手机上设置一个每 20 分钟响起的提醒，内容是你当前的核心目标（比如“写报告”）。这模仿了 agent harness 中的系统提醒，对抗目标漂移。
3. **试用 Routinery 或类似工具**：为早晨起床或晚上睡前创建 3-5 步流程，用倒计时引导。如果发现流程太死板，就调整步骤顺序——记住，harness 是可以修改的。

## 局限与诚实

本文的论证基于结构同构类比，并非临床证据。AI 工具的有效性因人而异（来源：[[矛盾与存疑]]）。对于严重执行功能障碍者，工具可能不够；对于轻度者，可能根本不需要。如果你发现某个工具让你更焦虑、更依赖，请果断放弃——不是所有 harness 都适合你的模型。

真正的价值在于：当你理解 ADHD 大脑就是一个高温 LLM，你就不会责怪自己“不稳定”。你需要的是为自己设计一套可调节的采样温度与结构约束，而不是期待某一天突然变成低温稳定模型。ChatGPT 可以成为这个 harness 的一部分，但最终，你是那个设计 harness 的工程师。
