# 为什么Agent正在成为ADHD新的多巴胺来源？

> 先描述一个越来越常见的场景:一个 ADHD 者深夜和 AI agent 工作——不是在完成任务,而是在「看它跑」:发一个指令,看它思考、调工具、吐结果;再发一个,再看一轮。三小时过去,回头看真正的产出,可能只有半小时的量。这个体验的质感似曾相识:**和刷短视频、开新游戏、逛购物 App 是同一种「停不下来」**。这不是巧合。Agent 交互的结构,几乎精确命中了 ADHD 奖赏系统的每一个偏好——**它正在成为一种新的多巴胺来源;而这件事同时是机会和陷阱,取决于你有没有看清它的机制。**

先拆机制,为什么 agent 对 ADHD 的奖赏系统如此对口。ADHD 奖赏加工的核心特征(神经影像与行为研究的一致方向):**对延迟奖励的折扣陡峭,对即时、新奇、变动的刺激敏感**。对照 agent 交互的结构:**①即时性**——指令发出,几秒内就有响应流出来(对比:自己做事,奖励在几小时外);**②变率性**——每次输出都不完全可预测,有时惊艳有时翻车,这正是变率强化(variable reinforcement)的结构,行为科学里最强的成瘾配方(老虎机、刷信息流,同款);**③低启动成本**——「发一句话」可能是人类历史上启动阻力最低的「工作」动作;**④代理感**——看着一个系统按你的意志行动,这种指挥的快感本身就是奖励。四个特征叠加,对一个奖赏系统本就偏好即时与新奇的大脑,吸引力是结构级的。

然后是关键的分岔:**同一个多巴胺来源,可以是燃料,也可以是漏斗——分界线在于奖励绑定的对象**。绑定在「产出」上,它是燃料:agent 把「写报告」这种延迟奖励任务,拆成了一轮轮几分钟粒度的「指令→结果」循环——**相当于给任务本身装了 reward shaping(Habitica 篇的机制,但这次奖励来自真实工作的中间产物)**:ADHD 者历史上最难启动的那类长任务,第一次有了即时反馈的版本,这是真实的、值得庆祝的可及性革命。绑定在「交互」上,它是漏斗:快感来自「看它跑」本身——于是出现新型拖延:**用「让 agent 再试一版」回避「我来做决断」,用无限的生成回避有限的交付**;产出没有增加,只是拖延第一次有了「我在工作」的完美伪装。

判据一条就够:**这一轮交互结束时,你离交付更近了,还是只是又看了一场表演?** 更近——燃料;原地——漏斗。诚实地回答一周,你就知道自己在哪边。

给两个把漏斗改回燃料的手法:**第一,给交互定「回合上限」**——同一个产出物,agent 最多迭代三轮,第三轮后必须由你做决断(用/改/弃)——**把「再试一版」的无限循环截断在决断点上**(这正是 human-in-the-loop 的自我应用:你是那个必须在环上的人);**第二,把快感显式记账**——每轮交互后问一句「这轮的快感来自结果,还是来自看它跑?」——和 Brain.fm 篇的内感受训练同理,**能分辨,才能调节**;分不清的快感,默认按漏斗处理。

## 边界

同构强度 B 级:ADHD 的延迟折扣与奖赏加工差异有扎实的实验与影像文献,变率强化的行为机制是行为科学的成熟结论,「agent 交互命中这些偏好」是结构分析而非已验证的实证(这个现象太新,直接研究还没长出来)。声明:本文说的是行为模式,不是诊断——「停不下来」若已侵蚀睡眠、工作与关系,达到行为成瘾的功能损害标准,请专业求助;另外多巴胺是通俗化的简写,实际机制远比「一种化学物质的多少」复杂,别拿本文当神经科学教材。

## 今天就能试的 3 件事

1. **做一周的「更近了吗」记账**:每次 agent 会话结束,一个字:近/原地——周末看比例。
2. **给下一个任务设三轮上限**:第三轮后强制决断——用、改、还是弃,你来说。
3. **记一次快感来源**:今晚那次交互,爽的是结果,还是看它跑?——写下来,这是你的调节起点。

Agent 把即时反馈装进了慢工作,这对 ADHD 是一个时代级的礼物——**但礼物和陷阱是同一个东西,区别只在奖励绑在产出上,还是绑在表演上**。多巴胺不挑对象,你得替它挑。
