# ADHD 家长如何用「给生成核心套 harness」的思路应对陪伴 ADHD 孩子里的任务启动困难、不会拆解？

> 把 LLM 工程里的「agent 用 planner-executor 循环分解长任务」迁移到ADHD 家长的陪伴 ADHD 孩子场景，双域证据作答。

你站在孩子房间门口，看着他在书桌前发呆，作业摊开，笔没动。你提醒了三次，他回你一句“我知道，但我就是开始不了”。你试过列清单、定奖励、甚至发脾气，都没用。这不是懒，是 ADHD 大脑的典型困境——任务启动困难与不会拆解。

而另一边，在 AI 工程领域，工程师们正为一个类似问题焦头烂额：LLM 能生成惊艳的文本，但让它执行一个多步骤任务（比如“帮我规划一次旅行”），它要么漏掉关键步骤，要么中途跑偏。解决方案是给它套上 **agent scaffolding**——一个 planner-executor 循环，让 LLM 先生成计划，再一步步执行，每步检查进度。

这两个场景，本质是同一个问题：一个强大的生成核心，缺乏可靠的内置调度层。ADHD 大脑与 LLM 在结构上同构——都是“高产但缺执行调度层的生成核心”（来源：ADHD 大脑与 LLM 的同构）。因此，为 LLM 搭建脚手架的方法论，完全可以迁移到 ADHD 孩子的陪伴中。

## 同构的真相：生成核心与调度层

ADHD 大脑的核心特征是高产但无序的生成能力。想法像喷泉一样涌出，但执行功能——工作记忆、任务启动、时间管理——常常故障，导致“想法很多，行动困难”（来源：ADHD 大脑与 LLM 的同构）。实证研究甚至表明，ADHD 个体的工作记忆容量可能超过常人，但认知灵活性与注意控制存在核心缺陷（来源：ADHD 大脑与 LLM 的同构）。

LLM 同样如此：它是强大的生成核心，但缺乏可靠的内置调度与执行控制。在构建 AI agent 时，需要“agent scaffolding”——即“围绕 LLM 构建的软件架构和工具，使其能够执行复杂、目标驱动的任务”（来源：ADHD 大脑与 LLM 的同构）。这种架构常采用“复合 AI 系统”，将 LLM 置于与记忆、工具和决策逻辑的控制循环中。

这告诉我们：**不要指望孩子“自己学会规划”，就像不指望 LLM 单靠提示词就能完成复杂任务。你需要成为那个“外部调度层”。**

## 历史人物的“外部调度层”

孔子和辛弃疾，都是典型的 ADHD 大脑，但他们各自找到了自己的“harness”。

孔子精力旺盛，周游列国 14 年坐不住，思维跳跃，《论语》全是场景化语录，无系统著作。他的 harness 是“克己复礼”——用外在的“礼”作为行为边界，每日反省，“吾日三省吾身”。这本质上是一个 **定时 re-grounding 的调度器**：每天早上设定行为准则，晚上检查执行情况。这与 LLM 工程中的“定时 checkpoint”异曲同工。

辛弃疾 21 岁带 50 人冲几万人敌营抓叛徒，一辈子想北伐但坐不住，被罢官后闲居 20 年。他的 harness 是“以词和剑为核心”——用固定的创作和练习来锚定注意力。这相当于 LLM 的 **外部记忆系统**：把发散的能量注入一个固定的输出管道。

他们的成就证明：**harness 不是限制，而是释放。** 就像 agent scaffolding 让 LLM 从“乱生成”变成“可靠执行”，孔子的“礼”和辛弃疾的“词”让他们的 ADHD 特质从障碍变成了创造力引擎。

## 如何构建孩子的 planner-executor 循环

具体来说，家长可以这样做：

**1. 把任务拆解成可执行的“步骤序列”**

ADHD 孩子面对“写一篇作文”这种开放式任务时，大脑会直接死机。你需要像 agent 的 planner 一样，把任务分解成原子步骤：
- 第一步：选一个题目（3 分钟）
- 第二步：列出 3 个要点（5 分钟）
- 第三步：写开头段（10 分钟）
- ……

工具推荐：**Goblin Tools** 的“任务分解”功能就是专门做这个的，它能把“打扫房间”拆成“捡起地上的衣服”“把书放回书架”等具体动作（来源：Goblin Tools 页面）。你也可以手动做，但 AI 工具能帮你省下认知负荷。

**2. 用视觉化时间线让时间“可见”**

ADHD 孩子常有“时间盲”——感受不到时间流逝。**Tiimo** 这类视觉化规划应用，通过将时间转化为可见的色块和进度条，直接应对时间盲问题（来源：Tiimo 页面）。你可以和孩子一起用纸笔画出时间线，或者用 Tiimo 这类 app，让每个任务块变成可以“看到”的实体。

**3. 建立“执行-检查”循环**

LLM agent 每执行一步都会检查结果，再决定下一步。家长可以扮演这个“检查者”：每完成一个步骤，和孩子一起回顾：“这一步做完了吗？感觉怎么样？下一步是什么？”这不仅是监督，更是帮孩子建立自我监控的神经通路。神经可塑性研究表明，训练诱导的抑制控制脑可塑性有助于正常化执行功能（来源：神经可塑性）。

**4. 用自动排程工具减少决策负担**

**Motion** 这类自适应规划器能自动根据任务优先级和截止日期动态调整日程，消除“下一步该做什么”的决策负担（来源：Motion 页面）。对于 ADHD 孩子，你可以用类似逻辑：提前规划好每天的固定时段（比如 4-5 点做数学作业），让执行变成习惯而非每次都要决策。

## 脚手架 vs 拐杖：一个必须警惕的边界

这里必须诚实指出：**过度依赖外部调度层可能削弱孩子内在执行功能的发展。** 这就是“脚手架 vs 拐杖”的争议（来源：矛盾与存疑）。AI 工具可能从“脚手架”退化为“永久拐杖”，让孩子的自我管理能力永远无法建立。

我的判断是：**关键在于“逐步撤除”。** 就像训练 LLM agent 时，随着模型能力提升，逐渐减少人工干预。家长应该从“全程陪伴”过渡到“只在关键节点介入”，最终让孩子自己掌握 planner-executor 循环。

## 今天就能试的 3 个行动

1. **一次拆解练习**：选一个孩子最抗拒的任务，用“第一步…第二步…”的形式拆成不超过 5 个步骤，写在纸条上，让他每完成一步就划掉一条。
2. **画一条时间线**：用一张 A4 纸画一条横线，标出“开始”和“结束”，中间分成几段，每段写一个步骤。贴在书桌前。
3. **设置一个“检查点”闹钟**：每 15 分钟响一次，提醒孩子停下来问自己：“我刚才在做什么？下一步该做什么？”

这些方法不需要任何 app，但已经嵌入了 planner-executor 循环的核心逻辑。当你和孩子一起执行时，你就在做一件和 AI 工程师同样的事：**给一个强大的生成核心套上 harness，让它从混乱中长出秩序。**
