# 用 Claude 给 ADHD 补上外部执行功能层，是脚手架还是拐杖？

> Claude 接进 ADHD 工作流：哪些交给它、哪些必须自己留着。

如果你是一位 ADHD 患者，你可能熟悉这种感觉：脑子里有无数绝妙想法，但就是无法把它们变成行动清单；或者你是一位 AI 工程师，正在为 agent 搭建 harness——编排工具、管理记忆、设置护栏，防止它跑偏。这两件事，其实是同一道题的两个版本。

## 同一个核心，同一个缺口

ADHD 大脑与 LLM 在结构上高度同构：两者都是“高产但缺执行调度层的生成核心”。ADHD 的生成核心（发散思维、创意、多任务并行）产出极高，但执行功能（任务启动、持续专注、抑制冲动）薄弱，导致“高产但不可靠”（来源：人在回路与身体在场）。LLM 本身是强大的生成核心，但缺乏可靠的执行调度层，直接输出可能偏离目标、陷入循环或产生危险行为（来源：人在回路与身体在场）。

这种同构在神经心理学层面得到印证：最新研究发现，Transformer 在 Stroop 冲突任务中，注意力随序列变长在冲突试次上骤降到随机水平——无法抑制优势反应、无法解决认知冲突，这与 ADHD 执行功能缺陷的核心标志一一对应（来源：Deficient Executive Control in Transformer Attention）。LLM 呈现“强记忆、弱控制”剖面：工作记忆容量甚至超过常人，但认知灵活性与注意控制存在核心缺陷——无法灵活切换任务集、无法抑制自动化反应，这正是 ADHD 的经典神经心理模式（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。

## 外部执行功能层：脚手架还是拐杖？

AI 工具充当的是“外部执行功能层”——补偿工作记忆不稳定、任务启动困难和注意力分散（来源：外部执行功能层）。但问题在于：它是可拆卸的脚手架，还是永久依赖的拐杖？

答案取决于设计哲学。以曾国藩为例，他是一位典型的注意力不集中型 ADHD 患者：30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”。他的解决方案是“日课十二条”——黎明即起、读书不二、谨言、写日记反省等，用最笨最稳的方法抵消冲动（来源：人物案例）。这套系统本质上就是一套“外部执行功能层”，与 LLM agent harness 中的定时 re-grounding、外部调度器完全同构。曾国藩的日课是脚手架，因为他最终内化了自律，而非终身依赖。

同样，蔡元培的“思想自由，兼容并包”也是一种 harness——他把不同观点的人请到北大，用制度而非意志力来维持开放。他的成就不是靠天生执行功能，而是靠外部系统。

## 真实工具的证据

当前 AI 工具正在实践这一思路：
- **Goblin Tools** 的 Magic ToDo 将模糊任务分解为具体子步骤，用户可调节“辣度”控制粒度（来源：Goblin Tools）。这直接对应 LLM harness 中的“任务分解与规划”模块。
- **Saner.AI** 通过本地记忆和快速检索减少搜索循环，对抗工作记忆不足（来源：Saner.AI）。这相当于 LLM 的向量数据库 + 记忆检索。
- **Motion** 自动排程并动态调整，消除“下一步做什么”的决策负担（来源：Motion）。这类似 agent harness 中的调度器。
- **Focusmate** 利用身体在场效应：有另一个人在场时，执行功能障碍被绕过（来源：人在回路与身体在场）。这对应 LLM harness 中的 human-in-the-loop 监督。

## 争议与边界

尽管同构框架诱人，但必须承认局限：
1. **过度依赖风险**：AI 工具可能成为永久拐杖而非可拆卸脚手架，但尚无长期研究验证（来源：矛盾与存疑）。
2. **个体差异**：AI 辅助效果依赖个体差异，且缺乏长期验证（来源：矛盾与存疑）。
3. **证据强度**：当前多为概念类比和初步实验，缺乏大规模随机对照试验（来源：矛盾与存疑）。
4. **认知负荷**：部分用户反映学习使用 AI 工具本身增加认知负荷，净效果因人而异（来源：矛盾与存疑）。

## 核心判断

我的观点是：**AI 工具作为外部执行功能层，本质是脚手架而非拐杖，但前提是用户有意识地设计“退出机制”**。曾国藩的日课不是依赖，而是内化；蔡元培的制度不是替代，而是赋能。同样，好的 AI 工具应该像 agent harness 一样，允许用户逐步减少干预。

## 今天就能试的行动

1. **用 Goblin Tools 分解一个拖延任务**：输入“清理厨房”或“写报告”，调节辣度到 3，看它拆成多少步。
2. **设置一个 Focusmate 会话**：每天 25 分钟与人视频共处，体验身体在场效应。
3. **用 Motion 自动排程一天**：输入 3-5 个任务和截止日期，观察 AI 如何分配时间。
4. **写一条“退出机制”**：在笔记中记录“如果我用这个工具 30 天后仍无法独立完成该任务，就减少使用频率”。

ADHD 大脑不是缺陷，LLM 也不是 bug。它们只是缺少一层 harness。而 harness 的最终目的，是让你不再需要它。
