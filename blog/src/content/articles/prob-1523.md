# 如果把 ADHD 大脑当成生成核心，Inflow 能不能当它的重锚定与目标漂移？

> 用 agent 工程的思路重新理解 Inflow 对 ADHD 的真正价值。

你打开 Inflow，想用它规划今天最重要的三件事。十分钟后，你发现自己正在研究它的 UI 配色方案，或者刷起了 App Store 的评论。目标漂移了——不是因为你懒，而是因为你的大脑和 LLM 一样，是个“高产但缺执行调度层的生成核心”。

## 同构：ADHD 大脑与 LLM 共享的困境

ADHD 大脑的生成核心（发散思维、创意、多任务并行）产出极高，但执行功能（任务启动、持续专注、抑制冲动）薄弱，导致“高产但不可靠”（来源：AI 与 ADHD 的专注力）。LLM 同样如此：它能在对话中生成精彩内容，但直接输出可能偏离目标、陷入循环或产生危险行为，所以需要 agent harness（脚手架）来提供外部调度——设置护栏、加入人工检查点、在关键节点重锚定（来源：人在回路与身体在场）。

这种同构意味着，ADHD 需要的不是“更强大的生成器”，而是类似 agent harness 的外部执行功能层。曾国藩的“日课十二条”就是一套手工 harness：黎明即起、读书不二、谨言、写日记反省——用最笨最稳的方法抵消自己的冲动（来源：人物案例）。他年轻时“无恒”“浮躁”，日记里天天骂自己，但正是这套外部纪律让他从浮躁青年变成“半个圣人”。这套 harness 的结构与 LLM agent 的“定时 re-grounding”完全同构：日课就像每轮对话前重新注入 system prompt，写日记反省就像日志遥测，而“结硬寨打呆仗”则相当于设置 token 预算和工具调用上限。

## Inflow 能否成为 ADHD 的重锚定系统？

Inflow 的设计理念正是提供这种重锚定：它将任务分解为可管理的步骤，设置提醒和专注计时器以提高生产力（来源：重锚定与目标漂移）。它试图充当“待办应用、教练和支持性朋友”（来源：重锚定与目标漂移）。这些功能对应的是对抗 ADHD 内在的执行功能缺陷——工作记忆不稳定、任务启动困难、时间盲。

但这里有一个关键缺陷：“传统提醒系统需要你主动设置提醒，而设置提醒本身就是一个执行功能任务”（来源：重锚定与目标漂移）。ADHD 个体恰恰在这方面薄弱。Inflow 如果要求用户先规划好所有步骤，它就已经失败了——因为规划本身是执行功能的核心。

更理想的重锚定系统应该是“无需主动设置的提醒”。RescueTime 在这方面做得更好：它自动记录时间使用，识别注意力漂移，并能在必要时主动阻断分心网站（来源：RescueTime）。它不需要用户记住“我要开始专注了”，而是直接揭示“时间到底去了哪里”。Focusmate 则通过虚拟身体在场提供另一种重锚定：有另一个人在场时，执行功能障碍被绕过，任务得以启动和维持（来源：人在回路与身体在场）。它的 AI 匹配算法优化了配对效率（来源：Focusmate），但核心机制仍然是社会问责，而非 AI 生成。

## 脚手架还是拐杖？一个必须回答的问题

AI 工具既可能成为永久依赖的拐杖，也可能成为逐步内化的脚手架（来源：AI 与 ADHD 的专注力）。目前没有长期追踪研究能回答这个问题（来源：矛盾与存疑）。但我们可以从 agent harness 工程中看到线索：好的 harness 不是替 agent 做决定，而是让 agent 在关键节点暂停、反思、重新锚定。同样，好的 ADHD 工具应该逐步训练用户的内化能力，而不是永远替代执行功能。

Brain.fm 提供了另一种思路：它通过神经锁相技术影响大脑信息处理方式，帮助用户进入专注状态（来源：Brain.fm）。这更像是直接调节生成核心的内部参数，而非增加外部调度。但它在 ADHD 人群中的独立临床证据仍有限（来源：Brain.fm）。

## 争议与局限

同构性是一个有用的类比，但并非经过验证的科学事实（来源：矛盾与存疑）。ADHD 大脑与 LLM 的相似性可能止于功能层面，而非神经机制。此外，ADHD 的异质性意味着同一工具对不同亚型效果迥异（来源：AI 与 ADHD 的专注力）。Inflow 可能对注意缺陷为主型有效，但对多动冲动型可能不够。

另一个矛盾是：超聚焦是 ADHD 的双刃剑——它能让用户沉浸于任务，但也会导致忽略整体目标（来源：矛盾与存疑）。Inflow 的提醒机制可能打断超聚焦，反而破坏创造力。

## 今天就能试的行动

1. **设置一个“无需主动设置”的提醒**：用 RescueTime 或类似工具自动追踪时间，而不是手动规划。先观察一周，找出你的目标漂移模式。
2. **尝试一次 Focusmate 会话**：预约 25 分钟，和一个陌生人一起工作。体验身体在场效应如何绕过执行功能启动困难。
3. **设计你的“日课十二条”**：像曾国藩一样，写下一套外部纪律。不需要完美，只需要三条：比如“每天第一个任务固定为写日志”“每工作 25 分钟起身喝水”“睡前回顾一次目标是否漂移”。
4. **测试 Brain.fm 的专注音乐**：在需要深度工作时使用，看它是否能降低你被环境带偏的概率。注意记录效果，因为个体差异很大。

Inflow 能不能当 ADHD 的重锚定？能，但前提是它不要求你先执行“设置提醒”这个执行功能任务。最好的重锚定系统，是那些在你还没意识到自己漂移时，就已经把你拉回来的系统。
