# 用 ChatGPT 给 ADHD 补上外部执行功能层，真的能像 agent 那样工作吗？

> 把 ChatGPT 当作 ADHD 的外部执行层来实测：它补的是哪一块短板？

## 问题：执行功能短板，AI 能补吗？

你是否有过这样的体验：脑子里想法翻涌，但就是无法启动任务；计划写满整页，却总在第一步卡住；工作记忆像漏水的桶，刚想做的事转眼就忘。这是 ADHD 人群的日常——执行功能（计划、组织、启动）先天偏弱。与此同时，AI 工程师们正在为 LLM 搭建 agent scaffolding：给模型配上记忆、工具和编排逻辑，让它能可靠地完成多步骤任务。这两件事听起来风马牛不相及，但若深入结构，会发现它们共享同一套困境：**高产但缺执行调度层的生成核心**。

## 同构：ADHD 大脑与 LLM 的惊人相似

ADHD 大脑的生成核心——发散思维、联想能力、超聚焦——与 LLM 的文本生成能力类似，但两者都缺乏可靠的内置执行调度层（来源：ADHD 大脑与 LLM 的同构）。ADHD 的工作记忆缺陷与 LLM 的无状态性都导致无法可靠保留跨会话上下文，必须依赖外部记忆系统（来源：无状态与外部记忆）。注意力管理上，ADHD 大脑和 LLM agent 都面临被环境带偏或上下文膨胀导致推理退化的问题（来源：上下文工程）。

这种同构意味着，为 LLM 搭建 agent scaffolding 与为 ADHD 搭建执行功能支架是同一类工程（来源：生成核心与调度层）。LLM 需要外部 harness 来提供规划、任务分解、工具接口和验证循环（来源：Agent Scaffolding: Architecture and Design Patterns for Agentic AI）；ADHD 大脑同样需要外部系统来补偿工作记忆不稳定、任务启动困难和注意力分散（来源：外部执行功能层）。

## 人物案例：曾国藩的“日课”与 LLM harness 的同构

晚清名臣曾国藩，是典型的注意力不集中型 ADHD：30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”，读书慢到“他人目下二三行，余或疾读不能终一行”。他的解决方案是“日课十二条”：黎明即起、读书不二、谨言、写日记反省……这套严格的日常流程，本质就是一套外部执行功能层——用固定节奏和外部记录来补偿内在调度缺陷。其中“写日记反省”相当于 agent 的日志与复盘循环，“读书不二”相当于单任务模式，“结硬寨打呆仗”则是用最笨最稳的方法对冲冲动决策。

这与 LLM harness 的结构惊人一致：日课 ↔ 定时 re-grounding，日记 ↔ 状态记录与验证循环，秘书 ↔ 外部调度器。曾国藩的成就——平定太平天国、发起洋务运动——证明这套 harness 能释放生成核心的潜力。同样，LLM 加上 harness 后，能从“聊天玩具”变成可靠的工作代理。

## 实测：ChatGPT 能补哪块短板？

当前已有多个 AI 工具在实践这一思路。**Goblin Tools** 的 Magic ToDo 功能能自动将模糊任务（如“清理厨房”）分解为可管理的小步骤，用户可调节“辣度”控制粒度（来源：Goblin Tools）。这直接对应 LLM harness 中的任务分解模块——将复杂目标拆解为原子步骤。**Saner.AI** 通过增强知识回忆，减少搜索循环和标签切换，对应 LLM 的检索增强生成（RAG）和记忆缓冲区（来源：Saner.AI）。**Motion** 则自动创建并动态调整每日计划，当日程被打乱时重新安排，相当于 agent 的动态规划与重调度循环（来源：Motion）。

这些工具的共同点：**规划、排序和跟踪发生在大脑之外，让大脑自由去做实际工作**（来源：Using an AI Assistant to Manage ADHD: A Practical Guide）。这正是 agent scaffolding 的核心原则——将 LLM 置于与记忆、工具和决策逻辑的控制循环中，增强其裸能力（来源：Agent Scaffolding）。

## 关键判断：脚手架 vs 拐杖

然而，这一同构也带来警示。曾国藩的日课是“可拆卸的脚手架”——他晚年仍保持自省，但已内化为习惯。而 AI 工具若被永久依赖，可能成为拐杖，导致能力退化（来源：拐杖与脚手架）。当前缺乏长期研究验证过度依赖风险（来源：矛盾与存疑）。此外，个体差异显著：有人需要强力打断（如番茄钟），有人需要柔性引导（来源：超聚焦）。AI 工具的学习成本本身可能增加认知负荷（来源：矛盾与存疑）。证据层面，同构假设多为类比推理，缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。
