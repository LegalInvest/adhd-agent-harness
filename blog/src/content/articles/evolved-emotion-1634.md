# 为什么用 Saner.AI 治 ADHD 的情绪失调，和给 agent 套 会褪去的脚手架 是一回事？

> Saner.AI 实测：同一套 harness 思路，ADHD 与 LLM 两边都成立。

## 问题：情绪一触即发，项目一拖再拖

你明明知道 deadline 就在明天，却瘫在沙发上刷了三个小时短视频；你刚下决心开始工作，一条微信通知就让你彻底偏离轨道。ADHD 的大脑像一台没有刹车的超跑——引擎轰鸣，却无法控制方向。情绪更是如此：一句批评就能引爆怒火，一次小挫折就能坠入绝望。这不是“性格不好”，而是执行功能缺陷在作祟：前额叶皮层（大脑的 CEO）调控不足，导致情绪生成系统（杏仁核）过度反应（来源：AI 与 ADHD 的情绪调节）。

与此同时，另一个领域正经历同样的困境：LLM（大语言模型）能写出流畅的诗篇，却会在多步任务中突然“忘记”上下文，或者调用错误的工具。工程师们发现，“单个 LLM 不足以可靠地完成多步骤任务、与业务工具交互或适应领域特定逻辑”（来源：Agent Scaffolding: Architecture and Design Patterns for Agentic AI）。于是他们给 LLM 套上“脚手架”（scaffolding）——一个外部的调度层，负责规划、记忆、工具调用和验证循环。

这两个场景看似风马牛不相及，实则共享同一个深层结构：**高产的生成核心 + 缺失的调度层**。ADHD 大脑与 LLM 是同构的。而 Saner.AI 这类工具，正是为 ADHD 大脑搭建的“脚手架”——一种会褪去的、可调节的外部执行功能层。

## 同构：ADHD 大脑与 LLM 的孪生困境

ADHD 大脑的情绪调节问题，往往源于底层执行功能缺陷：工作记忆不稳、时间感知偏差、任务启动困难（来源：AI 与 ADHD 的情绪调节）。这就像 LLM 的“无状态性”——每次对话都像失忆，需要外部记忆来维持上下文。AI 工具通过日志、情绪追踪等机制提供“外部记忆”，帮助 ADHD 用户识别情绪模式（来源：AI 与 ADHD 的情绪调节）。

再看注意力：ADHD 的注意力易被带偏，类似 LLM 的“上下文膨胀”——信息越多，越容易偏离核心任务。AI 通过“上下文工程”聚焦当前情绪任务，减少无关刺激（来源：AI 与 ADHD 的情绪调节）。这恰如 agent harness 中的“会话状态记录工具结果、完成的子任务和进度笔记”（来源：规划循环与任务分解），将 LLM 的注意力锁定在目标上。

更关键的是“规划循环”。ADHD 大脑的规划功能先天不足，传统工具（纸质计划、便签）是静态的，无法适应实际行为，导致“被遗弃的规划器、应用和习惯的墓地”（来源：规划循环与任务分解）。而自适应 AI 规划工具（如 Motion）能动态调整日程，学习用户的生产力模式（来源：规划循环与任务分解）。其核心原则是：“规划、排序和跟踪发生在你的大脑之外，让你的大脑自由去做实际工作”（来源：规划循环与任务分解）。这在 agent harness 中对应的是“外部规划循环”——LLM 本身不擅长规划，需要 harness 提供任务分解、工具接口和验证循环（来源：规划循环与任务分解）。

## 案例：释迦牟尼的“脚手架”

释迦牟尼——这位两千多年前的王子，29 岁抛妻弃子出家求道，先后跟随两位仙人学习，苦行六年，不断试错，最终在菩提树下证悟。他一生游行说法，从不著书立说，所有教导都是场景化比喻。这背后是典型的 ADHD 特质：冲动（放弃王位）、超专注（六年苦行）、兴趣驱动（不断切换法门）。但他建立了一套精密的“脚手架”来管理自己的大脑：**八正道 + 持戒 + 正念**。

“八正道”是行为准则，相当于 agent harness 中的“规则引擎”，约束 LLM 的输出范围；“持戒”是外部纪律，类似定时 re-grounding 的“日课”；“正念”则是注意力聚焦工具，相当于 LLM 的“上下文工程”。释迦牟尼甚至明确说：“自依止法依止，不依赖权威”——这正是“脚手架”与“拐杖”的边界：工具是临时的，最终要内化。他通过这套系统，将 ADHD 的冲动和超专注转化为持续 49 年的传法动力，影响亚洲文化 2500 年。

对比 LLM 侧的 harness：同样需要规则（提示词模板）、外部记忆（向量数据库）、工具调用（API 接口）和验证循环（输出检查）。释迦牟尼的“八正道”就是他的 agent harness——一个会褪去的脚手架，最终目标是“自依止”，即内在的自我调控。

## Saner.AI：ADHD 的 agent harness

Saner.AI 是一款专注于“知识回忆”和“本地记忆”的 AI 任务管理器，旨在减少搜索循环和标签切换（来源：Saner.AI）。它被描述为“专为对抗执行功能障碍、任务瘫痪和认知超载而设计的全能 ADHD 友好型 AI 生产力和知识助手”（来源：Saner.AI）。其核心功能包括：AI 笔记捕获、自动组织任务、脑暴工作流、每日规划，以及通用收件箱（从邮件、文档、日历中提取待办事项）（来源：Saner.AI）。

这本质上就是一个 agent harness：它将 ADHD 大脑的“生成核心”（创意、冲动、情绪）与外部“调度层”（记忆、规划、工具）连接起来。当你情绪失控时，Saner.AI 的笔记捕获快速记录触发事件，自动组织成情绪模式，下次类似情境时提醒你——这相当于 LLM 的“记忆缓冲区”。当你任务瘫痪时，它的任务分解功能（类似 Goblin Tools 的 Magic ToDo）将模糊目标拆解为可执行步骤（来源：Goblin Tools），降低启动门槛。

但 Saner.AI 不是万能的。它擅长处理结构化情绪问题（如任务焦虑），但对深层情绪（如创伤、自我否定）的干预效果证据不足（来源：AI 与 ADHD 的情绪调节）。更重要的是，它可能成为“永久拐杖”——长期依赖外部调度，可能阻碍内在情绪调节能力的发展（来源：AI 与 ADHD 的情绪调节）。这正是“脚手架 vs 拐杖”的边界：脚手架是临时支撑，最终要拆除；拐杖是永久替代，会削弱自身能力。

## 争议与局限

同构命题虽然诱人，但“ADHD 大脑与 LLM 同构”目前仍是理论类比，缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。此外，个体差异显著：ADHD 亚型（注意力不集中型 vs. 冲动型）对 AI 工具的反应不同，现有研究样本量小，结论难以泛化（来源：AI 与 ADHD 的情绪调节）。长期依赖风险也缺乏实证——没有纵向研究告诉我们，使用 Saner.AI 一年后，用户的内在调节能力是提升了还是退化了。

## 今天就能试的行动

1. **给情绪装一个“外部记忆”**：下次情绪爆发时，用 Saner.AI 或任何笔记工具快速记录“触发事件→身体感觉→想法→行为”。一周后回顾，识别模式。这相当于给 LLM 加一个日志缓冲区。
2. **用“辣度”分解任务**：打开 Goblin Tools 的 Magic ToDo，输入一个让你焦虑的任务（如“写报告”），调节“辣度”直到步骤清晰可执行。这相当于 agent harness 的任务分解组件。
3. **建立“日课”脚手架**：像释迦牟尼的“持戒”一样，每天固定时间做三件事：清晨规划（用 Motion 或 Saner.AI 的每日规划）、中午复盘（检查情绪日志）、傍晚回顾（清理工作记忆）。坚持两周，观察变化。
4. **测试“褪去”**：第三周开始，逐步减少工具使用频率——比如从每天三次规划改为两次。如果情绪和效率没有崩溃，说明脚手架正在内化；如果崩溃，则调回原频率，并思考哪些环节需要更多练习。

## 结论

Saner.AI 治 ADHD 的情绪失调，和给 agent 套会褪去的脚手架，本质是同一件事：用外部调度层补偿缺失的执行功能。释迦牟尼的“八正道”是两千年前的 harness，今天的 Saner.AI 是数字化的 harness。它们都不是终点，而是通往自我调控的桥梁。工具会褪去，但你学会的技能不会。
