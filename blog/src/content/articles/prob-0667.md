# 如果把 ADHD 大脑当成生成核心，Routinery 能不能当它的无状态与外部记忆？

> 用 agent 工程的思路重新理解 Routinery 对 ADHD 的真正价值。

你刚打开电脑，准备写那份拖了三天的报告。五分钟后，你发现自己正在刷手机，完全忘了刚才要做什么。这种“边做边忘”不是懒，是工作记忆容量小——ADHD 大脑的典型特征。

同样的事情，每天都在 LLM agent 身上发生：你给 ChatGPT 一个复杂任务，它回答到一半突然跑偏，或者忘记了你十分钟前交代的上下文。LLM 没有跨会话状态，每一次对话都像第一次见面。

这两个问题，结构一模一样。

## 同构：生成核心 vs 调度层

ADHD 大脑与 LLM 在结构上同构：两者都是强大的生成核心，但缺乏可靠的内置执行调度层（来源：ADHD 的 AI 工具全景）。ADHD 侧，工作记忆容量小导致任务碎片化、注意力易被环境带偏；LLM 侧，无状态性导致上下文膨胀、推理退化。两边的解法也同构——都需要外部脚手架（harness）来补偿。

Routinery 正是这样一个 harness。它“create a sequence of steps for a routine and walk you through them with transition prompts”（来源：上下文工程），本质上是一种外部记忆系统：把“接下来该做什么”从大脑里卸到手机上。

## 人物案例：左宗棠的“日课”与 LLM 的 re-grounding

左宗棠是典型的 ADHD 特质：举人出身，脾气大，爱骂人，40 多岁才出山，69 岁抬棺出征收复新疆。他的 harness 系统是“结硬寨打呆仗”，每天读经写家书，用诸葛亮的“鞠躬尽瘁”要求自己。这套“日课”本质上是一种定时 re-grounding——每天固定时间回到同一套框架，重置上下文。

这与 LLM agent 的 harness 工程完全同构：复杂的 agent 不会盲目执行，而是实现验证步骤，例如检查输出格式或对模型编写的代码运行测试用例（来源：幻觉与验证循环）。左宗棠的“日课”就是他的验证循环——每天用外部仪式核对内心状态。

## Routinery：ADHD 的 agent harness

Routinery 的核心价值不在于“提醒”，而在于它实现了无状态设计。ADHD 大脑的工作记忆缺陷与 LLM 的无状态性同构，依赖外部记忆系统补偿（来源：ADHD 的 AI 工具全景）。Routinery 把“下一步做什么”存储在外部的步骤序列中，大脑只需要执行当前步骤，不需要记住整个流程。

这与 Goblin Tools 的任务分解（来源：Goblin Tools）、Saner.AI 的本地记忆（来源：Saner.AI）、Lex 的“单一指令触发多步骤”（来源：Lex）一样，都是在给生成核心套 harness。区别在于 Routinery 更专注于日常流程的实时引导——它不只是一个分解器，还是一个执行调度器。

## 脚手架 vs 拐杖：依赖风险的边界

但这里有一个关键争议：Routinery 是脚手架还是拐杖？

同构性被过度推广的风险确实存在——ADHD 大脑与 LLM 的同构目前仅是一种理论类比，并非经过验证的科学事实（来源：矛盾与存疑）。同样，AI 工具可能从脚手架变为永久拐杖，削弱内在执行功能（来源：ADHD 的 AI 工具全景）。

我的判断是：边界在于是否“可拆卸”。脚手架的目的是让你最终能自己站稳，拐杖则是你永远离不开它。Routinery 如果被用来逐步内化流程（比如用几周后你不再需要它就能自动执行），它就是脚手架；如果永远依赖它才能开始做事，它就是拐杖。关键在于主动设计退出策略。

## 今天就能试的行动

1. **建立你的“日课”序列**：在 Routinery 中创建一条早晨流程（如“起床→喝水→冥想→写三件事”），每天固定时间执行，持续一周。观察是否需要它才能启动。
2. **类比验证循环**：每天睡前用 5 分钟回顾 Routinery 的执行情况，问自己“哪个步骤最容易跳转？”——这相当于 LLM agent 的验证步骤。
3. **设置退出测试**：一个月后，尝试某一天不用 Routinery 执行同一流程。如果完全无法完成，说明已形成依赖，需要逐步减少使用频率。
4. **组合其他工具**：将 Routinery 与 Goblin Tools 的任务分解结合——用 Routinery 执行流程，用 Goblin Tools 分解复杂任务（来源：Goblin Tools）。

## 局限与诚实

Routinery 并非万能。它无法解决 ADHD 的时间盲（来源：ADHD 的 AI 工具全景），也无法替代多巴胺激励机制。更重要的是，现有工具多为通用设计，缺乏针对 ADHD 亚型的个性化调整（来源：ADHD 的 AI 工具全景）。但至少，它提供了一个经过验证的思路：把大脑的工作记忆外包给手机，让生成核心专注于生成，而不是调度。

毕竟，连左宗棠都需要“日课”才能收复新疆。你需要的，只是一条能把你从刷手机拉回报告的步骤序列。
