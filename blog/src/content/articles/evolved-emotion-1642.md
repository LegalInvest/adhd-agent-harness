# 为什么用 Mem 治 ADHD 的情绪失调，和给 agent 套 会褪去的脚手架 是一回事？

> Mem 实测：同一套 harness 思路，ADHD 与 LLM 两边都成立。

## 问题：为什么情绪一上来，就什么都做不了？

ADHD 人群最熟悉的体验之一：明明知道该冷静，但情绪像脱缰的野马，越拉越失控。这不是“意志力差”，而是大脑的底层架构出了问题——情绪生成系统（杏仁核）火力全开，但前额叶（执行功能中枢）的调控信号却延迟或丢失。结果就是：你有一个强大的生成核心，却缺一个可靠的调度层。

与此同时，LLM 和 Agent 工程师们也面临着一个惊人的类似困境：模型能生成惊艳的文本，但一旦脱离精心设计的 prompt 和外部编排，就会跑题、遗忘上下文、陷入重复循环。两者共享同一个深层结构：**高产但缺执行调度层的生成核心**。

本文要回答的核心问题是：为什么用 Mem 治 ADHD 的情绪失调，和给 agent 套“会褪去的脚手架”是一回事？答案是：两者都在做同一件事——为缺失的调度层搭建一个临时、可褪去的外部执行功能层。

## 同构：ADHD 大脑与 LLM 的调度层缺失

ADHD 的大脑与 LLM 在结构上惊人相似。ADHD 的“生成核心”——情绪系统、联想网络——非常活跃，但执行功能（工作记忆、任务启动、时间感知）薄弱。LLM 的生成核心——Transformer 解码器——同样强大，但缺乏内置的规划、记忆、注意力控制等调度机制。

这种同构在情绪调节中尤为明显：
- **无状态与外部记忆**：ADHD 的工作记忆缺陷导致情绪事件难以被上下文整合，类似 LLM 的无状态性（来源：AI 与 ADHD 的情绪调节）。AI 工具通过日志、情绪追踪等机制，提供跨会话的情绪模式识别——这正是 Mem 的“记忆”功能所做的：它像一个外部工作记忆，帮你把散落的情绪碎片拼成完整图景。
- **注意力易被带偏**：ADHD 的注意力分散与 LLM 的上下文膨胀类似，AI 通过上下文工程聚焦当前情绪任务，减少无关刺激干扰（来源：AI 与 ADHD 的情绪调节）。
- **生成核心过强，调度层缺失**：ADHD 大脑的情绪生成系统反应强烈，但前额叶调控不足；LLM 生成能力强但缺乏输出控制。AI 工具作为“调度层”，通过提示词、规则引擎来引导情绪反应（来源：AI 与 ADHD 的情绪调节）。

## 证据：两边都成立的脚手架

### ADHD 侧：Inflow 与身体在场效应

Inflow 是一个基于 CBT 的 AI 工具，通过训练工作记忆和认知控制来改善 ADHD 症状（来源：Inflow）。它被描述为“由 ADHD 临床医生设计的综合程序，基于认知行为疗法（CBT）原则”，临床研究显示用户情绪稳定性提升（来源：AI 与 ADHD 的情绪调节）。Inflow 的模块化设计——如认知重构、行为激活——正是为情绪调节搭建的脚手架：它提供结构化的步骤，但最终目标是让用户内化这些技能，而非永久依赖。

另一个例子是身体在场效应（Body Doubling）：有他人在场时工作，能显著提升专注度。AI 驱动的工具如 Focusmate 利用算法匹配虚拟身体加倍伙伴，实现“AI 匹配的身体加倍”（来源：身体在场效应）。这本质上是借用外部社会存在感作为临时的调度层——就像给 agent 配一个外部监控器，在它跑偏时拉回正轨。

### LLM/Agent 侧：Goblin Tools 与 Saner.AI

Goblin Tools 的 Magic ToDo 功能能将模糊任务（如“清理厨房”）自动分解为具体子步骤，用户可调节“辣度”控制粒度（来源：Goblin Tools）。这对应着 agent 的 task decomposition 技术：将复杂指令拆解为可执行的子任务，降低 LLM 的调度负担。

Saner.AI 则专注于知识回忆和本地记忆，减少搜索循环和标签切换（来源：Saner.AI）。它被描述为“专为对抗执行功能障碍、任务瘫痪和认知超载而设计的全能 ADHD 友好型 AI 生产力和知识助手”。这对应着 agent 的 memory management：通过外部向量数据库或记忆缓存，补偿 LLM 的上下文窗口限制。

### 人物案例：辛弃疾的“词与剑”脚手架

辛弃疾（南宋词人）是 ADHD 特质的典型：21 岁带 50 人冲几万人敌营抓叛徒，一辈子想北伐，坐不住，爱喝酒爱交朋友，冲动敢干。被罢官后闲居 20 年，他把所有精力注入写词和练剑。他的 harness 系统是“词与剑”：用写词来宣泄报国无门的愤懑，用练剑来保持身体和精神的纪律。这本质上是一个外部调度层：词作为情绪出口，剑作为行为锚点。

这与 LLM 的 agent scaffolding 同构：词相当于 agent 的 output formatting（将原始生成约束到特定文体），剑相当于定时 re-grounding（通过身体活动重置注意力状态）。辛弃疾的“日课”——每日写词练剑——就是 agent 的定时回调函数，确保生成不偏离核心目标。

## 核心观点：脚手架 vs 拐杖的边界

本文的核心判断是：**AI 工具作为外部执行功能层，其价值在于“会褪去”**。脚手架之所以是脚手架，是因为它最终会被拆除——使用者需要逐步内化那些技能，而不是永远依赖外部支撑。

但这也引出了争议：AI 工具是“外部执行功能层”还是“永久拐杖”？过度依赖可能导致能力退化（来源：矛盾与存疑）。目前缺乏长期追踪研究验证适度使用的边界。此外，ADHD 亚型（注意力不集中型 vs. 冲动型）对 AI 情绪调节工具的反应不同，现有研究样本量小，结论难以泛化（来源：AI 与 ADHD 的情绪调节）。

对于 LLM/Agent 侧，同样的问题：agent scaffolding 是否会让模型变得“懒惰”，不再发展内在的调度能力？目前的理论类比缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。

## 今天就能试的行动

1. **用 Mem 记录情绪触发点**：每次情绪波动时，快速记下触发事件、感受和想法。一周后回顾，识别模式。这相当于给情绪系统加一个外部日志，补偿工作记忆的缺失。
2. **设置“身体加倍”时段**：用 Focusmate 或找一位朋友，每天固定 25 分钟一起工作。利用社会存在感降低分心阈值，提升任务启动效率。
3. **将大任务分解为“辣度”子任务**：用 Goblin Tools 的 Magic ToDo 或类似工具，将一个模糊任务拆成 5-10 个小步骤。每完成一步打勾，获得即时多巴胺反馈。
4. **建立“词与剑”日课**：每天固定时间做一件输出性的事（写日记、画画、编程）和一件身体性的事（散步、拉伸、跳舞）。这既是情绪出口，也是行为锚点。

## 结语

ADHD 的情绪失调与 LLM/Agent 的调度缺失，本质上是同一个问题的两面：一个强大的生成核心，配了一个不匹配的调度层。AI 工具作为会褪去的脚手架，不是要取代人的内在能力，而是要在缺失的地方临时搭桥，直到桥自己站稳。

下次当你被情绪淹没时，记住：你不是“坏掉了”，你只是缺一个外部调度层。而 Mem、Goblin Tools、Focusmate——这些工具就是你的脚手架。用它们，然后学会在它们褪去后，依然能走稳。
