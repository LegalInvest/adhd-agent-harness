# Claude 之于 ADHD，是脚手架还是拐杖？

> 从「同构」视角评测 Claude：用久了会让 ADHD 更强还是更依赖？

## 问题：你是在用 Claude，还是在被 Claude 用？

ADHD 人群很容易陷入一个循环：打开 Claude，输入一个模糊任务，得到一份漂亮的大纲，然后——关掉窗口，什么也没做。或者更糟：让 Claude 替你写邮件、做计划、拆任务，久而久之，你发现自己连“先做什么”都懒得想了。

这不是你的错。ADHD 大脑的工作记忆容量小、边做边忘，而 Claude 这类 LLM 恰好也是“无状态”的——每次对话都像失忆重启。两者在结构上惊人地同构：都是强大的生成核心，但缺乏可靠的执行调度层（来源：ADHD 的 AI 工具全景）。

问题在于：Claude 到底是帮你搭建了可撤除的脚手架，还是给了你一根永远离不开的拐杖？

## 同构：ADHD 大脑与 LLM 的“无状态”困境

ADHD 大脑的典型体验是：脑子里有无数好点子，但一到执行就卡壳。计划写了半页纸，转头就忘；任务拆成三步，做到第二步就迷失方向。这不是懒惰，而是执行功能的缺陷——特别是工作记忆容量小，导致无法同时 hold 住目标、步骤和进度。

LLM 面临同样的困境。一个 Claude 会话，你给了它系统提示、上下文、工具定义，它一开始表现得像个天才。但聊到第 20 轮，它开始“忘记系统提示”，行为偏离初始指令（来源：重锚定与目标漂移）。这就是 LLM 版本的“目标漂移”——和 ADHD 患者做着做着跑去刷手机，本质上是同一回事。

解决方案也同构：外部记忆和重锚定。

ADHD 一侧，工具如 Goblin Tools 通过“魔法待办”将“整理房间”分解为“捡起衣服、擦桌子”等小步骤，降低启动门槛（来源：Goblin Tools）。Saner.AI 提供本地知识回忆，减少搜索循环（来源：Saner.AI）。Lex 允许单一指令触发多步骤任务，减少决策负担（来源：Lex）。这些工具本质上是在大脑外部搭建了一个“执行功能层”——帮你记住、帮你启动、帮你保持方向。

LLM/agent 一侧，工程师通过 harness 工程做同样的事：设计上下文交付、工具接口、规划工件和验证循环（来源：规划循环与任务分解）。事件驱动的系统提醒对抗“指令消退”（来源：重锚定与目标漂移）。向量数据库和 Mem 这类工具提供持久记忆，让 agent 跨会话保持上下文（来源：无状态与外部记忆）。

两边的解法一模一样：给生成核心套上一个外部调度层。

## 脚手架 vs 拐杖：边界在哪里？

脚手架是临时结构，建筑完成后撤除；拐杖是永久替代，拿掉就无法行走。

当前大多数 ADHD 工具被设计为长期使用，未提及撤除机制（来源：矛盾与存疑）。Goblin Tools 的分解功能用久了，你可能发现自己不再练习“自己拆任务”的能力。Saner.AI 替你记住一切，你的工作记忆可能更少被锻炼。Claude 帮你写邮件，你的表达组织能力可能退化。

但这不是工具的错。关键在于：你是否意识到自己在使用“外部执行功能层”，并主动设计使用方式？

真正的脚手架用法是：
- 用 Claude 拆解任务后，自己尝试复述步骤，而不是直接复制粘贴。
- 用 Saner.AI 记录想法后，定期回顾，训练自己的回忆能力。
- 用 Lex 触发流程后，观察 AI 的决策逻辑，下次尝试自己预判。

拐杖用法则是：
- 任何任务都先丢给 Claude，等它给出答案再执行。
- 离开工具就无法启动任何事。
- 工具一停，生产力归零。

## 争议与局限：证据还远远不够

必须诚实地说，目前关于 AI 工具对 ADHD 长期效果的研究几乎空白。现有证据多来自用户评价和综述文章，缺乏随机对照试验（来源：矛盾与存疑）。ADHD 大脑与 LLM 同构的命题，主要建立在概念类比和工具案例上，缺乏大规模实证（来源：矛盾与存疑）。

更复杂的是，ADHD 异质性极高。Goblin Tools 的分解功能对某些人有效，对另一些人可能反而增加认知负荷。Claude 的对话式交互对某些人是灵感源泉，对另一些人则是分心陷阱。

此外，多巴胺干预的有效性存在争议（来源：矛盾与存疑）。一些工具声称通过神经科学原理改善执行功能，但缺乏透明验证。

所以，Claude 之于 ADHD，既不是万能药，也不是洪水猛兽。它是一块材料——你可以用来搭脚手架，也可以用来做拐杖。

## 今天就能试的 3 件事

1. **刻意“断奶”**：用 Claude 拆解一个任务后，关掉窗口，凭记忆写出步骤。对比 Claude 的版本，看看自己漏了什么。这是训练工作记忆的微练习。

2. **设置使用边界**：规定自己每天最多 3 次将任务丢给 Claude 直接生成。其余时间，先用纸笔写下自己的想法，再用 Claude 补充。

3. **建立外部记忆系统**：用 Saner.AI 或 Mem 记录关键信息，但每周做一次“无工具回顾”——不搜索，先凭记忆复述。这能防止外部记忆完全替代内在能力。

Claude 可以是脚手架，也可以是拐杖。区别不在于工具本身，而在于你如何使用它——以及你是否记得，脚手架终有一天是要拆掉的。
