# ADHD 学生视角：为什么「治好 ADHD 的执行功能（计划/组织/启动）先天偏弱」和「让 LLM 不跑飞」其实是同一道工程题？

> agent 的编排/记忆/工具层就是「外部执行功能」——同一套 harness 思路，两边都成立。

### 从两个看似无关的崩溃说起

一个 ADHD 学生坐在书桌前，计划写一篇 2000 字的文章。打开文档，思绪如泉涌，但刚写两行，手机通知弹出一个新闻标题，点进去，半小时后发现自己正在浏览某个无关维基百科页面。回来时，文档还停留在那两行，而截止时间已经逼近。另一边，一位工程师在调试一个 LLM agent，它最初完美地遵循指令，但经过几轮工具调用后，开始遗忘系统提示，输出偏离目标，甚至陷入无限循环。两个场景，同一个核心问题：**生成核心（大脑或 LLM）产出极高，但缺乏可靠的执行调度层来引导输出**。

### 同构：ADHD 大脑与 LLM 的共享结构

当前最站得住脚的核心论点是：ADHD 大脑与 LLM 在结构上同构——都是“高产但缺乏可靠执行调度层的生成核心”（来源：ADHD × AI 的科学与研究前沿）。ADHD 一侧，默认模式网络过度活跃带来发散思维和创意，但前额叶执行控制不足，导致计划、组织、启动困难（来源：执行功能）。LLM 一侧，生成能力强大，但缺乏稳定的上下文控制器，长程任务中“开始忘记系统提示”（来源：重锚定与目标漂移），行为偏离初始指令。两者都需要外部调度层来引导输出。

这种同构不是空洞类比。神经科学证据显示，ADHD 的多巴胺功能失调导致动机和注意力缺陷，而 LLM 的奖励机制（如 RLHF）也面临类似的“动机”调节问题（来源：ADHD × AI 的科学与研究前沿）。认知实验中，ADHD 患者在需要执行控制的任务中表现不稳定，与 LLM 在长上下文任务中的性能退化模式相似（来源：ADHD × AI 的科学与研究前沿）。AI 工具的有效性进一步支持这一框架：基于外部执行功能层的工具（如任务管理、提醒系统）在 ADHD 用户中报告有效（来源：ADHD × AI 的科学与研究前沿）。

### 外部执行功能层：同一套 harness 思路

既然问题同构，解法也同构。ADHD 群体早已自发使用“外部执行功能层”：笔记系统、提醒、身体搭档（body doubling）。Focusmate 等工具通过 AI 匹配虚拟身体搭档，提供外部问责（来源：人在回路与身体在场）。LLM/agent 工程中，这被称为“harness”或“脚手架”：通过 human-in-the-loop 监督、护栏（token 预算、工具调用次数上限）、人工检查点来提供外部调度（来源：人在回路与身体在场）。其核心是“形式化规划与护栏，使代理输出真正有用且正确”（来源：人在回路与身体在场）。

具体工具层面，Goblin Tools 的 Magic ToDo 将复杂任务分解为可管理的小步骤（来源：Goblin Tools），相当于为 ADHD 大脑提供一个“任务分解 harness”。Saner.AI 通过增强知识回忆减少搜索循环（来源：Saner.AI），相当于为无状态的 LLM 提供外部记忆系统。Motion 自动排程并动态调整计划（来源：Motion），相当于为 agent 提供动态规划层。这些工具的设计逻辑与 agent harness 如出一辙：**在生成核心之外构建一个调度层，负责上下文维护、目标锚定、任务分解和启动**。

### 脚手架 vs 拐杖：必须划清的边界

然而，同一套思路也共享同一套风险。AI 工具作为外部执行功能层，可能成为永久拐杖而非脚手架。多个页面警告“过度依赖可能削弱内在时间感知能力”（来源：矛盾与存疑），且“缺乏如何避免依赖的具体指导”（来源：矛盾与存疑）。对于 ADHD 用户，如果工具完全替代了自身执行功能的锻炼，长期可能恶化内在能力。对于 LLM/agent，如果 harness 过于僵化，可能限制模型本身的适应性和创造力。关键在于，外部层应设计为“补偿”而非“替代”，逐步培养用户或模型的内在调度能力。但目前纵向研究缺乏，这一边界仍属理论推演（来源：ADHD × AI 的科学与研究前沿）。

### 争议与局限

同构视角虽具解释力，但需诚实承认局限。首先，ADHD 的异质性（注意力主导型 vs. 冲动主导型）与 LLM 的架构差异（不同模型）是否影响同构的普适性？目前研究样本有限（来源：ADHD × AI 的科学与研究前沿）。其次，LLM 的“注意力机制”与 ADHD 的注意控制缺陷是否真正同构？目前仅是功能类比，缺乏神经生物学层面的直接证据（来源：ADHD × AI 的科学与研究前沿）。最后，AI 工具的有效性证据主要来自用户报告，缺乏严格对照试验（来源：矛盾与存疑）。工具如 Brain.fm 被归类为 ADHD 辅助，但其效果缺乏独立临床研究（来源：矛盾与存疑）。

### 今天就能试的行动

1. **任务分解实验**：下次面对复杂任务时，用 Goblin Tools 的 Magic ToDo 或 ChatGPT 将任务分解为 5-10 个小步骤，每完成一步打勾。这相当于为大脑加装“任务启动 harness”。
2. **外部记忆系统**：用 Saner.AI 或类似工具记录所有想法和待办，减少“搜索循环”。相当于为无状态大脑提供向量数据库。
3. **动态规划测试**：用 Motion 或类似工具自动排程一天的任务，观察是否减少“下一步做什么”的决策疲劳。相当于为 agent 加装动态调度层。
4. **身体搭档模拟**：尝试 Focusmate 或找一位朋友一起工作，利用“身体在场效应”提升执行力。相当于 human-in-the-loop 监督。

### 结语

ADHD 学生与 LLM 工程师面对的其实是同一道工程题：如何为一个高产但不可靠的生成核心构建可靠的外部执行调度层。理解这种同构，不仅能让 ADHD 群体更理性地选择工具，也能让 agent 工程师从人类认知补偿策略中汲取灵感。但别忘了，脚手架是为了最终能拆除，而不是永远依赖。
