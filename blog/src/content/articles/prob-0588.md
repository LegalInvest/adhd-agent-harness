# 用 ChatGPT 给 ADHD 补上采样温度与表现波动，真的能像 agent 那样工作吗？

> 把 ChatGPT 当作 ADHD 的外部执行层来实测：它补的是哪一块短板？

## 问题：为什么你的状态像 LLM 的采样温度一样不可控？

你打开 ChatGPT，想写一篇报告。前几分钟思路如泉涌，输出流畅；十分钟后，你开始刷 Twitter，接着点开购物网站，最后关掉文档——什么都没完成。这种“状态日内大幅波动”是 ADHD 人群的日常：你拥有强大的生成能力，但执行功能像 LLM 的采样温度一样随机，高时爆发，低时瘫痪。

工程师们熟悉这个困境：LLM 在高温采样下输出多样但不可靠，低温则重复乏味。为了让 LLM 稳定产出，他们搭建 agent scaffolding——用结构约束（如任务分解、状态机、记忆回放）来驯服随机性。那么，把 ChatGPT 当作 ADHD 的外部执行层，真的能补上这块短板吗？

## 同构：ADHD 大脑与 LLM 是同一类“高产但缺调度层”的生成核心

最新研究揭示了惊人的结构同构。明尼苏达大学测试 LLM 的执行功能，发现“强记忆、弱控制”剖面：工作记忆容量甚至超过常人，但认知灵活性与注意控制存在核心缺陷——无法灵活切换任务、无法抑制自动化反应（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。这正是 ADHD 的经典神经心理模式：你记得住细节，却管不住注意力。

耶鲁大学进一步发现，自注意力机制本身导致工作记忆容量限制：随任务复杂度增加，注意力分数熵增、注意力弥散，与人类注意分散机制同源（来源：Self-Attention Limits Working Memory Capacity of Transformer-Based Models）。换句话说，ADHD 大脑和 LLM 共享同一个瓶颈：不是容量/算力不足，而是 orchestration 层不稳定。

这解释了为什么 ADHD 人群需要外部脚手架。孔子周游列国 14 年，精力旺盛但坐不住，他用“克己复礼”作为行为边界，每日反省——这本质上是一个外部调度系统：用“礼”的结构来约束“仁”的冲动。曾国藩 30 岁前浮躁无恒，日记里天天骂自己“无恒”“浮躁”，后来靠“日课十二条”（黎明即起、读书不二、谨言等）建立刚性流程，用最笨最稳的方法抵消冲动。他们的 harness 系统，与 LLM agent 的 scaffolding（定时 re-grounding、任务分解、状态机）结构同构：都是把“该做什么、现在做哪一步”从大脑内部搬到外部系统。

## 实测：ChatGPT 补的是调度层，不是温度

回到问题：用 ChatGPT 给 ADHD 补上采样温度与表现波动，真的能像 agent 那样工作吗？

答案是：能，但前提是你把它当作外部执行功能层，而不是情绪调节器。

工具如 Routinery 通过可视化倒计时和过渡提示，把流程从大脑内部搬到外部（来源：Routinery）。Goblin Tools 的 Magic ToDo 将模糊任务分解为可执行步骤，降低启动门槛（来源：Goblin Tools）。Saner.AI 则用本地记忆减少搜索循环和标签切换（来源：Saner.AI）。这些工具的共同逻辑是：不替用户思考，而是用结构约束来稳定输出。

ChatGPT 可以扮演类似的角色。例如，你可以让它把“写季度报告”分解为“列出大纲 → 写引言 → 分析数据 → 总结”，并为每一步设定时间限制。这相当于 agent 中的任务分解和状态机——用外部结构驯服生成核心的随机性。但关键区别在于：ChatGPT 本身没有内置调度层，它需要你手动搭建 scaffolding。如果你只是问“我该怎么办”，它可能给你一堆建议，但不会帮你执行。

## 脚手架 vs 拐杖：边界在哪里？

这里有一个诚实争议：过度依赖 AI 工具是否会导致能力退化，成为永久拐杖？（来源：矛盾与存疑）目前缺乏纵向研究，但基于同构视角，我们可以划定一条边界：脚手架是增强你的调度层，拐杖是替代你的生成核心。

当你用 ChatGPT 分解任务、设定计时、记录进度时，你是在训练自己的执行功能——就像孔子的“每日反省”和曾国藩的“日课”，它们最终内化为习惯。但如果你让 ChatGPT 替你写完整篇文章、替你做所有决策，你的生成核心（创造力、判断力）就会萎缩。

## 今天就能试的 3 个行动

1. **用 ChatGPT 做任务分解**：输入一个模糊任务（如“清理书房”），要求它按“辣度”分解为 5-10 个步骤。然后逐项执行，每完成一项就标记。这模仿了 Goblin Tools 的逻辑，但更灵活。
2. **设定“外部调度器”**：让 ChatGPT 每隔 25 分钟提醒你回到当前任务。你可以说：“每 25 分钟问我一次‘你还在做 X 吗？’”。这相当于 agent 的定时 re-grounding。
3. **建立“日课”模板**：模仿曾国藩的日课十二条，让 ChatGPT 每天早上列出 3 件优先事项，并在晚上回顾完成度。这训练了你的调度层，而不是替代它。

## 结论

ChatGPT 不能直接补上 ADHD 的采样温度波动，但它可以作为外部调度层，像 agent scaffolding 一样约束你的生成核心。关键不在于工具本身，而在于你如何搭建脚手架。记住：你不需要一个更聪明的 AI，你需要一个更稳定的执行系统。
