# ADHD 家长如何用「给生成核心套 harness」的思路应对陪伴 ADHD 孩子里的工作记忆容量小、边做边忘？

> 把 LLM 工程里的「LLM 无跨会话状态，靠外部记忆/向量库续命」迁移到ADHD 家长的陪伴 ADHD 孩子场景，双域证据作答。

## 问题：你的孩子边做边忘，你的大脑也在边做边忘

你正在陪孩子写作业，刚讲完一道数学题，转身去倒杯水，回来发现孩子已经对着下一题发呆，完全忘了刚才的思路。你深吸一口气，重新讲一遍，但自己也开始忘——刚才讲到哪里了？孩子的作业进度卡在哪？你明明记得有个重要的提醒事项，现在却怎么也想不起来。

这不是你的错，也不是孩子的错。ADHD 大脑的工作记忆就像一台没有持久化存储的服务器：每次会话都是新的，上下文随时可能丢失。

如果你恰好也熟悉 LLM 工程，你会立刻意识到——这不就是无状态 API 调用吗？LLM 每次对话独立，不保留跨会话记忆，全靠外部向量库和 prompt 工程来维持上下文。ADHD 大脑与 LLM 是同一类“高产但缺执行调度层的生成核心”（来源：ADHD 的 AI 工具全景）。

## 同构的解法：给生成核心套上 harness

无论是 ADHD 大脑还是 LLM，核心矛盾都是：生成能力强，但调度、记忆、任务启动能力弱。LLM 工程中，解决方法是给无状态的 LLM 套一个 harness——外部记忆（向量数据库）、任务分解（agent 工作流）、上下文管理（prompt 模板）。ADHD 场景中，对应的就是外部工具作为“执行功能层”来补偿工作记忆、任务启动和时间感知缺陷。

### 无状态与外部记忆：从跨会话遗忘到持久上下文

LLM 的标准调用是无状态的：每次对话都是独立的，模型不保留跨会话记忆（来源：无状态与外部记忆）。生产级 agent 系统通过外部记忆（如向量数据库）解决溢出问题，组合使用会话内上下文、长期检索向量库和结构化知识库（来源：无状态与外部记忆）。

ADHD 大脑同样如此：工作记忆容量有限且不稳定，跨会话任务难以持续，每天需要重新回忆项目上下文（来源：无状态与外部记忆）。工具如 **Mem** 和 **Saner.AI** 提供“AI 第二大脑”，通过持久记忆和自动回忆减少重复输入上下文的摩擦（来源：ADHD 的 AI 工具全景）。Saner.AI 专注于知识回忆和本地记忆，帮助用户快速找回信息，减少搜索循环和标签切换（来源：Saner.AI）。

**家长行动**：用 Saner.AI 或 Mem 记录孩子的学习进度、作业难点和当日提醒，每次打开时自动呈现，不再靠大脑硬扛。

### 任务分解：从压倒性到可执行

ADHD 孩子面对“整理房间”或“完成数学作业”这类大任务时，启动困难源于工作记忆无法同时容纳所有子步骤。LLM 同样需要分解：没有 harness 的 LLM 只是一个无状态的 API 调用（来源：无状态与外部记忆），而 **Goblin Tools** 的 Magic ToDo 功能能自动将任何任务分解为更小、更易管理的步骤（来源：Goblin Tools）。它被评价为“将压倒性的事情变成一系列不压倒性的事情”（来源：Goblin Tools）。类似地，**Lex** 允许通过单一指令触发复杂、多步骤的任务序列，降低决策疲劳（来源：Lex）。

**家长行动**：用 Goblin Tools 将“完成作业”分解为“拿出课本→翻到第10页→读题→列算式→计算→检查”，每完成一步打勾，孩子获得即时反馈。

### 上下文工程：主动设计“当下注意什么”

ADHD 大脑易被环境带偏，LLM 易上下文膨胀。主动设计“当下注意什么”是关键。**Reflect** 的结构化反思和 **Otter.ai** 的实时转录，提供了即时外部反馈，类似于“身体在场效应”的虚拟版本（来源：ADHD 的 AI 工具全景）。

**家长行动**：陪孩子学习时，用 Otter.ai 录音并自动生成文字摘要，结束后和孩子一起回顾“我们刚才做了什么”，强化记忆锚点。

## 脚手架 vs 拐杖：核心判断

AI 工具应作为脚手架促进能力发展，而非永久拐杖（来源：拐杖与脚手架）。真正的脚手架应帮助使用者“建造”，但使用者仍需自己“攀登”（来源：拐杖与脚手架）。例如，Otter.ai 减轻笔记负担，但过度依赖可能削弱主动记笔记的能力（来源：ADHD 的 AI 工具全景）。

**我的核心判断是**：ADHD 家长使用 AI 工具时，必须刻意设计“退出机制”——比如先用 Goblin Tools 分解任务，但逐步让孩子自己尝试分解；先用 Saner.AI 记录进度，但慢慢训练孩子自己回忆。工具是临时支架，不是永久支撑。

## 诚实面对局限

现有证据主要来自用户报告和概念类比，缺乏大规模对照实验（来源：矛盾与存疑）。过度依赖 AI 工具可能削弱内在能力（来源：矛盾与存疑）。个体差异导致效果不一，同一工具对某些孩子是福音，对另一些可能无效甚至增加焦虑（来源：矛盾与存疑）。

因此，不要迷信任何单一工具。把 AI 看作实验——试一周，观察效果，调整或放弃。

## 今天就能试的 3 条行动

1. **安装 Goblin Tools**：今晚就把“整理书包”分解成5个小步骤，让孩子逐项完成。
2. **用 Saner.AI 建一个“孩子学习档案”**：记录今日作业、难点、明日提醒，每天打开自动回顾。
3. **陪孩子用 Otter.ai 录一次学习过程**：结束后一起看文字摘要，问“我们刚才做了什么？”，强化记忆。

记住：你不是在给孩子装一个“大脑外挂”，而是在帮他搭建脚手架。总有一天，他要自己爬上去。
