# ADHD 创业者视角：为什么「治好 ADHD 的怕过度依赖工具、又怕没有支撑就崩」和「让 LLM 不跑飞」其实是同一道工程题？

> harness 应是会褪去的脚手架，而非永久拐杖——同一套 harness 思路，两边都成立。

## 问题：同一个恐惧，两种面孔

如果你是 ADHD 创业者，你一定熟悉这个循环：找到一个工具（比如 Goblin Tools 帮你分解任务，Tiimo 帮你可视化时间），效率飙升，但紧接着恐惧袭来——"我是不是太依赖它了？没了它我还能正常运转吗？" 于是你刻意戒断，结果再次崩盘。

如果你是 Agentic Harness 工程师，你也熟悉这个循环：给 LLM 加一层编排层（ReAct、Plan-and-Execute），让它不跑飞，但紧接着担心——"这层 harness 会不会让模型失去自主性？撤掉之后它会不会原地崩溃？"

两群人，同一个恐惧：
- **怕过度依赖工具**：怕 harness 变成永久拐杖，撤掉后能力归零。
- **怕没有支撑就崩**：怕撤掉 harness 后，生成核心（ADHD 大脑或 LLM）立刻跑飞。

这个恐惧不是心理问题，而是工程问题。

## 同构：生成核心与调度层的分离

ADHD 大脑与 LLM 共享同一个底层架构：一个强大的生成核心（ADHD 的智力/创造力，LLM 的算力/知识库），配上一个不可靠的调度层（ADHD 的执行功能，LLM 的上下文窗口与编排）。（来源：ADHD 大脑与 LLM 的同构）

证据在两边都很清晰：
- **ADHD 侧**：工作记忆缺陷导致情绪信息难以维持，任务启动困难引发拖延和自责，时间盲导致时间压力感失调——这些都是调度层故障。（来源：工作记忆、任务启动、时间盲）
- **LLM 侧**：长上下文丢失焦点，多步推理中遗忘中间结果，工具调用顺序混乱——同样是调度层故障。

两边的解法因此同构：**提供外部调度层——harness/脚手架——来补偿生成核心的弱点**。

## 脚手架 vs 拐杖：边界在哪？

核心判断：**脚手架是可撤除的临时结构，拐杖是永久替代。**

- **脚手架**：在建造阶段提供支撑，完成后可拆除，建筑本身获得独立站立的能力。
- **拐杖**：永久替代受损的腿，撤掉后无法行走。

在 ADHD 侧，好的工具应该是脚手架。例如 Goblin Tools 将“整理房间”分解为“捡起地板上的衣服”“擦桌子”等步骤（来源：Goblin Tools）。这个分解过程可以逐步内化——用户学会自己分解任务后，可以不再依赖工具。而 Saner.AI 通过本地记忆减少搜索循环（来源：Saner.AI），如果用户依赖它记住所有信息而不训练自己的工作记忆，它就变成了拐杖。

在 LLM 侧，好的 harness 应该是脚手架。例如，一个 ReAct 模式在推理时逐步调用工具，但最终目标是让模型学会在单次推理中完成多步规划。如果 harness 永远不撤，模型就永远无法独立。

边界在于：**设计时是否包含撤除路径**。

## 争议与局限

当前多数工具并未设计撤除机制。「拐杖与脚手架」概念强调脚手架应可逐步撤除，但多数工具如 Goblin Tools、Saner.AI 设计为长期使用，未提及撤除路径（来源：矛盾与存疑）。这导致过度依赖风险真实存在。

此外，ADHD 大脑与 LLM 同构命题目前主要基于概念类比和工具案例，缺乏大规模实证（来源：矛盾与存疑）。情绪调节干预的有效性也需更多验证（来源：AI 与 ADHD 的情绪调节）。

## 同一道工程题的解法

既然问题是同构的，解法也可以互相借鉴：

1. **显式设计撤除阶段**：像训练 LLM 时逐渐减少 prompt 中的示例一样，ADHD 工具应逐步减少自动分解的粒度，让用户自己完成更多步骤。

2. **用时间盲设计对抗时间盲**：Tiimo 的视觉时间表（来源：时间盲）可以同时用于 LLM——给 Agent 一个可视化的进度条，让它知道还剩多少步骤。

3. **身体在场效应的虚拟化**：Focusmate 的 AI 匹配身体加倍（来源：身体在场效应）对 ADHD 有效，对 LLM 呢？也许可以设计一个“观察者 Agent”在后台默默记录，但不干预，只提供隐性问责。

## 今天就能试的行动

1. **检查你的工具**：列出你依赖的所有 ADHD 工具，对每个工具问：如果明天它消失了，我还能正常运转吗？如果答案是“不能”，说明它可能是拐杖。

2. **设计一个撤除实验**：选一个工具（比如 Goblin Tools），下周只在一半的任务中使用它，另一半任务尝试手动分解。记录差异。

3. **为你的 Agent 加一个“脚手架日志”**：在 harness 中记录每一步的决策，但逐渐减少日志的详细度，直到 Agent 能自主决策。

4. **试验身体在场效应**：用 Focusmate 或类似工具，每天 15 分钟，观察专注度变化。同时，为你的 LLM Agent 加一个“旁观者 Agent”，只记录不干预，观察效果。

## 结语

ADHD 创业者与 Agent 工程师面对的是同一个恐惧。但恐惧本身不是敌人——它提醒我们：harness 应是会褪去的脚手架，而非永久拐杖。设计可撤除的支撑，才是真正的自由。
