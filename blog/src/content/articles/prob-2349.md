# ADHD 学生如何用「给生成核心套 harness」的思路应对考试与作业里的任务启动困难、不会拆解？

> 把 LLM 工程里的「agent 用 planner-executor 循环分解长任务」迁移到ADHD 学生的考试与作业场景，双域证据作答。

## 问题：为什么你明明有能力，却总卡在第一步？

你盯着空白的文档、堆积的试卷、或一个模糊的“复习计划”，大脑像死机一样。不是不想做，而是不知道从哪里开始——任务太大、太模糊，启动需要消耗的意志力远超你能调动的。ADHD 学生对此再熟悉不过：**任务启动困难**和**不会拆解**是执行功能障碍的两座大山（来源：执行功能障碍）。与此同时，在 AI 工程界，工程师们也在面对类似的困境：给 LLM 一个复杂的长任务，它要么跑偏，要么中途崩溃，无法可靠地完成多步推理。两边看似毫不相干，但问题的结构却惊人地一致。

## 答案：给生成核心套上 harness

本文的核心观点是：ADHD 大脑与 LLM 都是**高产但缺执行调度层的生成核心**。ADHD 大脑能爆发超专注（ADHD 组超专注时长 14.5 小时 vs 对照组 8.2 小时，来源：Pace Ventures Research 2026），LLM 能生成惊艳的文本，但两者都缺乏可靠的内置调度层来规划、分解和监控长任务。因此，为 LLM 搭建脚手架（如 agent 框架、提示工程）与为 ADHD 搭建执行功能支架是同一类工程：**都是给生成核心套上 harness**（来源：AI 与 ADHD 的时间管理）。这个 harness 的核心是一套 planner-executor 循环：先由 planner 将大任务分解成可执行的小步骤，再由 executor 逐一执行，并在循环中不断调整。

## 历史人物的 harness：孔子与曾国藩

孔子和曾国藩是 harness 系统的典范。孔子 ADHD 特质明显：精力旺盛、坐不住、冲动爱骂人、思维跳跃（《论语》全是场景化语录）。他的 harness 是“克己复礼”：用外在的“礼”作为行为边界，每日反省，“吾日三省吾身”。这相当于一个**外部调度器**——用固定的仪式和规则来约束冲动的生成核心。曾国藩则更极端：30 岁前浮躁坐不住，日记里天天骂自己“无恒”。他的 harness 是“日课十二条”和“结硬寨打呆仗”。日课十二条是**定时 re-grounding** 机制，每天固定时间做固定的事，类似 LLM 的 checkpoint 和 grounding 步骤；“结硬寨打呆仗”则是用最笨最稳的方法分解任务，避免冲动决策。这两个系统与 LLM agent 的 planner-executor 循环同构：**日课 ↔ 定时 re-grounding，礼 ↔ 外部约束，打呆仗 ↔ 稳健的任务分解**。

## 现代工具的 harness：从 Motion 到 Goblin Tools

今天的 AI 工具正在将这种 harness 产品化。**Motion** 通过自动排程消除“下一步该做什么”的决策负担，直接帮助用户克服时间盲和任务启动困难（来源：Motion）。它的 AI 核心是自动排程与动态调整：持续评估任务优先级、截止日期和可用时间，实时重建日程。这相当于一个**外部 planner**，替你完成分解和排序。**Reclaim.ai** 则更强调“保护”时间，通过创建深度工作智能块，自动防御会议侵占——这是**外部调度器的防御模式**，防止外部干扰打乱执行循环（来源：Reclaim.ai）。**Tiimo** 用视觉化时间线让时间流逝变得可感，直接应对时间盲问题（来源：Tiimo）。**Goblin Tools** 则聚焦于任务分解：它能将“写论文”自动拆成“找资料、列提纲、写初稿、修改”等子步骤，这是**planner 的分解功能**的具象化。这些工具共同构成了一个“外部大脑”，帮助用户从混乱中建立秩序（来源：AI 与 ADHD 的时间管理）。

## 争议与局限：脚手架还是拐杖？

然而，这种 harness 并非万能。最大的争议是**过度依赖风险**：AI 工具可能从“脚手架”退化为“永久拐杖”，削弱用户内在执行功能的代偿发展（来源：矛盾与存疑）。目前缺乏长期研究评估这种依赖的后果。此外，多数工具缺乏独立临床验证（如 Motion），其效果可能受安慰剂效应影响（来源：矛盾与存疑）。个体差异也很大：某些用户对视觉化工具（如 Tiimo）反应良好，而另一些则更受益于自动排程（如 Reclaim.ai），但缺乏个性化匹配的指导原则（来源：AI 与 ADHD 的时间管理）。成本问题同样突出：多数 AI 工具采用订阅制，可能加剧数字鸿沟（来源：AI 与 ADHD 的时间管理）。

## 今天就能试的 3 条行动

1. **用 Goblin Tools 分解一个模糊任务**：把“复习期末考试”或“写论文”输入 Goblin Tools，让它自动拆成子步骤，然后挑第一个子步骤开始做。
2. **用 Motion 或 Reclaim.ai 自动排程一天**：将你的待办事项输入，让 AI 替你规划时间，观察启动是否变容易。如果不想付费，可以用 Todoist 手动创建时间块，模仿外部 planner。
3. **模仿曾国藩的“日课”**：每天固定 3 件事（如早起、专注 25 分钟、写日记），用手机定时提醒，坚持一周。这不是靠意志力，而是靠外部 re-grounding 机制。

## 结语：你不需要修复大脑，只需要给它一个 harness

ADHD 大脑不是坏的，它是高产的生成核心——缺的只是一个调度层。无论是孔子的“礼”、曾国藩的“日课”，还是 Motion 的自动排程，本质上都是给核心套上 harness。LLM 工程的经验告诉我们：不要试图让生成核心变“乖”，而是设计一个可靠的 planner-executor 循环来驾驭它。对 ADHD 学生来说，这意味着：**你不是懒，也不是能力不足，你只是需要一个外部调度器**。今天就开始，给自己套上 harness。
