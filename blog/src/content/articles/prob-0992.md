# 如果把 ADHD 大脑当成生成核心，Lex 能不能当它的重锚定与目标漂移？

> 用 agent 工程的思路重新理解 Lex 对 ADHD 的真正价值。

你打开 Lex，准备写一篇深度文章。十分钟后，你发现自己正在研究 Lex 的字体设计，又过了十分钟，你在读一篇关于宋代官制的论文。你的大脑像一台没有 system prompt 约束的 LLM：生成能力极强，但上下文窗口被无关 token 填满，目标早已漂移。

这并非巧合。ADHD 大脑与 LLM 在架构上存在深层同构：两者都是**高产但缺乏可靠执行调度层的生成核心**（来源：AI 与 ADHD 的专注力）。ADHD 的创意发散与 LLM 的文本生成同源，而执行功能缺陷与 LLM 的上下文控制缺失对应。当 LLM 在长程任务中发生目标漂移时，工程师会重锚定 system prompt；当 ADHD 大脑在写作中走神时，它也需要一个外部重锚定系统。

## 重锚定：从曾国藩的日课到 LLM 的 system prompt

曾国藩 30 岁前是典型的注意力不集中型 ADHD：日记里天天骂自己“无恒”“浮躁”，读书“他人目下二三行，余或疾读不能终一行”。他的解法是**日课十二条**——黎明即起、读书不二、谨言、写日记反省。这套系统本质上是一个外部调度层：每天固定时间做固定事，用最笨最稳的方法抵消冲动。

这与 LLM agent 工程中的“定时 re-grounding”结构同构。在长程 agent 任务中，工程师会在每 N 步后插入一个重锚定步骤，将当前上下文与原始目标对齐。曾国藩的“读书不二”就是他的重锚定 system prompt：一本书没读完，绝不碰第二本。

现代工具也在做类似的事。Focusmate 通过实时视频配对，提供外部问责——这相当于一个“人类 system prompt”，每 25 分钟强制你汇报进度（来源：Focusmate）。RescueTime 自动记录时间使用，揭示注意力漂移路径，相当于一个“日志审计系统”，让你看到自己的 token 流去了哪里（来源：RescueTime）。

## Lex 的独特位置：重锚定而不打断超聚焦

但 ADHD 面临一个矛盾：超聚焦状态一旦进入，强行打断可能适得其反。部分用户反馈番茄钟的强制打断反而有效，而柔性引导可能不够强力（来源：矛盾与存疑）。Lex 的独特价值在于它不打断你的流，而是提供一种**轻量级的上下文锚点**。

当你用 Lex 写作时，它的极简界面和实时预览就像一条隐形的 system prompt：“写你的文章，不要研究字体。”它不限制你查阅资料，但通过将注意力锚定在文本输出上，减少上下文膨胀。这与 Brain.fm 的神经锁相音乐异曲同工——后者通过音频模式影响大脑信息处理方式，帮助进入专注状态（来源：Brain.fm），但 Lex 更直接：它本身就是你当前的“注意力容器”。

## 脚手架，不是拐杖

这里必须诚实指出争议：AI 工具作为外部执行功能层，长期使用是否会导致内在能力退化？拐杖与脚手架的边界尚不明确（来源：矛盾与存疑）。曾国藩的日课是脚手架——他通过刻意练习内化了自我管理能力；而依赖 Focusmate 实时配对，如果从不尝试独立启动，就可能变成拐杖。

明尼苏达大学的研究发现，LLM 表现出“强记忆、弱控制”的执行功能剖面——工作记忆容量甚至超过常人，但注意控制存在核心缺陷（来源：工作记忆）。这正是 ADHD 的经典模式。对 LLM，我们不会因为它的工作记忆强就放弃 system prompt 工程；对 ADHD，我们也不应该因为大脑生成能力强就拒绝外部重锚定。关键是用脚手架来增强调度层，而不是用拐杖替代执行功能。

## 今天就能试的行动

1. **在 Lex 中设置“重锚定提示”**：每次打开 Lex，在文档顶部写一句你的核心目标（如“写一篇关于 Lex 与 ADHD 的文章”），写偏了就回来读一遍——这就是你的 system prompt。

2. **用 RescueTime 做一次“日志审计”**：开启自动追踪一周，看看你的 token 流去了哪里。识别出最常导致目标漂移的网站，设置主动阻断（来源：RescueTime）。

3. **尝试 Focusmate 的“定时重锚定”**：预约一个 25 分钟时段，结束时向伙伴汇报进度。这相当于给大脑加一个外部 re-grounding 步骤（来源：Focusmate）。

4. **写“日课”而不是待办清单**：像曾国藩一样，每天固定 3 件最重要的事，做完再碰其他。这不是计划，是重锚定仪式。

## 局限与边界

ADHD 大脑与 LLM 的同构终究是隐喻而非严格等价。人类的注意力涉及情绪、多巴胺、身体状态，远比 transformer 的注意力机制复杂（来源：矛盾与存疑）。Lex 不是灵丹妙药——它无法解决多巴胺失调或时间盲，但作为重锚定工具，它恰好对齐了 ADHD 与 LLM 共享的那个核心问题：生成能力不缺，缺的是把注意力锁在目标上的调度层。
