# Brain.fm 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 Brain.fm：它到底补上了哪一层执行功能？

你打开 Brain.fm，戴上耳机，音乐响起。前 5 分钟，你感觉注意力被温柔地拉回屏幕。但 20 分钟后，你发现自己正在刷手机——音乐还在播，大脑却早已漂移。

这不是 Brain.fm 的错。是你用错了。

## 同构：ADHD 大脑与 LLM 共享一个核心缺陷

ADHD 大脑与 LLM 在结构上高度同构：两者都是“高产但缺执行调度层的生成核心”（来源：ADHD 大脑与 LLM 的同构）。ADHD 侧，工作记忆容量甚至可能超过常人，但认知灵活性与注意控制存在核心缺陷——无法灵活切换任务集、无法抑制自动化反应（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。LLM 侧，用经典 Stroop 冲突任务测试 Transformer 注意力，发现随序列变长，模型在冲突试次上骤降到随机水平——无法抑制优势反应、无法解决认知冲突，这与 ADHD 执行功能缺陷的核心标志一一对应（来源同上）。

两者都需要外部脚手架来补偿调度缺陷。对 LLM 而言，是“agent scaffolding”——围绕 LLM 构建的软件架构和工具，使其能够执行复杂、目标驱动的任务（来源：Agent Scaffolding: Architecture and Design Patterns for Agentic AI）。对 ADHD 大脑而言，则是外部执行功能层：通过补偿工作记忆不稳定、任务启动困难和注意力分散，帮助用户实现目标导向行为（来源：AI 与 ADHD 的专注力）。

## Brain.fm 补上了哪一层？

Brain.fm 利用 AI 生成音乐，通过神经锁相技术影响大脑信息处理方式，帮助用户进入专注状态（来源：Brain.fm）。在 2026 年一项对 44 款 ADHD 应用的测试中，它被列为最佳 9 款之一（来源：ADHD Apps: We tested 44 Apps and Here're the Best 9 in 2026）。

但它的作用仅限于“状态调节”——就像 LLM 的 system prompt 中的“你是一个专注的助手”指令。它告诉你应该专注，却无法告诉你专注什么、如何保持、何时切换。它补上了“进入专注状态”这一层，却无法阻止目标漂移。

这正是“有人用错了”的地方：把 Brain.fm 当成完整的解决方案，就像只给 LLM 一个 system prompt 却不提供任何工具和记忆。

## 真实人物的 harness：曾国藩与刘晶

曾国藩是典型的注意力不集中型 ADHD：30 岁前浮躁坐不住，天天串门喝酒看杀人，日记里天天骂自己“无恒”“浮躁”；读书慢，“他人目下二三行，余或疾读不能终一行”。他的 harness 是“日课十二条”：黎明即起、读书不二、谨言、写日记反省等。这十二条本质上是外部调度系统——每天固定时间执行固定动作，用最笨最稳的方法抵消冲动。这对应 LLM harness 中的“定时 re-grounding”：每 N 步重置上下文，重新明确目标。

ADHD 创业者刘晶（General Partner, Kleiner Perkins Labs, China）则用 systems thinking 构建自己的 harness：高能量、创造力、同理心，但需要外部结构来管理。她的投资组合包含 56+ 家公司，公司增长速度 ADHD 组是对照组的 1.6 倍（来源：Internal Analysis）。她的 harness 可能包括秘书、日程系统、决策框架——对应 LLM harness 中的“外部调度器”和“工具调用”。

## 脚手架 vs 拐杖：边界在哪里？

AI 工具既可能成为永久依赖的拐杖，也可能成为逐步内化的脚手架（来源：拐杖与脚手架）。关键在于：是否在搭建完整的 harness，而不仅仅依赖单一工具。

Brain.fm 是脚手架的一部分，但不是全部。完整的 ADHD harness 需要：
- **状态调节**（如 Brain.fm）
- **任务启动**（如 Focusmate 的虚拟身体在场，通过实时同伴问责提供外部结构和社会压力，来源：Focusmate）
- **时间追踪**（如 RescueTime 自动记录时间去向，补偿时间盲与工作记忆负担，来源：RescueTime）

这三者共同构成外部执行功能层，就像 LLM harness 需要：
- **system prompt**（状态指令）
- **工具调用**（执行动作）
- **记忆系统**（上下文管理）

## 争议与局限

必须诚实指出：Brain.fm 在 ADHD 人群中的独立临床证据仍有限（来源：Brain.fm）。同构性说法目前仅是一种理论类比，并非经过验证的科学事实（来源：矛盾与存疑）。依赖风险与益处并存：过度依赖外部工具可能阻碍内化执行功能，但缺乏长期追踪研究（来源：AI 与 ADHD 的专注力）。

## 今天就能试的行动

1. **组合使用**：用 Brain.fm 进入状态后，立刻打开 RescueTime 追踪时间，并预约一个 Focusmate 时段。
2. **写“日课”**：像曾国藩一样，列出 3-5 条每日固定动作（如“9:00 打开 Brain.fm + 写今日三件事”）。
3. **定期 re-grounding**：每 25 分钟（一个番茄钟）暂停，问自己“我现在在做什么？目标是什么？”——就像给 LLM 重发 system prompt。
4. **警惕单一依赖**：如果发现离开 Brain.fm 就无法工作，说明它已变成拐杖。尝试每周一天不用，用其他方法（如白噪音、身体加倍）替代。

ADHD 大脑不是缺陷，而是一种替代架构。问题是，你给它装上了正确的 harness 吗？
