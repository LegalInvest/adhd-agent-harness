# ADHD 想把执行功能（计划/组织/启动）先天偏弱外包给 ChatGPT，该怎么设计这套「人 + AI」的回路？

> 把 ChatGPT 接进 ADHD 工作流：哪些该交给它，哪些必须自己留着？

## 问题：当大脑的“执行调度层”先天偏弱

你有没有过这样的体验：脑子里同时冒出十个好点子，但就是无法启动任何一个任务；日程表上写满了截止日期，却总在最后一刻才恐慌地开始；明明知道该做什么，身体却像被钉在椅子上，手指划着手机屏幕，心里骂着自己“快动啊”。这不是懒，不是意志力差——这是ADHD大脑的执行功能缺陷：计划、组织、启动能力先天偏弱。工作记忆容量有限且不稳定，时间感知扭曲（时间盲），任务启动像推一堵墙（来源：[[执行功能]]、[[工作记忆]]、[[时间盲]]、[[任务启动]]）。

但有趣的是，这种大脑并非一无是处。ADHD个体往往拥有极强的发散思维、超聚焦能力和创造力——他们的大脑是一个“高产但缺执行调度层的生成核心”。这听起来像什么？像一个大语言模型（LLM）：能生成惊艳的文本，但无状态、缺乏内置调度层，需要agent scaffolding（外部执行功能层）来编排、记忆、调用工具。

## 同构：ADHD大脑与LLM是同一类“生成核心”

这不是比喻。最新研究揭示了ADHD大脑与LLM在认知机制上的深层平行：
- **工作记忆干扰**：ADHD患者在工作记忆任务中易受干扰，而LLM同样表现出类似人类的干扰特征——随着记忆负荷增加表现下降，受近因效应影响，成功回忆依赖于主动抑制无关内容（来源：Human-like Working Memory Interference in Large Language Models）。
- **自主组织困难**：ADHD患者在无外部结构时难以自主组织任务；类似地，LLM在没有严重依赖手动修正提示的情况下，难以自主发现最佳问题解决模式（来源：Working Memory Identifies Reasoning Limits in Language Models）。
- **前额叶-纹状体门控与Transformer自注意力**：认知神经科学中，执行功能依赖于前额叶-纹状体机制进行选择性门控；而Transformer在训练后，自注意力机制发展出模仿该门控的操作，证明了计算同构性（来源：TRANSFORMER MECHANISMS MIMIC FRONTOSTRIATAL GATING OPERATIONS WHEN TRAINED ON HUMAN WORKING MEMORY TASKS）。

这意味着：ADHD大脑和LLM都需要一个“外部执行功能层”——对ADHD来说是脚手架（如日程表、清单、身体在场），对LLM来说是agent harness（编排、记忆、工具）。而ChatGPT这类AI工具，恰好可以充当这个外部层。

## 历史回响：曾国藩的“日课”与agent harness的同构

晚清名臣曾国藩，极可能是典型的ADHD患者。30岁前他浮躁坐不住，天天串门喝酒看杀人，日记里骂自己“无恒”“浮躁”；读书慢，“他人目下二三行，余或疾读不能终一行”；一辈子严重银屑病，情绪不稳定，打败仗就跳水自杀。但他硬是靠一套“外部执行功能层”逆天改命：**日课十二条**（黎明即起、读书不二、谨言、写日记反省等），以及“结硬寨打呆仗”的军事策略——用最笨最稳的方法抵消自己的冲动。

这套系统与LLM agent harness结构同构：
- **日课↔定时re-grounding**：每天固定时间做固定事，类似agent的定期上下文重置，防止注意力漂移。
- **写日记↔外部记忆**：每天记录反思，相当于向量数据库，存储跨日上下文。
- **结硬寨↔工具调用**：用标准化流程（如扎营必挖壕沟）替代临时决策，类似agent调用预设工具。

曾国藩的成就证明：**执行功能可以外包给系统，但意图和判断必须自己保留**。

## 设计回路：哪些交给AI，哪些必须自己留着？

基于同构视角，我们可以设计一套“人+AI”回路，核心原则是：**AI处理执行调度，人类保留意图与元认知**。

### 1. 任务启动：AI分解，人类选择
ADHD最怕的是“开始”。Goblin Tools的Magic ToDo功能能将模糊任务（如“清理厨房”）自动分解为具体小步骤，用户可调节“辣度”控制粒度（来源：Goblin Tools）。这降低了启动门槛。但人类必须自己决定“现在做哪个子任务”——AI不能替你做价值判断。

### 2. 工作记忆：AI作为第二大脑
ADHD大脑的工作记忆不稳定，跨会话任务容易丢失上下文。Saner.AI提供本地记忆存储和快速检索，用户无需在多个标签页间切换即可获取信息（来源：Saner.AI）。这相当于LLM的向量数据库。但人类需要定期“清理”记忆——删除过时信息、更新优先级，否则AI记忆会变成垃圾堆。

### 3. 时间管理：AI动态规划，人类设定边界
Motion能自动根据任务优先级、截止日期和可用时间动态调整日程，当任务面临延期风险时提前警告（来源：Motion）。这解决了ADHD的时间盲。但人类必须设定“硬边界”（如休息时间、深度工作时段），否则AI会填满所有时间，导致 burnout。

### 4. 注意力管理：AI引导超聚焦，而非打断
ADHD的超聚焦是一把双刃剑。AI应提供“gentle structure”引导超聚焦到优先事项，而非强行打断（来源：[[超聚焦]]）。例如，可以设置AI每45分钟提醒“检查是否仍在做计划任务”，而不是用番茄钟强行中断。

## 脚手架 vs 拐杖：边界在哪？

这是最大的争议。AI工具是“外部执行功能层”还是“永久拐杖”？专家警告：若工具替代了治疗或自我管理，可能造成依赖（来源：[[拐杖与脚手架]]）。真正的脚手架应帮助使用者“建造”，但使用者仍需自己“攀登”。

我的判断是：**区分标准在于是否保留了元认知**。如果你依赖AI告诉你“该做什么”，那就是拐杖；如果你用AI执行“你决定要做的事”，那就是脚手架。例如，用Motion自动排程没问题，但如果你不再自己思考“什么最重要”，就危险了。

## 局限与争议

- **证据强度**：当前多为概念类比和初步实验，缺乏大规模随机对照试验证明同构假设或AI工具的临床有效性（来源：[[矛盾与存疑]]）。
- **个体差异**：AI辅助效果依赖个体差异，且缺乏长期验证（来源：[[任务启动]]）。
- **学习成本**：部分用户反映学习使用AI工具本身增加认知负荷（来源：[[矛盾与存疑]]）。
- **过度依赖风险**：尚无长期研究验证AI工具是否导致能力退化（来源：[[拐杖与脚手架]]）。
