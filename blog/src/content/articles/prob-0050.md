# 为什么「治好 ADHD 的不擅长精确、重复、状态维护类任务」和「让 LLM 不跑飞」其实是同一道工程题？

> agent 通过 function calling 把精确计算外包给工具——同一套 harness 思路，两边都成立。

你有没有过这样的经历：脑子里同时转着三个项目、两封未回邮件和一个灵感，但身体就是瘫在椅子上刷手机——因为启动任何一件事都像要翻越一座山。或者，你是个写 Agent 的工程师，看着 LLM 在复杂任务里跑飞，输出一堆看似合理但完全偏离目标的文字，气得想砸键盘。

这两件事，表面上一个关乎神经多样性，一个关乎 AI 工程，但底层是同一道题：一个高产的生成核心，缺少一个可靠的执行调度层。

## 同构的病灶：高产但缺调度

ADHD 大脑被描述为“高产的生成核心”——想法丰富、联想活跃，但缺少稳定、可靠的调度层来筛选、组织、启动和坚持任务（来源：ADHD 的 AI 工具全景）。大语言模型（LLM）同样具备强大的生成能力，却在长程上下文、目标保持与执行调度上存在天然缺陷。两者共享同一个结构困境：**生成能力强，执行可靠性弱**。

这不是比喻。Pace Ventures Research 2026 的数据显示，ADHD 组的超专注工作时长平均为 14.5 小时/周，而对照组仅为 8.2 小时——ADHD 大脑在“产出”上并不逊色，甚至更强。问题出在调度：无法稳定地把生成能力导向目标。

## 历史人物的 harness：孔子与毛泽东

两千多年前的孔子，身高近一米九，精力旺盛到周游列国 14 年坐不住，冲动起来骂弟子“朽木不可雕”，对音乐可以“三月不知肉味”，但对种地等俗事零耐心。他的解决方案是“克己复礼”——用外在的“礼”作为行为边界，每日反省（“吾日三省吾身”），直到 70 岁才“从心所欲不逾矩”。这套系统本质上是一个 **外部行为调度层**：用固定的仪式和反省机制，把冲动的生成核心约束到社会认可的轨道上。

毛泽东同样如此。他小时候是问题儿童，爱读“闲书”，一辈子在动——从井冈山到延安到北京，永远在调查研究。他冲动敢赌，四渡赤水、抗美援朝都是险棋。他的 harness 是“批评与自我批评”“民主生活会”“民主集中制”——用集体决策和制度制衡自己的冲动，用“党指挥枪”把个人意志纳入组织系统。

这两个人的 harness，和今天 LLM agent 的 harness 结构同构：**日课 ↔ 定时 re-grounding，秘书/集体 ↔ 外部调度器，反省机制 ↔ 日志与回溯**。

## 现代工具的 harness 实践

现代 ADHD 工具和 LLM agent harness 共享同一套工程逻辑。

### ADHD 侧：外部执行功能层

- **Goblin Tools** 的 Magic ToDo 功能把模糊任务（如“清理厨房”）分解成可执行的小步骤，用户可调节“辣度”控制粒度，降低启动门槛（来源：Goblin Tools）。
- **Saner.AI** 通过本地记忆和快速检索减少搜索循环，补偿工作记忆不足（来源：Saner.AI）。
- **Lex** 允许用户通过单一指令触发多步骤任务序列，类似 agent 的规划机制（来源：Lex）。
- **Otter.ai** 实时转录会议和灵感，减轻笔记负担；**Reflect** 通过结构化提问帮助反思（来源：ADHD 的 AI 工具全景）。

这些工具的核心不是替代大脑，而是搭建一个“外部执行功能层”——把 ADHD 大脑接入一个可维护的调度系统（来源：ADHD 的 AI 工具全景）。

### LLM/Agent 侧：Agent Harness

LLM agent 的 harness 同样在做这件事。Agent harness 通过 human-in-the-loop 监督提供外部调度：设置护栏（token 预算、工具调用次数上限）、加入人工检查点（暂停并询问后再执行）、集成 MCP 工具和遥测（来源：人在回路与身体在场）。其核心是“形式化规划与护栏，使代理输出真正有用且正确”（来源：What is an agent harness in the context of large-language models?）。

一个典型的 agent harness 包括：任务拆解（类似 Goblin Tools）、外部记忆（类似 Saner.AI）、对话式执行（类似 Lex）、捕获与反思（类似 Otter.ai/Reflect）。结构完全对应。

## 核心观点：脚手架，不是拐杖

我的判断是：**ADHD 大脑和 LLM agent 需要的不是“治愈”或“修复”，而是外部脚手架——一种可拆卸的、可定制的调度层**。

但这里有一个争议：过度依赖 AI 工具是否会导致能力退化，成为永久拐杖？（来源：矛盾与存疑）同样，过度依赖 agent harness 是否会让 LLM 失去泛化能力？答案是：**边界在于脚手架是否可拆卸**。好的脚手架帮助用户学会自己搭建，坏的脚手架让用户永远离不开它。孔子 70 岁才“从心所欲”，说明他的 harness 最终内化了；毛泽东的“批评与自我批评”也旨在培养自我监督能力。

目前的证据多为短期有效，长期效果未知（来源：矛盾与存疑）。个体差异极大——有人需要强力打断（番茄钟），有人需要柔性引导。

## 今天就能试的行动

1. **用 Goblin Tools 的 Magic ToDo 分解一个你拖延已久的任务**：输入“整理邮件”，调节辣度到你觉得可行的粒度，然后从第一步开始做。
2. **给你的 LLM agent 加一个“人工检查点”**：在关键决策点前设置 pause，让人类确认后再继续。
3. **用 Saner.AI 或类似工具建立“第二大脑”**：把所有想法、笔记、待办丢进一个收件箱，让 AI 自动组织，减少搜索循环。
4. **尝试一次“身体在场”工作**：用 Focusmate 或找个朋友视频连线，各自做各自的事，利用 social accountability 降低启动阻力。

## 局限

本文的“ADHD↔LLM 同构”论点目前多为类比推理，缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。此外，AI 工具的学习成本本身可能增加认知负荷，净效果因人而异。

但至少，当你下次因为启动不了任务而自责，或者看到 agent 跑飞而沮丧时，可以记住：**这不是你的错，也不是模型的错——只是缺少一个合适的 harness。**
