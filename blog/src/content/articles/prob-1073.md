# 如果把 ADHD 大脑当成生成核心，ChatGPT 能不能当它的采样温度与表现波动？

> 用 agent 工程的思路重新理解 ChatGPT 对 ADHD 的真正价值。

如果你是个ADHDer，你一定经历过这种崩溃：早上还能量爆棚，能一口气写完三页方案；下午却连回一条微信都要做十分钟心理建设。你的大脑像一台性能炸裂但散热乱飘的服务器——算力惊人，但输出质量全看“温度”。

如果你是个做Agentic Harness的工程师，你一定也熟悉这种困境：LLM本身才华横溢，但采样温度一高就胡言乱语，温度一低又变成复读机。你必须用prompt模板、记忆模块、工具调用这些“脚手架”把它框住，才能让它可靠地完成任务。

这两件事，其实是同一个问题。

## 同构：两个“高产但缺调度”的生成核心

ADHD大脑与LLM在结构上高度同构：两者都是强大的生成核心，但缺乏可靠的执行调度层（来源：ADHD 大脑与 LLM 的同构）。ADHD一侧，默认模式网络（DMN）过度活跃，想法奔涌，但前额叶执行功能薄弱——工作记忆差、任务启动困难、时间感知失灵。LLM一侧，Transformer能生成惊艳的文本，但内置的注意力机制在长上下文中会退化到随机水平，无法抑制优势反应、无法解决认知冲突（来源：ADHD 大脑与 LLM 的同构）。

实证研究进一步揭示了这种同构：用经典Stroop冲突任务测试LLM，发现短上下文中表现正常，但随序列变长，模型在冲突试次上骤降到随机水平——这与ADHD执行功能缺陷的核心标志一一对应（来源：ADHD 大脑与 LLM 的同构）。ADHD表现出“强记忆、弱控制”的神经心理模式：工作记忆容量甚至超过常人，但认知灵活性与注意控制存在核心缺陷（来源：ADHD 大脑与 LLM 的同构）。

所以，当你说“我今天状态不好，脑子像一团浆糊”时，你其实是在说：我的“采样温度”失控了，输出质量波动太大。

## ChatGPT 能当这个温度调节器吗？

答案是：能，但前提是你得把它当成“外部调度层”，而不是“万能大脑”。

ChatGPT 本身也是LLM，它和你一样有“温度”问题。但它的价值在于：你可以通过prompt工程和任务分解，为你的大脑构建一个外部执行功能层——包括任务启动辅助、时间感知补偿、工作记忆外挂和注意力引导（来源：AI 与 ADHD 的日常生活）。

比如，Goblin Tools 的 Magic ToDo 功能能把“清理厨房”这种模糊任务自动分解成“把碗放进洗碗机”“擦台面”“拖地”等具体步骤，用户可调节“辣度”控制分解粒度（来源：Goblin Tools）。这直接对应ADHD的任务启动困难——当大任务被拆成小步骤，启动阈值就降低了。

再比如，Routinery 通过可视化倒计时和过渡提示，帮你把“早上起床”变成一条可执行的流水线（来源：Routinery）。倒计时提供时间锚点，过渡提示减少任务切换的认知负荷——这本质上就是agent工程里的“定时re-grounding”机制。

Habitica 则用游戏化提供即时的多巴胺奖励，弥补内在动机不足（来源：Habitica）。这相当于给LLM加了一个“奖励模型”来引导输出方向。

## 历史人物的“脚手架”：孔子与左宗棠

孔子可能是个ADHDer。他身高1米9，精力旺盛，周游列国14年坐不住；冲动爱骂人，见南子急得对天发誓，骂宰予“朽木不可雕”；但对音乐超专注到“三月不知肉味”。他的专属harness是“克己复礼”——用外在的“礼”作为行为边界，每日反省，“吾日三省吾身”。70岁才做到“从心所欲不逾矩”，一辈子和自己的冲动做斗争。

左宗棠也是。举人出身，脾气大，爱骂人，和曾国藩闹别扭；40多岁才出山，69岁抬棺出征收复新疆。他的harness是“身无半亩心忧天下，读破万卷神交古人”，外加“结硬寨打呆仗”——每天读经写家书，用诸葛亮的“鞠躬尽瘁”要求自己。

这两套系统，本质上和LLM的agent scaffolding同构：孔子的“礼”相当于prompt模板，左宗棠的“日课”相当于定时re-grounding。他们都是用外部结构来稳定一个高产但易波动的生成核心。

## 脚手架 vs 拐杖：边界在哪？

这里必须诚实地说：AI工具可能从脚手架变成永久拐杖，削弱内在执行功能（来源：矛盾与存疑）。目前缺乏长期追踪研究来证明这种工具使用不会导致依赖。而且，ADHD症状异质性大，同构理论可能过于简化——并非所有ADHD患者都受益于同一类工具（来源：AI 与 ADHD 的日常生活）。

我的核心观点是：**ChatGPT对ADHD的真正价值，不是替代你的大脑，而是让你意识到“你的大脑和LLM是同一类东西”——然后你可以用同样的工程思维，为自己构建一个外部调度层。** 这个调度层必须是你主动设计、定期调整的，而不是被动依赖的。

## 今天就能试的行动

1. **用Goblin Tools分解一个你拖延了一周的任务**：把“写报告”扔进去，调高“辣度”直到步骤细到你觉得“这我总能做吧”。然后只做第一步。
2. **用Routinery设计一个早晨流程**：从起床到出门，设5-7个步骤，每个步骤配倒计时。执行三天，观察你的启动时间是否缩短。
3. **把ChatGPT当“外部工作记忆”**：当你脑子乱时，把当前想法全部丢进对话框，让它帮你整理成清单。然后告诉它：“请每完成一项就提醒我下一项。”
