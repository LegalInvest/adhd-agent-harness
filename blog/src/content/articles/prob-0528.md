# 3 个 LLM 工程概念，彻底重写了我对 ADHD 注意力容易被环境带偏的理解

> LLM 受上下文窗口内容左右，需要上下文工程 如何反向照亮 ADHD 的注意力容易被环境带偏。

## 问题：为什么我的注意力总像被环境“劫持”？

如果你是个ADHD患者，你一定经历过这样的场景：打开电脑准备写报告，结果被一条弹窗新闻带走，半小时后你在看某个陌生博主的开箱视频，完全忘了最初要干什么。你的注意力像一艘没有舵的船，被环境中的任何风吹草动带偏。

如果你是个在做Agentic Harness的工程师，你一定也经历过：你精心设计的LLM agent，因为上下文窗口里多了一句无关的日志，输出就开始跑偏，甚至产生幻觉。你不得不花大量精力做上下文工程，确保agent只看到该看的信息。

这两个问题，表面上是不同领域的事，但它们的底层结构惊人地同构。我花了很长时间才意识到：ADHD大脑和LLM/agent本质上是同一类系统——**高产但缺执行调度层的生成核心**。而LLM工程中的三个概念，彻底重写了我对ADHD注意力易被环境带偏的理解。

## 概念一：验证循环——从“想当然”到“接地”

ADHD大脑的一个典型特征是容易“想当然”。比如在情绪调节中，ADHD个体可能脱离环境线索做出不恰当反应（来源：ADHAPT: An AI Tool for Emotional Regulation）。这就像LLM在没有验证循环时，自信地输出幻觉。LLM工程中，复杂的harness不会盲目执行工具，而是实现验证步骤，例如检查输出格式或对模型编写的代码运行测试用例（来源：What is an agent harness...）。

这种验证循环对ADHD同样关键。晚清名臣曾国藩，30岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”。他的解法是“日课十二条”：黎明即起、读书不二、谨言、写日记反省。这个“写日记反省”就是一个每日的验证循环——把当天的行为与目标核对，校正偏差。曾国藩的“结硬寨打呆仗”，本质上是用最笨最稳的方法抵消自己的冲动。这正好对应LLM harness中的“规划-验证-迭代”模式：每一步都做一次接地，避免幻觉累积。

**同构对应**：曾国藩的日课 ↔ LLM agent的定时re-grounding；他的日记反省 ↔ 验证与CI集成。

## 概念二：确定性控制流——用外部结构稳定波动

ADHD个体的表现常呈现显著的日内波动，注意力在不同时间点差异很大（来源：采样温度与表现波动）。这种波动本质上是“生成核心”缺乏稳定的执行调度层所致。LLM的输出同样具有随机性，采样温度越高，输出越随机。工程实践强调“确定性状态转换”和“计划-执行”分离模式，来强制单向、确定性的控制流（来源：AI Agents in 2026: Tools, Memory, Evals, and Guardrails）。

对于ADHD，外部结构就是确定性控制流。辛弃疾21岁带50人冲几万人敌营抓叛徒，但被罢官后闲居20年，他的解法是“以词和剑为核心”——每天练剑写词，把报国无门的愤懑全部注入词里。这个“练剑写词”的固定日程，就是他的确定性控制流，稳定了他波动巨大的情绪和注意力。

工具层面，Focusmate通过虚拟身体在场提供外部结构：用户预约时段，平台匹配伙伴，通过视频会议共同工作（来源：Focusmate）。这种模式弥补了内在多巴胺调节不足，相当于为ADHD大脑套上了一个外部调度器。

**同构对应**：辛弃疾的练剑写词日程 ↔ LLM的确定性工作流；Focusmate的配对 ↔ 外部调度器。

## 概念三：采样温度控制——调节输出的多样性

LLM的采样温度控制着输出的多样性：温度越高，输出越随机；温度越低，输出越确定。ADHD大脑同样缺乏这种“温度调节”——有时超聚焦到无法切换任务，有时注意力散漫到无法聚焦。Brain.fm利用AI生成音乐，通过神经锁相技术影响大脑信息处理方式，帮助用户进入专注状态（来源：Brain.fm）。这相当于从外部调节大脑的“采样温度”。

但要注意，这种调节不能过度依赖。工具既可能成为永久依赖的拐杖，也可能成为逐步内化的脚手架（来源：拐杖与脚手架）。RescueTime自动追踪时间并阻断分心网站，但过度关注数据可能引发焦虑（来源：RescueTime）。关键是要有意识地使用，而不是让工具完全接管。

## 核心观点：脚手架而非拐杖

我的核心判断是：**ADHD大脑与LLM的同构，不是让我们把自己当成机器，而是让我们用工程思维主动设计外部脚手架，逐步内化为自己的能力**。就像曾国藩的日课，最终内化成了他的行为习惯；就像LLM agent的验证循环，最终成为系统的一部分。边界在于：你是在主动使用工具，还是被工具控制。

## 争议与局限

必须诚实指出，ADHD大脑与LLM的同构目前仅是一种理论类比，缺乏实证基础（来源：矛盾与存疑）。Brain.fm等工具在ADHD人群中的独立临床证据仍有限（来源：Brain.fm）。个体差异也很大，同一工具对不同ADHD亚型效果可能迥异（来源：AI 与 ADHD 的专注力）。

1. **设置一个“验证循环”**：每天睡前花5分钟，写一句今天最偏离目标的事，再写一句明天如何纠正。像曾国藩写日记一样。
2. **用Focusmate或类似服务**：预约一个30分钟的虚拟共同工作时段，体验外部调度器的力量。
