# 为什么用 Goblin Tools 治 ADHD 的情绪失调，和给 agent 套「会褪去的脚手架」是一回事？

> Goblin Tools 是个任务拆解工具,和「情绪」八竿子打不着——但 ADHD 社区里最动人的用户反馈偏偏是情绪层面的:「它是第一个不让我觉得自己是废物的工具。」这句话背后藏着一条被低估的因果链,以及一个所有辅助工具都逃不掉的问题:脚手架要不要拆、什么时候拆。

先把那条因果链摆直。ADHD 的情绪失调有相当一部分不是独立症状,是**执行失败的下游产物**:任务启动不了→自责(「这么简单都做不到」)→情绪淹没→更启动不了——一个每天转好几圈的循环。研究者把 ADHD 的情绪调节困难列为核心特征之一,而临床观察反复看到:**挫败-羞耻循环是后天叠加的那部分,恰恰是可干预性最高的那部分**。Goblin Tools 切进来的位置很刁:它不处理情绪,它把「启动不了」这个循环的第一环拆掉——任务被拆到可执行,执行发生了,自责失去了燃料。它的界面语言也在做同一件事:不打分、不催促、无成就系统、无红色逾期标记——**一个从设计上就不生产羞耻的工具**。所谓「治情绪」,治的是循环,不是感受。

再说「会褪去的脚手架」。教育心理学里 scaffolding 的完整定义包含两半:**提供支撑,和逐步撤除**——只有前半的叫依赖,不叫教学。agent 工程里也有同款讨论:给弱模型的辅助结构(超细的步骤分解、密集的检查点),随着模型能力上升要逐步撤掉,否则结构本身成为天花板。Goblin Tools 用在 ADHD 身上,同样站在这个岔路口:

## 让脚手架「工作」的用法

**用它中断循环,并且说出来。** 拆解→执行→完成的每一次,都在给「我做得到」积累证据。关键的一步是**把证据显性化**:完成后对自己说一句「看,拆对了就做得到——问题在粒度,不在我」。这句归因矫正是整个干预里最值钱的部分,工具只是它的载体。

**观察自己的「辣度曲线」。** 状态差时用最细的拆解,状态好时用粗的——几周后回看,你会发现某些任务类型已经不需要拆了。**这条曲线就是脚手架该褪去的地图**:能自己拆的任务,试着不用工具直接写三步;还不能的,继续用,不羞耻。

**把「拆解思维」内化为提问习惯。** 工具的终极产出不是那张子任务列表,是你脑中长出的那个问题:「这个任务的下一个物理动作是什么?」——哪天你发现自己在没打开工具的情况下自动问出这句,脚手架已经完成了它教学的那一半。

## 让脚手架变成「拐杖化」的用法

**每个任务都拆、永远最细粒度**:包括早已熟练的日常——支撑无限扩张,能力反而在支撑下退化;**用拆解代替感受**:情绪上来就疯狂拆任务,把工具当情绪回避的通道——情绪失调本身需要被正面处理(命名、呼吸、必要时专业帮助),拆解只能拆掉「执行失败」这一个情绪源;**把工具当裁判**:列表没清完就判自己失败——那是把羞耻生产线换了个厂址。

**一个诚实的补充**:「褪去」不是对所有人成立。ADHD 是发育性差异,部分支撑可能长期需要——就像近视者不谋求「褪去眼镜」。判断标准不是「还在用工具」,是**功能和自我叙事的方向**:用着工具、事情在推进、自责在减少,就是好系统;这和「脚手架必须拆」并不矛盾——**拆的是「本可自己走」的那部分,留的是补发育差异的那部分**,两者的边界只能靠你自己的辣度曲线试出来。

## 边界

同构强度 B 级:scaffolding 的「支撑-撤除」结构在教育心理学有扎实传统,ADHD 情绪失调与执行失败的关联有临床文献,Goblin Tools 本身无对照研究。声明:本文说的「情绪改善」限于挫败-羞耻循环这个下游部分;ADHD 的情绪失调还有更底层的成分(情绪反应强度、恢复速度),那部分需要的可能是治疗、药物与技能训练(如 DBT 类技能),不是任务工具;持续的情绪淹没、易怒或低落,请专业评估。

## 今天就能试的 3 件事

1. **今天完成一次拆解-执行后,说出那句归因矫正**:「问题在粒度,不在我。」
2. **开始记辣度曲线**:哪些任务要最细,哪些已经不用拆——一张纸就够。
3. **挑一个熟练任务「裸奔」**:不开工具,自己写三步。测一测哪部分脚手架已经可以褪了。

Goblin Tools 治情绪的秘密,是它根本不碰情绪——它拆掉了羞耻的燃料供应。而所有好脚手架的宿命都一样:一部分内化成你的思维习惯,一部分留下来补真实的差异。分清这两部分,是每个 ADHD 工具用户早晚要做的功课。
