# ADHD 职场人如何用「给生成核心套 harness」的思路应对项目推进与汇报里的任务启动困难、不会拆解？

> 把 LLM 工程里的「agent 用 planner-executor 循环分解长任务」迁移到ADHD 职场人的项目推进与汇报场景，双域证据作答。

## 问题：任务启动不了，拆解不会——ADHD 职场人的日常死循环

你盯着项目大纲，光标闪烁，大脑一片空白。不是不想做，是启动按钮像被锁死。好不容易开始，又卡在“把这件事拆成几步”上——要么拆得太粗（“完成报告”），要么拆得太细（“打开电脑、新建文档、输入标题……”），最后在焦虑中刷手机。

这不是意志力问题。ADHD 的核心缺陷之一——执行功能障碍——让计划、组织、任务启动变得异常困难（来源：执行功能障碍）。大脑像一台没有调度层的生成器：创意喷涌，但无法自主排序、分步、执行。

而另一边，LLM 领域的工程师正面临同样的问题：一个强大的语言模型，如果不加任何调度框架，给它一个“写一份季度汇报”的指令，它要么输出一堆泛泛而谈，要么陷入无意义的细节循环。于是有了 agent 架构——用 planner 和 executor 的循环来分解长任务。

## 同构：ADHD 大脑与 LLM 共享同一套“生成核心 + 缺失调度层”架构

ADHD 大脑与 LLM 都是高产的生成核心，但缺乏可靠的内置调度层（来源：AI 与 ADHD 的时间管理）。LLM 需要提示工程、agent 框架来引导输出；ADHD 大脑需要外部执行功能层——即一套“harness”——来管理时间、拆解任务、启动行动。

这种 harness 在 LLM 工程里叫 planner-executor 循环：planner 把大任务分解成子步骤，executor 逐一执行，planner 再根据反馈调整下一步。在 ADHD 职场场景中，这套循环可以手工搭建：用外部工具充当 planner（如 Todoist 列出步骤），用计时器或身体动作充当 executor（如番茄钟），再定期回顾调整。

## 证据：工具如何扮演外部调度层

市面上已有不少工具在实践这一思路。Motion 和 Reclaim.ai 通过 AI 自动排程，动态调整每日计划，直接消除“下一步该做什么”的决策负担（来源：Motion；Reclaim.ai）。Tiimo 则用视觉化时间线让时间流逝变得可感，直接应对时间盲——ADHD 者常无法感知时间，而 Tiimo 将时间变成可见元素（来源：Tiimo）。这些工具共同构成了一个“外部大脑”，帮助用户从混乱中建立秩序（来源：AI 与 ADHD 的时间管理）。

但工具只是 harness 的一部分。真正的系统需要人机协同：planner（工具/人）负责分解，executor（人/工具）负责执行，re-grounding（日课/回顾）负责校准。

## 人物案例：孔子的“克己复礼”与苏轼的“豁达系统”

孔子是典型的 ADHD 大脑：精力旺盛、冲动、思维跳跃、对感兴趣的事超专注（三月不知肉味），对俗事零耐心。他的 harness 是“克己复礼”——用外在的“礼”作为行为边界，每日反省（吾日三省吾身），70 岁才做到从心所欲不逾矩。这套系统对应 LLM 的 planner：礼是预设的规则框架，反省是定时 re-grounding。

苏轼同样：兴趣极广，对无聊行政零耐心，但搞工程效率极高。他的 harness 是“豁达”——不管贬到哪里，都用美食、写作、交朋友转移注意力，每日读书写字，不因挫折停止创作。这对应 agent 的 executor：持续输出，不因局部失败中断整体循环。

两人都通过外部系统（礼/日课/习惯）补偿了内在调度层的缺失，成就斐然。

## 核心观点：harness 是脚手架，不是拐杖

本文的核心判断是：**ADHD 职场人需要的不是更多工具，而是一个可复用的 planner-executor 循环框架**。这个框架必须满足三个条件：
1. **分解可执行**：把“完成项目”拆成“写大纲→找资料→写初稿→修改”等具体步骤，每步有明确产出和时限。
2. **反馈及时**：每完成一步，有视觉或听觉奖励（如 Todoist 完成音效），利用多巴胺机制维持动机（来源：AI 与 ADHD 的时间管理）。
3. **定期 re-grounding**：每天/每周回顾，调整计划，避免偏离（类似孔子的“三省吾身”）。

但必须诚实指出局限：过度依赖工具可能从“脚手架”退化为“永久拐杖”，削弱内在执行功能的代偿发展（来源：矛盾与存疑）。目前多数工具缺乏独立临床验证，效果可能受安慰剂效应影响（来源：矛盾与存疑）。此外，同构性本身只是理论类比，并非科学事实（来源：矛盾与存疑），不能过度推广。

## 今天就能试的行动

1. **用 Goblin Tools 的“魔法任务分解”**：输入“写季度汇报”，它会自动拆成 5-8 个子步骤，直接复制到 Todoist。
2. **设置两个番茄钟**：第一个用于“规划”（planner 模式），第二个用于“执行”（executor 模式），中间休息 5 分钟用于 re-grounding。
3. **每天写一句“三省吾身”**：睡前问自己“今天 planner 有效吗？executor 执行了吗？明天怎么调？”，记录在手机备忘录。
4. **如果使用 Motion 或 Reclaim.ai**：每周手动检查一次自动排程结果，确认它没有把重要但不紧急的任务无限推迟。

## 结语

ADHD 大脑不是缺陷，而是未经调优的生成核心。你不是缺少能力，而是缺少一个 harness。像孔子和苏轼那样，为自己搭建一套 planner-executor 循环，让生成核心在轨道上运行。记住：工具是脚手架，不是拐杖——最终，你要学会自己调度自己。
