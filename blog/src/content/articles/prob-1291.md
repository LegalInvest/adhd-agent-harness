# Saner.AI 之于 ADHD，就像 planner-executor 任务分解 之于 LLM——但有人用错了

> 从同构视角实测 Saner.AI：它到底补上了哪一层执行功能？

## 问题：为什么自动排程器救不了你的“启动困难”？

你打开 Saner.AI 或 Motion，自动排程填满了日历——但你还是盯着“写年终报告”这个任务，大脑一片空白，手指无法敲下第一个字。这不是意志力问题，而是你的大脑缺少一个关键层：**任务分解调度层**。

ADHD 大脑与 LLM 在结构上是同构的：两者都是强大的“生成核心”，但缺乏可靠的内置调度层（来源：AI 与 ADHD 的时间管理）。LLM 面对“写一篇万字论文”的指令会胡编乱造，除非你用 planner-executor 循环把它拆成“写大纲→搜资料→写引言→……”；ADHD 大脑面对“整理房间”也会宕机，因为它无法自动将大目标解构成可执行的子步骤。**Saner.AI 之于 ADHD，就像 planner-executor 任务分解之于 LLM——但很多人用错了**：他们只用了自动排程（相当于只给 LLM 一个调度器，却没教它怎么拆任务），而忽略了最核心的分解功能。

## 同构的真相：两个“生成核心”都缺一个 harness

证据来自两边。ADHD 侧：执行功能研究指出，ADHD 患者的前额叶皮层活动不足，导致任务启动、计划和排序困难（来源：AI 与 ADHD 的时间管理）。LLM/agent 侧：agent 框架（如 ReAct、Plan-and-Solve）的核心就是让 LLM 先规划子步骤，再逐个执行，否则长任务必然失败。两边用的是同一类解法：**给高产生成核心套上外部 harness**。

历史上，孔子和曹操早就建好了自己的 harness。孔子有 ADHD 特质：精力旺盛、冲动（骂宰予“朽木不可雕”）、对音乐超专注（三月不知肉味）、对俗事零耐心。他的 harness 是“克己复礼”——用外在的“礼”作为行为边界，每日反省（“吾日三省吾身”），相当于一个**定时 re-grounding 机制**，与 LLM 的“系统提示+定期检查”同构。曹操的 harness 是“唯才是举”——他把行政全丢给荀彧，自己专注军事文学，相当于**外部调度器**，与 agent 框架中“将任务委派给子 agent”同构（来源：人物案例）。这两人都没靠“内在意志力”硬扛，而是搭建了外部结构。

## 工具对比：哪些真的补上了分解层？

当前 AI 时间管理工具可粗分为三类：

1. **自动排程器**（Motion、Reclaim.ai、Tiimo）：帮你自动安排日程，但假设你已经有明确的任务列表。Motion 能根据截止日期动态调整（来源：Motion），Reclaim.ai 能保护深度工作时间（来源：Reclaim.ai），Tiimo 用视觉化时间线让时间盲变得可感（来源：Tiimo）。这些工具解决了“什么时候做”，但没解决“怎么做”——你依然需要自己把大任务拆成小步骤。

2. **任务分解器**（Goblin Tools、部分 Saner.AI 功能）：Goblin Tools 的“魔法待办”能自动将“打扫房间”拆成“收拾桌面→扫地→拖地”等子任务。这才是真正对应 planner-executor 循环的工具。Saner.AI 如果只做情绪调节和日程排布，就错失了分解这一核心功能。

3. **身体加倍**（Focusmate）：通过虚拟共在提供隐性问责，激活多巴胺通路。研究表明定期问责检查可将目标达成可能性从 25% 提升至 95%（来源：身体在场效应）。这相当于给 LLM 加了一个“人类评审员”在 loop 中。

**核心判断：真正有效的 ADHD harness 必须包含“分解+调度+问责”三层**。只做调度（如 Motion）或只做问责（如 Focusmate）都不够。Saner.AI 的潜力在于它能否整合这三层——但目前多数用户只用了它的调度功能，忽略了分解。

## 争议与局限：脚手架还是拐杖？

必须诚实指出：**同构性仍是一种理论类比，缺乏实证基础**（来源：矛盾与存疑）。ADHD 大脑与 LLM 的神经机制并不相同，类比是为了启发设计，而非科学结论。此外，过度依赖 AI 工具可能从“脚手架”退化为“永久拐杖”，削弱内在执行功能的代偿发展（来源：AI 与 ADHD 的时间管理）。目前 Motion 等工具缺乏独立临床验证，效果可能受安慰剂效应影响（来源：Motion）。最后，个体差异巨大：有人对视觉化工具（Tiimo）反应好，有人更受益于自动排程（Reclaim.ai），目前缺乏个性化匹配指导（来源：AI 与 ADHD 的时间管理）。

## 今天就能试的 3 条行动

1. **用 Goblin Tools 拆解一个你拖延已久的大任务**：输入“整理书房”，看它拆出的子步骤是否让你觉得“这个我能做”。对比只用 Motion 自动排程的效果。
2. **设置一个“外部调度器”**：像曹操用荀彧一样，找一个朋友每天花 5 分钟帮你确认今日任务分解是否合理。或者用 Focusmate 约一个虚拟身体加倍伙伴，在有人“在场”时启动最难的第一步。
3. **在 Saner.AI 中主动创建“分解模板”**：不要只让它自动排程，而是手动输入“写报告”的分解步骤（如“写大纲→写引言→写数据部分……”），然后让工具按这个模板调度。这相当于给 LLM 写好 planner 提示词。

记住：你的大脑不是坏掉了，它只是缺一个外部调度层。选对工具，搭建你的 harness，而不是只买一个更贵的日历。
