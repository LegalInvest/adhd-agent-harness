# 用 ChatGPT 给 ADHD 补上无状态与外部记忆，真的能像 agent 那样工作吗？

> 把 ChatGPT 当作 ADHD 的外部执行层来实测：它补的是哪一块短板？

## 问题：ADHD 的“边做边忘”与 LLM 的“无状态”是同一个坑

你是否有过这样的经历：打开 ChatGPT，想继续昨天没写完的方案，却发现自己完全忘了当时思考的细节，不得不从头开始解释背景？这种“每次打开都是新会话”的挫败感，对于 ADHD 大脑来说，几乎是日常生活的缩影——工作记忆容量有限且不稳定，导致跨会话任务难以持续（来源：无状态与外部记忆）。

而 LLM 本身也是无状态的：每次对话都是独立的，模型不保留跨会话的记忆（来源：The Anatomy of an AI Agent）。没有外部记忆体，LLM 就像一个永远失忆的天才，每次都要重新认识你。

这正是本文要回答的核心问题：用 ChatGPT 给 ADHD 补上无状态与外部记忆，真的能像 agent 那样工作吗？答案是：**能，但有条件**。

## 同构：ADHD 大脑与 LLM agent 共享同一套缺陷

ADHD 大脑被描述为“高产但缺乏可靠执行调度层的生成核心”（来源：ADHD 的 AI 工具全景）。LLM 同样如此——它是强大的生成引擎，但缺乏内置的执行调度和持久记忆。两者的核心困境在于：

- **工作记忆**：ADHD 的工作记忆容量保留但控制弱（易被干扰），LLM 同样有强记忆容量但弱上下文控制（来源：ADHD 的 AI 工具全景）。
- **任务启动**：ADHD 的任务启动困难对应 LLM 的“冷启动”问题。
- **时间盲**：ADHD 的时间感知缺陷与 LLM 的时序推理弱点同构。
- **上下文工程**：ADHD 易被环境带偏，LLM 易上下文膨胀（来源：上下文工程）。

因此，AI 帮助 ADHD 的本质是**为生成核心套上 harness（脚手架）**，即通过外部执行功能层补偿内在的调度缺陷（来源：ADHD 的 AI 工具全景）。

## 证据：ChatGPT 作为外部记忆体的实际效果

### ADHD 侧：用户报告支持

现实中，ADHD 个体已在使用 ChatGPT 作为“外部对话记忆”，减少标签切换（来源：ADHD 的 AI 工具全景）。例如，用户反馈称 ChatGPT 能“将混乱的待办清单转化为清晰的下一步行动”（来源：Goblin Tools）。更系统的工具如 Saner.AI 专注于知识回忆和本地记忆，帮助用户快速找回信息，减少搜索循环（来源：Saner.AI）。Mem 则被称为“AI 第二大脑”，提供持久化上下文。

这些工具的共性是为 ADHD 大脑提供一个“永远在线”的外部记忆体，补偿内部工作记忆的波动（来源：无状态与外部记忆）。

### LLM/agent 侧：工程实践验证

生产级 agent 系统通过外部记忆（如向量数据库）解决状态丢失问题，并组合使用会话内上下文记忆、长期检索向量库和结构化知识库（来源：The Anatomy of an AI Agent）。检索增强和工具调用将决策锚定在外部证据和持久状态上（来源：AI Agent Systems）。这恰恰是 ChatGPT 作为 agent 的架构——通过外部记忆（如自定义指令、知识库）来维持跨会话一致性。

### 局限：证据仍以定性为主

必须诚实指出，现有证据主要来自用户报告和工具设计逻辑，缺乏大规模随机对照试验（来源：矛盾与存疑）。例如，Goblin Tools 被评价为“简单有用”，但无独立 RCT 支持（来源：Goblin Tools）。Saner.AI 的效果基于用户反馈，可能无法覆盖所有 ADHD 症状（来源：Saner.AI）。

## 关键判断：脚手架 vs 拐杖

我的核心观点是：**ChatGPT 作为外部记忆体，其价值取决于使用方式——是作为脚手架促进能力发展，还是作为永久拐杖导致依赖。**

当 ChatGPT 被用来存储和检索关键上下文（如项目背景、个人偏好），它就像 agent 的向量数据库，是高效的脚手架。但当用户开始依赖它记住所有细节，甚至放弃主动记忆，它就变成了拐杖。

这一矛盾在资料中被反复提及：过度依赖 AI 工具可能削弱内在能力（来源：矛盾与存疑）。例如，Otter.ai 减轻笔记负担，但过度依赖可能削弱主动记笔记的能力（来源：ADHD 的 AI 工具全景）。工具设计者声称是“补偿”而非“替代”，但缺乏如何避免依赖的具体指导（来源：矛盾与存疑）。

## 今天就能试的三件事

1. **为 ChatGPT 创建“项目记忆”提示词**：在每次对话开头，粘贴一个固定格式的上下文摘要（如“项目：XX；上次进展：XX；下一步：XX”），让 ChatGPT 自动记住关键信息。这相当于手动搭建一个轻量级外部记忆体。

2. **用 Goblin Tools 分解一个让你拖延的任务**：输入一个模糊目标（如“整理房间”），观察 AI 如何将其拆解为小步骤。注意启动焦虑是否降低。

3. **设置“上下文检查”闹钟**：每 30 分钟提醒自己“当前在做什么？”，对抗注意力飘移。这类似于 LLM 的上下文窗口管理——主动压缩膨胀的上下文。

## 结论：同构启示下的理性期待

ADHD 大脑与 LLM agent 在无状态与外部记忆上的同构，为我们提供了一个有力的思考框架：ChatGPT 确实能像 agent 的外部记忆体那样补偿工作记忆不足，但它不是万能药。效果取决于你是否主动进行上下文工程——正如 agent 需要精心设计的检索和工具调用，ADHD 用户也需要主动管理“当下注意什么”（来源：上下文工程）。

记住：工具是脚手架，不是拐杖。用得好，它是你的第二大脑；用不好，它只是另一个让你分心的标签页。
