# Goblin Tools 之于 ADHD，是脚手架还是拐杖？

> 从「同构」视角评测 Goblin Tools：用久了会让 ADHD 更强还是更依赖？

## 问题：ADHD 的边做边忘，与 LLM 的无状态困境

你是否有过这样的体验：打开冰箱想拿牛奶，却在门口愣住，忘了自己要干什么？或者，刚读完一段文字，翻页后内容就模糊了？这并非粗心，而是 ADHD 大脑的典型特征——工作记忆容量小（来源：执行功能障碍）。ADHD 大脑被描述为一个高产的生成核心：想法丰富、联想活跃，但缺少稳定、可靠的调度层来筛选、组织、启动和坚持任务（来源：ADHD 的 AI 工具全景）。

无独有偶，大语言模型（LLM）同样具备强大的生成能力，却在长程上下文、目标保持与执行调度上存在天然缺陷。每个对话会话都是独立的，模型无法跨会话记住之前的交互，必须依赖外部记忆（如向量数据库）和 agent scaffolding 来维持状态。

## 同构：ADHD 大脑与 LLM agent 共享一个架构

ADHD 大脑与 LLM 在结构上可被视为同类系统：底层都拥有一个高产生成核心，但都缺乏可靠的内置执行调度层。两者都需要外部脚手架——ADHD 需要外部执行功能层，LLM 需要 agent scaffolding（来源：ADHD 的 AI 工具全景）。

历史人物孔子就是这一同构的生动案例。孔子身高1米9，精力旺盛，周游列国14年坐不住；冲动爱骂人，思维跳跃，《论语》全是场景化语录，无系统著作。他的专属 harness 是“克己复礼”——用外在的“礼”作为行为边界，每日反省（“吾日三省吾身”）。这相当于一个外部调度层：用固定的仪式和规则来约束生成核心的冲动，70岁才做到从心所欲不逾矩。

在 LLM 侧，这对应着 agent 的“re-grounding”机制：定时将模型拉回目标轨道，防止生成偏离。孔子的“日课”就像 LLM 的定时上下文刷新，而“礼”则像约束生成行为的系统提示。

## Goblin Tools：是脚手架还是拐杖？

Goblin Tools 是一套 AI 驱动的工具套件，核心功能包括 Magic ToDo（AI 任务分解）、语气检查器和时间估计工具（来源：Goblin Tools）。它专为 ADHD 和自闭症用户构建，将复杂任务分解为可管理的小步骤，降低启动门槛（来源：Goblin Tools）。

从同构视角看，Goblin Tools 扮演的正是“外部调度层”角色：它把模糊的任务意图（如“清理厨房”）转化为具体的行动列表，用户可调节“辣度”控制分解粒度（来源：Goblin Tools）。这相当于 LLM agent 中的“规划机制”——将复杂目标分解为一系列步骤（来源：Lex 工具页）。

那么，它是脚手架还是拐杖？

**脚手架**意味着：工具在用户发展自身执行功能的过程中提供临时支持，用户最终能内化这些策略，减少依赖。**拐杖**则意味着：用户永久依赖工具，一旦失去，功能就会退化。

当前证据显示，Goblin Tools 的短期效果积极：用户反馈称“Goblin Tools 在这方面很棒”（来源：Goblin Tools）。但长期依赖风险缺乏实证（来源：矛盾与存疑）。

我的判断是：**Goblin Tools 是脚手架还是拐杖，不取决于工具本身，而取决于用户如何使用它。** 如果用户仅仅用它来“完成当前任务”，而不反思分解策略、不尝试在没有工具时复制类似方法，那么它就是拐杖。如果用户将其作为学习如何拆解任务的训练辅助，逐步内化这种能力，那么它就是脚手架。

这与 LLM agent 工程面临同样的问题：agent scaffolding 是让模型变得更强大，还是让模型失去了在没有脚手架时自主规划的能力？目前尚无定论。

## 争议与局限

首先，ADHD 大脑与 LLM 的同构目前多为类比推理，缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。其次，AI 工具对任务启动的长期效果未知，部分用户反映学习使用 AI 工具本身增加认知负荷（来源：矛盾与存疑）。最后，Goblin Tools 的 Magic ToDo 虽然降低启动门槛，但可能削弱用户主动分解任务的能力——这正是拐杖风险的来源。

## 今天就能试的行动

1. **用 Goblin Tools 分解一个小任务**：选择一个你拖延已久的任务（如“整理书桌”），用 Magic ToDo 分解成 5-10 步，然后逐项执行。完成后，反思分解逻辑，尝试在脑中复述步骤。
2. **对比有/无工具的启动差异**：明天早上，先尝试不用任何工具写一个待办列表，然后对比用 Goblin Tools 分解同一个任务，感受认知负荷的变化。
3. **设置“无工具日”**：每周选择一天，强制自己不用任何 AI 工具进行任务管理，依靠纸笔或脑内规划。这能测试你内化了多少分解策略。
4. **将工具作为训练轮**：使用 Goblin Tools 时，刻意记录它分解出的步骤，并尝试在类似任务中主动回忆和应用。

Goblin Tools 是一面镜子，照出的不是工具的好坏，而是我们如何使用工具。对于 ADHD 大脑和 LLM 而言，外部脚手架是必要的，但关键在于——我们是否在搭建脚手架的同時，也在建造自己的肌肉。
