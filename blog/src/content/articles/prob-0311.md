# 用 ChatGPT 给 ADHD 补上上下文工程，是脚手架还是拐杖？

> ChatGPT 接进 ADHD 工作流：哪些交给它、哪些必须自己留着。

### 问题：你的注意力，是否像 LLM 一样被上下文绑架？

如果你是 ADHD 患者，你一定熟悉这种场景：打开电脑准备写报告，却被通知栏的新闻标题吸引，点进去后又被关联链接带偏，半小时后你发现自己正在研究某个远古文明的饮食结构，而报告一个字没动。你的大脑就像一台没有调度器的生成器——创意不断，却无法聚焦。

如果你是做 Agentic Harness 的工程师，你同样熟悉另一幕：你精心设计的 LLM agent，因为上下文窗口塞进了无关日志，开始胡言乱语；或者因为历史对话过长，它忘记了最初的任务指令。你不得不写一堆 prompt engineering 代码，来“约束”这个强大的生成核心。

这两类人看似毫不相干，但底层困境惊人地同构：ADHD 大脑与 LLM 都是**高产但缺乏可靠执行调度层的生成核心**（来源：ADHD 大脑与 LLM 的同构）。ADHD 的注意力易被环境带偏，LLM 的生成易被上下文窗口内容左右——两者都需要“上下文工程”：主动设计“当下注意什么”的方案。

### 同构脊柱：为什么 ChatGPT 是天然的解？

ADHD 的注意力分散，本质是执行功能缺陷——尤其是工作记忆容量有限，无法同时记住“当前任务”和“干扰抑制指令”。这就像 LLM 的无状态性：每次对话都是新的开始，除非你把上下文显式塞进 prompt。

因此，AI 帮助 ADHD 的本质是**给生成核心套上 harness（外部执行功能层）**（来源：AI 与 ADHD 的专注力）。ChatGPT 恰好能扮演这个角色：你告诉它“我要写报告，每 10 分钟提醒我一次当前目标”，它就变成了一个外部调度器。这就像工程师为 LLM 搭建的 agent harness——包裹模型，处理它本身做不到的执行控制（来源：ADHD 大脑与 LLM 的同构）。

但 ChatGPT 能做的远不止提醒。你可以把整个任务上下文“卸载”给它：比如“我正在写一篇关于 ADHD 的文章，已经完成了引言，下一段需要讨论工具证据。如果我发现跑题，请拉我回来。” 这样，你的工作记忆负担就转移到了外部，就像 LLM 依赖向量数据库来维持连续性（来源：AI 与 ADHD 的专注力）。

### 真实证据：哪些工具已经验证了这条路？

并非所有工具都平等有效。根据现有证据，我们可以分级来看：

- **强证据**：Focusmate 利用“身体在场效应”，通过 AI 匹配实时问责伙伴，多项用户研究显示能显著提升任务启动和维持（来源：AI 与 ADHD 的专注力）。它的机制正是外部执行功能层——社会压力替代了内在的多巴胺调节。
- **中等证据**：RescueTime 自动追踪时间使用，帮助 ADHD 用户对抗“时间盲”，有初步研究支持其改善时间感知（来源：AI 与 ADHD 的专注力）。它的 AI 自动分类减轻了工作记忆负担。
- **弱证据**：Brain.fm 声称通过神经锁相技术影响大脑状态，但缺乏针对 ADHD 的独立临床研究（来源：Brain.fm）。

这些工具的共同点：它们不是替代你的大脑，而是为你的生成核心套上 harness。Focusmate 提供外部问责，RescueTime 提供外部记忆，ChatGPT 则是一个可定制的上下文工程平台。

### 脚手架还是拐杖？边界在哪里？

这里存在一个关键争议：长期使用 AI 工具，是否会导致 ADHD 患者内在执行功能进一步退化？这就像 LLM agent 过度依赖 hard-coded prompt 而失去泛化能力。

我的判断是：**边界在于工具是否设计为“过渡性”的**。脚手架的目的是在建筑建成后拆除；拐杖则是永久替代。

- 如果你用 ChatGPT 只是“帮我写邮件”，那你就是在用拐杖——你失去了练习写作的机会。
- 但如果你用 ChatGPT 来“列出邮件要点，然后我自己写”，或者“每 10 分钟提醒我当前任务”，那你就是在搭脚手架——你在训练自己的任务启动和注意力维持能力。

现有证据警告：过度依赖 AI 工具可能削弱内在能力，比如时间感知和任务启动（来源：矛盾与存疑）。但神经可塑性研究告诉我们，大脑可以通过训练重塑（来源：神经可塑性）。因此，关键不是用不用 AI，而是**怎么用**。

### 今天就能试的 3 条行动

1. **用 ChatGPT 做“上下文锚点”**：在开始任何任务前，先向 ChatGPT 描述你的目标、当前进度和可能的干扰。让它每 15 分钟问一次“你现在还在做 X 吗？” 这相当于给 LLM 加了一个 context window 刷新机制。
2. **把 Focusmate 作为启动仪式**：预约一个 50 分钟的 Focusmate 时段，告诉你的伙伴你今天的核心任务。身体在场效应会迫使你启动，就像 agent harness 强制 LLM 执行特定工作流。
3. **用 RescueTime 识别你的“上下文膨胀”模式**：查看每日报告，找出那些你被带偏的高频时间段。然后针对性地设计 ChatGPT 提示，比如“下午 3 点提醒我检查是否跑题”。

### 局限与诚实

我必须指出，本文的论证建立在“ADHD 大脑与 LLM 同构”这一隐喻之上，而这一隐喻并非严格等价——LLM 的注意力机制与人类注意力有本质差异（来源：AI 与 ADHD 的专注力）。此外，现有工具的证据多来自用户报告，缺乏大规模随机对照试验（来源：矛盾与存疑）。个体差异也很大：对某些 ADHD 亚型，Focusmate 可能因社交焦虑而适得其反。

因此，不要盲目套用。最好的方法是把自己当作一个 agent 系统，持续实验、测量、调整。ChatGPT 是你的 harness，但只有你才能决定它是脚手架还是拐杖。
