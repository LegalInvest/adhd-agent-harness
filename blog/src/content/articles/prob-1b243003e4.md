# 两个互不引用的领域都在研究「工作记忆」——ADHD 文献和 LLM harness 文献为什么得出了同一个结论？

> 一位 ADHD 研究生同时读两类文献：白天读 ADHD 综述，晚上帮实验室调 agent。某天她把两篇论文摊在一起——一篇讲 ADHD 工作记忆容量受限导致多步任务失败，一篇讲 LLM 上下文窗口溢出导致长任务崩溃——两篇文献的参考文献列表零重叠，但解决方案几乎逐条对应。她的第一反应不是兴奋，是怀疑：这是巧合、是我过拟合，还是真的有结构在？

收敛：本文只回答一个问题——**两个互不引用的领域在「工作记忆/上下文」上得出趋同结论，这个趋同的成色有多真，能支撑多重的实践迁移？**

## 穿透：趋同的具体内容

ADHD 侧文献的稳定结论：工作记忆缺损是 ADHD 的核心特征之一（多项荟萃分析支持），后果是多步任务中间状态丢失——走到厨房忘了来干什么、对话中丢掉自己要说的点。干预方向高度一致：**外部化**——写下来、清单化、把中间状态存到环境里而不是脑子里。

Harness 侧文献的稳定结论：LLM 的上下文窗口有限且「无状态」，长程任务中间状态会被挤出窗口，后果是 agent 在多步任务中忘记早期约束。解决方案同样收敛于外部化：scratchpad、外部记忆库、检索增强、状态摘要——把中间状态存到模型之外。

两边互不引用（文献计量上确认的 Swanson 结构：A 与 C 不相连，通过中间概念 B「有限工作容量下的任务完成」相通），却各自独立走到了「外部记忆」这同一扇门前。这正是文献发现方法（LBD）里最有价值的信号：独立趋同比互相引用的重复更有证据力，因为排除了学术时尚的传染。

但要挖风险：趋同发生在**计算层**（都是有限容量系统的调度问题），不在实现层——人脑的工作记忆是神经回路，不是 KV cache。把类比下探到实现层的说法（「你的大脑就是个 transformer」）是修辞，不是科学。

## 验证：趋同能证伪吗

能。如果两个领域的最优解在某处分叉——比如 harness 侧发现「无限上下文」能根治而 ADHD 侧的外部化只是补偿——类比就该在那里止步。目前确实有一条分叉的苗头：LLM 的上下文可以工程性扩容，人的工作记忆训练扩容证据薄弱（工作记忆训练的远迁移效应在荟萃分析中不乐观）。这条分叉恰恰给出实践指引：**别练容量，练外部化**——这是两边文献共同支持、且对人更现实的那一半。

## 决策

做什么：把「外部化中间状态」当作一等公民协议——任何超过三步的任务，第一步是建一个外部 scratchpad（一张纸、一个便签、一条置顶消息），中间状态即时写入。

不做什么：不要购买声称能「扩容工作记忆」的训练产品来解决任务失败——证据不支持远迁移；不要把本文类比当作诊断依据。

先做什么：今天正在进行的最复杂的一件事，现在就给它建 scratchpad。

## 边界

本同构有双域文献的独立支撑，属于机制层类比中较强的一档，但本文未做正式 A/B/C 分级；「工作记忆↔上下文窗口」的对应停留在计算层。工作记忆问题的评估与干预请以临床专业意见为准，本文不构成诊断建议。

## 今天就能试的 3 件事

1. 给当前最复杂的任务建一张「中间状态卡」：写上现在进行到哪一步、下一步是什么——放在看得见的地方。
2. 设一条协议：被打断时的第一个动作不是回应打断，而是花 10 秒把当前状态写下来——像 agent 保存 checkpoint。
3. 睡前把明早第一件事写在便签上贴在显眼处——把「跨夜的中间状态」也外部化。
