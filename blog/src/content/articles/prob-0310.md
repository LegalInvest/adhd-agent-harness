# ChatGPT 之于 ADHD，就像 上下文工程 之于 LLM——但有人用错了

> 从同构视角实测 ChatGPT：它到底补上了哪一层执行功能？

ADHD 的日常是一场与“环境噪音”的拉锯战。你打开文档准备写报告，手机通知一闪，顺手点进去，半小时后发现自己躺在购物 app 里。这不是意志力弱——是你的大脑缺乏一个能主动过滤干扰的“上下文工程”层。

同样的事发生在 LLM 身上：一个 agent 被喂了一堆历史对话，上下文窗口膨胀后，它开始胡言乱语，忘记最初的任务。工程师们为此设计了确定性工作流、工具契约、状态机——这些统称为 **harness**，即外部执行调度层。

**核心论点：ChatGPT 之于 ADHD，就像上下文工程之于 LLM——但多数人用错了。** 他们把它当作一个“聪明的大脑”来对话，期待它自动解决一切，却忘了它本身和 ADHD 大脑一样：高产、发散、缺乏稳定的调度层。正确的用法是：把 ChatGPT 当作 harness 的一部分，用结构化提示、外部记忆、定时重定向来搭建一个“外部执行功能层”。

## 同构：两个高产但失控的生成核心

ADHD 大脑与 LLM 在结构上高度同构：两者都是**高产但缺乏可靠执行调度层的生成核心**（来源：AI 与 ADHD 的专注力）。ADHD 的创意发散像高采样温度下的 LLM 输出——丰富但随机；而 LLM 的无状态性则对应 ADHD 的工作记忆缺陷，都需要外部记忆系统来维持连续性（来源：AI 与 ADHD 的专注力）。

这种同构在“波动性”上尤为明显。ADHD 个体的表现日内波动极大，注意力、执行功能在不同时间点差异显著；LLM 的输出也受采样温度控制，温度越高越随机。工程师通过“确定性状态转换”和“计划-执行分离”来稳定输出（来源：采样温度与表现波动），而 ADHD 患者则需要外部结构——如日程、提醒、环境设计——来稳定表现。

## 人物案例：曾国藩的“日课”就是最早的 harness

晚清名臣曾国藩是典型的注意力不集中型 ADHD：30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”，读书慢到“他人目下二三行，余或疾读不能终一行”。他的解决方案是**日课十二条**：黎明即起、读书不二、谨言、写日记反省……这套系统本质上就是一个外部执行调度层，用固定的流程和反省机制来对冲内在的冲动。

曾国藩的“日课”对应 LLM harness 中的**定时 re-grounding**：每天固定时间点重新锚定注意力，就像 agent 工作流中定期检查上下文是否偏离。他的“写日记”则是**外部记忆**，记录状态和反思，弥补工作记忆的不可靠。

## 工具实测：ChatGPT 到底补上了哪一层？

现有 AI 工具对 ADHD 的支持，本质上都在提供外部执行功能层。但效果取决于是否精准对应特定短板，且需避免形成永久依赖（来源：AI 与 ADHD 的专注力）。

- **Focusmate**：基于虚拟身体在场的实时问责，通过 AI 匹配伙伴（来源：Focusmate）。它补的是**任务启动**——ADHD 最难的“开始”这一步。与曾国藩的“读书不二”同构：有人看着你，你就没法跑题。
- **RescueTime**：自动追踪时间，揭示“时间到底去了哪里”（来源：RescueTime）。它补的是**时间盲**——ADHD 常低估或高估时间流逝。相当于 LLM 的 trace 日志，记录每一步消耗。
- **Brain.fm**：用 AI 生成音乐，通过神经锁相影响大脑状态（来源：Brain.fm）。它补的是**注意力维持**——类似 LLM 中通过提示工程固定输出风格。但独立临床证据仍有限（来源：Brain.fm）。

这些工具的共同逻辑是：**不要指望 AI 替你思考，而是让它替你记住、提醒、启动。** 就像工程师不会让 LLM 自己管理上下文，而是给它一个向量数据库、一套工具调用契约。

## 批判与争议：脚手架 vs 拐杖

当前最大的争议是：AI 工具是“外部执行功能层”还是“拐杖”？长期使用是否会导致内在执行功能进一步退化？（来源：矛盾与存疑）

我的判断是：**关键在于可撤除性。** 一个好的 harness 应该像脚手架——在建筑建成后可以拆除。曾国藩的日课最终内化为习惯，他不再需要每天写“无恒”来提醒自己。同样，Focusmate 的使用者应该逐渐学会在没有伙伴时也能启动任务。如果工具变成了“没有它我就无法工作”，那就是拐杖。

另一个局限是证据质量。多数工具仅有用户报告或小样本研究，缺乏大规模随机对照试验（来源：矛盾与存疑）。Brain.fm 的神经锁相机制在 ADHD 中是否有效尚存疑。个体差异也极大——对某些人有效的番茄钟，对另一些人可能造成焦虑。

## 今天就能试的行动

1. **用 ChatGPT 做“启动提示”**：每天早上让它列出今天最重要的 3 件事，并设定一个 5 分钟倒计时。不要和它聊天，只让它当计时器。
2. **模仿曾国藩的“日课”**：在 ChatGPT 里建一个“日课模板”，每天固定时间让它问：“你今天做了什么？偏离目标了吗？” 记录回答，一周后回顾。
3. **尝试一次 Focusmate 会话**：预约一个 25 分钟时段，和陌生人一起工作。感受“被看着”的启动效果。
4. **用 RescueTime 追踪一周**：不看数据，只让它后台记录。一周后打开报告，你会震惊于“时间到底去了哪里”。
