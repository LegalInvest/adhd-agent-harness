# Mem 之于 ADHD，是脚手架还是拐杖？

> 从「同构」视角评测 Mem：用久了会让 ADHD 更强还是更依赖？

## 问题：为什么 ADHD 用户总在“边做边忘”？

如果你是 ADHD 患者，你一定经历过这样的场景：打开浏览器想查资料，结果被一条通知带走，半小时后发现自己正在刷视频，而最初的任务早已蒸发。这不是意志力问题——是你的**工作记忆**在“漏”。ADHD 的工作记忆容量保留但控制弱，极易被干扰（来源：ADHD 的 AI 工具全景）。就像一台 RAM 很大的电脑，但操作系统调度器坏了，随时可能跑偏。

同样的问题也困扰着 LLM agent 的工程师：LLM 本身没有跨会话状态，每次对话都是“冷启动”。为了让 agent 记住用户偏好、历史任务，工程师必须搭建外部记忆系统——向量库、缓存、持久化上下文。否则，agent 就会像 ADHD 患者一样，每次都要重新“认识”用户。

## 同构：ADHD 大脑与 LLM agent 是同一类生成核心

ADHD 大脑与 LLM 在架构上惊人相似：两者都是“高产但缺可靠执行调度层的生成核心”（来源：ADHD 的 AI 工具全景）。ADHD 大脑能产生大量创意和想法，但缺乏执行功能来组织、启动和监控；LLM 能生成流畅文本，但缺乏内置的上下文控制、时序推理和验证机制。

因此，两边都需要一个**外部 harness**（脚手架）来补偿内在缺陷。对 LLM agent 而言，harness 提供上下文传递、工具接口、验证循环和记忆系统（来源：幻觉与验证循环）。对 ADHD 而言，AI 工具作为外部执行功能层，通过分解任务、管理上下文、增强记忆和提供实时反馈来改善执行功能（来源：ADHD 的 AI 工具全景）。

Mem 正是这个同构的典型产物。

## Mem：是脚手架还是拐杖？

Mem 定位为“AI 第二大脑”——它能自动记录、整理和回忆你的笔记、想法和任务，相当于一个永不遗忘的外部记忆库。对 ADHD 用户来说，这完美补偿了工作记忆的“漏”：你不用再担心忘记刚才的想法，Mem 会帮你记住。对 LLM agent 工程师来说，Mem 就像向量数据库 + 自动摘要器，让 agent 能跨会话保持上下文。

但这里存在一个关键边界：**脚手架 vs 拐杖**。

脚手架是临时辅助，帮助用户发展内在能力；拐杖则是永久替代，长期使用反而削弱原有能力。Mem 的设计逻辑是“补偿”而非“替代”（来源：矛盾与存疑），但实际使用中很容易滑向拐杖。例如，Otter.ai 减轻笔记负担，但过度依赖可能削弱主动记笔记的能力（来源：ADHD 的 AI 工具全景）。同样，如果用户完全依赖 Mem 来“记住一切”，而不训练自己的检索和组织策略，那么工作记忆的控制能力可能进一步退化。

这个边界在 LLM agent 一侧同样存在。如果 agent 过度依赖外部记忆库，每次推理都直接从库中拉取完整上下文，而不训练模型自身的上下文压缩和选择性注意能力，就会导致“上下文膨胀与推理退化”（来源：上下文工程）。优秀的 harness 工程会设计验证循环和记忆裁剪，迫使模型在有限上下文中高效推理——这正是脚手架的做法。

## 证据：Mem 的效果与局限

现有证据主要来自用户报告和工具设计逻辑。Mem 和 Saner.AI 通过持久记忆和自动回忆，补偿工作记忆不足（来源：ADHD 的 AI 工具全景）。用户反馈称，Mem 减少了“搜索循环和标签切换”（来源：Saner.AI 页）。但缺乏大规模随机对照试验，效果未严格验证（来源：矛盾与存疑）。此外，个体差异大：有人觉得 Mem 是救星，有人觉得信息组织方式反直觉。

## 核心判断：Mem 是脚手架，但需要主动使用

我认为，Mem 本质上是脚手架，但能否避免沦为拐杖取决于用户是否**主动使用其外部记忆来训练内部工作记忆的检索与组织能力**。具体来说：

- **做脚手架**：用 Mem 记录想法后，定期（如每天）主动回顾、分类、关联，训练大脑的检索路径。这类似于 LLM agent 的“反思循环”——不是直接复用旧记忆，而是重新加工。
- **做拐杖**：只存不回顾，需要时直接搜索，大脑从不参与记忆的编码与提取。这就像 agent 每次都用全量上下文，不加裁剪。

## 诚实指出争议与局限

1. **证据不足**：Mem 对 ADHD 的效果主要基于用户报告，缺乏对照实验（来源：矛盾与存疑）。可能被夸大。
2. **依赖风险**：过度依赖可能削弱内在记忆能力，但目前缺乏如何避免依赖的具体指导（来源：矛盾与存疑）。
3. **个体差异**：并非所有 ADHD 用户都适合 Mem。部分用户可能发现其信息组织方式增加认知负荷（来源：Goblin Tools 页的局限）。

## 今天就能试的 3 条行动

1. **用 Mem 记录一个项目，每天花 5 分钟主动回顾并整理标签**，而不是只靠搜索。这是训练大脑检索能力的关键。
2. **对比使用 Mem 前后的“遗忘次数”**：记录一周内忘记的重要事项数量，看 Mem 是否真正减少了认知负荷。
3. **设置“无 Mem 时段”**：每天选 1 小时关闭 Mem，强迫大脑自己记忆，避免依赖。这对应 LLM agent 的“上下文裁剪”练习。

## 结语

Mem 之于 ADHD，就像外部记忆库之于 LLM agent：它是强大的脚手架，但只有主动使用才能促进能力发展。同构视角告诉我们：工具的价值不在于“替代”，而在于“训练”。下一次你用 Mem 时，不妨问自己：我是用它来增强，还是用它来逃避？答案决定了它是脚手架还是拐杖。
