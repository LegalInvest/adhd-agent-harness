# Mem 之于 ADHD，是脚手架还是拐杖？

> 从「同构」视角评测 Mem：用久了会让 ADHD 更强还是更依赖？

你打开 Mem，快速记下一个念头，关上。三小时后回来，它还在。你不需要记住刚才想了什么，Mem 替你记住了。这种感觉对 ADHD 大脑来说，像溺水的人抓住了浮板。但问题是：这块浮板，会让你学会游泳，还是让你永远离不开它？

这个问题，不只 ADHD 患者在问。做 LLM agent harness 的工程师也在问同样的问题：当我们把记忆外挂给向量库，agent 是在变得更自主，还是在变得更依赖外部 crutch？

## 同构的困境：高产生成核心与缺失的调度层

ADHD 大脑常被描述为一个“高产的生成核心”——想法丰富、联想活跃，但缺少稳定、可靠的调度层来筛选、组织、启动和坚持任务（来源：ADHD 的 AI 工具全景）。LLM 同样具备强大的生成能力，却在长程上下文、目标保持与执行调度上存在天然缺陷。两者都需要外部脚手架来把生成能力约束到目标轨道上。

这个同构不是比喻，而是工程事实。ADHD 个体使用外部工具来管理“当下注意什么”，比如 Routinery 创建步骤序列并引导你一步步执行（来源：上下文工程）。LLM agent 同样需要上下文管理，防止“上下文膨胀与推理退化”（来源：上下文工程）。两边的解法结构一致：用外部系统补偿内部调度缺陷。

## Mem：外部记忆的两种面孔

Mem 被定位为“AI 第二大脑”，通过知识回忆、本地记忆与可复用工作流，补偿 ADHD 的工作记忆不稳定（来源：ADHD 的 AI 工具全景）。它像一块永远不丢的便签纸。但便签纸有两种用法：一种是把所有事记下来，大脑从此不记任何事；另一种是用便签纸辅助思考，最终内化出记忆策略。

关键区别在于：Mem 是否被用来构建可迁移的认知策略。如果只是被动地“存进去、搜出来”，它就是拐杖——你依赖它，一旦没有它，你比之前更糟。但如果用它来练习“捕捉-整理-回顾”的循环，逐步训练自己识别什么值得记、如何组织信息、何时回顾，它就是脚手架——支撑你成长，最终可以撤掉一部分。

这个边界在 LLM agent harness 里同样清晰。纯依赖外部向量库的 agent，一旦上下文窗口变化或记忆库被清空，就会“失忆”并产生更多幻觉和重复工作（来源：幻觉与验证循环）。而好的 harness 会设计验证循环和规划机制，让 agent 学会在每一步自我检查，逐步减少对外部记忆的依赖。

## 历史案例中的同构：孔子的“日课”与毛泽东的“民主生活会”

孔子是典型的 ADHD 大脑：精力旺盛、周游列国 14 年坐不住，冲动爱骂人（“朽木不可雕”），思维跳跃，《论语》全是场景化语录，无系统著作。他的 harness 是“克己复礼”——用外在的“礼”作为行为边界，每日反省（“吾日三省吾身”）。这套系统相当于一个外部调度器：每天固定时间 re-grounding，用仪式化的步骤约束冲动的生成核心。

毛泽东同样如此：一辈子读书不辍、永远在动、思维极度跳跃。他的 harness 是“批评与自我批评”和“民主生活会”——让别人提意见，用集体决策制衡自己的冲动。这相当于 agent harness 中的“验证与 CI 集成”：系统不会盲目执行，而是通过外部反馈循环校正行为。

这些历史案例的共同点：他们构建了可迁移的策略，而不是依赖某个固定工具。孔子的“日课”可以迁移到任何环境，毛泽东的“调查研究”成为思维习惯。这正是脚手架与拐杖的分界线。

## 证据与局限

目前支持 AI 工具作为外部执行功能层的核心论点是：AI 并不是在“治愈”ADHD，而是在 ADHD 大脑与外部世界之间搭建一个临时的外部执行功能层（来源：ADHD 的 AI 工具全景）。但“临时”有多临时？长期依赖风险缺乏实证（来源：矛盾与存疑）。

更具体的矛盾：部分 ADHD 用户反馈，强行打断（如番茄钟）反而有效，而柔性引导可能不够强力（来源：矛盾与存疑）。这意味着 Mem 的“温柔提醒”可能对某些人失效。此外，学习使用 Mem 本身可能增加认知负荷，净效果因人而异（来源：矛盾与存疑）。

在 LLM 侧，纯对话系统常因幻觉、缺乏接地和无法执行/验证行动而失败（来源：幻觉与验证循环）。Mem 提供的记忆如果缺乏验证机制，同样可能存储错误信息并放大幻觉。

## 今天就能试的行动

1. **设置“回顾触点”**：在 Mem 中每周固定时间回顾一周笔记，尝试不搜索直接回忆——训练内部记忆肌肉。
2. **用 Mem 做“捕捉-整理-输出”循环**：每天捕捉 5 条想法，整理成 3 条行动，输出 1 条总结。坚持两周，观察自己是否开始主动分类信息。
3. **对比实验**：一周完全依赖 Mem 记录所有事，另一周只记录关键节点（如会议结论、待办），其余靠大脑记忆。记录哪一周的完成度和焦虑感更高。
4. **为 Mem 添加验证步骤**：对重要信息（如截止日期、数据），手动核对一次后再存入。这对应 LLM harness 中的验证循环，能减少“记忆幻觉”。

Mem 本身是中性的。它可以是脚手架，也可以是拐杖。区别不在于工具，而在于你如何使用它——以及你是否在用它训练自己，最终可以少用它。
