# 如果把 ADHD 大脑当成生成核心，Focusmate 能不能当它的外部执行功能层？

> 用 agent 工程的思路重新理解 Focusmate 对 ADHD 的真正价值。

## 问题：ADHD 大脑缺了什么，Focusmate 补了什么？

ADHD 人群的核心痛点是执行功能缺陷——规划、组织、任务启动困难，工作记忆不稳定，时间盲（来源：[[执行功能]]、[[工作记忆]]）。这些缺陷让大脑像一台高性能但缺操作系统的计算机：想法奔涌，却无法可靠地调度、排序、执行。工程师在构建 agent 时遇到同样的问题：LLM 是强大的生成核心，但单独使用时无法可靠完成多步骤任务，需要外部 harness——即编排、记忆、工具层——来引导和扩展其能力（来源：Agent Scaffolding: Architecture and Design Patterns for Agentic AI）。

Focusmate 是一个简单的视频结对工作平台：你和另一个人约定时间，打开摄像头，各自安静工作。它没有复杂的算法，没有 AI 调度，但大量 ADHD 用户报告它显著提升了专注和启动能力。为什么？因为 Focusmate 提供了 ADHD 大脑最缺的东西：一个外部执行功能层。

## 同构：ADHD 大脑与 LLM 都需要外部调度层

ADHD 大脑与 LLM 在结构上同构：两者都是“高产但缺执行调度层的生成核心”（来源：[[ADHD 大脑与 LLM 的同构]]）。ADHD 大脑的发散思维、联想能力与 LLM 的文本生成能力类似，但都缺乏可靠的内置调度机制。因此，AI 帮助 ADHD 的本质不是修复缺陷，而是“给生成核心套上 harness”（来源：[[生成核心与调度层]]）。

Focusmate 的机制恰好映射了 agent harness 的核心组件：

- **启动触发器**：约定时间相当于 agent 的定时触发。ADHD 大脑难以自主启动任务，Focusmate 的预约时间创造了外部启动信号，类似 LLM agent 中的“定时 re-grounding”（来源：[[任务启动]]）。曾国藩的“日课十二条”中“黎明即起”也是同样的原理——用外部纪律抵消内部冲动。

- **身体在场效应**：有他人在场时，ADHD 大脑的专注度显著提升（来源：[[身体在场效应]]）。Focusmate 的摄像头对方制造了“虚拟身体在场”，相当于 agent harness 中的“观察者循环”——系统持续监控执行状态并反馈。

- **工作记忆卸载**：Focusmate 不需要你记住“接下来该做什么”，因为 session 结构已经定义了“现在就是工作时间”。这类似于 Saner.AI 通过本地记忆减少搜索循环（来源：Saner.AI），也类似于 LLM agent 中的“记忆缓冲区”存储会话状态（来源：Agent Scaffolding）。

毛泽东的“批评与自我批评”民主生活会也是一种 harness：用集体监督制衡个人冲动，用外部反馈校准方向。这与 agent harness 中的“验证循环”同构——系统通过外部检查点确保生成结果符合目标。

## 证据：Focusmate 的实效与局限

用户反馈表明，Focusmate 对任务启动和持续专注有效。一位 ADHD 创业者描述：“Focusmate 让我从‘想工作’变成‘正在工作’。” 这符合“身体在场效应”的实证——他人在场提升多巴胺水平，补偿 ADHD 的动机缺陷（来源：[[多巴胺]]）。但效果依赖个体差异：部分用户发现摄像头本身是干扰，而另一些则需要更结构化的工具（如 Motion 的自动排程，来源：Motion）。

从 agent 工程角度看，Focusmate 相当于一个“单通道 harness”：它提供启动和在场感，但缺乏任务分解、优先级排序、记忆检索等高级组件。相比之下，Goblin Tools 的 Magic ToDo 能自动将模糊任务分解为小步骤（来源：Goblin Tools），Motion 能动态调整日程（来源：Motion），Saner.AI 能快速检索信息（来源：Saner.AI）。这些工具组合起来才能构成完整的“外部执行功能层”。

## 核心判断：Focusmate 是脚手架，不是拐杖

Focusmate 的价值在于它是可拆卸的脚手架，而非永久拐杖。ADHD 大脑需要外部结构来启动，但长期依赖可能削弱内在执行功能（来源：[[拐杖与脚手架]]）。Focusmate 的 session 结构强制你在固定时间内工作，之后脚手架拆除——你仍然需要自己完成后续任务。这与 agent harness 的设计原则一致：harness 引导 LLM，但不取代其生成能力。

矛盾在于：部分用户反馈，Focusmate 的柔性能量不如番茄钟的强行打断有效（来源：[[矛盾与存疑]]）。这说明个体差异极大，且缺乏对比研究。同样，ADHD 大脑与 LLM 同构的论点目前基于类比推理，缺乏神经科学或计算机科学的直接证据（来源：[[矛盾与存疑]]）。

## 今天就能试的行动

1. **预约一次 Focusmate session**：选一个你拖延的任务，约定30分钟，打开摄像头，关掉其他通知。观察启动难度是否降低。
2. **组合使用 Focusmate + 任务分解**：在 session 开始前，用 Goblin Tools 的 Magic ToDo 将任务分解为3-5个小步骤，打印出来放在桌前。
3. **记录自己的“harness 需求”**：连续三天记录哪些外部条件（时间、他人、工具）帮你启动了任务。这相当于 agent 工程的“需求分析”。
4. **尝试逆向工程 Focusmate**：如果你是工程师，思考如何用 AI 模拟身体在场效应——例如，一个虚拟伙伴在固定时间“出现”并保持沉默，只提供眼神接触。这比视频结对更轻量，但可能同样有效。

Focusmate 不是万能药，但它提供了一个清晰的同构案例：ADHD 大脑和 LLM agent 都需要外部执行功能层，而最简单的脚手架往往最有效。真正的挑战不是技术，而是找到适合你个人的“harness 配置”。
