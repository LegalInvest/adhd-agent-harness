# 3 个 LLM 工程概念，彻底重写了我对 ADHD 想理解自己的大脑到底是怎么回事的理解

> LLM 是「高产但缺执行调度层的生成核心」 如何反向照亮 ADHD 的想理解自己的大脑到底是怎么回事。

你有没有过这样的体验：脑子里同时冒出十几个绝妙想法，但不知道该先做哪个；开会时灵感喷涌，散会后却忘了该推进什么；明明能力不差，却总在“启动任务”这一步卡住，直到截止日期逼近才爆发。如果你是一名被 ADHD 困扰的人，这种“高产但失控”的感觉可能贯穿一生。如果你是一名在做 Agentic Harness 工程的开发者，你大概也熟悉另一种“高产但失控”——LLM 能瞬间生成大段文本、代码、方案，但一旦脱离精心设计的 prompt 链，它就会跑题、遗忘上下文、陷入无限循环。

这两个群体——ADHD 大脑和 LLM agent——看似毫无关联，但近年来的跨学科研究揭示了一个惊人的同构：**ADHD 大脑与 LLM 在结构上是同一类“高产但缺执行调度层的生成核心”**（来源：ADHD × AI 的科学与研究前沿）。这意味着，工程师为 LLM 搭建的“脚手架”，恰恰是 ADHD 人群需要的“外部执行功能层”。下面三个 LLM 工程概念，将彻底重写你对 ADHD 的理解。

## 1. 无状态与外部记忆

**LLM 侧**：标准的 LLM 调用是无状态的，每次对话都是独立的，模型不保留跨会话的记忆（来源：无状态与外部记忆）。没有 harness 的 LLM 只是一个无状态的 API 调用。

**ADHD 侧**：ADHD 大脑的工作记忆容量有限且不稳定，导致跨会话任务难以持续。典型表现为：每天需要重新回忆项目上下文，否则容易偏离轨道（来源：无状态与外部记忆）。

**同构解法**：LLM 需要外部记忆系统（如向量数据库）来持久化上下文；ADHD 大脑同样需要“第二大脑”工具来存储客户历史、项目背景、质量标准。Saner.AI 就是这样一个工具——它利用 AI 实现本地记忆存储和快速检索，减少搜索循环和标签切换（来源：Saner.AI）。本质上，ADHD 大脑需要一个“永远在线”的外部记忆体来补偿内部工作记忆的波动（来源：无状态与外部记忆）。

**人物案例**：北宋文豪苏轼，一生被贬，却到哪都能适应。他的“外部记忆”是什么？是每日读书写字、用诗词记录见闻。这些文字成为他跨地域、跨情境的“持久上下文”。苏轼的日课，本质上就是他的“外部记忆系统”——与 Saner.AI 的自动笔记捕获功能异曲同工。

## 2. 上下文工程

**LLM 侧**：LLM agent 面临上下文膨胀与推理退化的问题——随着对话变长，模型容易丢失早期信息或陷入重复（来源：上下文工程）。有效的 agent 需要严格的安全控制和高效的上下文管理，防止上下文膨胀导致推理退化（来源：上下文工程）。

**ADHD 侧**：ADHD 大脑的上下文管理同样脆弱——容易被环境带偏，时间感知扭曲，任务碎片化（来源：上下文工程）。使用通用聊天机器人时，它们“often add another inbox, another prompt, and another place to lose the thread”（来源：上下文工程）。

**同构解法**：工程师通过 prompt 链、状态机、上下文窗口裁剪来管理 LLM 的注意力；ADHD 人群则需要外部工具来主动管理“当下注意什么”。Goblin Tools 的 Magic ToDo 功能正是这样一个“上下文工程”工具——它将模糊任务（如“清理厨房”）分解为具体、可执行的子步骤，用户可调节“辣度”控制粒度（来源：Goblin Tools）。这相当于为大脑的生成核心提供了一个“微步骤执行计划”，防止注意力漂移。

**人物案例**：毛泽东一辈子读书不辍，行军路上也带着书；他的调查研究方法——“没有调查就没有发言权”——本质上是一种上下文工程：通过实地调研获取第一手信息，确保决策基于当前真实情况，而非空想。这与 Goblin Tools 将模糊任务分解为可执行步骤的逻辑一致：先理解上下文，再行动。

## 3. 外部调度层

**LLM 侧**：没有 harness 的 LLM 只是一个生成核心，缺乏执行调度。Agent scaffolding 的核心就是为 LLM 加上调度层——决定何时调用工具、如何规划步骤、如何回退。

**ADHD 侧**：ADHD 的执行功能缺陷——任务启动困难、时间盲、计划困难——本质上就是缺乏可靠的内置调度层（来源：ADHD × AI 的科学与研究前沿）。

**同构解法**：Motion 这类 AI 日程规划应用，就是 ADHD 的“外部调度层”。它自动根据任务、会议和截止日期创建并动态调整每日计划，当日程被打乱时自动重新安排（来源：Motion）。Motion 通过消除“下一步该做什么”的决策负担，直接帮助用户克服时间盲和任务启动困难（来源：Motion）。

**人物案例**：毛泽东的“民主集中制”和“党指挥枪”原则，本质上就是为革命事业搭建的外部调度层——用集体决策制衡个人冲动，用组织系统管理军队和国家。这与 Motion 的自动排程逻辑同构：用外部规则代替内部冲动决策。

## 争议与局限

同构视角虽然强大，但需警惕过度依赖风险。AI 工具是“外部执行功能层”还是“永久拐杖”？核心论点认为 AI 作为外部脚手架补偿缺陷，但长期依赖可能导致能力退化（来源：矛盾与存疑）。此外，同构假设目前多为类比推理，缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。个体差异也很大——部分 ADHD 用户反馈强行打断（如番茄钟）反而有效，而柔性引导可能不够强力（来源：矛盾与存疑）。
