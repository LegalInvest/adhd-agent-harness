# Claude

## 是什么

Claude 是由 Anthropic 开发的对话式 AI 工具，与 ChatGPT、Gemini 类似，属于通用型 AI 助手（来源：Harnessing Artificial Intelligence to Live Better with ADHD - CHADD）。其终端原生代理 Claude Code 在真实软件工程任务中可匹配或超越 IDE 集成工具（来源：Building AI Coding Agents for the Terminal: Scaffolding, Harness, Context Engineering, and Lessons Learned）。Claude 也被用于构建更复杂的系统，例如 Second Brain AI 即基于 Claude Code 构建（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。

## 与 ADHD 的关系

Claude 可帮助 ADHD 用户简化决策任务，例如审查合同和制定商业计划，从而减轻[[执行功能]]负担（来源：Harnessing Artificial Intelligence to Live Better with ADHD - CHADD）。它还能将混乱的待办清单转化为清晰的下一步行动，类似 [[Goblin Tools]] 的功能（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。

## AI 如何介入

- **任务执行**：通过 Claude Code 或 Codex 的文件系统，让助手在结构化工作空间中行动（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。Claude Code 作为终端原生代理，在软件工程任务中表现突出（来源：Building AI Coding Agents for the Terminal: Scaffolding, Harness, Context Engineering, and Lessons Learned）。
- **对话式整理**：将杂乱的待办事项转化为可执行的步骤，缓解[[工作记忆]]压力（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。
- **代理工程**：Anthropic 提供了关于代理架构、工具设计、评估框架等指南，例如《Building Effective Agents》《Writing Effective Tools for Agents》《Demystifying Evals for AI Agents》等（来源：GitHub - ai-boost/awesome-harness-engineering: Awesome list for AI agent harness engineering: tools, patterns, evals, memory, MCP, permissions, observability, and orchestration. · GitHub）。这些资源可帮助开发者构建更可靠的 ADHD 辅助代理。

## 证据

摘录中提及 Claude 是 ADHD 用户可选的 AI 工具之一（来源：6 ways AI can help you manage ADHD symptoms - Understood.org），但未提供具体实证研究数据。Anthropic 的行业基准报告指出，基础设施配置（如 harness 设置）可影响基准测试结果 5 个百分点以上（来源：GitHub - ai-boost/awesome-harness-engineering: Awesome list for AI agent harness engineering: tools, patterns, evals, memory, MCP, permissions, observability, and orchestration. · GitHub）。

## 局限与争议

> ⚠️ 矛盾：来源 [5] 称 Claude 由 AI-PRO 开发，但其他来源（如 [1][2]）暗示 Claude 与 Anthropic 相关（Claude Code）。实际 Claude 由 Anthropic 开发，此处摘录可能存在错误。

Claude 作为通用 AI，其 ADHD 专用功能不如 [[Goblin Tools]] 等工具聚焦，且依赖用户主动使用。
