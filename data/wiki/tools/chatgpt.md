# ChatGPT

## 是什么

ChatGPT 是 OpenAI 开发的对话式 AI 模型，能够生成文本、回答问题、提供建议等。它被描述为“认知协作者”，帮助用户梳理思路、准备沟通、完成任务（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。

## 与 ADHD 的关系

ADHD 成人从 ChatGPT 中独特受益，将其作为[[执行功能]]工具（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。它可以帮助弥补[[工作记忆]]和[[时间盲]]带来的挑战，例如将混乱的待办事项转化为清晰的下一步（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。然而，用户也容易陷入“迷人的岔路”——比如本应写客户提案，却花了45分钟研究罗马高架渠的历史（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。这提示 ChatGPT 可能加剧[[多巴胺]]驱动的注意力分散。

## AI 如何介入

- **沟通准备**：ChatGPT 帮助用户“把事情理清楚，从而在与人交流时感到更准备充分、舒适和自信”（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。
- **任务启动**：提供草稿起点，克服启动困难（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。
- **简化复杂信息**：例如将医学检验结果翻译成通俗语言（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。
- **工作辅助**：教师用于报告卡评语建议，产品设计师频繁使用（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。
- **语言输出**：加速强制性任务的语言生成（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。

## 证据

用户报告显示，ChatGPT 可作为自助工具、问题解决者、编辑和助手——前提是用户核实其准确性（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。ADDitude 读者提供了多种使用案例（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。

## 局限与争议

- **准确性风险**：必须双重检查 ChatGPT 的输出（来源：“A Cognitive Collaborator:” How Adults with ADHD Are Using ChatGPT）。
- **注意力陷阱**：容易引发过度探索，偏离原定任务（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。
- 与其他工具如[[Goblin Tools]]和[[Saner.AI]]相比，ChatGPT 更通用但缺乏专门针对 ADHD 的结构化设计。
