# AI 与 ADHD 的时间管理

## 核心论点

ADHD 大脑与 LLM 在结构上同构——都是“高产但缺乏可靠执行调度层的生成核心”（[[ADHD 大脑与 LLM 的同构]]）。因此，AI 工具帮助 ADHD 进行时间管理的本质，是“给生成核心套 harness”（[[外部执行功能层]]）。当前最站得住脚的核心论点是：**AI 工具作为外部执行功能层，通过补偿时间盲、工作记忆缺陷和任务启动困难，显著提升 ADHD 用户的时间管理表现，但效果依赖于工具与用户个体特征的匹配，且存在过度依赖的风险。**

## 同构视角

从同构视角看，ADHD 大脑的[[执行功能]]缺陷（尤其是[[工作记忆]]容量有限但控制弱、[[时间盲]]、[[任务启动]]困难）与 LLM 的上下文控制缺陷（[[上下文工程]]）对应。两者都需要外部脚手架（[[拐杖与脚手架]]）来补偿。具体而言：
- **时间盲**：ADHD 用户难以感知时间流逝，对应 LLM 缺乏对时间约束的固有理解。AI 工具如 [[Tiimo]] 和 [[Structured]] 通过视觉化时间线将时间转化为可见元素，类似为 LLM 提供时间戳或进度条。
- **工作记忆缺陷**：ADHD 的工作记忆容量保留但控制弱（[[工作记忆]]），与 LLM 的无状态性（[[无状态与外部记忆]]）同构。工具如 [[Todoist]] 通过外部提醒和优先级排序充当外部记忆，类似向量数据库为 LLM 提供持久上下文。
- **任务启动困难**：ADHD 的启动困难源于执行功能失调，AI 工具如 [[Motion]] 和 [[Reclaim.ai]] 通过自动调度和智能时间块降低启动门槛，类似为 LLM 提供 prompt 模板和任务分解。

## 证据脉络

现有证据主要来自用户报告和初步研究，支持 AI 工具的有效性：
- **Motion**：用户报告其动态日程调整功能显著减少决策疲劳，但缺乏对照研究。
- **Reclaim.ai**：通过保护深度工作块，帮助用户维持注意力，类似[[身体在场效应]]的虚拟版本。
- **Tiimo** 和 **Structured**：视觉化时间线被用户认为能改善时间感知，但 AI 功能细节尚不明确（[[Structured]]）。
- **Todoist**：智能优先级排序基于任务属性和截止日期，但算法透明度低。

此外，[[多巴胺]]机制可能解释部分效果：AI 工具通过即时反馈和完成感提供多巴胺奖励，但证据链不完整。

## 仍存争议

1. **过度依赖风险**：AI 工具可能成为“永久拐杖”（[[拐杖与脚手架]]），削弱用户内在的时间感知和规划能力。目前缺乏长期追踪研究。
2. **个体差异**：ADHD 亚型（注意力缺陷 vs. 多动冲动）对工具响应不同，现有工具多面向注意力缺陷型。
3. **证据质量**：多数证据基于用户自我报告，缺乏随机对照试验。AI 功能的“黑箱”特性（如 [[Todoist]] 的优先级算法）使得机制难以验证。
4. **同构理论的实证**：ADHD 大脑与 LLM 的同构仍属理论类比，神经科学和计算机科学的交叉验证不足。
