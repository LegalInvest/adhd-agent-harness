# 总览：ADHD × AI 知识库

## 核心论点：ADHD 大脑与 LLM 的同构

本知识库的脊柱命题是：**ADHD 大脑与 LLM 在结构上同为“高产但缺乏可靠执行调度层的生成核心”**。ADHD 的执行功能缺陷——如工作记忆容量有限、注意力易分散、时间感知困难、任务启动障碍——与 LLM 的上下文窗口限制、注意力分散、幻觉、目标漂移等问题在计算层面高度同构。因此，**「AI 帮 ADHD」的本质不是“治疗”，而是「给生成核心套上 harness（外部执行功能层）」**——就像为 LLM 搭建 prompt 模板、RAG 系统、agent 框架一样，AI 工具为 ADHD 大脑提供结构化的外部脚手架，补偿其内部调度层的不足。
