# 拐杖与脚手架

## ADHD 一侧

ADHD 大脑缺乏内在的执行功能脚手架，表现为工作记忆容量有限、时间感知困难、任务启动与切换障碍。外部工具（如 AI 助手）可以充当临时脚手架，外化思维，减轻认知负荷（来源：Harnessing Artificial Intelligence to Live Better with ADHD - CHADD）。但专家警告过度依赖风险：若工具替代了治疗或自我管理，可能造成依赖（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。真正的脚手架应帮助使用者“建造”，但使用者仍需自己“攀登”（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。最新研究进一步揭示了 ADHD 与 LLM 在认知机制上的深层平行：
- **工作记忆干扰**：ADHD 患者在工作记忆任务中易受干扰，而 LLM 同样表现出类似人类的干扰特征——随着记忆负荷增加表现下降，受近因效应和刺激统计偏差影响，且成功回忆依赖于干扰控制（主动抑制任务无关内容）（来源：Human-like Working Memory Interference in Large Language Models）。
- **自主组织困难**：ADHD 患者在无外部结构时难以自主组织任务；类似地，LLM 在没有严重依赖手动修正提示的情况下，难以自主发现最佳问题解决模式（来源：Working Memory Identifies Reasoning Limits in Language Models）。
- **前额叶-纹状体门控与 Transformer 自注意力**：认知神经科学中，执行功能依赖于前额叶-纹状体机制进行选择性门控；而 Transformer 在训练后，自注意力机制发展出模仿该门控的操作，证明了计算同构性（来源：TRANSFORMER MECHANISMS MIMIC FRONTOSTRIATAL GATING OPERATIONS WHEN TRAINED ON HUMAN WORKING MEMORY TASKS）。
- **自动模式与抑制监控**：ADHD 的自动反应模式与 LLM 的自动完成机制平行，两者都在有限执行监督下生成复杂行为，且易出现虚构/幻觉等错误，需要外部解释者提供意义（来源：Automatic Minds: Cognitive Parallels Between Hypnotic States and Large Language Model Processing）。
- **医疗系统障碍 vs AI 赋能**：第一人称报告指出，AI 工具的灵活性和即时性帮助 ADHD 作者搭建任务脚手架、打破障碍，而 ADHD 医疗系统却制造访问障碍（来源：Executive Dysfunction by Design: A Cognitive Accessibility Analysis of AI Support vs. Healthcare Barriers）。

## LLM/harness 一侧

LLM 本身是生成核心，缺乏可靠的执行调度层。Harness 工程正是设计围绕 LLM 的脚手架——包括上下文传递、工具接口、规划工件、验证循环、记忆系统和沙箱——决定代理在真实任务中的成败（来源：GitHub - ai-boost/awesome-harness-engineering）。脚手架在首次提示前组装代理（系统提示、工具模式、子代理注册），而 harness 在运行时编排工具调度、上下文管理和安全执行（来源：Building AI Coding Agents for the Terminal）。

LLM 的工作记忆限制与 ADHD 高度相似：
- **工作记忆干扰**：LLM 尽管可完全访问上下文，仍表现出工作记忆限制，且复现人类干扰特征（来源：Human-like Working Memory Interference in Large Language Models）。
- **自主推理困难**：LLM 在复杂任务中难以自主发现最佳策略，类似 ADHD 患者（来源：Working Memory Identifies Reasoning Limits in Language Models）。
- **门控机制涌现**：Transformer 在训练中发展出类似前额叶-纹状体的门控机制（来源：TRANSFORMER MECHANISMS MIMIC FRONTOSTRIATAL GATING OPERATIONS WHEN TRAINED ON HUMAN WORKING MEMORY TASKS）。
- **自动性与幻觉**：LLM 的自动完成机制与 ADHD 的自动模式平行，均需外部监督（来源：Automatic Minds: Cognitive Parallels Between Hypnotic States and Large Language Model Processing）。
- **执行功能发展**：LLM 在已知任务上表现出类似执行功能的能力（如规划、工作记忆），但在新任务上有限（来源：Artificial Neuropsychology: Are Large Language Models Developing Executive Functions?）。

## 工程启示（可迁移的做法）

1. **渐进式撤除**：ADHD 工具应设计为随用户执行功能增强而减少辅助；LLM harness 应支持从强干预（完整提示链）到弱干预（仅安全约束）的过渡。
2. **上下文工程**：ADHD 侧通过外部记忆（如笔记、提醒）扩展工作记忆；LLM 侧通过记忆系统和上下文窗口管理实现（来源：Long-Term Memory Architectures for AI Agents - Redis）。
3. **验证循环**：ADHD 侧需要自我检查清单或他人反馈；LLM 侧通过验证循环和一致性检查（如传递性、可逆性）确保输出可靠（来源：Measuring the Inconsistency of Large Language Models in Preferential Ranking）。
4. **动机与注意力**：ADHD 侧神经可塑性依赖动机和注意力（来源：Harnessing neuroplasticity for clinical applications）；LLM 侧可通过奖励信号或注意力机制引导生成。
5. **干扰控制训练**：借鉴 LLM 干扰控制机制，ADHD 侧可设计训练提升抑制无关信息能力（来源：Human-like Working Memory Interference in Large Language Models）。
6. **自主发现策略**：ADHD 侧可学习 LLM 提示策略，通过结构化提示辅助自主组织（来源：Working Memory Identifies Reasoning Limits in Language Models）。
7. **门控模拟**：利用 LLM 门控机制启发 ADHD 执行功能训练（来源：TRANSFORMER MECHANISMS MIMIC FRONTOSTRIATAL GATING OPERATIONS WHEN TRAINED ON HUMAN WORKING MEMORY TASKS）。
8. **自动模式管理**：ADHD 侧通过外部监督减少自动错误；LLM 侧通过验证循环减少幻觉（来源：Automatic Minds: Cognitive Parallels Between Hypnotic States and Large Language Model Processing）。
9. **灵活工具设计**：借鉴 AI 工具赋能经验，ADHD 工具应灵活即时，避免僵化医疗系统（来源：Executive Dysfunction by Design: A Cognitive Accessibility Analysis of AI Support vs. Healthcare Barriers）。
10. **已知任务自动化**：ADHD 侧可将已知任务自动化，节省认知资源；LLM 侧对已知任务可减少脚手架（来源：Artificial Neuropsychology: Are Large Language Models Developing Executive Functions?）。

## 类比的边界与反例

- **主观报告 vs 客观评估**：ADHD 诊断过度依赖主观报告（来源：Frontiers | Artificial intelligence in ADHD assessment），而 LLM 输出可客观验证，但评估其“内部状态”同样困难。
- **依赖与成瘾**：ADHD 侧有 stimulant 依赖风险（来源：Diagnosis and Treatment of Attention-Deficit/Hyperactivity Disorder in Children and Adolescents），LLM 侧无生理依赖，但存在工具锁定风险。
- **时间尺度**：ADHD 神经可塑性变化缓慢（来源：Harnessing neuroplasticity for clinical applications），而 LLM 脚手架可即时调整。
- **执行功能发展**：LLM 在已知任务上表现类似执行功能，但新任务有限；ADHD 患者执行功能可通过训练改善，但机制不同（来源：Artificial Neuropsychology: Are Large Language Models Developing Executive Functions?）。
- **门控机制本质**：Transformer 门控是计算涌现，而前额叶-纹状体门控是生物机制，虽功能平行但物理实现不同（来源：TRANSFORMER MECHANISMS MIMIC FRONTOSTRIATAL GATING OPERATIONS WHEN TRAINED ON HUMAN WORKING MEMORY TASKS）。
- **自动性差异**：ADHD 自动反应受情绪、动机影响，LLM 自动完成纯统计，缺乏主观体验（来源：Automatic Minds: Cognitive Parallels Between Hypnotic States and Large Language Model Processing）。

> ⚠️ 矛盾：ADHD 侧强调“脚手架帮助建造，但需自己攀登”（来源：AI Tools for ADHD），而 LLM 侧 harness 有时完全接管执行，可能削弱代理自主性。两者都需警惕“永久拐杖”陷阱。
