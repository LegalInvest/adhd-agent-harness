# 无状态与外部记忆

## 无状态与外部记忆

### ADHD 一侧
ADHD 大脑的工作记忆容量有限且不稳定，导致跨会话任务难以持续。典型表现为：每天需要重新回忆项目上下文，否则容易偏离轨道。为此，ADHD 个体常借助“第二大脑”工具——例如基于 AI 的外部记忆系统，能存储客户历史、项目背景、质量标准和个人偏好（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。这些工具提供持久记忆、清晰的下一步行动、日历支持，并减少每日重复输入上下文的摩擦（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。本质上，ADHD 大脑需要一个“永远在线”的外部记忆体来补偿内部工作记忆的波动（来源：How AI Helps ADHD Brains Work Smarter, Not Harder）。

### LLM/harness 一侧
标准的 LLM 调用是无状态的：每次对话都是独立的，模型不保留跨会话的记忆（来源：The Anatomy of an AI Agent: Memory, Tools, Planning, and Execution Explained - DEV Community）。没有 harness 的 LLM 只是一个无状态的 API 调用（来源：Building an AI Agent Harness from Scratch: The Architecture Between LLM and Agent - DEV Community）。生产级 agent 系统通过外部记忆（如向量数据库）解决溢出问题（来源：The Anatomy of an AI Agent: Memory, Tools, Planning, and Execution Explained - DEV Community），并组合使用会话内上下文记忆、长期检索向量库和结构化知识库（来源：The Anatomy of an AI Agent: Memory, Tools, Planning, and Execution Explained - DEV Community）。检索增强和工具调用将决策锚定在外部证据和持久状态上（来源：AI Agent Systems: Architectures, Applications, and Evaluation）。

### 同构对应
| ADHD 大脑 | LLM/Agent 系统 |
|-----------|----------------|
| 工作记忆缺陷，无法可靠保留跨会话上下文 | 无状态 API 调用，不保留跨会话记忆 |
| 依赖外部第二大脑（如 AI 工具）存储项目历史、偏好 | 依赖外部记忆（向量数据库、知识库）存储长期信息 |
| 需要低摩擦的上下文恢复机制（如自动加载项目文件） | 需要检索增强（RAG）和工具调用以绑定证据 |
| 使用“第二大脑”作为持久记忆层（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar） | 使用向量存储作为长期记忆层（来源：The Anatomy of an AI Agent: Memory, Tools, Planning, and Execution Explained - DEV Community） |

两者都面临相同的核心挑战：**生成核心（大脑/LLM）本身不具备可靠的跨会话状态管理，必须由外部系统补偿**。

### 工程启示（可迁移的做法）
1. **为 ADHD 设计外部记忆系统**：模仿 agent 的检索增强架构，为 ADHD 个体构建“第二大脑”工具，自动保存和恢复上下文，减少认知负荷。
2. **低摩擦上下文恢复**：像 agent 的会话内记忆一样，ADHD 工具应提供“一键重启”功能，避免每日重复输入（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。
3. **组合记忆层次**：借鉴 agent 的混合记忆策略（会话记忆+向量库+知识库），为 ADHD 设计分层记忆系统：短期任务列表、长期项目档案、个人规则库。
4. **可审计的推理过程**：agent 使用“草稿板”或思维链记录推理（来源：The Anatomy of an AI Agent: Memory, Tools, Planning, and Execution Explained - DEV Community），ADHD 个体可借助外部工具记录决策过程，减少遗忘。

### 类比的边界与反例
> ⚠️ 矛盾：ADHD 大脑的工作记忆缺陷是神经生物学层面的，而 LLM 的无状态性是设计选择（可通过微调或记忆模块改变）。ADHD 无法通过“升级硬件”修复，但 LLM 可以通过架构改进（如 Transformer 的上下文窗口扩展）部分缓解无状态问题。

此外，ADHD 的外部记忆工具通常需要主动使用和习惯养成，而 agent 的外部记忆是自动集成的。类比在“补偿机制”层面有效，但在“根本原因”和“解决方案可修改性”上存在差异。
