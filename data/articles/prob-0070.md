# Goblin Tools 之于 ADHD，是脚手架还是拐杖？

> 从「同构」视角评测 Goblin Tools：用久了会让 ADHD 更强还是更依赖？

## 问题：边做边忘的循环

你是否经历过这样的场景：打开浏览器想查资料，却突然忘了要查什么；走进房间，却想不起要拿什么；刚读完一段文字，转头就忘了内容。对 ADHD 人群来说，这种“工作记忆容量小、边做边忘”是日常。与此同时，LLM 工程师也面临一个类似困境：大语言模型在每次对话中都是“无状态”的，它记不住上一轮说了什么，必须靠向量数据库或外部记忆模块来“续命”。

这两个问题看似无关，实则同构。ADHD 大脑与 LLM 都是“高产但缺执行调度层的生成核心”——前者能产生无数创意，后者能生成流畅文本，但都缺乏一个稳定的“执行调度层”来管理上下文、启动任务、维持时序。因此，AI 工具对 ADHD 的帮助，本质上是在为这个生成核心套上 harness（脚手架），即通过外部执行功能层补偿内在的调度缺陷（来源：ADHD 的 AI 工具全景）。

## 同构：无状态与外部记忆

ADHD 的工作记忆容量保留但控制弱，容易被干扰；LLM 同样有强记忆容量但弱上下文控制，容易在长对话中“跑偏”（来源：ADHD 大脑与 LLM 的同构）。两者的解法结构同构：都需要外部记忆系统来持久化上下文。

对 ADHD 来说，工具如 Saner.AI 提供“知识回忆”功能，通过本地记忆存储和快速检索，减少搜索循环和标签切换（来源：Saner.AI）。对 LLM 来说，向量数据库和检索增强生成（RAG）实现类似效果。Mem 的“AI 第二大脑”概念正是这种同构的产物——它同时服务于 ADHD 用户和需要持久记忆的 AI 系统（来源：ADHD 的 AI 工具全景）。

## Goblin Tools：脚手架还是拐杖？

Goblin Tools 的核心功能是 Magic ToDo，能将复杂任务自动分解为小步骤，例如“整理房间”变成“捡起地板上的衣服”“擦桌子”等（来源：Goblin Tools）。用户报告称这降低了启动焦虑（来源：ADHD 的 AI 工具全景）。这看似是完美的脚手架：它补偿了工作记忆不足，让任务变得可执行。

但问题在于：**用久了，你会不会失去自己分解任务的能力？** 如果每次都需要工具帮你拆解，你可能会越来越依赖它，就像拐杖一样离开就无法行走。这正是“脚手架 vs 拐杖”的边界：脚手架是暂时支撑，让你最终能自己站立；拐杖是永久替代，让你永远需要它。

从同构视角看，LLM 工程师也面临类似选择：是让 agent 每次都依赖外部检索，还是教会它“记住”并复用策略？前者快速但导致依赖，后者需要更多训练但更可持续。

## 证据与局限

现有证据主要来自用户报告和工具设计逻辑，缺乏大规模随机对照试验（来源：矛盾与存疑）。Goblin Tools 被评价为“简单有用”（来源：Goblin Tools），但个体差异大——有人觉得分解步骤太细反而烦琐，有人则离不开它（来源：任务启动）。

更关键的是，过度依赖 AI 工具可能削弱内在能力。例如，Otter.ai 减轻笔记负担，但过度依赖可能削弱主动记笔记的能力（来源：ADHD 的 AI 工具全景）；时间管理工具可能削弱内在时间感知（来源：时间盲）。这些警告与“拐杖与脚手架”概念一致：关键在于避免成为永久拐杖（来源：拐杖与脚手架）。

## 核心判断：Goblin Tools 是脚手架，但需要主动使用

我的核心观点是：**Goblin Tools 本身是中性的，它成为脚手架还是拐杖，取决于使用者是否主动内化其分解策略。** 如果你只是被动接受它给出的步骤，它就变成了拐杖；如果你观察它如何分解任务，并尝试自己先分解再对比，它就变成了脚手架。

这个判断与 ADHD 干预的黄金原则一致：工具应促进能力发展，而非替代。同时，它也呼应了 LLM agent 设计中的“微调 vs 检索”之争：微调让模型内化知识，检索则依赖外部库。最佳实践往往是混合模式。

## 今天就能试的行动

1. **用 Goblin Tools 分解一个你拖延已久的任务**，但先自己列出前 3 步，再对比工具的分解，看看差异在哪里。这能帮你学习分解策略。
2. **设置一个“无工具日”**：每周选一天，不用任何 AI 工具，强迫自己手动分解任务。这能测试你的依赖程度。
3. **用 Saner.AI 或 Mem 记录你的“遗忘模式”**：记录你经常忘记什么，然后让工具帮你建立外部记忆。但每周回顾一次，看能否减少依赖。
4. **与 LLM agent 类比**：如果你在用 ChatGPT 或 Claude 辅助工作，尝试在对话开始时明确告诉它“请记住这个上下文”，观察它是否比默认状态更有效。这能让你理解外部记忆的价值。

## 结语

Goblin Tools 不是魔法，它是 ADHD 大脑与 LLM 同构的产物。它既可以是脚手架，也可以是拐杖。关键在于你如何使用它——是让它成为你的一部分，还是让它替代你的一部分。记住：工具是 harness，不是义肢。
