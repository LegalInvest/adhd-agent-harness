# Mem 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 Mem：它到底补上了哪一层执行功能？

## 问题：你存了那么多，为什么还是失控？

你用过 Mem 吗？——那个号称「第二大脑」的 AI 笔记工具，自动抓取你的聊天记录、会议转录、随手想法，然后用向量搜索让你随时找回。ADHD 用户爱它，因为终于不用每天重新回忆项目上下文了（来源：无状态与外部记忆）。但用着用着，你发现不对劲：Mem 里堆满了笔记，可你每天打开它，依然不知道该先做什么。任务还是丢，目标还是漂。

这就像 LLM agent 做长程任务：你给它一个超长的 system prompt，里面塞满了历史上下文、规则、偏好。它记住了所有，但推理到第 50 步时，它忘了最初的目标，开始胡编乱造。工程师们管这叫「上下文膨胀与推理退化」（来源：上下文工程）。

问题出在哪？——你只解决了「记忆」，没解决「重锚定」。

## 同构：ADHD 大脑与 LLM 共享同一个缺陷

ADHD 大脑被描述为「高产但缺执行调度层的生成核心」：你的工作记忆不稳定，像 LLM 的无跨会话状态，每天醒来都得重新加载上下文（来源：无状态与外部记忆）。LLM agent 也一样：它的上下文窗口有限，控制逻辑分散在框架默认值里，任务一长就漂移（来源：上下文工程）。

两边都需要一个「外部调度器」——不是存更多，而是定时把注意力拉回目标。曾国藩的日课十二条就是这个调度器：黎明即起、读书不二、谨言、写日记反省……他用最笨最稳的「结硬寨打呆仗」抵消自己的冲动（来源：人物案例）。这套 harness 的核心不是记忆，而是每天多次的「重锚定」——类似于 LLM agent 中每 N 步插入一个「re-grounding prompt」来检查当前任务是否偏离原始目标。

张旭则是另一个极端：他靠酒进入超专注状态，把所有情绪注入草书，但对其他事一概不关心（来源：人物案例）。他的 harness 是「以酒和草书为核心」，本质上是一种「环境工程」——主动设计上下文，让注意力只流向一个方向。这对应 LLM 侧的「上下文工程」：通过精心设计的提示词和外部工具来管理「当下注意什么」（来源：上下文工程）。

## Mem 到底补上了哪一层？

Mem 补的是「无状态」层——它解决了「记不住」的问题。但它没有补「重锚定」层——它不会主动告诉你「现在该做什么」。ADHD 用户和 LLM agent 真正缺的不是存储，而是「执行调度层」：一个能在你超聚焦跑偏时拉你回来，在你分心走神时推你一把的外部结构。

看看现有的工具怎么补这一层：
- **RescueTime** 自动追踪时间并阻断分心网站，直接补偿「时间盲」与执行功能缺陷（来源：RescueTime）。它不存你的笔记，但它告诉你「时间到底去了哪里」——这是重锚定的第一步。
- **Focusmate** 通过虚拟身体在场提供外部问责，弥补内在多巴胺调节不足，帮你启动和维持任务（来源：Focusmate）。它的 AI 匹配算法优化了配对效率，但核心机制是「社会压力」——一种外部调度脉冲。
- **Brain.fm** 用 AI 生成神经锁相音乐，影响大脑信息处理方式，帮你进入专注状态（来源：Brain.fm）。但它缺乏 ADHD 特异性研究，效果因人而异。
- **Forest** 用游戏化番茄钟增强动机，利用多巴胺机制让你专注（来源：AI 与 ADHD 的专注力）。

这些工具的共同点是：它们都提供「定时或事件驱动的重锚定」——要么是时间追踪的反馈，要么是同伴的注视，要么是音乐的状态切换。而 Mem 没有这个机制。它只是一个被动的存储库。

## 脚手架 vs 拐杖：你的 Mem 用对了吗？

「拐杖与脚手架」概念警告：AI 工具既可能成为永久依赖的拐杖，也可能成为逐步内化的脚手架（来源：矛盾与存疑）。Mem 如果只用做存储，就会沦为拐杖——你越存越依赖，但执行功能毫无长进。真正的用法是把 Mem 改造成「重锚定系统」：

1. **用 Mem 存储「日课」模板**：像曾国藩一样，每天在 Mem 里跑一遍「今天最重要的三件事」「当前项目目标」「上次卡在哪」。这不是记录，而是 re-grounding。
2. **设置定时提醒**：让 Mem 每小时弹出一个问题：「你现在做的事还在主目标上吗？」——这相当于 LLM agent 的「上下文压缩与重聚焦」。
3. **结合 RescueTime 或 Focusmate**：让 Mem 成为「记忆层」，让其他工具成为「调度层」。不要指望一个工具解决所有问题。

## 诚实面对争议与局限

必须承认：ADHD 大脑与 LLM 的同构目前只是功能类比，并非经过验证的科学事实（来源：矛盾与存疑）。我们没有证据证明大脑的神经机制与 transformer 架构有直接对应。另外，大多数 ADHD 工具缺乏大规模随机对照试验（来源：AI 与 ADHD 的专注力）。Brain.fm 在 44 款应用测试中位列最佳 9 款，但独立临床证据仍有限（来源：Brain.fm）。Focusmate 的效果主要依赖用户报告（来源：Focusmate）。

更重要的是，个体差异巨大：注意缺陷为主型 vs 多动冲动型对同一工具的反应可能迥异（来源：AI 与 ADHD 的专注力）。Mem 对某些人可能是救命稻草，对另一些人只是另一个数据坟墓。

## 今天就能试的行动

1. **清理你的 Mem**：删除所有超过两周没碰过的笔记。只保留当前项目上下文和日课模板。存储越少，重锚定越容易。
2. **设置每小时提醒**：在手机或电脑上设一个闹钟，每整点问自己：「我现在做的事，是我今天最重要的事吗？」——这是最简单的重锚定。
3. **试用 Focusmate 一次**：预约一个 50 分钟的时段，和陌生人一起工作。体验「身体在场」带来的启动推力。
4. **写一条「目标声明」贴在屏幕边缘**：用一句话写下你今天的主目标。每次分心时，读一遍。这比任何工具都便宜。

记住：工具是脚手架，不是拐杖。重锚定才是 ADHD 和 LLM agent 共同的解药。
