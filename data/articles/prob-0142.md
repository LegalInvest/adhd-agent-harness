# Mem 之于 ADHD，是脚手架还是拐杖？

> 从「同构」视角评测 Mem：用久了会让 ADHD 更强还是更依赖？

## 问题：你的工作记忆，是不是每天都在“断片”？

ADHD 的大脑像一台没有内存的超级计算机——创意源源不断，但下一秒就忘了刚才要干什么。你打开浏览器想查资料，却被通知吸引，半小时后发现自己站在厨房里，手里拿着遥控器。这种“边做边忘”的日常，根源在于工作记忆容量小，而外部世界又充满了“上下文切换”的陷阱。

LLM agent 的工程师们对此再熟悉不过：大模型本身没有跨会话状态，每次对话都是“失忆”的。为了让 agent 完成复杂任务，必须给它挂上外部记忆——向量库、缓存、提示模板——否则它会在第三步就忘记第一步的目标。这种“无状态与外部记忆”的结构，与 ADHD 大脑的困境惊人地同构（来源：ADHD 的 AI 工具全景）。

那么，Mem 这类“永久记忆”工具，对 ADHD 患者来说，究竟是帮助大脑搭建脚手架，还是让人彻底放弃自己记忆的拐杖？

## 同构证据：两边都在用外部记忆“续命”

### ADHD 侧：记忆工具如何减少搜索循环

ADHD 患者常陷入“搜索循环”：找文件→被邮件吸引→回消息→忘了要找什么。Mem 和 Saner.AI 这类工具，通过持久记忆和快速检索，直接降低认知负荷。Saner.AI 被评测为“减少搜索循环和标签切换”的关键工具（来源：Saner.AI）。用户不必记住“那个想法在哪”，只需告诉工具，它就能从本地记忆库里拉出上下文。这相当于给大脑装了一个“外挂工作记忆”，让生成核心（创造力）不必分心去管理存储。

但问题来了：如果每次忘记都依赖 Mem 提醒，大脑的“记忆肌肉”会不会萎缩？这正是拐杖与脚手架之争的核心。

### LLM/agent 侧：外部记忆如何防止推理退化

在 harness 工程中，agent 如果依赖纯对话，会因“上下文膨胀”导致推理退化（来源：上下文工程）。有效的 agent harness 必须包含“上下文传递、记忆系统、验证循环”（来源：幻觉与验证循环）。例如，一个复杂任务分解时，agent 需要记住子步骤的中间结果，否则会重复工作或产生幻觉。Mem 这类外部记忆正是为了解决“失忆”问题——它让 agent 能跨会话保持状态，就像给 LLM 装了一个“长期工作记忆”。

工程师们深知：没有外部记忆的 agent 是“瘫痪的”；但过度依赖外部记忆，agent 会失去自主规划能力，变成“只会查库的鹦鹉”。这个边界，与 ADHD 患者面临的困境完全一致。

## 核心观点：脚手架 vs 拐杖的边界在于“可撤除性”

我的判断是：Mem 对 ADHD 的价值，取决于它是否被设计为**可逐步撤除的脚手架**，而不是永久依赖的拐杖。脚手架是临时支撑，目的是帮助大脑建立内在的执行功能；拐杖则是永久替代，让大脑放弃自我管理。

目前，多数工具（如 Mem、Saner.AI）设计为长期使用，未提及撤除机制（来源：矛盾与存疑）。Goblin Tools 的任务分解功能虽然能降低启动焦虑，但用户可能永远依赖它来“把压倒性的事情变成不压倒性的事情”（来源：Goblin Tools），而不是学会自己分解。Lex 的“单一指令触发多步骤”则更危险——它让用户跳过所有规划步骤，直接获得结果，长期可能削弱任务分解能力（来源：Lex）。

真正健康的脚手架，应该像 Routinery 那样“创建步骤序列并逐步引导”（来源：上下文工程），同时允许用户逐渐减少提示频率。或者像精心设计的提示词（prompts）那样，既提供结构，又鼓励用户自己思考（来源：上下文工程）。

## 争议与局限：证据不足，个性化缺失

必须诚实指出，目前的证据存在三大局限：

1. **缺乏长期临床研究**：现有证据多来自用户评价和概念对齐，没有随机对照试验验证 AI 工具对 ADHD 核心症状的长期改善（来源：矛盾与存疑）。我们不知道使用 Mem 一年后，工作记忆是提升了还是退化了。

2. **个性化适配不足**：ADHD 异质性极高，单一工具难以覆盖所有亚型（来源：ADHD 的 AI 工具全景）。Mem 对“遗忘型”用户可能有用，但对“冲动型”用户可能反而增加分心。

3. **隐私问题**：Mem 存储个人记忆，数据安全和用户信任问题尚未解决（来源：ADHD 的 AI 工具全景）。把大脑的“外挂”交给云服务，风险不容忽视。

## 今天就能试的行动

1. **用 Mem 记录“上下文切换点”**：每次被打断时，花 10 秒在 Mem 里记下当前任务的关键词。这既是外部记忆，也是自我监控训练——逐步减少记录频率，尝试自己回忆。
2. **模仿 agent 的验证循环**：完成一个任务后，用 1 分钟“验证”结果——比如检查是否遗漏步骤。这相当于给大脑装一个“自我批评”模块（来源：幻觉与验证循环）。
3. **设计可撤除的提示**：使用 Goblin Tools 分解任务后，尝试自己先列出步骤，再对比 AI 的输出。逐步减少依赖，直到能独立分解。
4. **建立“记忆健身房”**：每周选一天不用外部记忆工具，强迫大脑自己回忆。就像肌肉训练，适度负荷才能增强能力。

## 结语

Mem 之于 ADHD，本质是“外部记忆层”与“内在执行功能”的博弈。同构视角告诉我们：LLM agent 需要外部记忆才能有效工作，但优秀的 harness 会设计验证循环和逐步撤除机制。同样，ADHD 患者需要工具来补偿工作记忆缺陷，但最终目标应该是让大脑学会自己调度——脚手架终要拆除，拐杖只应临时使用。工具是放大器，不是替代品。
