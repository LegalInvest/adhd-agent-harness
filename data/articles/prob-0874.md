# Otter.ai 之于 ADHD，就像 planner-executor 任务分解 之于 LLM——但有人用错了

> 从同构视角实测 Otter.ai：它到底补上了哪一层执行功能？

## 问题：为什么你买了 Otter.ai，却依然启动不了任务？

你打开 Otter.ai，录下一场会议，得到一份漂亮的转录摘要。然后呢？那个摘要躺在文件夹里，像一块压在你工作记忆上的石头——你知道该处理它，但大脑一片空白，不知道从哪里开始。

这不是你的错。ADHD 大脑和 LLM 一样，是“高产但缺调度层的生成核心”（来源：AI 与 ADHD 的时间管理）。Otter.ai 擅长生成，但不擅长分解——它给了你结果，却没给你“下一步怎么做”。

## 同构：ADHD 大脑与 LLM 共享同一个缺口

先看 ADHD 侧：时间盲让你无法感知任务耗时，执行功能缺陷让你无法把“写周报”拆成“打开文档→列出要点→写第一段”。（来源：时间盲）

再看 LLM/Agent 侧：一个 agent 接到“帮我订机票”的指令，如果它只有一个生成器（LLM）而没有 planner-executor 循环，它会直接输出一堆文字，但永远不会打开订票网站。工程师们早就知道：必须给 LLM 套上 harness——用 planner 拆解步骤，用 executor 执行动作，用 memory 记录中间状态。

两边的问题一模一样：**生成核心不缺创意，缺的是把创意转化为行动的调度层。**

## 错用：Otter.ai 被当成了“录音笔”，而不是“记忆-分解”调度器

大多数人用 Otter.ai 的方式是：录音→转录→存储。这就像给 LLM 一个无限长的上下文窗口，却不给它任何工具调用——它只能生成，不能行动。

而正确的用法是：**把 Otter.ai 的输出作为 planner-executor 循环的输入。**

具体来说：
- Otter.ai 提供“外部记忆”，弥补 ADHD 的工作记忆缺陷（来源：AI 与 ADHD 的时间管理）。
- 但你需要一个分解器，把转录中的要点拆成可执行的任务。

这正是 Motion 和 Reclaim.ai 在做的事：它们自动规划日程，把“下一步做什么”的决策负担从你身上卸掉（来源：Motion；Reclaim.ai）。但 Otter.ai 本身没有这个功能，所以你必须手动或借助其他工具补上这一层。

## 正确的同构解法：给“生成核心”套上 planner-executor harness

ADHD 侧的证据：
- Tiimo 通过视觉化时间线，把抽象时间变成可见元素，直接对抗时间盲（来源：Tiimo）。
- Focusmate 通过虚拟身体加倍，提供社会性脚手架，降低启动门槛（来源：身体在场效应）。
- Todoist 的智能提醒充当数字工作记忆，弥补上下文丢失（来源：AI 与 ADHD 的时间管理）。

LLM/Agent 侧的证据：
- 工程师在构建 agent 时，必须显式定义 planner（如 ReAct 循环）、executor（如工具调用）、memory（如向量数据库）。没有这些，LLM 会“幻觉”出步骤但无法执行。
- 同理，ADHD 大脑也需要外部 planner（如 Motion 自动排程）、executor（如 Tiimo 的视觉提醒）、memory（如 Otter.ai 的转录存档）。

**关键判断：工具是“脚手架”还是“拐杖”，取决于你是否保留了撤除路径。** 脚手架的设计目的是让你最终能独立站立；拐杖则让你永远依赖。目前多数工具（如 Goblin Tools、Saner.AI）设计为长期使用，未提及撤除机制（来源：矛盾与存疑）。这意味着，如果你只是把 Otter.ai 当录音笔，你就在用拐杖；但如果你用它来训练自己的分解能力——比如每次转录后，自己尝试先拆出 3 个动作，再用工具验证——那它就是脚手架。

## 争议与局限

- **多巴胺干预的有效性**：虽然工具声称通过即时反馈调节多巴胺，但证据尚弱，且可能引发成瘾（来源：矛盾与存疑）。
- **个体差异**：ADHD 亚型不同，对工具反应可能迥异。注意力缺陷为主者可能更需要视觉调度，多动冲动者可能更需要身体加倍（来源：AI 与 ADHD 的时间管理）。
- **同构命题的实证不足**：ADHD 大脑与 LLM 的同构目前主要是概念类比，缺乏大规模临床验证（来源：矛盾与存疑）。

## 今天就能试的行动

1. **用 Otter.ai 录一次会议后，强制自己写出 3 个“下一步动作”**，而不是只存档。这相当于手动执行 planner 步骤。
2. **试用 Motion 或 Reclaim.ai 的自动排程**，观察它是否减少了你的决策疲劳。注意：初始设置需要输入所有任务，对执行功能较弱的人可能困难——可以请朋友帮忙完成第一次设置。
3. **尝试 Focusmate 一次**，体验身体在场效应。如果社交焦虑严重，可以选择“静音模式”，只共享摄像头。
4. **反思你的工具使用模式**：你是在用工具“代替”执行功能，还是“训练”执行功能？如果是前者，考虑每周一天“无工具日”，练习自己规划和启动任务。
