# 为什么用 ChatGPT 治 ADHD 的不知哪些方法有用，和给 agent 套 human-in-the-loop 监督 是一回事？

> ChatGPT 实测：同一套 harness 思路，ADHD 与 LLM 两边都成立。

## 问题：为什么用 ChatGPT 治 ADHD 的不知哪些方法有用，和给 agent 套 human-in-the-loop 监督是一回事？

你打开 ChatGPT，让它帮你规划一天。它列出一份完美清单，但你盯着屏幕，依然不知道从哪一步开始。另一边，你部署了一个 AI agent 来自动回复邮件，结果它把客户投诉回复成了促销广告——你不得不加一个“人在回路”审核层。这两件事，看似风马牛不相及，实则共享同一个底层困境：**生成能力过剩，执行调度缺失**。

ADHD 大脑与 LLM/agent，本质上是一类“高产但缺调度层的生成核心”。ADHD 患者常有丰富的创意和冲动，却难以自主组织任务、维持工作记忆（来源：Working Memory Identifies Reasoning Limits in Language Models）；LLM 同样能生成流畅文本，但受限于工作记忆干扰和近因效应，在需要多步推理时表现下降（来源：Human-like Working Memory Interference in Large Language Models）。两者都天然缺少一个内在的“执行调度器”。

## 同构解法：外部脚手架 vs. human-in-the-loop

对付这种“生成核心+缺调度”的架构，解法不是换一个更强的核心，而是加一个外部监督层。对 ADHD 患者，这个监督层是“脚手架”——如 Goblin Tools 的 Magic ToDo 将模糊任务分解为可执行步骤（来源：Harnessing Artificial Intelligence to Live Better with ADHD - CHADD），或 Saner.AI 提供本地记忆以减少搜索循环（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。对 LLM/agent，这个监督层是 human-in-the-loop——工程师在关键节点插入人工审核，防止幻觉或偏离目标。

历史人物早已实践了这种同构。明朝兵部尚书于谦，在土木堡之变后力排众议坚守北京。他冲动敢言，但以“儒家气节”为核心，严格操练军队、日夜准备防务，写《石灰吟》明志。他的“日课”相当于定时 re-grounding，而他的“秘书”（幕僚）则像外部调度器，帮他过滤信息、安排行动。同样，清末状元实业家张謇，41岁中状元后辞官办实业，一生创办20多家企业和300多所学校。他的 harness 是“父教育母实业”的信仰，加上每天写日记反省、严格管理企业。日记是外部记忆，实业救国是长期目标锚点——对应 LLM 的持久记忆和系统提示。

## 脚手架 vs. 拐杖：边界在哪里？

但这里有一个危险：过度依赖。如果 AI 工具替代了治疗或自我管理，它就从脚手架变成了拐杖（来源：AI Tools for ADHD: Boosting Productivity and Reducing Burnout）。真正的脚手架应帮助使用者“建造”，但使用者仍需自己“攀登”。同理，human-in-the-loop 如果变成“human-in-the-loop 永远不学习”，工程师就只是在给 agent 擦屁股，而不是在训练它。

矛盾在于：许多工具页面（如 Motion）强调“消除决策负担”，却未充分讨论依赖风险（来源：矛盾与存疑）。Motion 能自动排程，但若用户从不手动调整，时间感知能力可能进一步退化。同样，给 agent 加太多人工审核，会削弱其自主性，甚至让工程师沦为“人工补丁”。

## 今天就能试的行动

1. **用 Goblin Tools 的 Magic ToDo 分解一件让你拖延的事**：输入“清理厨房”，调节“辣度”到合适粒度，然后只做第一步——比如“把碗放进水槽”。这一步相当于给 agent 一个最小可行步骤。
2. **建立“外部记忆”习惯**：用 Saner.AI 或 Obsidian 记录每天的项目上下文，避免重复输入。这就像给 agent 一个持久 memory 层，减少每次对话的 re-grounding 成本。
3. **设置“人在回路”检查点**：无论用 ChatGPT 写邮件还是 agent 处理任务，在发送前加一个“审核步骤”——问自己“这个回复是否符合我的目标？”这相当于 human-in-the-loop 的轻量版。
4. **警惕“万能工具”幻觉**：没有工具能替代你内部的执行功能。每周反思一次：这个工具是在帮我建造，还是让我更依赖？如果是后者，减少使用频率。

## 局限与争议

必须诚实指出：ADHD 大脑与 LLM 的同构性目前仅是理论类比，缺乏实证基础（来源：矛盾与存疑）。不同页面有时将其作为事实，有时作为假设，存在不一致。此外，身体在场效应（如 Focusmate 的虚拟共视）能否替代真实人际互动仍有争议。我们的论证基于已有研究，但读者应保持批判态度。

最终，无论是 ADHD 患者还是 AI 工程师，核心问题都不是“用什么工具”，而是“如何设计一个可持续的外部调度层”。工具只是脚手架，真正的攀登者是你自己。
