# Tiimo 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 Tiimo：它到底补上了哪一层执行功能？

## 问题：为什么你的 Tiimo 越用越焦虑？

你下载了 Tiimo，设好每个时段，震动响起时却只是机械地划掉——然后继续刷手机。或者，你正超聚焦于一个无关细节，Tiimo 的提醒像蚊子一样被拍掉。更糟的是，你开始依赖它，没了提醒就彻底失序。

这不是 Tiimo 的错。它暴露了一个更深层的结构问题：ADHD 大脑与 LLM 一样，是「高产但缺执行调度层的生成核心」（来源：AI 与 ADHD 的专注力）。Tiimo 如果只做定时提醒，就像只给 LLM 加一个闹钟，而不修复其上下文膨胀和目标漂移——它当然会失效。

## 同构脊柱：重锚定 vs. 目标漂移

ADHD 侧痛点：分心走神或超聚焦跑偏，丢失目标。LLM/Agent 侧对应：长程任务里 agent 会目标漂移，需要重锚定 system prompt。

真实证据：
- **ADHD 侧**：超聚焦工作时长，ADHD 组平均 14.5 小时 vs 对照组 8.2 小时（来源：Pace Ventures Research 2026）。但这种超聚焦是双刃剑——一旦跑偏，深度沉浸反而加速偏离目标。
- **LLM/Agent 侧**：长对话中，LLM 因上下文膨胀导致推理退化，忘记初始指令，需要定期重注入 system prompt（例如每 N 轮对话重新锚定）。

两者的结构同构在于：都拥有强大的生成能力，但缺乏可靠的内置执行调度层，因此都需要外部脚手架来弥补调度缺陷（来源：AI 与 ADHD 的专注力）。

## 曾国藩与彭玉麟：两个历史 harness 案例

**曾国藩**（晚清）——典型注意力不集中型 ADHD：30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”；读书慢，“他人目下二三行，余或疾读不能终一行”。他的 harness：日课十二条（黎明即起、读书不二、谨言、写日记反省等），“结硬寨打呆仗”。

**彭玉麟**（晚清）——冲动型特质：性格刚直，敢杀李鸿章的侄子；一生只爱梅姑，画梅花画了一辈子。他的 harness：每天画梅花磨练心性，严格军纪，“三不要”原则（不要官、不要钱、不要命）。

这些历史人物与 LLM harness 的同构对应：
- **日课 ↔ 定时 re-grounding**：曾国藩的每天反省，相当于 LLM 每轮对话后重新注入 system prompt。
- **画梅花 ↔ 外部调度器**：彭玉麟通过重复性行为锚定注意力，相当于 agent 中一个独立的调度模块定期检查目标是否漂移。

关键点：这些 harness 不是被动提醒，而是主动的结构化调度——它们改变了行为模式，而非仅仅添加外部刺激。

## Tiimo 用错了：脚手架 vs. 拐杖

当前 ADHD 工具生态中存在一个关键矛盾：

**拐杖与脚手架**：AI 工具既可能成为永久依赖的拐杖，也可能成为逐步内化的脚手架（来源：AI 与 ADHD 的专注力）。Tiimo 如果只提供定时提醒，用户就会依赖外部信号，而不发展内部时间感知——这就是拐杖。

**证据**：多数工具（如 Brain.fm、Focusmate、RescueTime）的证据来自用户报告或小样本研究，缺乏大规模随机对照试验（来源：AI 与 ADHD 的专注力）。例如 Brain.fm 虽在 2026 年一项 44 款 ADHD 应用测试中位列最佳 9 款之一，但 ADHD 人群独立临床证据仍有限（来源：Brain.fm）。

**我的判断**：Tiimo 的正确用法不是作为“定时器”，而是作为“重锚定 system prompt”——每次提醒时，用户应该停下来，重新确认当前目标，就像 LLM 重新读取 system prompt。如果只划掉不思考，工具就失效了。

## 局限与争议

1. **同构的边界**：ADHD 大脑与 LLM 的同构是功能类比，还是存在更深层的神经机制对应？目前缺乏实证基础（来源：矛盾与存疑）。
2. **依赖风险**：过度依赖外部脚手架可能阻碍神经可塑性——即大脑通过重组结构来适应新行为的能力（来源：神经可塑性）。目前没有长期追踪研究。
3. **个体差异**：ADHD 的异质性意味着同一工具对不同亚型效果迥异（来源：AI 与 ADHD 的专注力）。

## 今天就能试的行动

1. **Tiimo 升级用法**：每次提醒后，花 10 秒写下当前目标（一句话），再继续。这模拟了 LLM 的“重锚定”。
2. **Focusmate 替代方案**：如果 Tiimo 无效，试试 Focusmate——虚拟身体在场通过同伴问责提升启动与专注力（来源：Focusmate）。
3. **RescueTime 检查**：安装 RescueTime，每天看一次时间报告，识别注意力漂移模式（来源：RescueTime）。
4. **曾国藩式日课**：选一件小事（如每天写 50 字日记），固定时间做，作为你的“定时 re-grounding”。
