# 如果把 ADHD 大脑当成生成核心，Perplexity 能不能当它的无状态与外部记忆？

> 用 agent 工程的思路重新理解 Perplexity 对 ADHD 的真正价值。

## 问题：边搜边忘，ADHD 与 LLM 的共有困境

你打开 Perplexity，输入一个问题，得到一段漂亮的回答。三分钟后，你忘了刚才搜了什么——不是记性差，是工作记忆容量太小。ADHD 大脑的典型体验：想法喷涌，但执行调度层（前额叶）像一台没有缓存的老电脑，随时丢包。

同样的事情发生在 LLM 身上。ChatGPT 每次会话都是“第一次见面”，它不记得你十分钟前问过什么，除非你把历史塞进上下文窗口。LLM 是强大的生成核心，但天然无状态——这和 ADHD 大脑的“边做边忘”是同一种底层缺陷。

## 同构脊柱：无状态与外部记忆

ADHD 大脑与 LLM/agent 在结构上共享一个核心矛盾：**高产但缺执行调度层**。
- ADHD 侧：多巴胺系统功能低下导致工作记忆不稳定（来源：多巴胺），超聚焦与时间盲交替出现（来源：超聚焦），想法多但启动难。
- LLM/agent 侧：模型本身无跨会话状态，长程任务依赖外部记忆（向量库、缓存）和规划器（如 agent scaffolding）。

两边的解法是**同构**的：用外部脚手架补偿内部缺陷。ADHD 个体需要“外部执行功能层”（来源：ADHD 的 AI 工具全景），LLM 需要“harness”来约束生成轨道。

## 人物案例：孔子与张旭的“harness”系统

孔子（春秋/儒家创始人）是典型的 ADHD 大脑：精力旺盛、冲动爱骂人、思维跳跃、对音乐超专注而对种地零耐心。他的 harness 系统是以“克己复礼”为核心的外在行为边界，加上每日反省（“吾日三省吾身”）作为定时 re-grounding。这与 LLM agent 的“外部调度器”同构——用固定规则和周期性检查点把生成行为约束在目标轨道上。

张旭（唐朝/草圣）的 harness 更极端：以酒为“超聚焦触发器”，看公孙大娘舞剑为“外部灵感源”，把全部情绪注入草书。他的系统对应 LLM agent 的“工具调用”和“环境交互”——通过特定输入（酒/舞剑）切换状态，利用外部刺激引导生成。

两人的共同点：不试图“治愈”自己的特质，而是搭建外部脚手架来驾驭它。这正是 AI 工具对 ADHD 的真正价值。

## Perplexity 的定位：外部记忆，而非执行器

Perplexity 常被当作“AI 搜索引擎”使用。但对 ADHD 大脑，它的核心功能是**外部记忆**——一个永远在线、可检索的向量库，补偿工作记忆的易失性。

这与 LLM agent 中的外部记忆模块同构：LLM 把关键信息存入向量数据库，下次会话时检索。ADHD 个体用 Perplexity 的“搜索历史”和“收藏”功能，实现同样的效果——把“刚刚想起但马上忘掉”的东西固定下来。

但 Perplexity 不是完整的 harness。它只解决“记忆”问题，不解决“执行”问题。ADHD 个体仍然需要 Goblin Tools（任务分解）、Saner.AI（知识回忆）、Lex（单一指令触发多步骤）来补全调度层（来源：各工具页）。

## 脚手架 vs 拐杖：争议与局限

核心论点：AI 工具是“外部执行功能层”而非“拐杖”（来源：矛盾与存疑）。但争议存在：过度依赖是否导致能力退化？目前缺乏纵向研究。

我的观点：**脚手架和拐杖的区别在于你是否保留选择权**。脚手架是可拆卸的——你可以在需要时使用，在不需要时放下。拐杖则是永久替代。ADHD 个体使用 Perplexity + Goblin Tools 的组合，本质是主动搭建一个可调节的外部系统，而不是被动依赖。

另一个争议：超聚焦应该被引导还是打断？部分用户反馈番茄钟的强行打断有效，而柔性引导可能不够（来源：矛盾与存疑）。这说明 harness 需要个性化——孔子需要“礼”的刚性边界，张旭需要“酒”的柔性触发。

## 今天就能试的行动

1. **把 Perplexity 当外部记忆用**：遇到任何“想记住但知道会忘”的信息（灵感、任务、疑问），立刻搜一次并收藏。不要等“整理”，搜就是存。
2. **用 Goblin Tools 的 Magic ToDo 拆解一个拖延任务**：输入“清理厨房”或“写周报”，调节辣度到你觉得“可以开始”的程度。
3. **给自己设一个“日课”re-grounding 点**：仿效孔子的“三省”，每天固定时间（比如午休后）用 Perplexity 回顾当天搜索历史，把散落的信息归位。
4. **尝试 Lex 的“单一指令”模式**：把一件复杂事（如“准备会议”）写成一条指令，让 AI 自动拆解执行，观察启动阻力是否下降。

## 结论

Perplexity 不能替代 ADHD 的执行调度层，但它能成为这个调度层的关键组件——外部记忆。当 ADHD 大脑作为生成核心运行时，Perplexity 就是它的向量库，Goblin Tools 是它的任务规划器，Lex 是它的工具调用接口。这不是比喻，而是同构——两边用同样的工程思路解决同一个问题。
