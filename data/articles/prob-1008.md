# 用 Claude 给 ADHD 补上重锚定与目标漂移，真的能像 agent 那样工作吗？

> 把 Claude 当作 ADHD 的外部执行层来实测：它补的是哪一块短板？

## 同一种痛：目标漂移

如果你有 ADHD，你一定经历过这样的时刻：打开电脑准备写一份报告，却被一封邮件带偏，半小时后发现自己正在研究某款新游戏的攻略。你并没有“不想工作”，你的大脑只是像一艘没有舵的船，被每一个浪头带向不同方向。

如果你在做 Agentic Harness 工程，你一定也见过这样的场景：一个 LLM agent 被赋予“整理用户邮件并生成周报”的任务，却在执行过程中不断被无关的上下文吸引，最终产出一份跑偏的摘要。你试图通过更精细的 system prompt 来“重锚定”它，但它依然会漂移。

这两种痛，本质上是同一种痛。ADHD 大脑与 LLM 共享一个核心缺陷：**两者都是高产但缺乏可靠执行调度层的生成核心**（来源：AI 与 ADHD 的专注力）。ADHD 大脑的创意发散、LLM 的文本生成，都像一台功率强劲但缺少方向盘的引擎。而“目标漂移”——无论是分心走神、超聚焦跑偏，还是 agent 的上下文膨胀——正是这种缺陷最直接的体现。

## 重锚定：给生成核心套上 harness

解决目标漂移的关键，是**重锚定**。对 LLM agent 来说，这意味着在 system prompt 中嵌入动态的“当前目标”字段，定期将 agent 拉回主轨道。对 ADHD 大脑来说，这意味着需要一个外部系统来不断提醒“你现在该做什么”。

这正是 AI 工具可以发挥作用的领域。它们充当**外部执行功能层**（来源：AI 与 ADHD 的专注力），像 agent harness 一样，为 ADHD 大脑提供任务启动、时间感知和注意力维持。

以 **Focusmate** 为例。它通过实时视频匹配一位伙伴，双方约定在固定时段内共同工作。这种“身体在场效应”提供了外部问责，相当于给 ADHD 大脑装了一个“同伴监督”的 system prompt（来源：Focusmate）。**RescueTime** 则自动记录时间使用，生成可视化报告，帮助用户识别注意力漂移模式，弥补时间盲缺陷（来源：RescueTime）。**Motion** 和 **Reclaim.ai** 利用 AI 自动规划日程，并根据实际完成情况调整时间估计（来源：时间盲），这类似于 agent 的“动态重规划”能力。

这些工具的共同逻辑是：**将抽象的目标转化为具体的、可感知的外部信号**。对 LLM 来说，那是 system prompt 中的“当前任务”；对 ADHD 大脑来说，那是 Focusmate 伙伴的注视、RescueTime 的弹窗、Motion 自动调整的日程。

## 脚手架 vs 拐杖：边界在哪里？

然而，这种同构也带来一个危险的问题：**过度依赖外部系统是否会削弱内在能力？**

在 LLM 领域，如果 agent 完全依赖外部 prompt 来维持方向，一旦 prompt 缺失或出错，它就会彻底迷失。在 ADHD 领域，如果一个人长期依赖 Focusmate 才能启动任务、依赖 RescueTime 才能感知时间，那么他的内在执行功能是否会进一步退化？

这就是“脚手架 vs 拐杖”的边界问题。脚手架是暂时性的支持，旨在帮助用户最终独立完成任务；拐杖则是永久性的替代，用户一旦离开就无法行动。当前，AI 工具的设计者声称自己是“脚手架”，但实际使用中可能沦为“拐杖”。例如，**Brain.fm** 通过神经锁相音乐帮助专注，但缺乏针对 ADHD 的独立临床研究（来源：Brain.fm），用户可能形成对特定声音的依赖。**Focusmate** 的效果高度依赖配对伙伴的可靠性（来源：Focusmate），用户可能无法将这种外部问责内化为自我管理能力。

现有证据也表明，AI 工具的实际效果尚未得到严格验证。多数工具仅有用户报告或小样本研究，缺乏大规模随机对照试验（来源：矛盾与存疑）。**个体差异**同样显著：同一工具对不同 ADHD 亚型效果迥异（来源：矛盾与存疑），就像同一个 system prompt 对不同 LLM 模型效果不同一样。

## 同构的局限：隐喻不等于等价

尽管 ADHD 大脑与 LLM 在目标漂移问题上存在同构，但我们必须承认这种类比是**隐喻而非严格等价**。LLM 的“注意力机制”基于 transformer 架构的数学运算，而人类注意力涉及多巴胺、前额叶皮层等复杂的神经生物学过程（来源：AI 与 ADHD 的专注力）。多巴胺失调是 ADHD 的核心特征之一，它影响动机、奖赏和运动控制（来源：多巴胺），而 LLM 没有多巴胺系统。

因此，AI 工具不能完全“治愈”ADHD，它只能补偿某些执行功能短板。正如一个 agent harness 不能解决 LLM 的底层能力问题，AI 工具也不能替代药物治疗或行为疗法。

## 今天就能试的行动

1. **体验一次 Focusmate**：预约一个 25 分钟的时段，和一个陌生人一起工作。感受外部问责如何改变你的任务启动能力。
2. **安装 RescueTime 并运行一周**：不要手动调整分类，先观察自动报告如何揭示你的注意力漂移模式。
3. **尝试 Motion 或 Reclaim.ai 的自动日程**：让 AI 为你规划一天的任务，观察它是否减少了你的决策负担和时间盲。
4. **设置一个“重锚定”提醒**：用手机闹钟每 30 分钟问自己一次“我现在应该做什么？”——这是最简单的外部 system prompt。

## 结语

ADHD 大脑与 LLM agent 共享同一个痛点：强大的生成能力与脆弱的执行调度层之间的断裂。AI 工具作为外部 harness，确实能帮助重锚定，但它的有效性取决于是否精准对应你的特定短板，以及你是否在使用中保持“脚手架”意识。记住：**工具是让你更自由，而不是让你更依赖**。
