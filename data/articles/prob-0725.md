# 用 ChatGPT 给 ADHD 补上工具使用与认知卸载，是脚手架还是拐杖？

> ChatGPT 接进 ADHD 工作流：哪些交给它、哪些必须自己留着。

## 问题：ADHD 的“执行调度层”去哪了？

如果你是一个 ADHD 成年人，你一定熟悉这样的场景：脑子里有一堆好想法，但就是无法把它们变成精确、重复、有条理的行动。你擅长生成，却不擅长维护状态——比如定时回邮件、按步骤整理项目、估算时间。这不是懒，而是执行功能缺陷：工作记忆容量有限、任务启动困难、时间盲（来源：时间盲；任务启动）。

与此同时，如果你是一个在做 Agentic Harness 工程的开发者，你也会熟悉另一种“生成核心缺调度”的困境：LLM 能写出漂亮的代码或文案，但一旦需要精确计算、多步工具调用、维护上下文状态，就必须靠 function calling 把“精确部分”外包给外部工具。LLM 本身是“高产但缺执行调度层的生成核心”。

这两类人面对的其实是同一个结构问题：一个强大的生成核心，配了一个弱鸡调度层。ADHD 大脑与 LLM 在认知架构上同构——两者都需要一个外部“脚手架”来补上执行功能（来源：ADHD 大脑与 LLM 的同构）。

## 答案：脚手架 vs 拐杖，边界在于“谁保留理解”

那么，用 ChatGPT 来补上工具使用与认知卸载，到底是脚手架还是拐杖？我的判断是：**它既可以是脚手架，也可以是拐杖，边界在于你是否保留了对自己任务的理解与反思。**

### 脚手架的证据：AI 工具作为外部执行功能层

从 ADHD 侧看，用户报告和工具设计逻辑已经提供了真实证据：

- **任务分解与启动**：Goblin Tools 的 Magic ToDo 功能能将“整理房间”这样压倒性的任务自动拆成“捡起地板上的衣服”“擦桌子”等小步骤，降低启动门槛。用户反馈称“Goblin Tools 在这方面很棒”（来源：Goblin Tools）。Lex 则允许通过单一指令触发多步骤序列，减少决策疲劳（来源：Lex）。
- **记忆与上下文管理**：Saner.AI 通过增强知识回忆减少搜索循环和标签切换（来源：Saner.AI）。Mem 作为“AI 第二大脑”提供持久化上下文，补偿工作记忆不足（来源：ADHD 的 AI 工具全景）。
- **实时反馈与反思**：Otter.ai 的实时转录和 Reflect 的结构化提问提供即时外部反馈，类似于虚拟的“身体在场效应”（来源：ADHD 的 AI 工具全景）。

从 LLM/Agent 侧看，这些工具对应的正是 Agentic Harness 的 function calling 机制：把“精确计算”外包给外部工具。例如，Goblin Tools 的分解相当于 agent 的规划器，Lex 的单指令多步骤相当于 agent 的工作流引擎，Saner.AI 的本地记忆相当于 agent 的持久化上下文存储。两边都在做同一件事：**为生成核心套上执行调度层**。

### 拐杖的风险：过度依赖可能削弱内在能力

但 wiki 资料中的“矛盾与存疑”明确警告：过度依赖 AI 工具可能削弱内在时间感知能力、主动记笔记能力等（来源：矛盾与存疑）。例如，Otter.ai 减轻笔记负担，但长期依赖可能让你失去主动总结的能力。同样，如果 agent 工程完全依赖外部工具而从不理解内部逻辑，一旦工具失效，整个系统就瘫痪。

### 边界在哪里？

脚手架 vs 拐杖的边界在于：**你是否保留了对自己任务的理解与反思**。脚手架是你在使用它时仍然知道自己在做什么、为什么这么做；拐杖是你完全交给它，不再思考。例如，用 ChatGPT 分解任务时，如果你只看结果而不理解分解逻辑，那就是拐杖；如果你边看边想“原来这个任务可以这样拆”，那就是脚手架。

## 诚实指出局限

必须承认，现有证据主要来自用户报告和概念类比，缺乏大规模随机对照试验（来源：矛盾与存疑）。个体差异很大——对某些人有效的工具对另一些人可能无效。此外，Brain.fm 的 ADHD 效果缺乏独立临床研究，Structured 的 AI 功能细节不明确，这些都可能造成不切实际的期望（来源：矛盾与存疑）。

## 今天就能试的行动

1. **用 ChatGPT 分解一个你拖延已久的任务**：输入“帮我将 [任务] 分解成 5-10 个小步骤”，然后思考每一步的逻辑，而不是直接照做。
2. **用 Goblin Tools 的 Magic ToDo 测试时间估算**：输入一个任务，看它的估算是否比你的直觉更准，然后对比实际用时。
3. **用 Saner.AI 或 Mem 建立“第二大脑”**：把你常丢失的信息（如会议笔记、灵感）存入，训练自己定期回顾，而不是完全依赖搜索。
4. **设置一个“反思提醒”**：每天用 ChatGPT 问自己“今天哪些任务依赖了 AI？我是否理解了它们？”——这是防止拐杖化的关键。

## 结语

ADHD 大脑与 LLM 是同构的生成核心，都需要一个外部调度层。ChatGPT 等 AI 工具可以成为这个调度层——但前提是你主动保留对任务的理解。脚手架帮你盖楼，拐杖让你忘记怎么走路。选择权在你手里。
