# 你说 ADHD 和 LLM 都需要外部编排——那为什么我的待办清单总是坚持不下来，而 agentic harness 就能让 LLM 稳定工作？

> 这是一个必须正面回答的质疑:同样是「外部编排」,为什么工程师的 harness 真的管用,而 ADHD 者买过的第 14 个待办 app 又双叒叕吃灰了?答案藏在一个残酷的区别里——harness 对 LLM 有强制力,而清单对你只有建议权。

先把质疑的合理性说足。ADHD 者几乎人人有一部「工具坟场史」:待办清单、番茄钟、子弹笔记、GTD、各种日程 app——每个都在头三天闪闪发光,然后被遗忘。如果「外部编排」真是答案,为什么这些编排工具全军覆没?这个经验事实太普遍了,它必须被解释,否则整个同构框架就是站不住的漂亮话。

解释的钥匙在于拆开「编排」这个词的两层含义。工程师给 agent 的 harness 有两个组件:**决策组件**(计划、排序、拆解——决定该干什么)和**强制组件**(调度器真的会在那个时刻调用模型,模型无法拒绝、无法忘记、无法「等会儿再说」)。LLM 的 harness 之所以管用,一半功劳在后者:**模型被架构包裹着,执行流是焊死的**——planner 排好的下一步,executor 到点必然执行。

而 ADHD 者的待办清单只有决策组件,没有强制组件。清单写得再好,执行仍然完全依赖你「看到它、想起它、并说服自己动」——**恰恰是 ADHD 最弱的那三个环节**。换句话说:你买的从来不是 harness,只是 harness 的说明书。清单坟场证明的不是「外部编排无效」,是「只有决策层、没有强制层的编排无效」——这个结论对 LLM 同样成立:把完美计划写进 prompt 却不接调度器,agent 照样跑飞。

## 那么,人类版的「强制组件」是什么?

人不能被真的焊进架构(也不应该),但强制力有梯度,可以逐级借用:

**梯度一:时间强制**。日历+闹钟替代清单——清单是「有空看一眼」,日历通知是「时间到了它来找你」。把待办的每一项翻译成具体时段,让时间当调度器。这已经能救活一批人。

**梯度二:社会强制**。对真人的承诺(「周三发你初稿」)、互报搭档、body doubling——社会脑对失约成本的敏感,是 ADHD 者身上最接近「焊死」的回路。给重要任务全部接上真人等待方,强制力上一个台阶。

**梯度三:环境强制**。物理环境的改造:手机锁盒、网站拦截器、只放一件工作的桌面——不是靠意志抵抗诱惑,是让诱惑物理上够不着。**环境强制的本质是把「每次都要赢的意志战」改成「只需要赢一次的架构战」**(设置的那一次)。

**梯度四:金钱/后果强制**。承诺合约类工具(违约扣钱)、预付费的课程、和朋友的对赌——给放弃装上真实代价。

回头看工具坟场,死因清晰了:那 14 个 app 全部停留在梯度零(纯信息,零强制)。而幸存者也有共性——ADHD 社群里真正流传下来的方法(body doubling、和人interlock 的截止日、实体闹钟阵),清一色是带强制组件的。

## 这个质疑还教会我们一件事

「工具无效史」在 ADHD 者心里常被记成「我的失败史」——连清单都坚持不了。架构视角平反:**坚持使用一个纯自觉驱动的工具,本身就是在考执行功能——用你考砸的科目当考试工具,不及格是设计问题,不是人格问题。** 下次选工具,先问一句:它的强制组件在哪一级?没有的,直接略过,那是给神经典型者设计的。

## 边界

同构强度 B 级:harness 的调度强制是真实架构事实,ADHD 的工具依从性困难有临床与社群大量佐证,「强制力梯度」是本文的实践框架,无对照研究。注意:梯度四(金钱后果)对伴随焦虑的个体可能弊大于利(压力放大冻结),自测慎用;工具全线失效且伴随生活功能受损的,请专业评估,药物治疗改善的正是「内在强制力」那一层。

## 今天就能试的 3 件事

1. **验尸你的工具坟场**:列出用过的三个工具,标注各自的强制组件等级。你会看到清一色的零。
2. **给明天最重要的一件事升到梯度二**:找一个真人,预约一句「明天下午三点发你」。
3. **做一次环境强制改造**:把最大的分心源(通常是手机)在工作时段物理隔离。一次设置,长期生效。

清单没能救你,不是因为外部编排是谎言,而是因为清单只是编排的一半——缺的那一半叫强制力。工程师把它焊进架构,你把它接到时间、真人、环境和后果上:说明书配上执行器,harness 才算装完。
