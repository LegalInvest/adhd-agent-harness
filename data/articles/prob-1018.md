# Reflect 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 Reflect：它到底补上了哪一层执行功能？

你打开 Reflect，准备写一篇长文。十分钟后，你发现自己正在刷手机——不是分心，而是 Reflect 的 AI 刚刚弹出一条“你最近在思考什么？”的提示，你顺着它跑偏了。这不是你的错，是工具用错了。

## 问题：AI 重锚定，为何反而让你更跑偏？

ADHD 大脑和 LLM 共享同一个深层矛盾：两者都是**高产但缺乏可靠执行调度层的生成核心**（来源：主题综述）。你的大脑创意发散如泉涌，但执行功能——任务启动、持续专注、抑制冲动——常常“无人驾驶”（来源：执行功能）。LLM 同样：它能生成流畅文本，但直接输出可能偏离目标、陷入循环（来源：人在回路与身体在场）。

于是，ADHD 群体和 Agent 工程师都想到了同一个解法：**重锚定**。对 LLM，是给 Agent 套上 harness——设置护栏、加入人工检查点、用 system prompt 持续校准（来源：人在回路与身体在场）。对 ADHD，是使用 Reflect 这类工具——定期弹出“你在做什么？”，试图把你拉回轨道。

但问题在于：**Reflect 的重锚定，常常锚错了地方**。它问的是“你在思考什么？”，而你需要的是“你当前的目标是什么？”——前者是发散，后者是收敛。这就像给 LLM 的 system prompt 写成了“生成有趣的内容”而不是“回答用户问题”，Agent 自然会漂移。

## 同构：ADHD 与 LLM 共享的执行功能短板

最新研究揭示了这种同构的神经证据。在经典 Stroop 冲突任务中，Transformer 的注意力随序列变长在冲突试次上骤降到随机水平——无法抑制优势反应、无法解决认知冲突，这与 ADHD 执行功能缺陷的核心标志一一对应（来源：执行功能）。LLM 呈现“强记忆、弱控制”剖面：工作记忆容量甚至超过常人，但认知灵活性与注意控制存在核心缺陷——无法灵活切换任务集、无法抑制自动化反应，这正是 ADHD 的经典神经心理模式（来源：执行功能）。

这意味着，ADHD 大脑和 LLM 需要的不是“更多注意力”，而是**更精准的执行调度层**。对 LLM，harness 提供的是形式化规划与护栏（来源：人在回路与身体在场）。对 ADHD，工具应该提供的是**外部执行功能层**——补偿工作记忆、时间盲、任务启动等特定短板（来源：主题综述）。

## 答案：Reflect 的正确用法——锚定目标，而非思维

Reflect 的“你在思考什么？”提示，本质上是**思维发散锚点**——它适合创意阶段，但如果你已经进入执行阶段，它反而会拉你跑偏。正确的重锚定应该是**目标锚点**：

- **ADHD 侧**：在 Reflect 中设置“当前任务目标”字段，每次弹出时只显示这个目标，而不是让你回顾思维流。这类似于 Focusmate 的外部问责——对另一个人的责任感能绕过执行功能屏障（来源：人在回路与身体在场）。但 Reflect 缺少这种社会压力，所以你需要自己设计锚定句。
- **LLM 侧**：Agent harness 的核心是“人在回路”——设置暂停点，询问后再执行（来源：人在回路与身体在场）。Reflect 的弹出机制本身是一个暂停点，但它的询问内容决定了效果。

## 证据：哪些工具真正补上了执行功能？

现有工具的证据强度不一（来源：主题综述）：
- **强证据**：Focusmate 基于身体在场效应，多项用户研究显示其能显著提升任务启动和维持。RescueTime 通过客观时间记录提升时间觉察，有初步研究支持其改善时间盲。
- **中等证据**：Brain.fm 利用神经锁相技术，有神经科学理论支持，但针对 ADHD 的独立临床研究缺乏。
- **弱证据**：Forest 作为专注工具的证据不足，尽管在 ADHD 诊断分类中表现良好。

关键在于：**工具必须精准对应特定短板**。Focusmate 对应任务启动，RescueTime 对应时间盲，Brain.fm 对应注意力维持。Reflect 如果只做思维记录，对应的是工作记忆——但它没有像 RescueTime 那样自动化，也没有像 Focusmate 那样提供外部问责。

## 争议：脚手架还是拐杖？

所有 AI 工具都面临同一个风险：长期使用是否会导致内在执行功能进一步退化？（来源：矛盾与存疑）工具设计者声称是“脚手架”，但实际使用中可能沦为“拐杖”。例如，过度依赖 RescueTime 可能削弱内在时间感知能力（来源：矛盾与存疑）。Reflect 的频繁提示可能让你失去自我监控的习惯。

边界在哪里？脚手架是**在任务中提供临时支撑，任务完成后可移除**；拐杖是**永久替代内在能力**。Focusmate 的虚拟身体搭档是脚手架——你不会因为用了它而失去独自工作的能力。但 Reflect 的自动提示如果成为你唯一的目标锚定方式，那就是拐杖。

## 今天就能试的行动

1. **改造 Reflect 的提示语**：将“你在思考什么？”改为“你当前的目标是什么？”。每次弹出时，只写一句话的目标，而不是记录思维流。
2. **组合使用 Focusmate + Reflect**：在 Focusmate 时段中，用 Reflect 记录任务目标，利用身体在场效应维持执行。
3. **设置 RescueTime 的“目标漂移”警报**：当你在非目标应用上停留超过 5 分钟时，自动弹出 Reflect 的目标锚定。
4. **每周一次“无工具日”**：选择一天不用任何 AI 工具，仅凭自我监控完成任务，检测内在执行功能是否退化。

Reflect 不是坏工具，但用错了就是锚错了地方。记住：ADHD 大脑和 LLM 一样，需要的不是更多提示，而是**指向目标的提示**。
