# 如果把 ADHD 大脑当成生成核心，Focusmate 能不能当它的无状态与外部记忆？

> 用 agent 工程的思路重新理解 Focusmate 对 ADHD 的真正价值。

## 问题：为什么 ADHD 大脑和 LLM 都卡在「启动」这一步？

ADHD 大脑的核心困境不是「不想做」，而是「想做却动不了」。工作记忆容量小、边做边忘，任务启动时大脑像一台没有操作系统的超级计算机——生成能力爆棚，执行调度层却频繁崩溃。LLM 也是类似的：无跨会话状态，每次对话都是全新开始，靠外部记忆（向量库、工具调用）才能续命。两者都是「高产生成核心」+「缺失调度层」的组合（来源：ADHD 大脑与 LLM 的同构）。

那么问题来了：Focusmate——一个让你和陌生人视频对坐、各自干活的工具——它到底在解决什么？它真的能当 ADHD 大脑的「外部记忆」和「状态管理器」吗？

## 答案：Focusmate 是 ADHD 的「外部调度器」，不是记忆库

Focusmate 不存储任何任务上下文，它不记录你上次做到哪一步，不提醒你明天要交什么。它做的是更底层的事：**定时 re-grounding 和身体在场效应**。每次 session 开始，你和搭档互相说「我要做X」，结束时汇报「我做了Y」。这个简单的「声明-执行-汇报」循环，恰好补上了 ADHD 大脑缺失的调度层——它把「启动」这个高能耗动作外包给了外部节奏。

这跟 LLM agent 的 harness 工程一模一样。LLM agent 需要外部脚手架来提供上下文管理、规划工件、验证循环和记忆系统（来源：Building AI Coding Agents for the Terminal: Scaffolding, Harness, Context Engineering, and Lessons Learned）。Focusmate 就是 ADHD 的脚手架：它不替你思考，但帮你维持「接下来该做什么」的轨道。

## 人物案例：徐渭的「艺术 harness」与 Focusmate 的同构

明朝才子徐渭（徐文长）是典型的 ADHD 大脑。他诗书画戏军全能，却执行功能极差，8 次举人不中，情绪失调到杀妻、9 次自杀。他的 harness 系统是以书画为核心，把所有痛苦和精力注入艺术创作——用军事谋略帮胡宗宪抗倭，把超专注用在打仗上。注意：他的 harness 不是「管理时间」，而是 **「把生成能力定向到单一通道」**。

这跟 Focusmate 的原理一致。Focusmate 不帮你管理所有任务，它只在你面前放一个「当下该做的事」的锚点。徐渭的日课（每日作画）相当于定时 re-grounding，而他的「秘书」（如果有的话）相当于外部调度器。Focusmate 就是那个数字化的「秘书」——它不存储你的知识，但它确保你在每个 25 分钟或 50 分钟块里，有一个外部信号告诉你「现在该回到轨道了」。

## 证据：ADHD 侧与 LLM 侧的真实数据

ADHD 侧的证据是明确的：工作记忆容量可能正常甚至超常，但认知灵活性和注意控制存在核心缺陷（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。这解释了为什么 Focusmate 有效——它不依赖你的工作记忆去记住「接下来做什么」，而是通过外部节奏（定时提醒、搭档的目视）来维持注意轨道。

LLM 侧的证据同样惊人：用经典 Stroop 冲突任务测试 Transformer 注意力，发现短上下文中表现正常，但随序列变长，模型在冲突试次上骤降到随机水平——无法抑制优势反应、无法解决认知冲突（来源：Deficient Executive Control in Transformer Attention）。这正是 ADHD 执行功能缺陷的核心标志。Focusmate 的定时 re-grounding 相当于给 LLM 的「上下文窗口重置」——每 25 分钟刷新一次注意力锚点，防止认知冲突累积到崩溃。

## 核心判断：Focusmate 是「脚手架」不是「拐杖」

这里必须诚实指出争议。AI 工具被质疑是「外部执行功能层」还是「永久拐杖」（来源：矛盾与存疑）。Focusmate 同样面临这个边界问题：如果一个人没有 Focusmate 就无法工作，那它就是拐杖；如果它能帮助一个人逐步内化「定时启动」的能力，那它就是脚手架。

我的判断是：Focusmate 更接近脚手架，因为它不提供记忆或决策——它只提供节奏。而节奏是可以内化的。徐渭最终形成了自己的艺术 rhythm，孔子 70 岁才「从心所欲不逾矩」——他们的 harness 系统最终内化为习惯。Focusmate 的用户也可以从「需要搭档」过渡到「自己就能定时启动」。

## 局限与矛盾

必须承认，这个「同构」框架目前是理论推理，缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。Focusmate 对 ADHD 的长期效果也缺乏实证——用户反馈积极，但纵向研究不足。此外，个体差异极大：有人需要柔性引导，有人需要强行打断（来源：超聚焦矛盾），Focusmate 的固定 session 长度可能不适合所有人。

## 今天就能试的行动

1. **用 Focusmate 做一次「无状态任务」**：选一个你一直拖延的任务（比如写一份周报），在 Focusmate session 中只做这一件事，结束时记录进度。观察「声明-执行-汇报」循环是否降低了启动阻力。
2. **对比 Focusmate 与 AI 任务分解工具**：用 Goblin Tools 把同一个任务分解成步骤，然后单独做 vs 用 Focusmate 做。哪个让你更容易开始？哪个让你更容易持续？
3. **建立个人「re-grounding 仪式」**：如果不用 Focusmate，设定一个 25 分钟闹钟，每次响铃时写一句话总结「我刚才做了什么」，然后继续。这模拟了 Focusmate 的定时锚点。
4. **尝试「外部调度器」组合**：用 Saner.AI 或 Lex 作为外部记忆（存储任务上下文），同时用 Focusmate 作为执行调度器（提供节奏）。这相当于给 ADHD 大脑配了一个完整的 harness 系统——记忆由 AI 管，节奏由人管。
