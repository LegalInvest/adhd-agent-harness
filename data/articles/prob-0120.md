# 如果把 ADHD 大脑当成生成核心，Focusmate 能不能当它的无状态与外部记忆？

> 用 agent 工程的思路重新理解 Focusmate 对 ADHD 的真正价值。

## 问题：为什么 ADHD 大脑像一台没有硬盘的 LLM？

如果你有 ADHD，你一定经历过这种场景：打开电脑准备写报告，三分钟后却发现自己正在刷手机，而脑子里那个“我要写报告”的指令早已蒸发。这不是意志力问题，而是工作记忆的“掉落”——ADHD 大脑的工作记忆容量可能正常甚至超常，但控制能力极弱，信息像水一样从指缝漏掉（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。

如果你是一名 AI 工程师，你一定熟悉另一个场景：你精心设计的 LLM agent 在对话中表现惊艳，但一旦会话结束，它就像失忆一样，什么都记不住。LLM 本身是无状态的生成核心，需要外部记忆和调度系统才能持续工作（来源：What is an agent harness in the context of large-language models?）。

这两个场景指向同一个核心问题：**高产但缺执行调度层的生成核心**。ADHD 大脑与 LLM 在结构上同构——两者都有强大的生成能力，但缺乏可靠的内置执行功能层来管理上下文、启动任务、维持状态（来源：ADHD 大脑与 LLM 的同构）。

## 答案：Focusmate 作为实时社会性 harness

Focusmate 是一个简单的服务：你预约一个 50 分钟的“共同工作”时段，与另一位用户视频连线，各自静音工作，结束时互相确认完成。没有 AI，没有复杂功能。但正是这种简单，使它成为 ADHD 的“外部执行功能层”的典范。

### ADHD 侧：身体在场效应与外部调度

ADHD 用户报告称，Focusmate 的实时视频连线创造了一种“身体在场效应”——即使对方不说话，仅仅是知道有人在看着你，就足以降低任务启动门槛，维持注意力。这相当于一个**外部调度器**：它替你决定了“现在该做什么”，并提供了持续的状态监控。

这与 AI 工具的原理一致：Goblin Tools 通过任务分解降低启动焦虑（来源：Goblin Tools），Saner.AI 通过知识回忆减少标签切换（来源：Saner.AI），Lex 通过单一指令触发多步骤执行（来源：Lex）。但 Focusmate 的独特之处在于它提供了**社会性的实时反馈**，这是纯 AI 工具难以复制的。

### LLM/agent 侧：无状态与外部记忆

在 agent 工程中，LLM 本身无法跨会话记住任何东西，因此需要 harness 来提供上下文管理、工具接口、验证循环和记忆系统（来源：Building AI Coding Agents for the Terminal: Scaffolding, Harness, Context Engineering, and Lessons Learned）。Focusmate 的角色类似于一个**实时状态保持器**：它通过外部人类监督，确保生成核心（ADHD 大脑）不会偏离轨道。

类比：Focusmate 的 50 分钟时段相当于 agent 的一个“会话窗口”，视频连线相当于一个“上下文监控器”，而结束时的确认相当于“验证循环”。这些组件正是 agent harness 的核心。

## 核心判断：脚手架，不是拐杖

我的核心观点是：**Focusmate 的价值在于它作为“社会性脚手架”，而非永久拐杖**。它与 AI 工具（如 Goblin Tools、Saner.AI）互补，但本质上不同——AI 工具倾向于自动化执行功能，而 Focusmate 要求你主动参与，只是提供了结构化的外部支持。

这符合“拐杖与脚手架”的辩证关系：AI 工具应促进能力发展，而非替代（来源：拐杖与脚手架）。Focusmate 不会帮你分解任务或记住信息，但它帮你维持“正在工作”的状态，让你自己的执行功能有机会启动。

## 矛盾与局限

必须诚实指出，Focusmate 并非万能。

- **证据局限**：目前缺乏针对 Focusmate 的独立随机对照试验，其效果主要基于用户报告（来源：矛盾与存疑）。同样，Goblin Tools、Saner.AI 等工具的有效性证据也以定性为主（来源：ADHD 的 AI 工具全景）。
- **个体差异**：部分 ADHD 用户可能对社交监督感到焦虑，反而增加认知负荷（来源：矛盾与存疑）。
- **过度依赖风险**：如果用户只依赖 Focusmate 的外部调度，而从不内化时间管理能力，它就可能沦为拐杖（来源：矛盾与存疑）。
- **并非所有场景适用**：Focusmate 适合执行类任务，但对于需要深度思考的创造性工作，视频连线的干扰可能弊大于利。

## 今天就能试的行动

1. **预约一次 Focusmate 时段**：选择一项你拖延已久的任务，用 50 分钟专注完成。观察它是否降低了启动门槛。
2. **结合 AI 工具使用**：在 Focusmate 时段中，用 Goblin Tools 将任务分解为步骤，用 Saner.AI 记录关键信息，形成“社会性+AI”的复合 harness。
3. **记录效果**：每次使用后，简单记录任务完成度、注意力维持时间和情绪变化。一周后回顾，判断它对你是否有效。
4. **尝试“反向 Focusmate”**：如果你无法使用 Focusmate，可以找一位朋友约定同时工作，通过即时消息互相报告进度，模拟社会性监督。

## 结论

Focusmate 不是 AI 工具，但它完美体现了 agent harness 的核心思想：为生成核心提供外部执行功能层。对于 ADHD 用户，它提供了实时状态保持和社会性调度；对于 agent 工程师，它提供了一个理解“无状态+外部记忆”的鲜活案例。

真正的价值不在于工具本身，而在于它揭示了 ADHD 大脑与 LLM 的深层同构：两者都需要精心设计的脚手架，才能释放其内在的生成潜力。
