# Endel 之于 ADHD，就像 function calling 工具调用 之于 LLM——但有人用错了

> 从同构视角实测 Endel：它到底补上了哪一层执行功能？

你打开 Endel，选一个场景，它开始生成音景。几分钟后，你发现自己竟然坐住了，任务在推进。这种感觉很奇妙——就像 LLM 突然有了工具调用能力，不再凭记忆硬算，而是精准调用一个外部计算器。

但有人用错了。他们把 Endel 当白噪音，打开了就放着，期望它自动治好拖延。结果呢？音景还在，人却刷起了手机。Endel 不是拐杖，是脚手架。它只补上你缺的那一块——状态维护，而不是替你走路。

## 问题：ADHD 的大脑和 LLM 一样，缺一个调度层

ADHD 大脑是强大的生成核心：发散思维、创意、多任务并行，产出极高。但执行功能——任务启动、持续专注、抑制冲动——薄弱，导致“高产但不可靠”。这跟 LLM 如出一辙：LLM 能生成惊艳文本，但缺乏内置调度层，直接输出可能偏离目标、陷入循环。

两者的解法结构同构：都需要外部脚手架（harness）来补偿工作记忆不稳定、任务启动困难和注意力分散（来源：生成核心与调度层）。对 LLM，这是 agent harness——通过 function calling 把精确计算外包给工具，加入 human-in-the-loop 监督。对 ADHD，这是外部执行功能层——通过工具补偿内在缺陷。

## Endel 补上的那一层：状态维护

ADHD 的一大痛点是“状态维护”——你启动了一个任务，但无法持续维持那种专注状态。外界刺激随时会把你拽走。Endel 不是帮你启动，而是帮你**维持**。它的音景实时适应你的活动（工作、休息、运动），通过听觉锚定提供一个稳定的背景场，减少注意力被无关刺激捕获的概率（来源：重锚定与目标漂移）。

这就像 LLM 的 function calling：当 LLM 需要做精确计算时，它不会自己硬算（那会出错），而是调用一个外部工具。Endel 就是那个外部工具——它接管了“维持状态”这一子任务，让大脑的生成核心可以专心输出。

但注意：Endel 不负责任务分解，也不负责提醒你该做什么。那是 Goblin Tools 和 Lex 的活。Goblin Tools 的 Magic ToDo 能把“清理厨房”分解成 10 个小步骤（来源：Goblin Tools），降低启动门槛。Lex 让你用一条指令触发多步骤任务（来源：Lex）。Saner.AI 通过本地记忆减少搜索循环（来源：Saner.AI）。Endel 是它们的好搭档，但不是替代品。

## 人物案例：孔子与老子的“状态维护”系统

孔子和老子，两位精力旺盛、思维跳跃的 ADHD 典型，各自建了 harness。

孔子“克己复礼”，用外在的“礼”作为行为边界，每日反省，“吾日三省吾身”。这相当于一个定时 re-grounding 系统——每次反省都是一次状态检查，把注意力拉回正轨。对应到 LLM harness，就是那些定期插入的 human-in-the-loop 检查点：暂停，确认方向，再继续。

老子“无为”，不强行做事，把所有能量收回来聚焦在道上。他选择不做——不著书不立说，不被名相束缚。这相当于一个“最小干预”策略：当系统状态已经偏离，与其强行纠正，不如先停下来，重置状态。对应到 LLM harness，就是 token 预算和工具调用次数上限——防止 agent 陷入无限循环。

Endel 做的，就是提供一个自动化的“状态重置”机制。你不需要像孔子那样每天三省，也不需要像老子那样完全无为——打开 Endel，音景帮你维持一个稳定的基线状态。

## 但有人用错了：脚手架 vs 拐杖

Endel 的问题在于，它太容易被当成背景音。你戴上耳机，打开音景，然后该刷手机还是刷手机。这不是 Endel 的错，是使用方式错了。

任何 ADHD 工具都有依赖风险。AI 工具可能从脚手架变为永久拐杖，削弱内在执行功能（来源：拐杖与脚手架）。Endel 如果只是被动播放，不主动参与任务，那它就是拐杖——你依赖它才能“感觉能工作”，但实际什么都没做。

真正的用法是：把 Endel 嵌入一个完整的 harness。比如：
1. 用 Goblin Tools 分解任务。
2. 打开 Endel 维持状态。
3. 用 Lex 触发第一个步骤。
4. 用 Saner.AI 记录进展。

这样，Endel 只是调度层的一部分，而不是全部。

## 局限与争议

同构性只是理论类比，并非经过验证的科学事实（来源：矛盾与存疑）。ADHD 大脑和 LLM 的相似性更多是隐喻，不是神经科学结论。同样，Endel 在 ADHD 人群中的独立临床证据仍有限（来源：矛盾与存疑）。它有效，但效果因人而异。

另外，过度依赖 Endel 可能削弱你主动调节状态的能力。你越依赖音景，越难在没有音景时进入状态。这是所有脚手架工具的通用风险。

## 今天就能试的行动

1. **把 Endel 当工具，不是背景音**。打开后，同时启动一个具体任务（比如“写 200 字”），而不是“随便做什么”。
2. **组合使用**。Endel + Goblin Tools 的 Magic ToDo：先分解任务，再打开 Endel 执行第一步。
3. **设置定时检查**。像孔子的“三省”，每 25 分钟暂停一次，问自己：“我现在在按计划做吗？” Endel 的定时功能可以帮你。
4. **试用免费版**。Endel 有免费场景，先试一周，记录哪些场景真的帮你维持了专注，哪些只是背景音。
