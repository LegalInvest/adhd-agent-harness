# Inflow 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 Inflow：它到底补上了哪一层执行功能？

## 从同一个问题出发：你明明知道目标，却眼睁睁看着它漂走

ADHD 大脑最折磨人的体验之一，是「知道该做什么，但就是做不了」——不是能力问题，是执行调度层出了故障。你打开文档准备写报告，十分钟后却在维基百科里研究二战坦克型号；或者你超聚焦在某个细节上，三小时后才惊觉整体目标早已偏离。这种「目标漂移」不是意志力薄弱，而是大脑缺乏一个内置的重锚定机制。

有趣的是，长程 LLM agent 工程师们正在面对几乎一模一样的问题：一个 agent 被赋予复杂任务后，随着上下文膨胀，它会逐渐偏离初始目标，陷入局部最优或无关分支。工程师们为此设计了「重锚定 system prompt」——定期向 agent 注入原始目标，让它重新对齐。

于是有人提出：Inflow 之于 ADHD，就像重锚定 system prompt 之于 LLM。这个类比很迷人，但许多人用错了方向。

## 同构的真相：高产核心 + 缺失的调度层

ADHD 大脑与 LLM 在结构上高度同构：两者都是「高产但缺执行调度层的生成核心」（来源：AI 与 ADHD 的专注力）。ADHD 大脑能快速联想、产生创意，但无法稳定执行计划；LLM 能流畅生成文本，但缺乏内在的规划与验证机制。两者都需要外部脚手架来弥补调度缺陷。

历史上，最聪明的 ADHDer 早已直觉地发明了自己的脚手架。曾国藩（晚清名臣）30 岁前浮躁坐不住，日记里天天骂自己「无恒」「浮躁」。他的解法是「日课十二条」：黎明即起、读书不二、谨言、写日记反省——这些本质上是一套定时重锚定系统。每天写日记就是一次「重锚定 system prompt」，把自己拉回目标。辛弃疾被罢官后闲居 20 年，把所有精力注入练剑写词，以「词与剑」为外部调度器，将报国无门的愤懑转化为稳定产出。

这些历史案例与 LLM agent 工程完全同构：曾国藩的「日课」对应 agent 的定时 re-grounding 循环，辛弃疾的「剑与词」对应外部约束（如任务分解模板）。

## 现代工具：它们补上了哪一层？

今天的 AI 工具正在尝试扮演同样的角色。

**Focusmate** 通过虚拟身体在场提供外部问责：你预约一个时段，平台匹配一位伙伴，彼此监督进度。这相当于为 ADHD 大脑增加了一个「外部调度器」——你不需要自己启动任务，伙伴的存在替你完成了启动。其核心机制是「AI-Matched Body Doubling」（来源：Focusmate），算法优化配对效率。这与 LLM agent 中的「human-in-the-loop」完全同构：一个外部节点定期检查进度并纠正方向。

**RescueTime** 自动追踪时间使用，识别注意力漂移，并阻隔分心网站。它补偿的是「时间盲」——ADHD 大脑无法准确感知时间流逝，就像 LLM 无法感知上下文长度膨胀一样。RescueTime 的自动分类和报告相当于 agent 的「日志与监控系统」（来源：RescueTime）。

**Brain.fm** 通过神经锁相技术生成专注音乐，帮助进入状态。它不直接处理目标漂移，而是优化「启动状态」——相当于为 agent 提供一个「初始 prompt 模板」。（来源：Brain.fm）

**Motion** 和 **Reclaim.ai** 则更进一步：它们自动分解任务、动态调整日程。这直接对应 agent 的「任务分解与规划循环」，即把大目标拆为可执行步骤（来源：规划循环与任务分解）。

## 但有人用错了：脚手架 vs 拐杖

问题来了：这些工具到底是脚手架，还是拐杖？

脚手架是帮助用户逐步内化能力的临时结构；拐杖则是永远替代功能的依赖品。当前最大的争议在于：AI 工具是否会导致 ADHD 用户永久依赖外部提醒，而无法内化执行功能？（来源：矛盾与存疑）

一个典型的误用是：用户依赖 Inflow 等应用每天推送提醒，但从不建立自己的「重锚定习惯」。就像 LLM agent 工程师只给 agent 一个固定 system prompt，却不设计 re-grounding 循环——agent 会在长程任务中逐渐漂移。

真正的解法是：工具应该成为「重锚定系统」的一部分，而非全部。曾国藩的日课之所以有效，是因为他每天主动反省、调整，而不是被动等待提醒。同样，LLM agent 的最佳实践是定期注入原始目标，并结合验证机制。

## 诚实面对局限

必须承认，目前 ADHD × AI 领域的证据仍很薄弱。Brain.fm 虽在 2026 年一项 44 款应用测试中位列最佳 9 款之一，但缺乏针对 ADHD 的独立临床研究（来源：Brain.fm）。Focusmate 的效果基于「身体在场效应」，但 AI 能否真正替代真实人际互动，仍有争议（来源：矛盾与存疑）。

此外，同构性本身是一种理论类比，并非经过验证的科学事实（来源：矛盾与存疑）。ADHD 大脑与 LLM 的相似性可能是功能层面的，而非神经机制层面的。

## 今天就能试的行动

1. **设置「重锚定」闹钟**：每 25 分钟用手机闹钟提醒自己「当前目标是什么？」——这是最简单的外部 re-grounding。
2. **使用 Focusmate 一次**：体验虚拟身体在场如何降低任务启动阻力，注意它是否减少了你的目标漂移。
3. **尝试 RescueTime 的自动阻断**：设置一个「分心网站阻断」规则，观察它是否帮你保持轨道。
4. **写「日课」式日志**：每天花 5 分钟写下「今天偏离目标了吗？为什么？」——这是曾国藩式的自我反省，也是你自己的 system prompt。

工具只是脚手架，真正的重锚定能力，需要你亲自构建。
