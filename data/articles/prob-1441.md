# Reclaim.ai 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 Reclaim.ai：它到底补上了哪一层执行功能？

## 问题：ADHD 与 LLM 共享的「目标漂移」困境

如果你有 ADHD，你一定熟悉这个场景：早上打开电脑打算写一份报告，十分钟后却在研究今天午餐的食谱，两小时后发现自己刷了三个小时的短视频。这不是意志力问题——你的大脑像一台没有调度器的生成模型，输出能力极强，但缺乏内置机制来锚定初始目标。

做 Agentic Harness 的工程师对此感同身受：你设计的 agent 在长程任务中会逐渐偏离原始指令，上下文膨胀后推理退化，最终输出与初始目标相去甚远。解决方法是重锚定 system prompt——定期将 agent 拉回正轨。

ADHD 大脑与 LLM 在结构上同构：两者都是强大的生成核心，但缺乏可靠的内置执行调度层（来源：AI 与 ADHD 的专注力）。Reclaim.ai 声称能帮助 ADHD 用户管理时间，但许多人用错了——他们只把它当提醒工具，忽略了它作为「重锚定系统」的深层价值。

## 同构脊柱：重锚定与目标漂移

ADHD 侧的核心痛点是分心走神或超聚焦跑偏，丢失目标。LLM/Agent 侧对应的是长程任务中 agent 会目标漂移，需要重锚定 system prompt。两者的解法结构同构：都需要一个外部调度层，定期检查当前行为是否对齐初始目标，并在偏离时拉回。

Reclaim.ai 的 AI 调度功能正是这种外部调度层：它学习用户的时间模式，自动安排任务、会议和习惯，并在冲突时重新调整。但关键在于，它的价值不在于「提醒你做什么」，而在于「帮你确认你正在做的是你真正想做的」。这相当于 LLM 的 system prompt 重锚定——不是简单插入新指令，而是重新评估当前上下文与初始目标的一致性。

曾国藩的日课十二条提供了 ADHD 侧的真实证据。他 30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”。他的 harness 是“读书不二”——一本书不读完绝不翻下一本，以及每日写日记反省。这本质上是一种定时重锚定：每天通过写日记重新确认自己的行为是否对齐目标（来源：人物案例）。这与 LLM harness 中的定期 re-grounding 完全同构。

## Reclaim.ai 补上了哪一层执行功能？

执行功能障碍是 ADHD 的核心特征之一，包括计划、组织、时间管理、任务启动困难（来源：执行功能障碍）。Reclaim.ai 主要补偿的是时间盲——ADHD 用户难以准确估计和记录时间使用（来源：RescueTime 页面）。但 RescueTime 通过自动记录揭示时间去向，而 Reclaim.ai 更进一步：它主动预留时间块，并在冲突时重新安排。

然而，当前 Reclaim.ai 的用法多停留在“自动安排日程”，这相当于只给 LLM 一个定时器，而没有重锚定机制。真正的同构用法是：将 Reclaim.ai 视为一个外部调度器，定期触发“当前任务是否对齐周目标/月目标”的检查。这类似于 Focusmate 的虚拟身体在场——通过实时同伴问责提供外部结构（来源：Focusmate 页面），但 Reclaim.ai 的 AI 调度可以做得更智能。

## 争议与局限

我必须诚实指出：ADHD 大脑与 LLM 的同构目前仅是一种理论类比，缺乏实证基础（来源：矛盾与存疑）。Reclaim.ai 在 ADHD 人群中的独立临床证据同样有限。此外，工具可能成为拐杖而非脚手架——过度依赖外部调度可能阻碍执行功能的内化（来源：拐杖与脚手架概念）。

另一个矛盾是：Reclaim.ai 的自动调度可能增加认知负荷，尤其当规则不精准时，用户需要反复调整，反而加剧挫败感（来源：RescueTime 页面提及的局限）。

## 今天就能试的行动

1. **设置每周重锚定检查**：在 Reclaim.ai 中创建每周 30 分钟的“目标对齐”事件，期间回顾本周任务是否匹配你的核心目标。
2. **使用“任务意图标注”**：为每个任务添加一句“为什么做这个”（类似 LLM 的 system prompt），并在 Reclaim.ai 的日程中显示。
3. **组合使用 RescueTime 与 Reclaim.ai**：RescueTime 自动追踪时间去向，Reclaim.ai 根据数据调整日程——形成“监控-反馈-调整”闭环。
4. **尝试 Focusmate 作为重锚定触发器**：在 Reclaim.ai 安排的任务时段内，用 Focusmate 的虚拟身体在场增强启动与维持（来源：Focusmate 页面）。

Reclaim.ai 不是万能的，但它提供了一个结构：当它与重锚定思维结合时，ADHD 大脑和 LLM 都能从“高产但迷失”转向“高产且定向”。关键在于，不要只把它当提醒工具——让它成为你的外部调度层，定期问自己：“我现在做的事，和我最初的目标一致吗？”
