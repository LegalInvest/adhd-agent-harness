# 用 ChatGPT 给 ADHD 补上重锚定与目标漂移，真的能像 agent 那样工作吗？

> 把 ChatGPT 当作 ADHD 的外部执行层来实测：它补的是哪一块短板？

## 问题：你丢了多少次目标？

如果你是 ADHD 患者，你大概经历过这样的场景：打开电脑准备写报告，十分钟后却发现自己正在研究“如何养水母”——因为你只是好奇搜了一下“海洋生物”，然后就被超聚焦带跑了。你并不懒，你只是**丢失了目标**。

如果你是 Agent 工程师，你大概也见过类似的场景：你精心设计的 agent 在长程任务中越跑越偏，从“分析销售数据”变成了“生成一份关于鲸鱼迁徙的报告”——因为某个中间步骤的上下文膨胀了，agent 的注意力被带走了。

这两件事是同一种问题：**重锚定失败**。ADHD 大脑和 LLM 都是高产但缺乏可靠执行调度层的生成核心（来源：AI 与 ADHD 的专注力）。它们能生成无数想法，但无法稳定地锚定在目标上。那么，用 ChatGPT 来给 ADHD 补上这块短板，真的能像 agent harness 那样工作吗？

## 同构：ADHD 大脑与 LLM 的共享困境

先看证据。ADHD 一侧，执行功能缺陷包括工作记忆差、时间盲、任务启动困难（来源：AI 与 ADHD 的专注力）。你脑子里有一万个创意，但“从想法到行动”的管道却经常堵塞。LLM 一侧，agent 在长程任务中会因上下文膨胀而漂移，需要 harness 提供上下文传递、规划工件、验证循环等（来源：幻觉与验证循环）。两者都缺一个**外部锚点**。

这个同构不是比喻，而是功能上的映射。ADHD 的工作记忆缺陷与 LLM 的无状态性同构，都需要外部记忆系统来维持连续性（来源：AI 与 ADHD 的专注力）。ADHD 易被环境带偏，LLM 易被上下文膨胀干扰，都需要主动设计“当下注意什么”的工程方案（来源：同上）。因此，AI 工具的本质是给生成核心套上 harness——外部执行功能层（来源：同上）。

## 实测：ChatGPT 能补哪块短板？

理论上，ChatGPT 可以充当这个 harness。你可以在对话中让它帮你分解任务、提醒目标、记录进度——就像一个低配版的 agent 调度器。但实际效果如何？我们来看几个关键组件。

**重锚定**：你可以对 ChatGPT 说：“我的目标是写一篇 2000 字的文章，如果我发现跑题了，请提醒我回到提纲。”这相当于给它一个 system prompt。但问题在于，ChatGPT 没有真正的验证循环。它不会主动检查你是否偏离目标，除非你明确要求它“每隔 10 分钟提醒我一次”。而即使它提醒了，你作为 ADHD 患者，可能已经沉浸在超聚焦中，根本听不见——就像 agent 输出了一串错误代码，但 harness 没有捕获它（来源：幻觉与验证循环）。

**外部记忆**：RescueTime 通过自动时间追踪减轻工作记忆负担，帮助识别注意力漂移模式（来源：RescueTime）。ChatGPT 可以模拟这一点：你可以让它记录你的对话历史，并总结你偏离目标的次数。但 RescueTime 的自动化是持续的、被动的，而 ChatGPT 需要你主动启动和维持对话——这对执行功能弱的 ADHD 患者本身就是挑战。

**温度控制**：ADHD 的表现波动与 LLM 的采样温度同构（来源：采样温度与表现波动）。温度高时，输出随机、有创意；温度低时，输出确定、可重复。ADHD 的“高温度”状态就是超聚焦跑偏，而“低温度”状态就是僵化、无法启动。ChatGPT 的默认回复温度较高，容易生成发散内容，反而可能助长漂移。你需要手动降低温度（通过 prompt 如“请保持回答简洁、紧扣目标”），但这需要元认知——这恰恰是 ADHD 的短板。

**验证循环**：这是最关键的缺口。Agent harness 的核心是验证与 CI 集成，例如检查输出格式或运行测试用例（来源：幻觉与验证循环）。ADHD 也需要类似的验证步骤来校正认知偏差——比如冲动判断、脱离环境线索（来源：同上）。但 ChatGPT 不会主动验证你的行为。它只能提供建议，而你是否执行，取决于你自己。Focusmate 通过实时同伴问责提供外部验证（来源：Focusmate），但 ChatGPT 没有这种身体在场效应。

## 脚手架 vs 拐杖：边界在哪里？

这里有一个核心矛盾：AI 工具作为外部执行功能层，长期使用是否会导致内在能力进一步退化？（来源：矛盾与存疑）Brain.fm 宣称通过神经锁相帮助专注，但缺乏针对 ADHD 的独立临床研究（来源：Brain.fm）。Focusmate 有用户研究支持，但依赖实时视频，可能造成社交焦虑（来源：Focusmate）。RescueTime 的自动分类可能不精准，过度关注数据可能引发焦虑（来源：RescueTime）。

我的判断是：**ChatGPT 可以作为一个低成本的实验工具，但无法替代专用的 harness。** 它补的是“提醒”和“记录”这两块，但缺了“验证”和“温度控制”。如果把它当作拐杖，你会越来越依赖它的提醒，而失去自己重锚定的能力。如果把它当作脚手架，你需要在每次使用后反思：“这次 ChatGPT 帮我锚定了吗？我下次能自己做到吗？”

## 今天就能试的行动

1. **设定一个“锚定 prompt”**：在开始任何任务前，对 ChatGPT 说：“我的目标是 [具体目标]。请每隔 15 分钟问我一次：‘你还在做这件事吗？’如果我发现跑题，请立即让我回到目标。”运行一次，记录你偏离的次数。
2. **用 RescueTime 做一周时间审计**：不改变行为，只观察。看看你每天有多少时间花在“目标外”活动上。这能帮你建立对漂移模式的觉察（来源：RescueTime）。
3. **尝试一次 Focusmate 会话**：预约一个 50 分钟的时段，告诉伙伴你的目标。结束后，对比“独自工作”和“有人在场”时的锚定效果（来源：Focusmate）。
4. **给 ChatGPT 加一个验证步骤**：在完成任务后，让 ChatGPT 评估你的输出是否偏离原始目标。例如：“请对照我最初的目标，检查我是否跑题了。列出偏离点。”

## 诚实局限

本文的论证基于同构隐喻，但 ADHD 大脑与 LLM 的注意力机制本质不同（来源：AI 与 ADHD 的专注力）。所有工具的证据主要来自用户报告，缺乏大规模随机对照试验（来源：矛盾与存疑）。个体差异极大，同一工具对不同 ADHD 亚型效果迥异（来源：矛盾与存疑）。因此，任何建议都需要你亲自实验，找到适合自己的组合。

最后，记住：ChatGPT 是你的脚手架，不是你的大脑。真正的重锚定能力，仍然需要你从内部生长出来。
