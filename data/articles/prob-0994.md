# Mem 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 Mem：它到底补上了哪一层执行功能？

## 问题：你的大脑是个高产的生成核心，但没有调度层

如果你有 ADHD，你大概熟悉这个场景：你打开电脑准备写报告，五分钟后却在研究如何给盆栽换土。你的大脑像一台没有操作系统的超级计算机——算力惊人，但进程管理一塌糊涂。

如果你在搭建 Agent，你也会熟悉另一个场景：你的 LLM agent 启动一个长程任务，前几步完美，第十步时它开始生成与目标无关的代码——目标漂移发生了。

这两个问题，本质上是同一个问题。ADHD 大脑与 LLM 在结构上同构：两者都是**高产但缺乏可靠执行调度层的生成核心**（来源：AI 与 ADHD 的专注力）。ADHD 的执行功能缺陷对应 LLM 的上下文控制缺失；ADHD 的工作记忆缺陷对应 LLM 的无状态性；ADHD 的易分心对应 LLM 的上下文膨胀干扰。

## 答案：重锚定——给生成核心套上 harness

在 Agent 工程中，解决目标漂移的标准方案是**重锚定 system prompt**：在每一步或关键节点，将 agent 的注意力拉回原始目标，检查进度，修正方向。这个机制在 LLM harness 中被称为“会话状态记录工具结果、完成的子任务和进度笔记”（来源：Agent Scaffolding: Architecture and Design Patterns for Agentic AI）。

在 ADHD 一侧，对应的方案是**外部重锚定系统**：通过提醒、通知、身体在场等机制，将注意力拉回原任务。例如，Focusmate 通过实时视频配对提供外部问责，弥补内在多巴胺调节不足（来源：Revolutionizing ADHD Management with AI Assistants）。RescueTime 通过自动时间追踪揭示真实的时间去向，帮助识别注意力漂移（来源：Revolutionizing ADHD Management with AI Assistants）。

## 案例：曾国藩的“日课十二条”就是手动重锚定

曾国藩是典型的注意力不集中型 ADHD：30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”。他的解法是“日课十二条”——黎明即起、读书不二、谨言、写日记反省——每天用固定仪式反复锚定目标。这本质上是一个**手动重锚定系统**，与 LLM harness 中的定时 re-grounding 同构：日课就像每小时的 system prompt 刷新，写日记就像记录会话状态。

但曾国藩的日课之所以有效，是因为他**终身执行，但从未依赖单一工具**。他不断调整日课内容，甚至允许自己在特定情况下跳过——这正是“脚手架”而非“拐杖”的用法。

## 核心观点：Mem 的陷阱——有人用错了重锚定

Mem（一款 AI 笔记工具）的卖点是“永不遗忘的第二大脑”。对 ADHD 来说，它确实能补上工作记忆缺陷：自动记录、智能检索、上下文关联。但问题在于，**有人把 Mem 当成了永久拐杖**——他们把所有信息丢进 Mem，然后不再训练自己的记忆策略。

这与 LLM agent 工程中的常见错误同构：有人把重锚定 system prompt 写死，以为一次设定就能解决所有目标漂移，却忽略了 agent 需要**动态调整**锚定频率和内容。

正确的用法是：**把 Mem 当作可拆卸的脚手架**——在任务初期或高认知负荷时使用，同时逐步内化其工作模式（如定期回顾、结构化记录）。这与“规划、排序和跟踪发生在你的大脑之外，让你的大脑自由去做实际工作”的理念一致（来源：Using an AI Assistant to Manage ADHD: A Practical Guide）。

## 证据：哪些工具真正有效？

现有证据强度不一：
- **强证据**：Focusmate 基于身体在场效应，多项用户研究显示能显著提升任务启动和维持（来源：AI 与 ADHD 的专注力）。
- **中等证据**：RescueTime 通过客观时间记录提升时间觉察，有初步研究支持（来源：AI 与 ADHD 的专注力）。Brain.fm 利用神经锁相技术，但针对 ADHD 的独立临床研究缺乏（来源：Brain.fm）。
- **弱证据**：Motion 通过自动排程帮助克服时间盲，但缺乏独立临床证据，初始设置成本高（来源：AI 与 ADHD 的专注力）。

## 局限与争议

1. **依赖风险**：AI 工具作为外部执行功能层，长期使用是否会导致内在执行功能退化？目前缺乏纵向研究（来源：矛盾与存疑）。
2. **个体差异**：ADHD 异质性大，同一工具对不同亚型效果迥异。例如，超聚焦个体可能需要引导而非打断（来源：矛盾与存疑）。
3. **同构的局限性**：ADHD 大脑与 LLM 的同构是隐喻而非严格等价，LLM 的“注意力机制”与人类注意力存在本质差异（来源：AI 与 ADHD 的专注力）。

## 今天就能试的行动

1. **设置“重锚定提醒”**：在手机或电脑上设置每 25 分钟一次的提醒，内容写“当前目标是什么？”。这相当于手动刷新 system prompt。
2. **试用 Focusmate 一次**：预约一个 50 分钟的 session，体验身体在场效应。注意观察它是否帮你启动了原本拖延的任务。
3. **用 RescueTime 追踪一周**：不设任何阻断，只看数据。找出你最大的时间黑洞，然后思考：这个黑洞是休息需要，还是目标漂移？
4. **创建自己的“日课”**：模仿曾国藩，写下 3 条每天必须做的微小动作（如“写一句话总结今日进度”），坚持一周，观察是否减少目标漂移。

记住：工具是脚手架，不是拐杖。真正的目标不是永远依赖 Mem，而是训练你的大脑，让它最终能自己重锚定。
