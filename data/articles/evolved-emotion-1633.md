# 为什么用 Goblin Tools 治 ADHD 的情绪失调，和给 agent 套 会褪去的脚手架 是一回事？

> Goblin Tools 实测：同一套 harness 思路，ADHD 与 LLM 两边都成立。

## 问题：为什么情绪失调和 agent 工程是同一件事？

如果你是一个 ADHD 成人，你一定经历过这种时刻：一件小事——比如找不到钥匙、被同事打断——突然引爆了剧烈的愤怒或沮丧，理性完全失效。你事后知道反应过度，但当时就是控制不住。这种情绪失调不是性格问题，而是执行功能缺陷的直接后果（来源：情绪失调）。

如果你是一个在做 Agentic Harness 工程的工程师，你一定遇到过这种场景：一个强大的 LLM 在没有外部约束时，输出飘忽不定、频繁幻觉、上下文失控。你不得不给它套上精心设计的脚手架——prompt 模板、外部记忆、任务分解链——才能让它稳定产出。

这两个场景，表面看毫无关系。但本文要论证一个反直觉的同构：ADHD 大脑与 LLM 是同一类「高产但缺执行调度层的生成核心」，而 Goblin Tools 治情绪失调，和给 agent 套会褪去的脚手架，本质上是同一个操作。

## ADHD 情绪失调：生成核心缺少执行调度层

ADHD 的情绪调节困难根植于执行功能缺陷：工作记忆容量有限，导致情绪信息无法被有效加工；时间盲导致对情绪持续时间的误判；任务启动困难加剧挫败感（来源：AI 与 ADHD 的情绪调节）。最新研究发现，Transformer 在 Stroop 冲突任务中，注意力随序列变长在冲突试次上骤降到随机水平——无法抑制优势反应、无法解决认知冲突，这与 ADHD 执行功能缺陷的核心标志一一对应（来源：Deficient Executive Control in Transformer Attention）。同时，LLM 呈现「强记忆、弱控制」剖面：工作记忆容量甚至超过常人，但认知灵活性与注意控制存在核心缺陷（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。

这意味着，ADHD 大脑和 LLM 都是强大的生成引擎，但缺乏一个可靠的执行调度层来管理输出。情绪失调，本质上是这个生成核心在缺乏外部约束时产生的「幻觉」——当工作记忆过载或注意力被干扰时，情绪反应脱离理性轨道（来源：AI 与 ADHD 的情绪调节）。

## Goblin Tools：给 ADHD 大脑套上外部 harness

Goblin Tools 的核心功能 Magic ToDo，能将任何复杂任务自动分解为可管理的小步骤（来源：Goblin Tools）。用户输入「整理房间」，AI 输出「捡起地板上的衣服」「擦桌子」等具体行动。这种分解直接补偿了工作记忆不足和任务启动困难，降低了认知负荷。更重要的是，它通过外部化执行调度，让 ADHD 大脑的生成核心得以流畅运作，从而减少情绪波动。

类似地，Inflow 通过训练工作记忆和认知控制来强化「流入」中枢（背外侧前额叶皮层），其神经科学基础是动态因果脑回路研究（来源：Inflow）。Saner.AI 则通过增强知识回忆，减少搜索循环和标签切换，直接针对工作记忆瓶颈（来源：Saner.AI）。

这些工具的共同本质是：它们作为外部执行功能层，接管了调度任务，让 ADHD 大脑的生成核心专注于产出（来源：AI 与 ADHD 的情绪调节）。

## Agentic Harness：给 LLM 套上会褪去的脚手架

在 LLM/agent 工程中，我们做同样的事。当 agent 需要稳定完成复杂任务时，工程师会设计「会褪去的脚手架」——prompt 模板定义行为边界，外部记忆存储上下文，任务分解链保证步骤有序。这些脚手架不是永久性的，它们会在 agent 能力提升后逐渐移除。

Transformer 在工作记忆任务中自发发展出输入输出门控机制，模仿人类前额叶-纹状体回路——这正是 ADHD 核心受损的脑区系统（来源：TRANSFORMER MECHANISMS）。这意味着，LLM 和 ADHD 大脑共享类似的底层架构，都需要外部 harness 来补偿执行调度的缺失。

## 同构的核心：脚手架 vs 拐杖

关键区别在于：脚手架是临时辅助，目标是最终移除；拐杖是永久替代，会削弱内在能力。Goblin Tools 等 AI 工具的设计初衷是脚手架——它们补偿执行功能，但不应替代用户发展内在能力。然而，现有证据指出，过度依赖 AI 工具可能削弱内在时间感知能力和情绪调节能力（来源：矛盾与存疑）。

这恰恰是 agent 工程中的核心挑战：如何设计会褪去的脚手架？如何确保 agent 在依赖外部约束的同时，逐步内化这些能力？ADHD 用户和 agent 工程师面对的是同一道题：平衡补偿与训练，避免从脚手架滑向拐杖。

## 局限与争议

必须诚实指出，当前证据主要来自用户报告和概念类比，缺乏大规模对照实验（来源：矛盾与存疑）。Goblin Tools 等工具的随机对照试验尚未完成，其长期效果和神经可塑性变化未知。个体差异巨大——ADHD 亚型不同（注意缺陷型 vs. 多动冲动型），对 AI 工具的反应差异大（来源：矛盾与存疑）。此外，基于多巴胺的干预（如 Inflow 的游戏化反馈）可能强化成瘾行为，需谨慎设计（来源：AI 与 ADHD 的情绪调节）。
