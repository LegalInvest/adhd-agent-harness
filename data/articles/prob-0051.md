# 3 个 LLM 工程概念，彻底重写了我对 ADHD 不擅长精确、重复、状态维护类任务的理解

你有没有过这样的体验：脑子里涌出一百个绝妙想法，但一坐到书桌前就瘫了，连打开文档的力气都没有；或者明明知道该做什么，却像被一层透明的膜挡住，怎么也启动不了。更糟的是，那些需要反复核对、保持状态、持续维护的任务——比如填表格、记账、整理周报——简直像酷刑。

如果你是一个在做 Agentic Harness 工程的工程师，你可能会觉得这描述很熟悉：大语言模型（LLM）同样能生成惊艳的文本，但让它在 2000 词的上下文中保持一个目标、精确执行多步计算、不跑偏，你得像伺候一个天才却任性的小孩一样，给它搭 prompt template、写 function schema、做 grounding。

这两件事——ADHD 个体的日常挣扎与 LLM 工程的日常困境——本质上是同一个问题。三个 LLM 工程概念，彻底重写了我对 ADHD 不擅长精确、重复、状态维护类任务的理解。

## 概念一：外部执行功能层（External Executive Function Layer）

在 LLM 工程里，我们常说 LLM 是一个强大的生成核心，但缺少可靠的调度层。所以我们要给它搭 harness：定义好 function calling 的 schema，让模型把精确计算（比如数学、日期推算）外包给工具，自己只负责语义理解和生成。这个 harness 就是「外部执行功能层」。

ADHD 大脑的结构与此高度同构（来源：ADHD 大脑与 LLM 的同构）。ADHD 个体也拥有一个高产、联想的生成核心——想法多、直觉准、能看见大局——但内置的执行调度系统（前额叶皮层）效率低下，导致任务启动困难、工作记忆不稳定、时间感知扭曲（来源：执行功能障碍）。

AI 工具的价值不在于「治愈」ADHD，而在于搭建一个临时的外部执行功能层（来源：ADHD 的 AI 工具全景）。比如 Goblin Tools 的 Magic ToDo，它接受“清理厨房”这种模糊指令，自动分解成“把碗放进洗碗机”“擦台面”“拖地”等子步骤，用户还可以调节“辣度”控制粒度（来源：Goblin Tools）。这就像 LLM 的 function calling：把模糊意图拆解成可执行的原子操作，交给外部系统去完成。

## 概念二：Agent Scaffolding（代理脚手架）

在 LLM 工程中，agent scaffolding 是指围绕大语言模型构建软件架构和工具，使其能够执行复杂、目标驱动的任务（来源：Lex）。Lex 这款 ADHD 工具的设计理念与此完全一致：允许用户通过单一指令触发复杂、多步骤的任务序列（来源：Lex）。

历史人物曾国藩的「日课十二条」就是一个典型的个人版 agent scaffolding。他 30 岁前极度浮躁，日记里天天骂自己“无恒”“浮躁”，读书慢到“他人目下二三行，余或疾读不能终一行”。他的解决方案是：黎明即起、读书不二、谨言、写日记反省——用一套固定的、可重复的规则（脚手架）来约束自己的冲动（来源：人物案例）。这套系统与 LLM 的 agent scaffolding 同构：日课相当于定时 re-grounding 的 prompt，日记相当于日志和反思机制，而“结硬寨打呆仗”则相当于用最笨最稳的工具函数（如精确计算、状态检查）来抵消生成核心的冲动。

今天，你可以用 Saner.AI 搭建自己的数字脚手架。它通过本地记忆和知识回忆，减少搜索循环和标签切换（来源：Saner.AI）。你只需要一个指令，它就能从链接的邮件、文档或日历中提取待办事项，拆解为里程碑，并自动排程。这本质上就是 agent 的 tool use——把状态维护（记住截止日期、项目进展）外包给外部系统。

## 概念三：Function Calling 与认知卸载

LLM 通过 function calling 把精确计算外包给工具，这是它得以可靠执行复杂任务的关键。ADHD 个体同样需要这种认知卸载：把需要精确、重复、状态维护的任务——比如时间估算、任务分解、笔记整理——交给 AI 工具。

ChatGPT 被成人 ADHD 用户用作「外部执行功能」工具（来源：ADHD 的 AI 工具全景）。Claude 则以代码编写和结构化推理见长，帮助整理待办与简化任务。这些工具的价值不在于替代大脑，而在于把 ADHD 的「高产生成核心」接入一个可维护的「执行调度系统」（来源：ADHD 的 AI 工具全景）。

但这里有一个必须指出的争议：AI 工具是「脚手架」还是「拐杖」？脚手架帮助你在建造时站稳，但一旦建成可以撤掉；拐杖则让你永远依赖它。目前缺乏纵向研究证明长期使用 AI 工具是否会导致能力退化（来源：矛盾与存疑）。我的判断是：关键不在于是否使用工具，而在于你是否理解工具的原理，并有意识地训练自己逐步减少依赖。曾国藩一辈子写日记反省，直到 70 岁才做到“从心所欲不逾矩”——脚手架最终内化了。

## 今天就能试的 3 件事

1. **用 Goblin Tools 的 Magic ToDo 分解一个你拖延已久的任务**：把“写报告”变成“打开文档→写标题→列出三个要点→写第一段”。感受一下启动门槛降低后的心理变化。
2. **在 ChatGPT 或 Claude 里建一个“执行调度员” prompt**：告诉它“我容易分心，请帮我把以下目标拆成 10 分钟能完成的步骤，并在我做完一步后提醒我下一步”。试试看它能不能像曾国藩的日课一样帮你 re-ground。
3. **用 Saner.AI 或类似工具搭建一个“万能收件箱”**：把所有待办、笔记、灵感都丢进去，让 AI 自动分类、排期。观察一周内你的搜索循环减少了多少。
