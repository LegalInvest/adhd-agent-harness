# ADHD 家长如何用「给生成核心套 harness」的思路应对陪伴 ADHD 孩子里的注意力容易被环境带偏？

## 问题：陪伴 ADHD 孩子时，为什么你的注意力总被环境带偏？

你正在陪 ADHD 孩子写作业，手机响了，你瞥了一眼——然后开始回消息。孩子趁机走神，你回过神时，他已经开始玩橡皮。你懊恼，但下次还是这样。

这不是你的错。ADHD 大脑的注意力系统就像一台没有调度器的生成引擎：创意、冲动、担忧不断涌现，但缺乏一个“该关注什么”的执行层。当环境里出现任何新刺激（消息、噪音、孩子的动作），注意力就会立刻被捕获。

LLM 也有同样的毛病。一个裸 LLM 会被上下文窗口里的任何新 token 带偏：用户问一句“顺便问一下天气”，它就忘了原本在写代码。所以工程师会给它套上 harness：外部记忆、工具调用、规划循环——这些组件告诉 LLM“现在该处理什么，忽略什么”。

本文的核心观点是：**ADHD 家长陪伴 ADHD 孩子时，需要给自己和孩子都套上一个注意力 harness，其结构与 LLM 的 harness 同构。** 这不是比喻，而是基于同一类底层问题的工程方案。

## 同构：ADHD 大脑与 LLM 共享同一个缺陷

ADHD 大脑与 LLM 在结构上高度同构：两者都是 **高产但缺乏可靠执行调度层的生成核心**（来源：AI 与 ADHD 的专注力）。ADHD 的创意发散与 LLM 的文本生成一样强大，但缺乏内在的“上下文工程”能力——即主动设计“当下注意什么”的机制（来源：AI 与 ADHD 的专注力）。LLM 需要外部脚手架来规划、分解任务、记忆状态（来源：规划循环与任务分解），ADHD 大脑同样需要外部结构来补偿执行功能缺陷。

曾国藩就是最好的例子。他 30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”，读书慢到“他人目下二三行，余或疾读不能终一行”。他的解决方案是 **日课十二条**：黎明即起、读书不二、谨言、写日记反省——这是一套严格的个人 harness。每条日课就像 LLM harness 里的一个“定时 re-grounding”组件，每天强制重置注意力焦点。他的“结硬寨打呆仗”更是典型的“用最笨最稳的方法抵消冲动”——对应 LLM 工程中的“验证循环”和“任务分解”。曾国藩不是靠意志力硬扛，而是靠设计外部系统来管理自己的注意力。

## 设计你的陪伴 harness：三个同构组件

### 1. 外部启动锚点：Focusmate 式的身体在场

ADHD 孩子最难的是“开始”。LLM 同样需要 prompt 才能启动。Focusmate 利用虚拟身体在场提供实时问责（来源：Focusmate），通过 AI 匹配伙伴，共同工作。你可以和孩子约定“我们一起开始”：你坐在旁边，不监督，只是“在场”。这种身体在场效应（来源：Focusmate）能显著提升任务启动和维持——就像给 LLM 一个明确的 system prompt + 实时反馈。

### 2. 外部时间感知：RescueTime 式的自动记录

ADHD 的时间盲（来源：RescueTime）让孩子无法感知时间流逝。LLM 的“无状态性”也需要外部记忆来维持连续性（来源：AI 与 ADHD 的专注力）。RescueTime 自动记录时间去向，减轻工作记忆负担（来源：RescueTime）。你可以用一个简单的计时器或物理沙漏，让时间变得可见。每次孩子分心时，指一下沙漏——这相当于 LLM harness 里的“会话状态记录”（来源：规划循环与任务分解）。

### 3. 外部环境调节：Brain.fm 式的神经锁相

环境噪音是注意力的杀手。Brain.fm 利用 AI 生成音乐，通过神经锁相技术影响大脑状态（来源：Brain.fm）。你可以为孩子创建一个“专注环境”：白噪音、特定音乐，甚至只是关闭所有屏幕通知。这相当于给 LLM 的上下文窗口做清理——只保留相关 token，屏蔽干扰。

## 脚手架 vs 拐杖：一个必须明确的边界

所有 harness 都面临一个风险：过度依赖导致能力退化（来源：矛盾与存疑）。曾国藩的日课是脚手架——它最终内化为习惯，让他成为“半个圣人”。而如果家长永远替孩子做注意力调度，孩子就学不会自我调节。

关键在于：**harness 应该逐步撤除，而不是永久佩戴。** 就像 LLM 工程中，harness 最终要引导模型学会自主规划，而不是永远依赖外部循环。你可以先和孩子一起设计 harness，然后每周减少一个组件，观察孩子是否能够独立维持。

## 今天就能试的行动

1. **设置一个“开始仪式”**：每天陪孩子开始任务前，花 1 分钟做同一件事（深呼吸、倒计时、或说一句固定的话）。这相当于 LLM 的 prompt 模板。
2. **使用物理计时器**：买一个沙漏或机械定时器，让孩子自己设定时间。这相当于 RescueTime 的简化版。
3. **创建一个“专注歌单”**：和孩子一起选 3-5 首不会引起分心的音乐，只在专注时段播放。这类似 Brain.fm 的神经锁相原理。
4. **每周一次“无 harness 测试”**：选择一个任务，让孩子完全自己管理注意力，记录结果。如果失败，分析哪个组件缺失。

## 局限与争议

ADHD 大脑与 LLM 的同构是隐喻而非严格等价（来源：矛盾与存疑）。LLM 的“注意力机制”与人类注意力存在本质差异，且个体差异巨大：有些孩子对番茄钟反应好，有些则厌恶打断（来源：矛盾与存疑）。此外，多数工具（如 Brain.fm）缺乏针对 ADHD 的独立临床研究（来源：Brain.fm）。因此，本文提供的方案需要你根据孩子的反应持续调整——就像 LLM 工程师需要不断调优 prompt 一样。

最后，记住：你不是在“修复”孩子，而是在为他设计一个能发挥其天赋的外部系统。ADHD 大脑的生成能力是宝藏，harness 只是让宝藏能被有序开采的工具。
