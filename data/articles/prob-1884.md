# ADHD 家长如何用「给生成核心套 harness」的思路应对陪伴 ADHD 孩子里的分心走神或超聚焦跑偏，丢失目标？

> 把 LLM 工程里的「长程任务里 agent 会目标漂移，需要重锚定 system prompt」迁移到ADHD 家长的陪伴 ADHD 孩子场景，双域证据作答。

## 问题：分心走神与超聚焦跑偏，谁在偷走目标？

ADHD 孩子的日常，常常是两种极端：要么像蝴蝶一样从一件事跳到另一件事，无法启动或维持任务；要么像激光一样钉在某个兴趣上，完全忽略时间与优先级。家长在陪伴中反复提醒、催促、打断，不仅消耗自己，也让孩子感到被控制。这种“目标漂移”的困境，像极了长程任务中的 AI agent：没有外部 harness 的 LLM，会在多步推理中偏离初始指令，生成无关内容。而 ADHD 大脑，正如 wiki 资料所述，是“高产但缺乏可靠执行调度层的生成核心”（来源：AI 与 ADHD 的专注力）。那么，家长能否像工程师给 agent 套 harness 一样，给孩子套一个“外部执行功能层”？

## 同构：ADHD 大脑与 LLM 共享同一个 bug

从结构看，ADHD 大脑与 LLM 的相似性惊人。两者都有强大的生成能力——孩子的创意发散、LLM 的文本生成——但都缺乏内在的调度层。ADHD 的执行功能缺陷（规划、工作记忆、时间盲）对应 LLM 的上下文控制缺陷；ADHD 的“无状态”工作记忆对应 LLM 的无状态性，都需要外部记忆系统维持连续性（来源：AI 与 ADHD 的专注力）。因此，AI 帮助 ADHD 的本质是“给生成核心套上 harness”（来源：AI 与 ADHD 的专注力）。同理，家长的角色不是替代孩子的执行功能，而是搭建一个可拆卸的脚手架。

## 历史中的 harness：曾国藩的日课十二条

晚清名臣曾国藩，30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”，读书慢到“他人目下二三行，余或疾读不能终一行”。他的解决方案是“日课十二条”：黎明即起、读书不二、谨言、写日记反省……用最笨最稳的方法抵消冲动。这套系统本质上是给大脑套上的外部调度层——定时 re-grounding（每日反省）、任务分解（“结硬寨打呆仗”）、外部记忆（日记）。这与 LLM harness 中的“会话状态记录工具结果、完成的子任务和进度笔记”（来源：规划循环与任务分解）完全同构。曾国藩没有试图改变自己的大脑，而是用外部结构锚定目标。

## 现代工具：从 Focusmate 到 RescueTime 的 harness 逻辑

现代 AI 工具正是这种 harness 的数字化版本。Focusmate 通过算法匹配实时问责伙伴，利用“身体在场效应”弥补执行功能缺陷（来源：Focusmate）。它相当于 agent harness 中的“外部验证循环”：每次模型调用后检查结果是否对齐目标。RescueTime 自动追踪时间，揭示“时间到底去了哪里”，对抗时间盲（来源：RescueTime），相当于 LLM 的“外部记忆缓冲区”记录进度。Brain.fm 用 AI 生成音乐实现神经锁相，帮助进入专注状态（来源：Brain.fm），类似 agent 的“注意力锚点”。Motion 则动态调整日程，自动分解任务（来源：规划循环与 ADHD 一侧），对应 harness 中的“任务分解与规划循环”。

这些工具的关键在于：它们提供的是“脚手架”而非“拐杖”。脚手架在能力提升后可拆除，拐杖则造成永久依赖。目前学界对依赖风险的争议尚无定论（来源：矛盾与存疑），但一个可行的原则是：工具应“无缝嵌入日常流程，不增加认知开销”（来源：工具使用与认知卸载），且使用目标是让孩子逐渐内化这些结构，而非永远依赖。

## 行动：今天就能试的 3 条 harness 策略

1. **建立“日课锚点”**：和孩子一起设计 3 个固定动作（如早起后列 3 件事、晚饭前回顾完成度），用白板或便签可视化。这模仿曾国藩的日课和 agent 的定时 re-grounding。
2. **使用 Focusmate 或类似的身体在场机制**：每周安排 2-3 次 25 分钟的视频共学时段，让孩子与同伴或家长一起工作。这利用外部问责弥补内在启动困难。
3. **用 RescueTime 或手动时间日志揭示时间去向**：每周一次，和孩子一起看时间花在哪，不批评，只观察。这帮助孩子建立时间感知，对抗时间盲。

## 局限：同构是隐喻，不是科学

必须诚实指出，ADHD 大脑与 LLM 的同构目前是理论框架，缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。此外，个体差异巨大：有的孩子对柔性引导反应更好，有的则需要强力打断（来源：矛盾与存疑）。家长需要像工程师调试 agent 一样，不断观察、调整、迭代自己的“harness 方案”。最终，目标不是让孩子变成“不 ADHD”，而是让他们的高产生成核心，在一个可靠的外部调度层中，自由而有序地运行。
