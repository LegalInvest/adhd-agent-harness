# 如果把 ADHD 大脑当成生成核心，Mem 能不能当它的重锚定与目标漂移？

> 用 agent 工程的思路重新理解 Mem 对 ADHD 的真正价值。

## 问题：为什么你的大脑和你的 AI agent 一样容易“跑偏”？

你打开电脑，计划写一份方案。三小时后，你在一个关于“如何训练鹦鹉说话”的 YouTube 视频里醒来。这不是意志力问题——你的大脑是一个**高产但缺乏执行调度层的生成核心**：创意源源不断，但“任务启动”“时间感知”“目标维持”这些调度功能却像坏掉的舵。同样的问题正在困扰 AI 工程师：他们设计的 agent 在长程任务中会**目标漂移**——从“写报告”滑向“研究历史背景”，最终产出与初始指令无关的内容。两边面对的是同一个结构性问题：生成核心缺少**重锚定机制**。

## 同构：ADHD 大脑与 LLM 的共享困境

ADHD 大脑与 LLM 的深层同构体现在三个层面（来源：AI 与 ADHD 的专注力）：

1. **生成核心与调度层分离**：ADHD 的创意发散与 LLM 的文本生成都极其强大，但执行功能（ADHD 侧）或上下文控制（LLM 侧）薄弱，导致输出不稳定。
2. **无状态与外部记忆**：ADHD 的[[工作记忆]]缺陷与 LLM 的无状态性同构——两者都无法可靠地维持跨步骤的上下文，都需要外部记忆系统（如第二大脑、向量数据库）来保持连续性。
3. **上下文工程**：ADHD 易被环境带偏，LLM 易被上下文膨胀干扰，都需要主动设计“当下注意什么”的工程方案。

这意味着，为 ADHD 大脑设计工具，本质上就是在为生成核心搭建 **harness（外部执行功能层）**。这个 harness 不是拐杖，而是脚手架——它提供结构，但最终依赖用户主动使用。

## 历史回响：两位 ADHD 先驱的 harness 系统

**曾国藩**（晚清名臣）是典型的注意力不集中型 ADHD：30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”，读书慢到“他人目下二三行，余或疾读不能终一行”。他的解决方案是 **日课十二条**：黎明即起、读书不二、谨言、写日记反省等，用最笨最稳的方法（“结硬寨打呆仗”）抵消自己的冲动。这套系统本质上是一个**定时 re-grounding 协议**——每天固定时间重读自己的“system prompt”，检查是否偏离目标。

**张居正**（万历首辅）则是另一种类型：少年天才，工作狂，58 岁累死在任上。他的 harness 是 **考成法**——用严格的外部考核制度管别人也管自己。这相当于一个**外部调度器**，通过定期检查进度来防止目标漂移。

两人的系统与 LLM harness 同构：曾国藩的日课↔定时重注入 system prompt；张居正的考成法↔外部进度检查器。核心都是**在生成核心之外建立可操作的锚点**。

## Mem 能成为重锚定吗？

Mem 是一款 AI 笔记工具，其核心是**外部记忆系统**——自动记录、检索、连接信息。对于 ADHD 大脑，Mem 的价值不在于“记住一切”，而在于**在注意力漂移时提供一个可返回的上下文锚点**。当你在 Mem 中写下目标并让它自动关联相关笔记，你就建立了一个**可检索的 system prompt**——当超聚焦把你带偏时，你可以回到 Mem，搜索“我原本要做什么”，重新锚定。

这与 agent 工程中的做法完全一致：长程 agent 会在每个步骤后把当前状态写回外部记忆，并在下一步开始时读取最新上下文，防止目标漂移。Mem 对 ADHD 的作用，就是充当这个**外部上下文缓冲区**。

但这里有一个关键争议：**脚手架 vs 拐杖**。如果 Mem 只是被动记录，你从不主动回看，它就是拐杖——让你更依赖外部存储，而不是训练自己的定向能力。如果主动用它来“重读目标”并调整行动，它就是脚手架——帮你逐步建立内部锚定能力。证据显示，长期依赖外部执行功能层的风险尚不明确（来源：矛盾与存疑），但**主动使用**是区分脚手架与拐杖的边界。

## 证据与局限

现有工具中，**Focusmate** 通过虚拟身体在场提供实时问责，有用户研究支持其提升任务启动和维持（来源：Focusmate）；**RescueTime** 通过自动时间记录对抗时间盲（来源：RescueTime）；**Brain.fm** 利用神经锁相技术帮助进入专注状态（来源：Brain.fm）。但它们的共同局限是：缺乏针对 ADHD 的大规模随机对照试验，效果因个体差异而异（来源：AI 与 ADHD 的专注力）。Mem 作为外部记忆系统，同样缺乏 ADHD 专项研究，其有效性取决于用户是否主动使用“重锚定”功能。

同构视角本身也有局限：ADHD 大脑与 LLM 的注意力机制存在本质差异——人类的注意力受情绪、多巴胺、身体状态影响，而 LLM 的注意力是数学计算（来源：矛盾与存疑）。因此，类比只能提供设计思路，不能替代临床验证。

## 今天就能试的 3 件事

1. **在 Mem（或任何笔记工具）中创建一个“目标锚点”笔记**，每天开始时写一句话定义今天最重要的任务，并在每次分心后回看它。
2. **设置定时提醒**（类比曾国藩的日课），每 30 分钟检查一次：“我现在做的事与目标一致吗？”——这是最简单的 system prompt 重注入。
3. **使用 Focusmate 或类似工具**，让另一个人成为你的外部调度器——就像张居正的考成法，外部问责能有效防止目标漂移。

记住：你的大脑不是坏的，它只是缺少一个可检索的 system prompt。而 Mem，就是那个 prompt 的载体。
