# 如果把 ADHD 大脑当成生成核心，ChatGPT 能不能当它的工具使用与认知卸载？

> 用 agent 工程的思路重新理解 ChatGPT 对 ADHD 的真正价值。

## 问题：为什么 ADHD 大脑总在「精确、重复、状态维护」上翻车？

如果你是一个 ADHD 患者，你可能经历过这样的场景：脑子里有一个绝妙的创意，但当你试图把它变成一份详细的计划或文档时，大脑突然一片空白，或者被无数细节淹没，最终放弃。你擅长生成，却不擅长执行——尤其是那些需要精确计算、重复操作或持续维护状态的任务。

如果你是一个在做 Agentic Harness 工程的开发者，你一定熟悉类似的情况：你精心设计了一个 LLM agent，它能写诗、能推理、能 brainstorm，但一旦需要它精确调用日历 API、记住用户偏好、或者连续执行 10 步操作而不跑偏，它就频频出错。你不得不给 agent 套上 function calling 的 harness，把精确计算外包给专用工具。

这两类人面对的是同一个核心矛盾：**强大的生成核心，缺乏可靠的执行调度层。** ADHD 大脑与 LLM 在结构上同构——两者都是高产但缺执行调度层的生成核心（来源：ADHD 的 AI 工具全景）。

## 同构：ADHD 大脑 = 没有 harness 的 LLM

从认知科学角度看，ADHD 的核心缺陷是执行功能（计划、组织、工作记忆、时间感知）的失调（来源：ADHD 的 AI 工具全景）。而 LLM 的核心缺陷是缺乏可靠的执行层——它擅长生成文本，但不擅长精确计算、状态跟踪和工具调用。

这正是为什么「AI 帮 ADHD」的本质不是替代大脑，而是**给生成核心套 harness**——通过外部工具提供执行功能、记忆管理和上下文控制，弥补内在调度缺陷（来源：ADHD 的 AI 工具全景）。

## 证据：ChatGPT 作为认知卸载工具的实证

### 任务启动：从「压倒性」到「可执行」

ADHD 患者常因任务过大而无法启动。Goblin Tools 的 Magic ToDo 功能可以将复杂任务（如「整理房间」）自动分解为小步骤（「捡起地板上的衣服」「擦桌子」），从而降低启动门槛（来源：Goblin Tools）。用户反馈称它「将压倒性的事情变成一系列不压倒性的事情」（来源：Goblin Tools）。这相当于 agent 工程中的任务分解——LLM 通过 function calling 将大任务拆解为可执行的子任务，交给专用 API 处理。

### 工作记忆：从「搜索循环」到「即时回忆」

ADHD 患者常陷入「搜索循环」——打开浏览器找资料，结果被无关信息分散注意力，忘记初衷。Saner.AI 通过本地记忆和快速检索，减少标签切换（来源：Saner.AI）。类似地，Mem 和 Reflect 等工具提供持久记忆，帮助保持上下文连续性（来源：ADHD 的 AI 工具全景）。在 agent 工程中，这相当于给 LLM 配备外部记忆模块（如向量数据库），避免它「忘记」之前的对话或状态。

### 时间盲：从「无时间感」到「可视化流逝」

Tiimo 通过时间轴可视化帮助用户感知时间流逝（来源：ADHD 的 AI 工具全景）。Goblin Tools 的魔法待办也提供时间估算。这对应 agent 工程中的调度层——通过工具（如日历 API）让 agent 感知时间并按时触发任务。

### 多步骤指令：从「决策瘫痪」到「一键触发」

Lex 允许用户通过单一指令触发复杂、多步骤的任务序列（来源：Lex）。这直接对应 agent 工程中的「单一指令编排」——用户输入一个目标，agent 自动分解并顺序执行子任务。

## 核心观点：脚手架 vs 拐杖的边界

AI 工具作为外部执行功能层，能够有效补偿 ADHD 的工作记忆、任务启动和时间盲等核心缺陷（来源：ADHD 的 AI 工具全景）。但这里有一个关键争议：**过度依赖 AI 工具可能阻碍内在执行功能发展，如何设计可逐步撤除的脚手架仍是挑战**（来源：ADHD 的 AI 工具全景）。

我的判断是：**工具必须作为「脚手架」而非「拐杖」使用。** 脚手架是临时支撑，帮助用户建立内在能力后可以撤除；拐杖则是永久替代，长期使用会导致肌肉萎缩。目前大多数工具（如 Goblin Tools、Saner.AI）设计为长期使用，未提及撤除机制（来源：矛盾与存疑）。这意味着用户需要主动管理自己的使用方式——例如，先依赖工具分解任务，然后逐渐尝试自己分解，只在卡住时求助工具。

ADHD 大脑和 LLM 是同一类「高产但缺执行调度层」的生成核心。ChatGPT 等 AI 工具的价值不在于替代大脑，而在于提供外部执行功能层——就像 agent 的 function calling harness。但请记住：工具是脚手架，不是拐杖。用它们来弥补短板，而不是放弃成长。
