# 3 个 LLM 工程概念，彻底重写了我对 ADHD 有想法有能力，却卡在执行与落地的理解

> LLM 智力强但需要外部编排才能完成长任务 如何反向照亮 ADHD 的有想法有能力，却卡在执行与落地。

你有没有过这样的体验：脑子里同时冒出三个绝妙的项目方案，但一坐到电脑前，却盯着空白文档发呆，最后刷了两小时短视频。或者，你明明知道自己能搞定一项复杂任务，但就是无法把它拆解成一步步的行动——启动它比登天还难。

这不是意志力问题。这是执行功能缺陷——计划、组织、工作记忆、冲动控制等环节容易失稳。

如果你是一名 LLM 工程师，你可能会觉得这个描述很熟悉：LLM 智力强但需要外部编排才能完成长任务。无状态、易幻觉、难以自主管理多步骤任务——LLM 和 ADHD 大脑，共享同一个核心困境。

三个 LLM 工程概念，彻底重写了我对 ADHD「有想法有能力，却卡在执行与落地」的理解。

## 概念一：生成核心与调度层

ADHD 大脑与 LLM 在结构上同构：二者都具备强大的生成能力，却都缺乏稳定、可靠的内置执行调度层（来源：ADHD 大脑与 LLM 的同构）。对 LLM 而言，这表现为无状态、易幻觉、难以自主管理多步骤任务；对 ADHD 大脑而言，则体现为执行功能缺陷。

这意味着，ADHD 个体并不缺想法、创造力或信息处理能力，缺的是把想法稳定转化为可交付成果的「控制层」。AI 工具可以扮演这个控制层：把模糊目标拆解为可执行步骤、维持跨会话上下文、提供外部提醒与时间锚点，从而把「生成核心」的输出导入可靠的职业轨道（来源：生成核心与调度层）。

屠呦呦就是一个经典案例。这位诺贝尔奖得主身上有明显的 ADHD 特质：不喜欢应酬不喜欢采访，一门心思搞研究，自己试药得了肝病。她的专属 harness 是什么？严格的实验流程，反复筛选药物，筛选了2000多种中药380多个提取物。这个流程本质上是为她的「生成核心」（强大的科研直觉与创造力）加装了一个外部调度层——实验流程就是她的 LLM harness，确保每一步都有据可查、可复现。

## 概念二：外部记忆与上下文工程

ADHD 大脑的工作记忆缺陷与 LLM 的无状态性相似：二者都无法可靠保留跨会话上下文（来源：无状态与外部记忆）。明尼苏达大学系统测试 LLM 的执行功能，发现「强记忆、弱控制」剖面——工作记忆容量甚至超过常人，但认知灵活性与注意控制存在核心缺陷，这正是 ADHD 的经典神经心理模式（来源：工作记忆）。

耶鲁大学进一步发现，自注意力机制本身导致工作记忆容量限制：随 N-back 任务 N 值增加，注意力分数熵增、注意力弥散、表现下降，与人类注意分散机制同源（来源：工作记忆）。这为外部记忆/外挂脚手架的必要性提供了架构层面的解释。

Saner.AI 就是这样一个外部记忆系统。它利用 AI 实现本地记忆存储和快速检索，用户无需在多个标签页或应用间切换即可获取所需信息（来源：Saner.AI）。它被描述为「专为对抗执行功能障碍、任务瘫痪和认知超载而设计的全能 ADHD 友好型 AI 生产力和知识助手」（来源：Saner.AI）。

而上下文工程则关注注意力管理：ADHD 大脑与 LLM agent 都容易被环境线索带偏，或因上下文膨胀导致推理退化（来源：上下文工程）。因此，职业发展工具需要主动设计「当前应关注什么」，而非被动接收所有信息。

## 概念三：脚手架 vs 拐杖

这里有一个必须指出的争议：AI 工具是「外部执行功能层」还是「拐杖」？

核心论点认为，AI 作为外部执行功能层补偿缺陷（来源：外部执行功能层）。但存疑的是，过度依赖可能导致能力退化，成为永久拐杖（来源：拐杖与脚手架）。长期依赖风险缺乏实证，需更多纵向研究。

我的判断是：关键在于使用方式。如果 AI 工具帮你分解任务、提醒截止日期，同时你仍然保持对任务逻辑和优先级的主动思考，它就是脚手架——最终让你独立行走。但如果它替你做了所有决策，你只是被动执行，它就是拐杖——可能导致能力退化。

Motion 就处在这个边界上。它是一款 AI 驱动的日程规划应用，能够自动根据任务、会议和截止日期创建并动态调整每日计划（来源：Motion）。当任务面临延期风险时，Motion 会提前数天或数周发出警告（来源：Motion）。这大大减少了「下一步该做什么」的决策负担，直接帮助用户克服时间盲和任务启动困难（来源：Motion）。但如果你完全依赖它，不再培养自己的时间感知能力，它就可能变成拐杖。

## 局限与争议

需要诚实指出，ADHD 大脑与 LLM 同构的论点目前多为类比推理，缺乏神经科学或计算机科学的直接证据（来源：矛盾与存疑）。此外，AI 工具对任务启动的长期效果未知（来源：矛盾与存疑），个体差异极大。

但至少，这个视角提供了一个全新的工程框架：与其自责「意志力不够」，不如承认你的大脑是一个高产但缺调度层的生成核心，然后为它设计一个可靠的外部 harness。

这不仅是 ADHD 的自我管理策略，也是 LLM 工程的核心命题。
