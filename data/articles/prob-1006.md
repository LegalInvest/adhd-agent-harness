# Claude 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 Claude：它到底补上了哪一层执行功能？

你是否有过这样的体验：打开 Claude 准备写一份报告，却在 20 分钟后发现自己正在让它帮你写一封无关的邮件、规划一次旅行，甚至讨论哲学问题——而最初的报告纹丝未动？

这不是你的错，也不是 Claude 的错。这是两种“高产但缺乏可靠执行调度层”的生成核心相遇时的必然结果。ADHD 大脑与 LLM 在结构上同构：两者都拥有强大的生成能力（ADHD 的创意发散、LLM 的文本生成），但缺乏可靠的执行调度层（ADHD 的执行功能、LLM 的上下文控制），导致输出不稳定（来源：AI 与 ADHD 的专注力·核心论点）。当 ADHD 用户将 Claude 当作“执行功能外挂”时，他们实际上是在用一个同样容易漂移的系统去锚定另一个——结果往往是双重漂移。

## 同构的陷阱：目标漂移与重锚定

ADHD 的典型困境是：分心走神或超聚焦跑偏，丢失最初目标。LLM/Agent 在长程任务中同样会“目标漂移”——随着上下文膨胀，模型逐渐偏离原始指令。两者的解法在结构上同构：都需要一个外部重锚定机制。

对于 LLM，工程上通过定期重锚定 system prompt（例如每 N 轮对话插入原始指令摘要）来维持目标。对于 ADHD 大脑，历史上最成功的重锚定系统之一来自曾国藩。曾国藩 30 岁前浮躁坐不住，天天串门喝酒，日记里骂自己“无恒”“浮躁”。他的解法是“日课十二条”：黎明即起、读书不二、谨言、写日记反省。这套系统本质上是一个外部调度层——每天固定时间用固定动作将注意力重新锚定到核心目标上。用现代术语说，这就是一个定时 re-grounding 机制，与 LLM 的定期重锚定 system prompt 完全同构。

胡林翼则是另一个例子：年轻时是扬州花花公子，父亲去世后幡然醒悟，每日写日记反省，严格治军。他的“以做百姓之心做官，以治私事之心治官事”本质上是一种将外部结构内化的过程，与 LLM 通过 fine-tuning 将 system prompt 内化到参数中异曲同工。

## 工具：哪些真正提供了重锚定？

基于同构视角，我们可以区分两类 AI 工具：一类提供“外部调度层”（脚手架），另一类试图替代执行功能本身（拐杖）。

**Focusmate** 是最接近重锚定 system prompt 的工具。它通过算法匹配虚拟身体加倍伙伴，利用身体在场效应——定期问责检查可将目标达成可能性从 25% 提升至 95%（来源：身体在场效应）。这里的“定期检查”正是 LLM 工程中常用的 checkpoint 机制：每 25 分钟与同伴同步一次目标，相当于强制重锚定。

**RescueTime** 则扮演了“日志系统”的角色。它自动记录时间去向，帮助用户识别注意力漂移。对于 ADHD 的时间盲，这相当于 LLM 的 token-by-token 注意力记录——让你看到“时间到底去了哪里”（来源：RescueTime）。但它的局限在于：它只告诉你漂移了，不帮你锚定回去。

**Brain.fm** 通过神经锁相技术影响大脑状态，但它不提供目标重锚定，更像是 LLM 的“温度参数调节”——改变生成风格，但不改变方向。

**Goblin Tools** 的“魔法待办”将复杂任务拆解为小步骤，降低启动门槛（来源：任务启动）。这相当于 LLM 的 step-by-step reasoning prompt，但如果没有后续的锚定机制，拆解后的子任务同样会漂移。

## 脚手架 vs 拐杖：边界在哪？

核心判断是：**AI 工具的有效性取决于它是否增强而非替代用户的元认知监控**。Focusmate 之所以更接近脚手架，是因为它要求你主动向同伴声明目标——这训练了目标设定能力。而完全依赖 AI 自动排程（如 Motion）则可能削弱内在的时间感知能力。

争议在于：长期使用外部执行功能层是否会导致内在能力退化？（来源：矛盾与存疑）目前缺乏纵向研究。但基于同构视角，我们可以提出一个假设性边界：如果工具让你在“脱离工具后”仍然能执行相同任务，它是脚手架；如果脱离后你完全无法启动，它就是拐杖。

## 今天就能试的行动

1. **用 Claude 做“系统提示”而非“自动完成”**：每次对话开始前，让 Claude 先写一段你的目标声明（类似 system prompt），并在每 3 轮对话后要求它重述原始目标。

2. **尝试 Focusmate 的 25 分钟模式**：设定一个具体目标，与同伴共享屏幕，25 分钟后汇报进展。这相当于给大脑加了一个“重锚定 checkpoint”。

3. **用 RescueTime 做“注意力审计”**：运行一周，不看数据，然后突然查看——你可能会发现自己的目标漂移模式（例如总是在下午 3 点转向社交媒体）。

4. **模仿曾国藩的“日课”**：每天固定时间，用 Claude 写 3 句话：今天核心目标是什么？当前漂移到了哪里？下一步重锚定动作是什么？

## 诚实面对局限

ADHD 大脑与 LLM 的同构是隐喻而非严格等价——LLM 的“注意力机制”与人类注意力存在本质差异（来源：矛盾与存疑）。此外，个体差异极大：有些人对身体在场效应敏感，有些人则对声音干预更有效。当前证据多基于用户报告，缺乏大规模随机对照试验（来源：AI 与 ADHD 的专注力·仍存争议）。

真正的挑战不是找到“最好的工具”，而是构建一套**个性化的重锚定系统**——就像曾国藩的日课十二条，或者 LLM 的 prompt 模板——然后每天用它对抗目标漂移。Claude 可以是这套系统的一部分，但它不是系统本身。
