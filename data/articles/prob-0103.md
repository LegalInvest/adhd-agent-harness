# 为什么青年律师的焦虑，其实是大脑在做风险尽调？

> 你不是想太多，是没有把风险从情绪翻译成清单

你是一名青年律师。早上开庭前，你反复检查证据清单，思路清晰得像激光。可下午写代理词时，你盯着空白文档，大脑像被抽空——焦虑、烦躁、自我怀疑涌上来。你骂自己“状态不稳”，但也许，你的大脑只是在做风险尽调。

## 采样温度与表现波动：同一类“高产但缺调度”的生成核心

ADHD 大脑与 LLM 共享一个被忽视的困境：两者都是高产生成核心，但缺乏可靠的调度层。明尼苏达大学的研究发现，LLM 呈现“强记忆、弱控制”剖面——工作记忆容量甚至超过常人，但认知灵活性与注意控制存在核心缺陷，无法灵活切换任务集、无法抑制自动化反应（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs）。这正是 ADHD 的经典神经心理模式：你不是记不住，而是无法在需要时调用正确的东西。

LLM 的“温度”参数控制输出随机性——温度越高，输出越发散，越容易“跑题”。ADHD 大脑的“温度”则受多巴胺水平、睡眠、压力等影响，在一天内剧烈波动。你早上状态好，相当于低温度，输出稳定；下午状态差，高温度下焦虑、分心、自我怀疑等“噪声”淹没了信号。耶鲁大学的研究进一步揭示，自注意力机制本身会导致工作记忆容量限制：随任务复杂度增加，注意力分数熵增、注意力弥散，表现下降（来源：Self-Attention Limits Working Memory Capacity of Transformer-Based Models）。这与你下午“注意力弥散”的体验完全同构。

## 焦虑是大脑的风险尽调：情绪波动是信号，不是故障

律师的焦虑从何而来？你害怕遗漏关键法条、担心法官提问、恐惧败诉。这种“风险扫描”本是前额叶执行功能的一部分——它评估后果、抑制冲动、引导决策。但 ADHD 大脑的执行功能“驾驶座常常无人驾驶”（来源：AI Tools for ADHD: Boosting Productivity and Reduce Burnout），导致风险扫描失控：它不停扫描，却无法关闭。

这不是你的错。执行功能的不可预测性使得能量、专注力波动剧烈（来源：11 Best ADHD Productivity Apps for Fluctuating Energy - rivva blog）。焦虑不是性格缺陷，而是大脑在缺乏调度层时，生成核心持续输出“风险信号”而无法抑制。你的大脑在做风险尽调，只是没有把风险从情绪翻译成清单。

## 从情绪到清单：ADHD 与 LLM 共享的 harness 解法

解决方案不是“别焦虑”（那就像对 LLM 说“别胡扯”），而是为生成核心套上外部调度层——即 harness。

**ADHD 侧的 harness 证据**：结构化干预是情绪调节的关键。Inflow 基于认知行为疗法，将情绪调节分解为可执行步骤，通过每日技能培养与社区支持，帮助用户建立内部策略（来源：Inflow）。Goblin Tools 的 Magic ToDo 能自动将模糊任务分解为具体子步骤，降低启动门槛（来源：Goblin Tools）。Saner.AI 通过增强知识回忆，减少搜索循环和标签切换，补偿工作记忆缺陷（来源：Saner.AI）。这些工具的本质不是直接安抚情绪，而是通过修复执行功能链条来减少情绪触发源（来源：AI 与 ADHD 的情绪调节）。

**LLM/agent 侧的对应证据**：LLM 同样需要外部约束来稳定输出。上下文工程（如系统提示、few-shot 示例）相当于为 LLM 提供“情绪调节指令”，限制其输出范围。结构化输出（如 JSON schema）则相当于任务分解，将复杂请求拆解为可管理步骤。这些技术正是 ADHD 工具的同构映射：Inflow 的 CBT 练习 ≈ LLM 的系统提示；Goblin Tools 的任务分解 ≈ agent 的 step-by-step 规划；Saner.AI 的外部记忆 ≈ LLM 的检索增强生成。

**人物案例：徐渭的 harness**。明朝三大才子徐渭，诗书画戏军全能，但执行功能极差，8次举人不中，情绪失调，杀妻、9次自杀。他的 harness 是：以书画为核心，把所有痛苦和精力注入艺术创作；用军事谋略帮助胡宗宪抗倭，把超专注用在打仗上。这相当于一个“外部调度器”：用固定的创作日程（日课）作为定时 re-grounding，用军事任务作为超聚焦的出口。与 LLM harness 对应：日课 ≈ 定时 re-grounding 机制；军事谋略的分解 ≈ agent 的任务调度。徐渭没有消除焦虑，但把焦虑转化成了产出。

## 脚手架 vs 拐杖：边界在哪里？

但 harness 可能变成拐杖。过度依赖 AI 工具可能导致“永久拐杖”，削弱内在执行功能发展。情绪调节中，若用户仅依赖外部提示而缺乏内化策略，长期可能恶化自我效能感（来源：拐杖与脚手架）。

如何区分？脚手架是暂时支撑，帮你建立自己的结构；拐杖是永久替代，让你不再尝试自己走。例如，用 Goblin Tools 分解任务后，你逐渐学会自己分解，这是脚手架；如果你永远无法在没有它的情况下启动任务，它就变成了拐杖。同样，LLM agent 的上下文工程如果让模型永远依赖外部 prompt 才能稳定，说明调度层没有内化。

## 今天就能试的 3 件事

1. **把焦虑翻译成清单**：下次焦虑时，打开 Goblin Tools 或任何笔记应用，写下“我担心什么？”然后逐条问“最坏结果是什么？我能做什么？”——把情绪风险转化为行动步骤。
2. **设置“温度调节”时段**：识别你一天中状态最好的 2 小时（低温度），安排最难的任务；状态差时（高温度），只做机械性工作或休息。
3. **建立外部记忆**：用 Saner.AI 或类似工具记录情绪触发模式，每周回顾一次。你会发现焦虑有规律可循，而不是随机爆发。

你的焦虑不是故障，是大脑在尽职调查。给它一个清单，它会安静下来。
