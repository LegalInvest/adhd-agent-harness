# ADHD 家长如何用「给生成核心套 harness」的思路应对陪伴 ADHD 孩子里的不擅长精确、重复、状态维护类任务？

> 把 LLM 工程里的「agent 通过 function calling 把精确计算外包给工具」迁移到ADHD 家长的陪伴 ADHD 孩子场景，双域证据作答。

## 问题：ADHD 家长在陪伴 ADHD 孩子时，为什么总在“精确、重复、状态维护”上崩溃？

你是一个 ADHD 家长，孩子也是 ADHD。你充满创意、热情爆棚，但一旦涉及“每天固定时间喂药”“记录情绪变化曲线”“整理书包清单”这类精确、重复、需要持续状态维护的任务，你就像掉进了黑洞。你明明知道这些事重要，但大脑就是拒绝执行。这不是懒——这是 ADHD 的执行功能障碍在作祟（来源：执行功能障碍）。

同样的问题，在 AI 工程里有一个精确的镜像：LLM（大语言模型）擅长生成创意文本，但一旦需要精确计算、多步状态跟踪或重复操作，它就会“胡言乱语”或“忘记上下文”。工程师的解法是给 LLM 套上 **harness（脚手架）**——让 agent 通过 function calling 把精确计算外包给专用工具（如计算器、数据库、日历 API）。

## 核心答案：用“给生成核心套 harness”的思路，把不擅长的任务外包出去

ADHD 大脑与 LLM 是同构的——两者都是“高产但缺执行调度层的生成核心”（来源：ADHD 的 AI 工具全景）。因此，ADHD 家长需要的不是“更努力”，而是像工程师一样，为自己和孩子设计一个外部执行功能层：**把精确、重复、状态维护类任务，外包给 AI 工具或结构化系统，让大脑只负责生成与连接。**

具体来说，这个思路包含三个同构操作：

### 1. 任务分解：用 Goblin Tools 把“大任务”变成“小步骤”

ADHD 侧：孩子的“整理书包”听起来像一座山。家长同样会因启动困难而拖延。

LLM 侧：LLM 面对“写一份市场报告”也会生成一堆废话。

解法：**Goblin Tools 的 Magic ToDo 功能**能将任何模糊任务自动拆解为可管理的小步骤（来源：Goblin Tools）。例如输入“准备上学”，AI 输出：“穿上校服”“装好作业本”“带水壶”。这不仅降低了孩子的启动门槛，也解放了家长“必须记住所有步骤”的认知负荷。

证据：用户报告称 Goblin Tools 的分解功能“将压倒性的事情变成一系列不压倒性的事情”（来源：Goblin Tools）。

### 2. 记忆外包：用 Saner.AI 替代“我忘了”的循环

ADHD 侧：家长需要记录孩子的用药时间、情绪波动、作业完成情况，但工作记忆的脆弱让这些信息频繁丢失（来源：工作记忆）。

LLM 侧：LLM 的上下文窗口有限，一旦对话变长，它会忘记早期指令。

解法：**Saner.AI** 通过 AI 驱动的本地记忆和快速检索，减少搜索循环和标签切换（来源：Saner.AI）。家长可以用它记录“今天孩子几点吃药”“昨天情绪爆发前发生了什么”，需要时直接语音提问，不再靠大脑硬扛。

证据：Saner.AI 被列为最佳 ADHD 生产力 AI 工具之一，其知识回忆功能直接针对“信息丢失”问题（来源：Saner.AI）。

### 3. 单一指令触发多步骤：用 Lex 降低决策疲劳

ADHD 侧：每天重复的“刷牙-洗脸-吃早餐”流程，对家长和孩子都是决策负担。

LLM 侧：LLM 在每步都需要用户提示，效率低下。

解法：**Lex** 允许用户通过单一指令触发复杂、多步骤的任务序列（来源：Lex）。家长可以设定“晨间 routine”指令，Lex 自动按序提醒“刷牙”“洗脸”“吃早餐”，并记录完成状态。这就像 agent 的 function calling——把流程编排外包给工具。

证据：Lex 的设计理念与 ADHD 干预策略一致，通过降低启动门槛提升任务完成率（来源：Lex）。

## 脚手架 vs 拐杖：必须警惕的边界

以上工具是“脚手架”还是“拐杖”？关键在于使用方式。脚手架促进能力发展，拐杖则替代能力（来源：拐杖与脚手架）。例如，用 Goblin Tools 分解任务时，家长可以逐步让孩子自己学会分解；用 Saner.AI 记录时，定期回顾帮助孩子建立内在记忆策略。如果只是机械依赖，可能削弱孩子（和家长）的内在时间感知与记忆能力（来源：矛盾与存疑）。

另外，现有证据主要来自用户报告，缺乏大规模随机对照试验（来源：矛盾与存疑）。个体差异也很大——有的孩子对语音提醒敏感，有的则反感。所以不要“一刀切”，要像 A/B 测试一样尝试。

## 今天就能试的 3 个行动

1. **下载 Goblin Tools**，把明天最令你头疼的“陪孩子做作业”输入 Magic ToDo，观察分解后的步骤是否让你和孩子都松了口气。
2. **在 Saner.AI 中建立“孩子日志”**，每天语音记录 3 个关键事件（如情绪爆发、完成作业），一周后回顾模式。

## 结语

ADHD 家长不是“不够努力”，而是大脑的架构决定了你天生不擅长精确、重复、状态维护。承认这一点，像工程师一样为你的生成核心套上 harness，你才能把最宝贵的创造力、幽默感和爱，用在真正需要你的地方——而不是消耗在“记得喂药”这种破事上。
