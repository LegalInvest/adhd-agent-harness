# 为什么当所有ADHD都用同一个Agent模板，高方差优势会被抹平——你的Agent正在把你变成平均值

> 一个 ADHD 社群里流传着一份「神级 Agent 配置」:精心调校的系统提示词、任务模板、回复风格,几千人复制使用。三个月后,一位成员发帖说了句扫兴的实话:「用了之后我确实更有条理了——但我重看自己这三个月的产出,**每一篇东西都像同一个人写的,而那个人不是我。**我最值钱的那些『不像话的想法』,在这套模板里根本活不到输出环节:它们要么被 Agent 的『合理化』磨圆,要么被任务流程判定为跑题裁掉。我用一套别人的最优解,把自己修剪成了它的形状。」

收敛:本文只回答——**「高方差」为什么是 ADHD 的核心资产而不只是负担(先把这个说严谨)?统一 Agent 模板抹平方差的三个具体机制?怎么配置一个「保方差」的个人 harness?**

## 穿透:harness 的天职是降方差——所以它默认是你天赋的敌人

先把「高方差优势」说得比鸡汤严谨。ADHD 认知的统计形状是:输出的均值可能不占优(稳定性差),但**分布的尾部更肥**——那些远离均值的采样(不合常理的联想、错误得有趣的方案、无人想到的角度)出现率更高。创造力研究里发散思维与 ADHD 特征的正相关、创业者中 ADHD 特征的过度代表,都指向这个形状。关键推论:**高方差人群的价值实现方式是「靠尾部赢」**——一百个想法里九十个平庸、八个糟糕、两个惊人,而那两个惊人的,是神经典型认知很难采样到的区域。你的商业模式(不管你是创作者、创业者还是研究者)本质上是在卖尾部。

而 harness——任何 harness——的工程天职恰恰是**降方差**:让输出稳定、合理、可预期。这三个词对执行(交付、日程、沟通)是纯增益,对生成(想法、方案、风格)是纯税收。统一模板抹平方差的三个机制:①**提示词的均值引力**:「神级配置」的系统提示词必然编码了某种「好输出」的标准(清晰/结构化/专业)——LLM 会忠实地把你的一切输入向这个标准回归;几千人用同一份提示词=几千人的输出被拉向同一个均值点。②**流程的尾部过滤**:任务导向的 Agent 流程里,「跑题」是要被纠正的 bug——而你的尾部想法在流程眼里全是跑题(它们本来就不在任务的延长线上);前文「目标回路」篇给跑偏设计了心跳豁免,统一模板没有这种东西,它裁掉漂移的时候,顺手裁掉了你的肥尾。③**风格的复利同化**:每天读 Agent 整理过的自己的文字,你的语感会被反向训练——第一个月是它模仿你,第三个月是你模仿它(那位发帖成员的「不像我」,就是这个复利到期)。

保方差的配置原则,三条:①**分区 harness 化**——把人生按「执行区/生成区」分区:执行区(日程、交付、邮件、流程)全力 harness,降方差是纯赚;生成区(构思、写作初稿、方案发散)**禁止 Agent 进场或只许当速记员**(记录你的原话,不整理、不优化、不「合理化」——前文「首触权」的升级版:生成区里连加工权都不给);②**提示词的自留地**:如果用别人的模板,至少重写「风格与判断」部分——把你自己的三个怪癖写进去(「保留我跳跃的类比」「不要替我把绝对化的表达软化」)——**模板可以借,人格不能租**;③**定期方差体检**:每月把自己三个月前和现在的产出各抽三篇对比读——「越来越像同一个人写的」且那个人不是你,就是同化的警报。

## 验证

方差可粗测:把你 Agent 化前后的产出各若干篇,让不了解情况的朋友(或另一个 AI)做「作者一致性」和「意外性」盲评。可证伪:若你的产出在 Agent 化后意外性不降反升(有人确实用 AI 拓宽了发散——用它生成远端联想再自己筛),那你的用法已经是「保方差配置」,本文的警告对你失效;区别只在配置,不在工具。

## 决策

做什么:画执行区/生成区地图;生成区把 Agent 降级为速记员;借来的模板重写人格段;每月方差体检。

不做什么:不要因此拒绝 harness 化执行区(执行的方差没有价值,别为「保真」保留混乱的日程);不要崇拜自己的一切怪念头(尾部里也有真正的垃圾——保方差是保采样,筛选照做,只是筛选权必须在你)。

先做什么:检查你正在用的 Agent 配置:哪几行提示词在替你定义「好」?把「好」的定义改成你自己的。

## 边界

发散思维与 ADHD 的关联、创业者中的过度代表有文献支持(GRADE 中,效应量与机制仍有争论);「均值回归/尾部过滤/风格同化」为机制推理+社区观察(本文未做正式 A/B/C 分级)。高方差优势不豁免执行支持的必要性;本文讨论的是配置策略,不构成「ADHD 天赋论」的背书——尾部要能变现,前提仍是执行区的 harness 撑住底盘。

## 今天就能试的 3 件事

1. 画你的执行区/生成区清单——Agent 该进哪个门,先分清楚。
2. 把借来的提示词里「风格/标准」段落重写成自己的三个怪癖。
3. 抽三篇近期产出读一遍:是你写的,还是「它整理的你」写的?

本文服务于人生 Harness 金字塔的**价值层**:给马套上挽具是为了拉车,不是为了让所有马跑成同一个步态——你的执行需要缰绳,你的天赋需要旷野;好的人生 harness,是知道哪里该收,哪里必须放。
