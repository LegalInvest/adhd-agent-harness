# ADHD 家长如何用「给生成核心套 harness」的思路应对陪伴 ADHD 孩子里的分心走神或超聚焦跑偏，丢失目标？

> 把 LLM 工程里的「长程任务里 agent 会目标漂移，需要重锚定 system prompt」迁移到ADHD 家长的陪伴 ADHD 孩子场景，双域证据作答。

## 问题：陪伴 ADHD 孩子时，为什么“讲道理”总是失效？

如果你是一位 ADHD 孩子的家长，你大概率经历过这样的场景：孩子正在做数学作业，突然被窗外的一只鸟吸引，然后开始画鸟，半小时后作业本上只有三行字；或者，孩子对乐高产生了超聚焦，三个小时不吃不喝，完全忘记了还有英语阅读任务。你反复提醒“专注目标”，但第二天一切照旧。这不是孩子不听话，而是他的大脑缺少一个关键的调度层——就像一辆引擎强劲但方向盘和刹车失灵的赛车。

## 同构：ADHD 大脑与 LLM 是同一类“生成核心”

最新研究揭示了一个惊人的平行：ADHD 大脑与大型语言模型（LLM）在结构上同构。两者都是强大的生成核心——能产生丰富的想法、创意和行动——但都缺乏可靠的内置执行调度层。ADHD 的工作记忆缺陷与 LLM 的“无跨会话状态”同构，两者都依赖外部记忆系统来补偿内部状态管理不足（来源：无状态与外部记忆）。Transformer 的自注意力机制在训练后发展出模仿前额叶-纹状体门控的操作，证明了计算层面的同构性（来源：拐杖与脚手架）。简而言之，ADHD 大脑就是一个“生物版 LLM”：高产、发散，但需要外部 harness 来锚定目标。

## 答案：给生成核心套上 harness——重锚定与目标漂移

工程中，当 agent 执行长程任务时，会因上下文膨胀或环境干扰而发生“目标漂移”，需要定期重锚定 system prompt。同样，ADHD 孩子在任务中容易分心走神或超聚焦跑偏，本质上就是目标漂移。家长的角色就是那个“重锚定系统”。具体怎么做？以下三条来自真实人物与工具的证据。

### 1. 建立“日课”式的外部调度层

晚清名臣曾国藩是典型的注意力不集中型 ADHD：30 岁前浮躁坐不住，日记里天天骂自己“无恒”“浮躁”，读书慢到“他人目下二三行，余或疾读不能终一行”。他的解法是“日课十二条”：黎明即起、读书不二、谨言、写日记反省等。这套 rigid 的日程就是他的外部调度层，相当于给 LLM 设定一个固定的 system prompt 刷新周期。对 ADHD 孩子，家长可以设计类似的“日课”，比如每天固定时间、固定顺序完成几件核心任务，用外部结构补偿内部调度缺陷。这与 LLM 工程中“定时 re-grounding”完全同构。

### 2. 用外部记忆体对抗“无状态”

ADHD 孩子经常“转头就忘”，今天记得的明天全忘。这就像 LLM 无法跨会话保留上下文。工具如 RescueTime 能自动记录时间去向，揭示注意力漂移（来源：RescueTime）；Obsidian 被推荐为 ADHD 的“第二大脑”，用于映射超聚焦和复杂想法（来源：无状态与外部记忆）。家长可以和孩子共建一个“任务板”或“记忆本”，每次任务前回顾，相当于给孩子的“系统 prompt”注入上下文。工程上，这等同于为 agent 维护一个外部记忆库，每次调用时注入相关历史。

### 3. 引入“身体在场”作为实时重锚定

Focusmate 通过虚拟身体在场提升启动与专注力，其核心是“身体加倍”效应（来源：Focusmate）。对 ADHD 孩子，家长或同伴的物理在场本身就是一种重锚定信号。释迦牟尼的“八正道”和“持戒”本质上也是用群体共修和戒律来拴住乱跑的念头——一种集体身体在场。工程上，这类似于 human-in-the-loop：当 agent 出现目标漂移时，由人类介入重新校准。

## 核心观点：脚手架，不是拐杖

这里必须诚实指出争议：AI 工具和外部结构会不会让孩子永久依赖？答案是“取决于如何使用”。真正的脚手架应帮助使用者“建造”，但使用者仍需自己“攀登”（来源：拐杖与脚手架）。曾国藩的日课他坚持了一辈子，但从未声称自己“治好了”ADHD；释迦牟尼的戒律是修行工具，不是最终目的。同样，Focusmate 和 RescueTime 是补偿手段，不是治疗。关键在于，家长要逐步引导孩子内化这些结构，而不是永远替他调度。

## 局限与争议

- **临床证据不足**：Brain.fm 和 Endel 虽有神经科学基础，但在 ADHD 人群中的独立随机对照试验极少（来源：Brain.fm）。
- **个体差异**：ADHD 的异质性意味着同一方法对注意缺陷为主型 vs. 多动冲动型效果可能迥异（来源：矛盾与存疑）。
- **同构的边界**：ADHD 大脑与 LLM 的同构目前仍是一种理论类比，并非经过验证的科学事实（来源：矛盾与存疑）。

## 今天就能试的行动

1. **设立“日课三件事”**：每天固定时间，用同一顺序完成三件核心任务（如：10 分钟阅读、15 分钟作业、5 分钟整理书包），坚持一周，观察变化。
2. **使用身体在场**：陪伴孩子做作业时，你也在旁边做自己的事（看书、工作），不说话，但物理在场。这相当于给孩子的“系统 prompt”增加一个锚点。
3. **建立外部记忆本**：准备一个笔记本或电子文档，每天开始任务前，让孩子写下“今天要完成什么”“现在做到哪一步了”，任务结束后更新。这直接补偿工作记忆缺陷。

最后，记住：你不是要“修复”孩子的大脑，而是为他搭一个 harness，让他那台强大的生成核心能够稳稳地跑向目标。就像工程师为 agent 写 system prompt 一样，你的每一次重锚定，都是在帮他回到轨道上。
