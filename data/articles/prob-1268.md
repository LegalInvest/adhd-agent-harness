# 如果把 ADHD 大脑当成生成核心，Lex 能不能当它的外部执行功能层？

> 用 agent 工程的思路重新理解 Lex 对 ADHD 的真正价值。

## 问题：你的大脑是一座失控的核反应堆

如果你有 ADHD，你一定熟悉这种感觉：脑子里同时冒出十几个绝妙的想法，但就是无法启动任何一个任务；计划做得完美无缺，执行时却像被胶水粘住。你的大脑是一座核反应堆——能量爆棚，但缺少控制棒。

现在想象你是一个 Agent 工程师：你手头有一个强大的 LLM，它什么都能生成，但总是跑偏、忘记系统提示、陷入循环。你不得不给它套上一个 harness——编排层、记忆层、工具层、人在回路——来确保它输出可靠。

这两件事，本质上是同一个问题。

## 同构：高产生成核心 × 薄弱执行调度层

ADHD 大脑的默认模式网络过度活跃，前额叶执行控制却不足（[[生成核心与调度层]]）。LLM 的生成能力同样强大，但缺乏稳定的上下文控制器，长上下文任务中性能退化（[[上下文工程]]）。两者都是“高产但不可靠”的生成核心，都需要外部调度层来引导输出。

这个同构视角不是比喻，而是有真实证据支撑的认知结构对应。

### ADHD 侧的真实痛点
- **工作记忆脆弱**：ADHD 的工作记忆容量不低，但控制弱——容易被干扰，导致“刚想做的事转头就忘”（[[工作记忆]]）。
- **目标漂移**：注意力被无关刺激捕获，或陷入超聚焦而忽略整体目标（[[重锚定与目标漂移]]）。
- **时间盲**：无法感知时间流逝，任务启动困难（[[时间盲]]）。
- **任务分解障碍**：面对复杂任务时，大脑无法自动拆解为可执行的步骤，导致瘫痪（[[执行功能]]）。

### LLM/Agent 侧的镜像
- **无状态性**：LLM 跨会话遗忘，需要外部记忆系统（如向量数据库）维持连续性（[[无状态与外部记忆]]）。
- **指令消退**：长程交互中模型开始“忘记系统提示”，行为偏离初始指令（[[重锚定与目标漂移]]）。
- **缺乏时间感知**：LLM 无法自主规划时间线，需要外部提醒和调度。
- **任务分解需求**：复杂任务需要链式思考或工具调用分解步骤，否则输出质量下降。

## 答案：Lex 能否成为外部执行功能层？

回到标题的问题：如果把 ADHD 大脑当成生成核心，Lex 能不能当它的外部执行功能层？

答案是：**能，但有条件**。

Lex 本质上是一个 AI 编排层——它整合了任务管理、提醒、知识回忆等功能，相当于 Agent harness 中的工具层和记忆层。具体来看：

- **重锚定系统**：Lex 的提醒和通知能对抗目标漂移，类似 harness 中事件驱动的系统提醒（[[重锚定与目标漂移]]）。
- **外部记忆**：Saner.AI 的知识回忆功能减少搜索循环，对应 LLM 的向量数据库（[[Saner.AI]]）。
- **任务分解**：Goblin Tools 的 Magic ToDo 将复杂任务拆解为小步骤，降低启动门槛（[[Goblin Tools]]）。
- **动态规划**：Motion 自动调整日程，消除“下一步该做什么”的决策负担（[[Motion]]）。
- **人在回路**：Focusmate 的身体搭档效应提供外部问责，类似 harness 中的人工检查点（[[人在回路与身体在场]]）。

这些工具组合起来，就是 ADHD 大脑的 harness。

## 脚手架 vs 拐杖：关键的边界

然而，同构视角也揭示了一个危险：外部执行功能层可能从“脚手架”变成“拐杖”。

证据显示，过度依赖 AI 工具可能削弱内在时间感知能力（[[时间盲]]），且缺乏纵向研究验证长期效果（[[矛盾与存疑]]）。就像 Agent harness 不能替代模型自身的推理能力，AI 工具也不应替代执行功能的内在发展。

**我的判断是：脚手架与拐杖的边界在于“是否主动参与”。** 如果工具只是被动替你规划、替你记忆，你逐渐丧失主动规划的能力，那就是拐杖。如果工具是“协作式”的——比如你仍然需要决定优先级、确认步骤、反思结果——那就是脚手架。

目前多数工具（如 Motion）自动排程，用户只需输入任务，这容易导致被动依赖。而 Goblin Tools 的分解功能需要用户确认步骤，Saner.AI 的回忆需要用户主动查询，更接近脚手架。

## 局限与争议

必须诚实指出：
1. **证据不足**：AI 工具对 ADHD 的有效性主要基于用户报告，缺乏严格对照试验（[[矛盾与存疑]]）。
2. **个体差异**：ADHD 的异质性（注意力主导 vs 冲动主导）与 LLM 架构差异是否影响同构普适性？目前研究有限（[[矛盾与存疑]]）。
3. **神经机制对应**：LLM 的“注意力机制”与 ADHD 的注意控制缺陷仅是功能类比，缺乏神经生物学直接证据（[[矛盾与存疑]]）。

## 今天就能试的行动

1. **用 Goblin Tools 分解一个你拖延已久的事**：输入“整理房间”或“写报告”，看 AI 如何拆解步骤，然后执行前两步。
2. **设置一个“重锚定”提醒**：用手机或 AI 工具（如 Lex）设置每 30 分钟的提醒，问自己“我现在在做该做的事吗？”
3. **尝试一次“身体在场”**：使用 Focusmate 或找个朋友在线一起工作 25 分钟，体验外部问责的效果。
4. **建立外部记忆库**：用 Saner.AI 或类似工具记录临时想法，减少大脑负担。

ADHD 大脑不是缺陷，而是一个需要 harness 的生成核心。AI 工具是那个 harness——但记住，最好的脚手架是让你最终能自己搭建的脚手架。
