# ADHD 家长如何用「给生成核心套 harness」的思路应对陪伴 ADHD 孩子里的任务启动困难、不会拆解？

> 把 LLM 工程里的「agent 用 planner-executor 循环分解长任务」迁移到ADHD 家长的陪伴 ADHD 孩子场景，双域证据作答。

## 引言：从“启动不了”到“不会拆”的日常战场

“快去写作业！”——你喊了第三遍，孩子还盯着天花板发呆。作业本摊开，笔在手里转，就是落不下第一个字。更崩溃的是，即使开始了，他可能写了一行就跑去逗猫，或者把“整理书包”拆成“把书全倒出来再发呆十分钟”。

这不是懒，也不是态度问题。ADHD 大脑的核心困境是：**生成核心高产，但调度层失灵**。它像一个大语言模型（LLM），能滔滔不绝地联想、创造，却没有内置的 planner-executor 循环来把意图变成有序行动。

作为 ADHD 孩子的家长，你需要的不是更多催促，而是一套**外部 harness**——就像 LLM 工程里给 agent 套的脚手架那样，把“规划、拆解、执行、监控”从孩子混乱的内置系统里卸载出来，交给可见的工具和流程。

## 同构证据：ADHD 大脑与 LLM 的共享缺陷

为什么说 ADHD 大脑和 LLM 是“同类”？

- **ADHD 侧**：工作记忆容量有限且不稳定，跨会话任务难以持续，每天需要重新回忆项目上下文（来源：无状态与外部记忆）。研究显示 ADHD 组在命名物体和言语检索速度上更慢（来源：Neuropsychological profiles of adolescents with ADHD）。这导致任务启动时大脑一片空白——不是不想做，是“怎么开始”这个指令根本加载不出来。
- **LLM/agent 侧**：标准 LLM 调用是无状态的，每次对话独立，不保留跨会话记忆（来源：无状态与外部记忆）。没有 harness 的 LLM 只是一个无状态的 API 调用（来源：Building an AI Agent Harness）。要让它完成长任务，必须给它 planner-executor 循环：先规划子步骤，执行一步，观察结果，再规划下一步。

**核心判断**：ADHD 孩子需要的不是“更努力”，而是像 agent harness 一样的**外部调度层**——把“现在该做什么、做多久、何时切换”显式化、视觉化、自动化。

## 古代智慧的 harness 原型：孔子与于谦

这套思路并非新发明。孔子和于谦——两位很可能有 ADHD 特质的历史人物——早已用自制 harness 管理了自己的“生成核心”。

**孔子**：身高1米9，精力旺盛，周游列国14年坐不住；冲动爱骂人，思维跳跃，《论语》全是场景化语录，无系统著作。他的 harness 是“克己复礼”——用外在的“礼”作为行为边界，每日反省（“吾日三省吾身”）。这对应 LLM harness 中的**定时 re-grounding**：每天通过固定仪式（反省）把注意力拉回正轨。

**于谦**：性格刚直，冲动敢说话，土木堡之变后力排众议坚守北京。他的 harness 是“严格操练军队，日夜准备防务”——用**外部纪律**和**明确的日常任务清单**（操练、布防）来约束自己的冲动，同时用《石灰吟》作为**价值观锚点**。这对应 LLM agent 里的**外部调度器**：把长期目标（保卫北京）拆解为每日执行步骤。

他们的成就——孔子创立儒家，于谦延续明朝国祚——恰恰说明 harness 不是压制创造力，而是**让生成核心的高产能找到有序出口**。

## 现代工具：给 ADHD 孩子套上外部 harness

今天的 AI 工具已经把这套逻辑产品化了。

### 1. 任务启动：消除“第一步”的决策负担

ADHD 孩子面对任务时，决策负担是启动的最大障碍。**Motion** 通过自动排程消除“下一步该做什么”的决策：它持续评估任务优先级、截止日期和可用时间，实时重建日程（来源：Motion）。当任务面临延期风险时，Motion 会提前数天或数周发出警告。

**Reclaim.ai** 则更强调“保护”而非“填充”——它自动创建深度工作、习惯和专注时间的智能块，防御会议侵占，减少因突发事件导致日程崩溃的风险（来源：Reclaim.ai）。对 ADHD 孩子来说，这意味着“写作业时间”不会被临时冒出的杂事吃掉。

### 2. 任务拆解：把大象放进冰箱

“不会拆解”的本质是工作记忆无法同时容纳目标、步骤和顺序。**Goblin Tools** 的“任务分解”功能可以把“整理房间”自动拆成“把书放回书架、把衣服叠好、把垃圾扔掉”等微观步骤。**Tiimo** 则把时间转化为可见元素：视觉日程、颜色编码、时间块，直接应对时间盲（来源：Tiimo）。对受时间盲和视觉提示帮助的孩子，Tiimo 用动态、视觉化的流程替代静态待办清单，减少无结构时段带来的焦虑。

### 3. 外部记忆：让“第二大脑”永远在线

ADHD 大脑的跨会话记忆不可靠。**Obsidian** 被推荐为视觉思考者的“第二大脑”，映射超聚焦和复杂想法（来源：无状态与外部记忆）。家长可以帮助孩子建立一个“项目笔记本”——把作业要求、项目背景、个人偏好存储其中，每天开始前快速回顾，避免每次重新加载上下文。这对应 LLM 的**外部记忆系统**：提供持久记忆、清晰的下一步行动，减少每日重复输入上下文的摩擦（来源：无状态与外部记忆）。

## 脚手架 vs 拐杖：诚实面对局限

所有工具都有边界。wiki 资料中的『矛盾与存疑』指出：

- **过度依赖风险**：AI 工具作为外部执行功能层，长期使用可能导致能力退化，成为永久拐杖（来源：矛盾与存疑）。目前缺乏纵向研究验证长期效果。
- **超聚焦的引导困境**：一方认为应柔性引导超聚焦到优先事项，另一方认为强行打断（如番茄钟）反而有效。个体差异大，尚无定论（来源：矛盾与存疑）。
- **学习成本**：部分用户反映学习使用 AI 工具本身增加认知负荷（来源：矛盾与存疑）。

**我的观点**：Harness 和拐杖的区别在于**是否逐步内化**。孔子从“克己复礼”到“从心所欲不逾矩”用了70年——外部规则最终变成了内在习惯。同样，AI 工具应该作为**过渡性脚手架**：先帮孩子建立“规划-执行-回顾”的循环，然后逐渐减少依赖，直到孩子能部分内化这个流程。完全依赖工具而不培养元认知，就是拐杖。

## 今天就能试的 3 条行动

1. **给“启动”加一个外部触发器**：用 Routinery 创建一个“作业启动”流程——先倒水、摆好文具、打开作业本，每步有过渡提示（来源：上下文工程）。让孩子不再面对“空白开始”。
2. **用视觉日程替代口头催促**：在 Tiimo 或纸质时间表上画出“作业块”和“自由块”，让孩子看到时间结构。当孩子问“现在该做什么”，指时间表，不回答。
3. **建立每日“重载”仪式**：每天固定时间（如晚饭后）用 5 分钟回顾当天任务完成情况，并写下明天的 3 个关键任务。这模仿 LLM 的定时 re-grounding，帮助孩子把工作记忆中的碎片整理到外部。

## 结语

ADHD 不是缺陷，而是一种**高产能但缺调度层的架构**。作为家长，你的角色不是修复孩子的大脑，而是像工程师一样，为孩子搭建一个外部 harness——让他的创造力、好奇心和爆发力，沿着清晰的轨道释放。孔子和于谦用自制 harness 改变了历史；今天的 AI 工具让这个过程更简单、更精准。关键在于：**脚手架要逐步撤掉，但结构要留在心里**。
