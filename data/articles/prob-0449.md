# 用 ChatGPT 给 ADHD 补上幻觉与验证循环，是脚手架还是拐杖？

> ChatGPT 接进 ADHD 工作流：哪些交给它、哪些必须自己留着。

## 引言：你和他都容易“先开枪，后画靶”

你刚打开一个文档，脑中已经蹦出结论；你还没查完资料，消息已经发出去了。对 ADHD 读者来说，这叫“冲动下结论”“想当然”。对 Agent 工程师来说，这叫 LLM 的“幻觉”——它永远自信地给你一个看似合理的答案，却不一定经过验证。两种场景，同一种底层 bug：一个生成能力极强、但缺少执行调度与验证闭环的系统（来源：ADHD 大脑与 LLM 的同构）。

所以本文要回答的问题其实是：如果 ADHD 和 LLM 是同一类系统，那用 ChatGPT 给 ADHD 补上的到底是“脚手架”——帮你搭起缺失的那层结构，还是“拐杖”——让你永远依赖外力走路？

## 1. 同构：高产生成核心 + 缺位的调度层

ADHD 一侧的典型图景是“强记忆、弱控制”：工作记忆容量可能正常甚至超常，但认知灵活性、注意控制和任务切换存在核心缺陷，注意力资源呈弥散分配（来源：Strong Memory, Weak Control: An Empirical Study of Executive Functioning in LLMs；Self-Attention Limits Working Memory Capacity of Transformer-Based Models）。这意味着大脑能大量产出联想、创意和洞察，却难以稳定地抑制优势反应、维持上下文、按优先级执行。

LLM 一侧惊人地相似。Transformer 自注意力在长上下文下的冲突解决能力会退化到随机水平，无法抑制“最顺嘴”的答案；当任务负载增加，注意力分数熵增，容量受限（来源：Deficient Executive Control in Transformer Attention；Self-Attention Limits Working Memory Capacity of Transformer-Based Models）。因此，LLM 本身只是一个无状态的生成核心，必须靠外部“harness”或“agent scaffolding”来提供记忆、工具、规划、验证与沙箱（来源：Agent Scaffolding: Architecture and Design Patterns for Agentic AI；Building AI Coding Agents for the Terminal: Scaffolding, Harness, Context Engineering, and Lessons Learned）。

一句话：ADHD 大脑和 LLM 都是“高产但缺执行调度层”的生成核心。两个系统缺的不是创造力，而是执行调度层。

## 2. “想当然”与“幻觉”共享同一种处方：外部验证循环

冲动下结论，就是人脑版本的 LLM 幻觉。两者都产出一个流畅、自洽、甚至诱人的输出，却缺少“先查资料、再核对、再修订”的闭环。ADHD 的困难在于前额叶的执行控制不够稳定；LLM 的困难在于没有身体、没有世界模型、没有长期记忆，只能根据上下文概率继续“编”。

历史人物的管理系统恰好可以映射到 LLM 的 harness 设计。毛泽东的 ADHD 特质包括思维跳跃、冲动敢赌、爱读闲书、永远在动；他的 harness 是“批评与自我批评”民主生活会、让别人提意见，“没有调查就没有发言权”的调查研究，以及用民主集中制和集体决策制衡冲动（人物案例：毛泽东）。这正对应 LLM harness 里的自我批判模块、检索增强生成（RAG）和外部人类/投票机制：生成之后必须有一个“不站在你这边”的验证者，把结论拉回事实。

屠呦呦的系统同样如此。她沉浸古籍、专注实验，她的 harness 是“严格的实验流程”——筛选 2000 多种中药、380 多个提取物，并依靠团队合作而非个人独断（人物案例：屠呦呦）。这对应 Agent 工程里的可重复验证循环、自动化测试与多智能体检查：灵感可以跳跃，结论必须被实验和同行校验。

对 ADHD 个体来说，这些历史 harness 不是励志故事，而是可复制的架构：用外部系统把“生成”和“验证”拆开。

## 3. ChatGPT 接进工作流：哪些交给它，哪些必须自己留着

ChatGPT 及围绕它搭建的 AI 工具，可以充当“外部执行功能层”。它的价值不是“替你思考”，而是把下列高频负荷从摇摇欲坠的工作记忆里卸载出去：

- **任务分解**：Goblin Tools 的 Magic ToDo 把模糊任务拆成可执行步骤，降低启动门槛（来源：Goblin Tools）。
- **外部记忆**：Saner.AI 通过本地记忆和知识回忆减少搜索循环与标签切换，让上下文不再“掉线”（来源：Saner.AI）。
- **自动排程**：Motion 根据截止日期、优先级和可用时间动态重建日程，缓解“时间盲”和“下一步该做什么”的决策瘫痪（来源：Motion）。

这些功能对应 LLM harness 里的“上下文交付”“记忆系统”“规划工件”与“工具接口”。它们让生成核心（你的大脑或 LLM）不必自己调度一切，从而把算力留给真正需要判断的地方。

但有些东西不能外包。ChatGPT 可以帮你：

- 生成假设和候选结论；
- 列出反对意见、需要核实的证据、潜在风险；
- 把复杂信息整理成可验证的清单。

但它不能替你：

- 做最终事实判断；
- 承担价值观与伦理权衡；
- 替你向真实世界道歉或负责。

一个实用的使用原则是：把 ChatGPT 当成“草稿生成器 + 对抗验证器”，而不是“终审法官”。
