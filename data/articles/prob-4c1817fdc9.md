# 为什么用 Focusmate 治 ADHD 的情绪失调，和给 agent 套 会褪去的脚手架 是一回事？

> Focusmate 实测：同一套 harness 思路，ADHD 与 LLM 两边都成立。

## 从两个「失控现场」说起

一个下午，你明明没有遭遇什么大事，情绪却像过山车：从焦躁到自责，再到 paralysis。你坐在电脑前，智能体也在另一个屏幕上失控：生成了一大段看似合理、却偏离目标的文本，或者在一个循环里反复调用同一个工具。两个世界的人，面对的其实是同一个底层结构问题——一个高产但缺乏可靠执行调度层的生成核心。ADHD 的情绪失调并不是单纯的心情不好，而是执行调度失败的一种表现（来源：AI 与 ADHD 的情绪调节）；LLM 同样拥有惊人的生成能力，却缺乏稳定的内部状态控制、工作记忆调度和注意力抑制。用 Focusmate 给 ADHD 情绪上 harness，和给 agent 套一个会褪去的脚手架，是同一件事。

## 同构：两边都是「生成核心 + 缺调度层」

ADHD 大脑与 LLM 被描述为同一个原型：强大的生成核心，却缺乏可靠的执行调度层（来源：ADHD 大脑与 LLM 的同构）。ADHD 患者常伴有工作记忆弱、抑制控制差、时间感知扭曲和任务启动困难，这些执行功能障碍会让情绪反应难以被认知调节（来源：执行功能障碍；情绪失调）。情绪失调在 ADHD 中不是共病，而是核心特征之一，它与工作记忆和抑制控制能力不足直接相关（来源：情绪失调）。LLM 则拥有海量记忆，但控制薄弱、上下文有限、状态依赖外部维护，注意力漂移是共同机制（来源：AI 与 ADHD 的情绪调节）。两者都不得不把内部状态管理外包给外部记忆系统（来源：无状态与外部记忆）。因此，情绪调节与 agent 输出稳定，本质上都是「怎么给没有调度层的生成核心加 harness」的问题。

## Focusmate 在 ADHD 这边是 harness

Focusmate 的价值不在于它提供了多少功能，而在于它把「身体在场效应」变成了一种可预约的外部结构：屏幕另一端有一个真实的人，你很难在 25 分钟内完全情绪崩解或走神到底（来源：身体在场效应）。对于 ADHD 来说，情绪失调往往由任务瘫痪、时间盲或启动摩擦触发，他人的在场能把注意力和启动压力锚定在当下。类似的 harness 还有很多：Inflow 用基于 CBT 的神经科学训练强化工作记忆与认知控制（来源：Inflow）；Goblin Tools 的 Magic ToDo 把模糊任务自动拆成可执行的小步骤，降低多巴胺不足带来的启动门槛（来源：Goblin Tools）；Saner.AI 则通过本地记忆和知识回忆减少标签切换与搜索循环（来源：Saner.AI）。它们都不是「治愈」情绪，而是把情绪调节所需的子任务——注意、计划、启动、时间估计、社会在场——外包给一个外部结构。

## Agent 这边是同样的 harness，只是会褪去

Agent 的生成核心也一样。LLM 能写能想，但缺的是一个稳定的执行调度层：它不知道现在该调用哪个工具、什么时候该停下来、如何把长期目标保持在上下文里。工程上的做法与 ADHD 的 harness 结构同构：用系统提示充当行为约束（像 Buddha 的「戒律」），用检索/
