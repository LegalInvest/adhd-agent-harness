> 神经机制如何映射到脚手架的淡出算法

## 问题：我们害怕的到底是什么？

ADHD 群体常陷入两难：既怕过度依赖 AI 工具，又怕没有支撑就彻底崩盘。工程师设计 Agentic Harness 时同样纠结——harness 应该是会褪去的脚手架，还是永久绑定的拐杖？这个问题的核心在于：**我们能否让外部支持像教育脚手架一样，随着能力内化而逐步撤销，而不是简单减少 token 输出？**

## 同构脊柱：两个“生成核心”的困境

ADHD 大脑与 LLM 在结构上惊人地同构：两者都是高产的“生成核心”，但缺乏可靠的“调度层”（来源：ADHD 大脑与 LLM 的同构）。ADHD 的前额叶执行功能缺陷导致工作记忆不稳定、时间感知模糊、任务启动困难（来源：执行功能障碍）；LLM 同样会产生无关或有害输出，需通过上下文工程和外部约束来引导（来源：AI 与 ADHD 的情绪调节）。因此，AI 工具作为外部执行功能层，本质上是为这个生成核心套上 harness（来源：AI 与 ADHD 的情绪调节）。

有效的教育脚手架落在“最近发展区”内，并随能力内化而逐步撤销。神经可塑性研究表明，训练可以诱导抑制控制脑可塑性，正常化底层脑网络（来源：神经可塑性）。那么，LLM harness 的提示压缩、动态权限和工具调用频率，能否对应到神经可塑性层面的“逐步撤除”指标？

## 真实人物的 Harness：从释迦牟尼到张自忠

释迦牟尼的 ADHD 特质显而易见：29 岁说走就走放弃王位，先后跟两个仙人学、苦行 6 年，不断试错不断放弃。他的 harness 是“八正道”和“持戒”——用戒律管行为，用正念拴念头，并且强调“自依止法依止”，不依赖权威。这个系统对应 LLM harness 中的 **定时 re-grounding**（日课）和 **外部调度器**（戒律作为规则约束）。张自忠同样冲动敢战，他的 harness 是“严格治军，和士兵一起训练一起打仗”——用纪律和共同目标来调度自己的冲动。这对应 LLM harness 中的 **动态权限**（根据情境调整行动范围）和 **工具调用频率**（仅在必要时调用）。

这两个案例揭示了一个关键：**有效的 harness 不是消除特质，而是提供可内化的调度结构**。

## LLM Harness 的“逐步撤除”指标

要避免 harness 成为永久拐杖，必须设计可量化的撤除指标：

1. **提示压缩率**：随着用户能力提升，初始的详细提示可以逐步简化为关键词，类似教育中减少脚手架提示。神经可塑性层面，这对应背外侧前额叶皮层功能连接的增强（来源：Inflow）。
2. **动态权限衰减**：工具调用权限可以从“全自动”逐步降为“半自动”或“手动确认”，直到用户能自主调度。Inflow 通过个性化训练任务动态调整认知负荷（来源：Inflow），正是这一思路。
3. **工具调用频率下降**：Goblin Tools 的 Magic ToDo 允许调节“辣度”控制分解粒度（来源：Goblin Tools），用户熟练后应降低分解频率。Saner.AI 通过增强知识回忆减少搜索循环（来源：Saner.AI），长期使用后用户应能更快检索信息，减少依赖。

这些指标不是简单减少 token 输出，而是**在神经可塑性层面促进内化**：当用户的工作记忆和抑制控制改善后，外部支撑自然可以撤除。

## 争议与局限

必须诚实指出，这一框架存在争议：

- **同构假设的边界**：ADHD 大脑的生物复杂性远超 LLM，情绪调节涉及神经递质、环境交互等，单纯“脚手架”类比可能简化问题（来源：矛盾与存疑）。

- **个性化不足**：当前工具多为通用设计，未充分适配 ADHD 亚型（来源：AI 与 ADHD 的情绪调节）。

## 今天就能试的行动

1. **设置“提示压缩”实验**：用 ChatGPT 或 Claude 完成一项日常任务时，从详细提示开始，每周减少 20% 的指令细节，观察自己能否保持同等产出。
2. **使用 Goblin Tools 的“辣度”调节**：将任务分解粒度从“高辣度”逐步调低，记录自己独立完成子任务的能力变化。
