# ADHD 的前额叶执行功能缺陷与多巴胺奖赏通路、工作记忆容量有关，而有效的教育脚手架通常落在“最近发展区”内并随能力内化而逐步撤销。LLM harness 的提示压缩、动态权限和工具调用频率，能否对应到神经可塑性层面的“逐步撤除”指标，而不是简单减少 token 输出？

> 这是一个 STORM 视角轮里怀疑论者与工程师互相逼问出来的问题,值得原样接住,因为它同时踩住了两边最不舒服的地方:教育学里,脚手架(scaffolding)的定义包含「撤除」——支持落在最近发展区(ZPD)内,随能力内化而逐步淡出,不撤的脚手架叫依赖;而 LLM harness 的世界里,几乎没有人在做「撤除」——提示越写越厚、权限越给越宽、工具越挂越多,**harness 只增不减是行业默认**。问题问的是:如果要做撤除,撤除的判据能不能对应到「能力真的内化了」的实质指标,而不是「输出的 token 少了」这种表面量?

先诚实地拆「能否对应」的两半。**神经可塑性那一半,目前答案是:没有可用的指标**。ADHD 的执行功能训练研究里,「近迁移易、远迁移难」是反复出现的模式(工作记忆训练的荟萃分析方向:练什么涨什么,泛化有限);「脚手架撤除后能力保持」的神经层面标志物——哪个通路的什么变化预示「内化完成」——研究远未到可操作的程度。所以任何声称「基于神经可塑性的淡出算法」的产品,今天都是过度宣称。**但工程那一半,答案是:可以有行为层的替代指标,而且 harness 侧已经有原型**。

harness 侧的「撤除」实践,散落在几个真实场景里:**其一,提示压缩的判据**——一个 agent 的系统提示从 8000 token 压到 2000,合格的判据不是「更短」,是**压缩后在同一评估集上的任务成功率不降**:说明那 6000 token 承载的行为已经被模型「学会」(通过微调)或本来就冗余;**其二,动态权限的判据**——从「每步需要人批准」放宽到「仅高风险操作需要批准」,判据是**低风险操作的历史错误率足够低**;**其三,工具移除的判据**——撤掉一个校验工具,判据是**模型的裸输出在该维度的错误率已低于阈值**。三个案例共享同一个结构:**撤除的许可来自「无支持状态下的表现证据」,而不是来自时间表或直觉**——这恰好就是教育学里「渐退(fading)」的操作定义:支持强度随表现数据阶梯式下调,每降一级先验证表现保持。

把这个结构翻译回 ADHD 的 harness(也回应本系列反复出现的「拐杖焦虑」):**撤除感兴趣的人,应该测行为,不该等神经证据**——比如提醒系统从每日五次降到两次,判据是「两次提醒周里的任务完成率不降」;body doubling 从每天降到每周,判据是「独自会话的启动成功率」;**每一级淡出都是一个 A/B 自测,数据说话**。但还有一个更根本的追问,本系列的立场必须一致:**「撤除」本身不该是默认目标**。眼镜不撤、假肢不撤、神经典型者的日历和提醒也没人要求撤——**教育脚手架要撤,是因为它教的能力可以内化;而 ADHD 的执行功能差异,更接近「长期的硬件特性」而非「未完成的学习」**:对多数人,harness 的正确终点不是裸奔,是「更轻、更合身、维护成本更低的 harness」。撤除实验值得做(它告诉你哪些支持是冗余的),但把「最终不需要工具」设为成功标准,是把神经差异误当成了没学完的课。

## 边界

同构定位(本文未做正式 A/B/C 分级):ZPD 与脚手架渐退是教育学的成熟概念,提示压缩与动态权限是 harness 工程的真实实践,「表现数据驱动淡出」的结构对应清晰;神经可塑性层面的撤除指标目前不存在,任何此类宣称都超出证据。声明:执行功能训练的迁移证据有限;是否减少任何支持(包括药物),请与专业人士讨论,不要以「摆脱依赖」为由自行撤除有效支持。

## 今天就能试的 3 件事

1. **列出你的 harness 清单**:每项标注「上次验证它有用是什么时候」——有些支持可能已经冗余,有些远比你以为的关键。
2. **做一次单项淡出实验**:选一项支持降一级(五次提醒→两次),跑两周,看完成率数据——升回来或保持,都算成功。
3. **写下你的终点定义**:「更轻的 harness」还是「没有 harness」?——想清楚这个,拐杖焦虑会消掉一半。

脚手架该不该撤,教育学的答案是「看表现数据」;**而 ADHD 的 harness,先要问一个更前置的问题:它是脚手架,还是眼镜?**——前者撤除是毕业,后者撤除只是看不清。
