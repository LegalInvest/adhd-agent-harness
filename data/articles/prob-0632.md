# 用 ChatGPT 给 ADHD 补上无状态与外部记忆，真的能像 agent 那样工作吗？

> 把 ChatGPT 当作 ADHD 的外部执行层来实测：它补的是哪一块短板？

## 问题：边做边忘，ChatGPT 能当外部记忆吗？

你刚打开 ChatGPT，准备写一段代码，却在输入框里愣住——刚才想好的需求是什么来着？你翻看聊天记录，发现上一段对话已经淹没在三天前的某个深夜。这种“边做边忘”的体验，对 ADHD 大脑来说是日常；而对熟悉 LLM 的工程师来说，它听起来像极了 token 耗尽后模型对前文的无知。

ADHD 的工作记忆容量小、不稳定，像一块随时被清空的缓存（来源：执行功能障碍）。LLM 同样没有跨会话状态，每次对话都是一个独立的上下文窗口（来源：无状态与外部记忆）。两者都是高产生成核心，却缺乏可靠的内置执行调度层——这正是本文要回答的问题：用 ChatGPT 给 ADHD 补上外部记忆，真的能像 agent 那样工作吗？

## 同构：ADHD 大脑与 LLM agent 共享同一个“脚手架”困境

把 ADHD 大脑比作 LLM 并非空谈。ADHD 的执行功能缺陷——任务启动困难、工作记忆不稳定、注意力分散——与 LLM 的无状态、缺乏调度层在结构上同构（来源：外部执行功能层）。LLM 需要外部记忆（如向量库）和调度器（如 agent scaffolding）来维持多步骤任务的连贯性；ADHD 大脑同样需要外部脚手架来补偿工作记忆和任务管理。

这种同构在历史人物身上早有体现。孔子（春秋/儒家创始人）身高1米9，精力旺盛，周游列国14年坐不住；冲动爱骂人，思维跳跃，《论语》全是场景化语录，无系统著作。他的专属 harness 是“克己复礼”——用外在的“礼”作为行为边界，每日反省（“吾日三省吾身”），70岁才做到从心所欲不逾矩（来源：人物案例）。这套日课制度，本质上就是 ADHD 的“定时 re-grounding”机制，与 LLM agent 中定期将对话摘要写入外部存储、防止上下文漂移的设计如出一辙。

再看张旭（唐朝/草圣），每次喝醉后号呼狂走，以头濡墨作书，多相睡眠，灵感来了不管场合就写字。他的 harness 是以酒和草书为核心，把超专注注入创作，从自然中找灵感（如看公孙大娘舞剑悟笔法）。这对应了 LLM agent 中“外部调度器”的角色——通过特定触发条件（酒、舞剑）将生成核心导向目标输出，而非任由随机联想。

## 证据：ChatGPT 能否胜任外部执行层？

现有工具已经展示了 AI 补偿 ADHD 执行功能的多种路径。Goblin Tools 的 Magic ToDo 能将“清理厨房”这类模糊任务自动分解为具体步骤，用户可调节“辣度”控制粒度（来源：Goblin Tools）。这相当于 agent 的任务规划模块——将复杂目标拆解为可执行的子任务序列。Saner.AI 通过本地记忆存储和快速检索，减少用户在不同应用间切换的搜索循环（来源：Saner.AI），相当于 agent 的向量数据库。Lex 允许通过单一指令触发多步骤任务序列，其底层架构类似于 agent scaffolding（来源：Lex）。

但 ChatGPT 本身呢？它被广泛用作对话式 AI 辅助 ADHD，但存在注意力分散风险——用户可能被无关回复带偏（来源：ChatGPT）。关键在于，ChatGPT 能否像 agent 那样，通过外部记忆（如 Mem 的持久记忆）和上下文工程（如主动设计提示词）来维持任务连贯性？

## 核心判断：脚手架而非拐杖，同构但不等于

我的核心观点是：ChatGPT 可以作为 ADHD 的外部执行层，但前提是使用者主动将其设计为“脚手架”，而非依赖其自动完成一切。这要求用户像工程师搭建 agent 系统一样，有意识地管理上下文、定期 re-grounding（如每段对话开始前总结前文），并利用工具的外部记忆功能（如 Otter.ai 的实时转录、Mem 的持久存储）来补偿工作记忆（来源：Mem, Otter.ai）。

然而，同构性不能被过度推广。目前 ADHD 大脑与 LLM 的类比缺乏实证基础，仅是一种理论模型（来源：矛盾与存疑）。工具宣称的效果也缺乏严格临床验证，多数基于用户报告（来源：矛盾与存疑）。更重要的是，过度依赖 AI 可能导致内在执行功能萎缩——工具从脚手架变成永久拐杖（来源：拐杖与脚手架）。

## 行动建议：今天就能试的 3 件事

1. **用 ChatGPT 做“任务分解 agent”**：把模糊目标（如“准备下周汇报”）输入 ChatGPT，要求它分解为 5-10 个具体步骤，并每完成一步就回来更新进度。这模拟了 Goblin Tools 的 Magic ToDo 功能，也实践了 agent 的规划模式。

2. **建立“会话摘要”习惯**：每次与 ChatGPT 对话结束时，让它生成一段摘要并保存到备忘录。下次开始新对话前，先粘贴摘要作为上下文。这相当于 LLM agent 的持久记忆机制。

3. **尝试 Saner.AI 或 Otter.ai 作为外部记忆**：用这些工具记录会议、灵感和待办事项，减少依赖大脑缓存。注意避免工具本身增加认知负荷（来源：认知负荷）。

## 局限与争议

- 同构性类比尚未被科学验证，需谨慎使用（来源：矛盾与存疑）。
- 多数 AI 工具缺乏针对 ADHD 亚型的个性化调整，且存在注意力分散风险（来源：矛盾与存疑）。
- 依赖风险真实存在：AI 工具可能削弱而非增强内在执行功能。

最终，ChatGPT 能否像 agent 那样工作，取决于你把它当作“外部大脑”还是“拐杖”。同构的洞察提醒我们：ADHD 大脑和 LLM 都需要脚手架，但脚手架必须由使用者亲手搭建，而不是由 AI 代劳。
