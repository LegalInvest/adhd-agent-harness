# RescueTime 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> 从同构视角实测 RescueTime：它到底补上了哪一层执行功能？

你打开电脑，准备写一份方案。十分钟后，你发现自己正盯着某个无关页面，而方案文档还停在标题。这不是意志力问题——你的大脑像一台没有调度器的生成核心：能产出惊人的想法，却无法自主锚定目标。

ADHD 大脑与 LLM 共享同一个困境：两者都是“高产但缺执行调度层的生成核心”（来源：AI 与 ADHD 的专注力）。Transformer 在长序列中注意力会弥散、在冲突任务中控制力骤降到随机水平（来源：Deficient Executive Control in Transformer Attention）；ADHD 大脑则在任务切换中丢失目标、被环境带偏。两者的解法在结构上同构——都需要外部脚手架来提供重锚定。

这就是 RescueTime 的真正角色。它不是时间记录仪，而是你的“重锚定 system prompt”。

## RescueTime 补上了哪一层执行功能？

执行功能是大脑的“驾驶座”，ADHD 的核心障碍之一就是执行功能缺陷（来源：执行功能）。RescueTime 直接补偿了其中的两个关键子功能：**时间盲**与**工作记忆瓶颈**。

- **时间盲**：ADHD 患者难以感知时间的流逝，RescueTime 的自动追踪“让时间变得可见”，揭示真实去向（来源：RescueTime）。这相当于给 LLM 加了一个 token 计数器——当上下文过长时，模型会退化，你需要主动截断或重设。
- **工作记忆瓶颈**：ADHD 的工作记忆缺陷与 LLM 的无跨会话状态同构（来源：AI 与 ADHD 的专注力）。RescueTime 将“我今天做了什么”外化为可视化报告，相当于 LLM 的 external memory 系统。

但 RescueTime 更关键的功能是**分心阻断**：它能在你滑向无关网站时主动拦截。这对应了 LLM 中的“抑制优势反应”——Transformer 在 Stroop 任务中无法抑制冲突刺激（来源：执行功能），ADHD 大脑同样无法抑制分心冲动。RescueTime 的阻断，就是外部植入的“抑制控制层”。

## 同构的实证：曾国藩的“日课”与 LLM 的 re-grounding

曾国藩的自我管理系统是 ADHD harness 的经典案例。他 30 岁前“浮躁坐不住”，日记里天天骂自己“无恒”。他的解法是**日课十二条**：黎明即起、读书不二、谨言、写日记反省。

这套系统的核心不是“记录”，而是**定时重锚定**。每一条日课都是一个检查点：你现在在做什么？目标是什么？这完全对应 LLM agent 中的“定时 re-grounding system prompt”——在长程任务中，agent 会目标漂移，需要定期注入原始目标来重锚定。曾国藩的“写日记反省”就是他的 re-grounding 机制。

曾国藩的“结硬寨打呆仗”同样值得注意。他用最笨最稳的方法抵消冲动——这对应了 AI 工程中的“防御性 prompt 设计”：不依赖模型自身稳定性，而是通过外部约束（如分步验证、输出格式限制）来保证行为可控。

## 有人用错了：脚手架 vs 拐杖

许多人把 RescueTime 当“记录工具”——每天看报告，感叹“又浪费了时间”，然后明天照旧。这相当于给 LLM 加了一个日志系统，却从不使用它来修正行为。

RescueTime 的正确用法是作为**脚手架**：不是被动记录，而是主动利用其阻断和提醒功能来**在当下**拉回注意力。曾国藩的日课之所以有效，不是因为他记了日记，而是因为日记中的反省直接指导了第二天的行动。

但这里存在真实的争议：AI 工具是否会导致永久依赖而无法内化执行功能？（来源：AI 与 ADHD 的专注力）RescueTime 的自动分类和阻断规则可能不够精准，需要用户调整，这对执行功能较弱的用户仍是挑战（来源：RescueTime）。更根本的是，同构性本身是功能类比，并非经过验证的科学事实（来源：矛盾与存疑）。

## 今天就能试的行动

1. **设置每日“重锚定”闹钟**：在 RescueTime 中开启 FocusTime 功能，每 25 分钟自动阻断分心网站，并弹出提示“你现在应该做什么？”——这是你的 system prompt re-grounding。
2. **用“周报反向工程”代替自责**：不看“效率分数”，而是从 RescueTime 报告中找出一个最常分心的时段，下周在那个时间段主动安排一个 Focusmate 伙伴（来源：Focusmate）——用身体在场效应强化锚定。
3. **建立“曾国藩式”日课清单**：在 RescueTime 的每日总结中，强制自己填写三项：今天的目标是什么？实际做了什么？明天如何调整？——这是你的私有 re-grounding 协议。
4. **尝试 Brain.fm 作为专注环境**：在 RescueTime 阻断分心后，打开 Brain.fm 的专注音乐（来源：Brain.fm），利用神经锁相技术帮助大脑进入稳定状态——相当于给 LLM 的注意力头加一个噪声过滤器。

RescueTime 的价值不在数据，而在它提供的**重锚定机会**。ADHD 大脑和 LLM 一样，不缺生成能力，缺的是在长程任务中反复把自己拉回目标的能力。工具是脚手架，不是拐杖——用对了，它帮你建造自己的执行功能；用错了，它只是另一个让你自责的记录仪。
