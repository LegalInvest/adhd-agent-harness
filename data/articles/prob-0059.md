# 3 个 LLM 工程概念，彻底重写了我对 ADHD 执行功能（计划/组织/启动）先天偏弱的理解

> agent 的编排/记忆/工具层就是「外部执行功能」 如何反向照亮 ADHD 的执行功能（计划/组织/启动）先天偏弱。

## 问题：为什么 todo 清单总在吃灰？

你打开一个待办应用，写下“整理房间”或“写报告”，然后盯着屏幕，手指悬空——就是点不下去。这不是懒，是执行功能（计划、组织、启动）的先天偏弱。ADHD 大脑像一个高产的生成器：想法多、创意足，但缺一个调度层来把“生成”转化为“行动”。

同样的问题，在 AI 工程里也有名字。当你搭建一个 LLM agent，你会发现：模型能写诗、能编码，但让它按顺序执行多步任务？它迷路、忘记上下文、输出失控。工程师不得不给 agent 套上编排层、记忆层、工具层——一个“外部执行功能”。

## 三个 LLM 工程概念，反向照亮 ADHD

### 1. 编排（Orchestration）：从“下一步做什么”到“自动调度”

LLM 本身是无状态的：它不知道历史，也不规划未来。Agent 框架（如 LangChain、AutoGPT）通过编排层管理任务链、决策分支和重试逻辑。编排层就是 agent 的“执行功能”。

ADHD 侧：Motion 这款工具正是编排层的化身。它自动根据你的任务、会议和截止日期创建每日计划，当计划被打乱（比如临时会议）时动态重排（来源：11 Best ADHD Productivity Apps for Fluctuating Energy - rivva blog）。你不需要思考“接下来该做什么”——Motion 替你调度。

**同构点**：Motion 的编排算法 = LLM agent 的编排层。两者都解决了“生成核心缺乏调度”的问题。

### 2. 记忆（Memory）：把工作记忆外挂

LLM 的上下文窗口有限，对话一长就忘。Agent 工程通过短期记忆（对话历史）、长期记忆（向量数据库）和检索增强生成（RAG）来弥补。

ADHD 侧：工作记忆缺陷是核心瓶颈之一（来源：ADHD, Executive Functions, and AI: A New Era in Treatment | Psychology Today）。Saner.AI 正是数字工作记忆：它存储你的知识碎片，需要时快速召回，减少搜索循环和标签切换（来源：Best AI Tools for ADHD Productivity in 2026 (Honest Review) - Iwo Szapar）。

**同构点**：Saner.AI 的本地记忆 = LLM agent 的 RAG。两者都把“记住”从大脑/模型内部移到外部系统。

### 3. 工具（Tools）：把“怎么做”外包

LLM 不擅长精确计算或实时查询，所以 agent 通过工具（计算器、API）来执行具体操作。工具层是 agent 的“双手”。

ADHD 侧：Goblin Tools 的 Magic ToDo 功能，把“整理房间”自动分解成“捡起地板上的衣服”“擦桌子”等小步骤（来源：AI Tools for Kids with ADHD: A Complete Guide for Parents...）。这相当于把“如何启动”这个元任务外包给了 AI。

**同构点**：Goblin Tools 的任务分解 = LLM agent 的工具调用。两者都绕过核心的“执行障碍”，直接输出可操作步骤。

## 核心判断：脚手架，不是拐杖

三个概念揭示了一个真相：ADHD 大脑与 LLM/agent 是同一类“高产但缺执行调度层的生成核心”。AI 工具不是替代你的大脑，而是为这个核心套上 harness——外部执行功能层。

但这里有一条红线：脚手架 vs 拐杖。脚手架是临时支撑，可逐步撤除；拐杖是永久依赖。当前多数工具（如 Motion、Saner.AI）设计为长期使用，未提及撤除机制（来源：矛盾与存疑）。长期依赖是否会削弱内在执行功能？目前缺乏纵向研究（来源：拐杖与脚手架）。

我的观点：**AI 应像提示工程一样被使用——主动管理输入，而非被动接受输出**。例如，用 Motion 调度时，每周自己手动调整一次计划，保持对编排逻辑的感知；用 Saner.AI 时，定期尝试不检索而凭记忆回忆，锻炼工作记忆。

## 今天就能试的行动

1. **用 Goblin Tools 分解一个你拖延的任务**：输入“写周报”，看它输出什么步骤。然后只做第一步。
2. **用 Motion 或类似工具自动排程一天**：把所有任务和截止日期丢进去，让它调度。观察你的启动焦虑是否下降。
3. **用 Saner.AI 建立一个知识库**：把常忘的信息（如密码、流程）存进去，下次需要时先检索，记录检索次数，一周后对比。
