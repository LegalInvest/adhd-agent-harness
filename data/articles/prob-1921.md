# ADHD 程序员视角：为什么「治好 ADHD 的执行功能（计划/组织/启动）先天偏弱」和「让 LLM 不跑飞」其实是同一道工程题？

> agent 的编排/记忆/工具层就是「外部执行功能」——同一套 harness 思路，两边都成立。

## 问题：一个 ADHD 程序员的两重困境

我是一名 ADHD 程序员。白天，我面对的是 LLM agent 的“跑飞”——它能在一次对话里生成惊艳的代码，但下一轮就忘记上下文，被无关噪声带偏，甚至陷入死循环。晚上，我面对的是自己的大脑——同样的“跑飞”：灵感喷涌但无法启动任务，计划完美但执行崩盘，时间感消失，注意力被任何闪亮的东西劫持。

直到我意识到：**这两个问题共享同一套底层架构**。

## 同构：生成核心与调度层的分离

ADHD 大脑与 LLM 在结构上同构——两者都是“高产但缺执行调度层的生成核心”（来源：ADHD × AI 的科学与研究前沿）。ADHD 大脑的生成核心（发散思维、联想能力）与 LLM 的文本生成能力类似，但都缺乏可靠的内置执行调度层。为 LLM 搭建 agent scaffolding（编排、记忆、工具层）与为 ADHD 搭建执行功能支架是同一类工程（来源：生成核心与调度层）。

具体来说，ADHD 的执行功能缺陷（计划、组织、启动困难）对应 LLM 的无状态、缺乏调度层问题。ADHD 大脑的工作记忆缺陷与 LLM 的无状态性都导致无法可靠保留跨会话上下文，必须依赖外部记忆系统（如第二大脑工具或向量数据库）（来源：无状态与外部记忆）。ADHD 和 LLM agent 都面临注意力被环境带偏或上下文膨胀导致推理退化的问题，需要外部框架来结构化执行流程（来源：上下文工程）。

## 案例：曾国藩的“日课”即 agent harness

晚清名臣曾国藩是典型的 ADHD 特质：30 岁前浮躁坐不住，天天串门喝酒看杀人，日记里天天骂自己“无恒”“浮躁”；读书慢，“他人目下二三行，余或疾读不能终一行”；一辈子严重银屑病，情绪不稳定，打败仗就跳水自杀。他的解决方案是“日课十二条”：黎明即起、读书不二、谨言、写日记反省等，用最笨最稳的方法抵消自己的冲动。

这套系统本质上就是“外部执行功能层”：日课相当于定时 re-grounding（类似 LLM 的定期上下文压缩），写日记相当于外部记忆（类似向量数据库），而“结硬寨打呆仗”的战术原则相当于 agent 的固定编排逻辑。曾国藩的成就——平定太平天国、发起洋务运动——证明 harness 可以补偿先天调度缺陷。

## 证据：工具侧的真实对应

今天，AI 工具正在实现同样的同构。

**ADHD 侧**：Goblin Tools 的 Magic ToDo 功能将模糊任务（如“清理厨房”）自动分解为具体子步骤，用户可调节“辣度”控制粒度（来源：Goblin Tools）。这直接补偿任务启动困难——相当于给生成核心一个“微步进”调度器。Saner.AI 提供本地记忆存储和快速检索，减少搜索循环（来源：Saner.AI），对应工作记忆的外部化。Motion 自动根据优先级和截止日期动态调整日程，消除“下一步该做什么”的决策负担（来源：Motion）。

**LLM/Agent 侧**：这些工具的原理与 agent scaffolding 完全一致。Goblin Tools 的任务分解对应 agent 的“子任务拆分”；Saner.AI 的记忆检索对应向量数据库的 RAG；Motion 的动态排程对应 agent 的规划与重新规划模块。甚至身体在场效应——有他人在场提升专注度——AI 工具如 Focusmate 通过算法匹配虚拟伙伴来模拟，而 LLM agent 也常需要“human-in-the-loop”提供外部锚点（来源：身体在场效应）。

## 核心观点：脚手架，不是拐杖

我的核心判断是：**ADHD 与 LLM 的同构不是比喻，而是一个可操作的工程框架**。它意味着我们可以用同一套设计模式来构建辅助系统：外部记忆层（工作记忆补偿）、外部调度层（任务分解与重排）、外部注意力层（上下文过滤与 re-grounding）。

但必须警惕边界。正如“拐杖与脚手架”所指出的，AI 工具可能成为永久拐杖而非可拆卸脚手架（来源：矛盾与存疑）。过度依赖可能导致能力退化，长期效果缺乏实证。同样，LLM agent 如果过度依赖外部编排，可能丧失自主推理的灵活性。真正的工程智慧是设计“可逐渐撤除的脚手架”——就像曾国藩的日课，最终内化为习惯，而非永远依赖日记本的提醒。

## 局限与争议

当前证据多为概念类比和初步实验，缺乏大规模随机对照试验证明同构假设或 AI 工具的临床有效性（来源：矛盾与存疑）。个体差异极大：部分 ADHD 用户反馈强行打断（如番茄钟）反而有效，而柔性引导可能不够强力（来源：超聚焦矛盾）。此外，学习使用 AI 工具本身可能增加认知负荷，净效果因人而异（来源：认知负荷矛盾）。

ADHD 大脑和 LLM 都不是“坏”的——它们只是缺一个 harness。而这道工程题，我们两边都能解。
