# 为什么用 RescueTime 治 ADHD 的情绪失调，和给 agent 套 会褪去的脚手架 是一回事？

> 情绪失调系列里,这一篇处理的敌人最擅长伪装——它听起来像美德:**苛刻的自我叙事**。一天结束时,那个熟悉的声音开始播报:「今天又废了」「你一天到晚在摸鱼」「别人半天干的活你一周都没干完」。注意这些句子的共同点:**全是没有数据支撑的整体性判决**。心理学的观察是,ADHD 者的自我评估存在系统性偏差——时间盲让「刷了一会儿手机」在记忆里变成「荒废了一下午」,负性偏差让三小时的有效工作在回放里消失、二十分钟的走神被无限放大。**你每天都在用一份被污染的报表,给自己做绩效评估**——而情绪失调的很大一部分,正是这份假报表的读后感。RescueTime 的自动记录,在这里的角色是一台**叙事校准仪**;它的褪去,则指向一个意料之外的终点。

先看校准机制。日志篇讲过它的本职(采基线、找模式),情绪视角下它做的是另一件事:**用客观数据对质主观叙事**。「今天又废了」——打开报表:生产性时间 3 小时 42 分,和你的月均持平。「一天到晚在摸鱼」——数据:社交媒体 47 分钟,占比 11%。**判决和证据第一次当庭对质,而证据通常比检察官仁慈得多**。这个动作在 CBT 里有正式的名字:行为证据检验(examining the evidence)——对自动思维的标准反驳程序;RescueTime 把它自动化了:**证据不用回忆(回忆已被污染),直接调取**。工程对应:评估 agent 性能靠 trace 和指标,不靠「感觉这个模型最近变笨了」——主观印象在两边都是不可靠的证人。

再看褪去,它的路径是**叙事习惯的重写**:初期,每天用数据对质一次判决(手动、刻意、常常震惊于差距);中期,判决出现时,内部自动响起「等等,数据说不定不是这样」——**怀疑判决的习惯已内化,这比记住任何数据都值钱**;成熟期,苛刻叙事本身的音量下降——不是因为你说服了它,是因为它被证伪太多次,信用破产了。**校准仪可以继续跑着,但它最重要的产出——「我的自我判决不可信,要看证据」这个元认知——已经安装完成。** 此后报表回归日志篇的本职:优化工具,不再是心理防线。

误用两种:

**误用一:数据被苛刻叙事重新劫持(校准仪沦为新弹药库)。** 危险的转折:报表显示生产时间 3 小时 42 分——判决官立刻改口:「才三小时?正常人都是六小时!」**数据没有校准叙事,反而给了它更精确的自虐坐标**(和 Forest 篇的枯树同构:反馈层被羞耻滤镜劫持)。判据:看完报表,情绪更稳了还是更糟了?后者连续一周,先停用报表,只留记录——**并把这个现象本身带去和专业者谈:数据都能被用来打自己,说明问题不在数据层**。

**误用二:只校准坏日子,不校准好日子(单向使用,错过一半疗效)。** 只有自责时才查数据,自我感觉良好的日子从不看——**但叙事偏差是双向的:「今天状态爆棚超高效」的日子,数据可能显示产出平平**(躁性的高估同样是失真)。全谱校准才能建立完整的元认知:**我的自我感知,好坏两个方向都需要外部数据**——这才是「不信判决信证据」的完全体,也是把这套习惯迁移到生活其他领域(人际、能力评估)的基础。

## 边界

同构强度 B 级:ADHD 的时间感知与自我评估偏差有实验支撑,证据检验是 CBT 的标准技术,自动化行为数据的客观性优势真实;RescueTime 对情绪的作用无直接研究,「叙事校准」是 CBT 框架的应用推断。声明:苛刻叙事若已弥漫为持续的无价值感、或数据反复被用作自我攻击的弹药,这超出自助工具的辖区——那个判决官的来历(常常很早)值得和专业者一起查;报表校准得了昨天下午,校准不了童年。

## 今天就能试的 3 件事

1. **做一次当庭对质**:下次「今天又废了」响起时,立刻打开报表——判决 vs 证据,记下差距。
2. **也校准一个好日子**:自我感觉高效的一天,查查数据——偏差是双向的。
3. **观察一周报表读后感**:更稳还是更糟?更糟就停,并把这件事本身当成重要信息。

你脑内的检察官从不出示证据,而证据一直躺在后台——**每一次当庭对质,都在削它的信用;削够了,你就不再需要开庭**。数据最深的疗效,是教会你:对自己的判决,也要讲证据。
