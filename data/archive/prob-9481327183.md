# 为什么用 Brain.fm 治 ADHD 的情绪失调，和给 agent 套 会褪去的脚手架 是一回事？

> 情绪失调系列的第四个触发器:**唤醒度失控**。ADHD 的老观察:唤醒水平像没有恒温器的房间——低的时候昏沉烦躁(坐不住、易怒地无聊),高的时候过载崩溃(声音、消息、待办一起涌进来时的炸毛)。很多被贴上「情绪化」标签的时刻,底层其实是唤醒度冲出了工作区间。Brain.fm 这类功能性音频,本质是一个**外部唤醒度调节器**:专注模式托高、放松模式压低。把它放进「会褪去的脚手架」框架,会遇到和 Tiimo 篇同样的分岔——但答案更有意思:**这个脚手架褪去的不是调节,而是调节的「代理权」。**

先看撑法。工程对应从 agent 说起:模型的「状态」也需要外部调节——temperature、采样参数、系统负载控制,都是不指望模型自我调节、而由 harness 代为设定的旋钮。Brain.fm 就是人的音频旋钮:**你不需要知道自己怎么了,只需要选一个模式,声音把神经系统往那个方向推**。对情绪失调,这个「不需要知道」正是初期价值:失调最严重的时刻恰恰是自我觉察最不在线的时刻,一个不依赖觉察的外部调节器,是雪中送炭。

但这里埋着「褪去」的线索。用 Brain.fm 几个月的人常报告一个副产品:**为了选模式,你被迫每天问自己几次「我现在是需要托高还是压低?」——这个提问本身,就是内感受(interoception)训练**:辨认自己的唤醒状态,这恰恰是情绪调节的第一块基石(情绪粒度研究的核心主张:能分辨,才能调节)。于是脚手架的演化路径浮现:**初期,音频替你调节(代理);中期,选模式的动作训练你识别状态(教学);成熟期,你在没有耳机的场合也能识别「我现在过载了」并主动降载(内化)**——音频可以继续用,但「知道自己怎么了」的能力已经归你。这是补偿性工具里藏着发展性成分的典型:**工具本身可以永驻,它顺带教的技能会毕业。**

误用两种:

**误用一:全程无觉察地使用(只享受代理,错过教学)。** 一键最爱模式、全天循环——调节发生了,但「我现在是什么状态」从来没被问过:**用了三年,内感受零增长**,没耳机的场合照样失控。修法:把选模式变成显式的两秒仪式——**按下播放前,先说出(或默念)你判断的当前状态**(「我现在昏沉」「我现在炸毛边缘」);判断错了也没关系,校准本身就是训练。

**误用二:用音频调节掩盖失调的真实源头。** 唤醒度天天失控,音频天天救火——**但失控的源头可能是睡眠崩了、承诺超了(Reclaim 篇)、或未处理的焦虑**:调节器管症状,不管病灶;最坏的情况是调节太有效,病灶永远不疼、永远不治。工程对应:靠调低 temperature 掩盖训练数据的问题。修法:**每周看一次「救火频率」——音频从「增强状态」滑向「抢救状态」的比例升高,就该查上游**(睡眠、负载、情绪),而不是升级会员。

## 边界

同构强度 C 级:如实标注——唤醒调节异常在 ADHD 研究中有理论支撑(最优刺激模型),内感受与情绪调节的关联有文献,但 Brain.fm 对 ADHD 情绪失调无直接对照研究,「选模式=内感受训练」是合理推断而非已证实机制。声明:频繁的情绪失控若伴随持续低落、失眠、或人际功能受损,请专业评估(共病心境障碍在成人 ADHD 中不少见)——音频是旋钮,不是治疗;药物与心理治疗的事,和医生谈。

## 今天就能试的 3 件事

1. **建立两秒状态播报**:今天每次戴上耳机前,先说出你判断的唤醒状态——高了还是低了?
2. **数一次救火比例**:本周用音频是「锦上添花」多,还是「抢救崩溃」多?后者超过一半,查上游。
3. **做一次无耳机识别**:今天挑一个烦躁时刻,不开音频,只练习说出「我现在过载了/垮了」——识别本身就是第一步调节。

调节可以永远外包,识别不行——**让声音继续帮你转旋钮,但「现在该转哪个方向」,要练成你自己的判断**。脚手架撑住状态的同时,悄悄教会你读懂自己:这才是它最值钱的功能。
