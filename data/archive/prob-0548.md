# 3 个 LLM 工程概念，彻底重写了我对「ADHD 大脑到底是怎么回事」的理解

> 三十年的隐喻都在说「你坏了」：刹车失灵、电池漏电、发动机过热。LLM 工程给了第一个不含贬义的模型：高产的生成核心，缺一层调度。

确诊后我读过的所有 ADHD 科普，都在用损坏隐喻：大脑的刹车坏了、过滤器漏了、油门卡死了。这些隐喻科学上各有依据，但它们共享一个叙事：**你是一台故障机器**。直到我开始研究 LLM agent 工程，才第一次遇到一组不带贬义、却更精确的概念。

## 1. 「高产生成核心 + 缺失调度层」：一个不含道德判断的架构描述

LLM 是什么？一个惊人高产的生成核心：给它任何提示，它立刻涌出十个方向的联想——其中有天才的洞见，也有彻底的胡说，而它自己无法区分。它没有内置的目标坚持、没有跨会话记忆、没有自我校验。行业对此的回应不是修理模型，而是**造 harness**：编排层管目标、记忆层管状态、验证层管质量。模型负责涌现，harness 负责收敛。

把这个架构图贴到 ADHD 大脑上，吻合度令人不安：联想力过剩（生成核心强）、兴趣驱动的爆发力（采样温度高）、但计划/启动/坚持/校验系统性偏弱（调度层薄）。区别只在于：LLM 的调度层由一个万亿美元的行业在外部补齐，而 ADHD 者被要求用意志力在内部长出来。

## 2. 这个同构有真实的科学底座，不只是漂亮修辞

这不是自媒体类比。过去两年出现了一小批直接研究这个映射的论文：耶鲁团队发现自注意力架构本身产生工作记忆容量极限，负载上升时注意力熵增弥散，与人类注意瓶颈同源（Self-Attention Limits Working Memory Capacity, 2024）；明尼苏达团队给 LLM 做神经心理测评，得到「强记忆、弱控制」的执行功能剖面——正是 ADHD 测评的经典构型（Strong Memory, Weak Control, EACL 2026）；还有研究发现 Transformer 注意力在 Stroop 类冲突任务下的崩溃模式与执行功能缺陷同源（Deficient Executive Control in Transformer Attention, 2025）。两个从未通信的领域，在机制层长出了同一套问题和同一套解法——这正是 Swanson 文献发现理论里最值得深挖的「未被发现的公共知识」结构。

## 3. 架构描述改变努力的方向：从修内核到造 harness

损坏隐喻和架构隐喻的区别不是心情好坏——它们指向完全不同的行动。「刹车坏了」暗示你该修刹车（练专注、逼自律、恨自己），三十年的经验告诉你这条路走不通。「生成核心缺调度层」则直接给出工程路线图：

- 记忆层外置：唯一收件箱 + 外部索引，大脑退役为纯生成器；
- 编排层外置：外接 planner（AI 拆解、自动排程）+ 重锚定（目标可见化）；
- 验证层外置：human-in-the-loop（body doubling、承诺装置）+ 延迟校验（重要输出放 10 分钟）。

每一条都有对应的 LLM 工程成熟方案可以直接抄。这就是这个同构框架真正的实用价值：**agent 工程行业在用百亿美元试错，为 ADHD 义务研发自我管理方案**。

## 边界：类比的诚实边界在哪里

必须清晰：大脑不是 Transformer。多巴胺、情绪、睡眠、羞耻的积累、被误解的三十年——这些是硅基系统没有的层。同构强度 A 级指的是「双侧存在直接研究且机制可映射」，不是「两者是同一种东西」。把类比当身份认同（「我就是个 LLM」）和把类比当耻辱一样，都是把地图错当了领土。它是一张好地图——仅此，但这已经很多了。

## 今天就能试的 3 件事

1. **重写一句自我描述**：把「我自律性差」改写成「我的调度层薄，生成核心强」。观察这个改写对情绪的影响——那就是隐喻的力量。
2. **画出你的 harness 现状图**：记忆/编排/验证三层，各写下你现在的外置方案。空着的那层就是下一个改进点。
3. **读一篇原始论文**：从上面三篇里挑一篇读摘要。让「这是真的研究」这个事实本身替你抵抗自我怀疑。

「我的大脑到底是怎么回事」——三十年来你得到的答案都是各种口味的「坏了」。现在有一个新答案：它是一台没配 harness 的高产引擎。而 harness，是可以造的。
