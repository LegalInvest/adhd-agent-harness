# Inflow 之于 ADHD，就像 重锚定 system prompt 之于 LLM——但有人用错了

> function calling 那篇讲了 Inflow 当「函数教程」:教你拆解、启动、认知重构的手艺。这一篇讲学完之后的事——因为 CBT 类课程有一个残酷的衰减曲线:**学的时候句句醍醐灌顶,两周后一条都想不起来**。这不是你的问题,是所有「洞见类知识」的默认命运:它们不像技能那样用肌肉记住,只能靠反复出现来维持在场。工程上这正是 system prompt 要解决的事——**关键指令不能只说一次,要常驻、要重注入**。Inflow 的课程给了你 prompt 的内容,但注入系统要你自己搭;不搭,就是白学。

先看衰减的机制。CBT 的产出是两类东西:**领悟**(「原来我拖延是因为把任务和自我评价绑定了」)和**技法**(「启动前先做两分钟版本」)。领悟的半衰期极短——**顿悟当晚的那种清晰,第二天早高峰就没了**;技法稍好,但没有触发机制也会闲置。ADHD 的前瞻记忆弱势让这条衰减曲线更陡:「下次遇到 X 时要用 Y 技巧」这种 if-then 计划,恰恰依赖最不可靠的那块内存。工程对应清晰:模型在某轮对话里「理解」了什么不重要,**下一轮上下文里没有的东西,等于不存在**。误用四种:

**误用一:只上课,不提炼(没有把课程压成 prompt)。** 课程是几十节的流水,prompt 是几行的浓缩——**中间需要一道「提炼」工序:每学完一个真正击中你的点,用自己的话写成一行**(「任务≠我;烂开始>不开始」)。跳过提炼直接指望「学过就会记得」,等于指望模型记住三个月前的对话。修法:建一页「我的领悟清单」,只收真正震动过你的,一行一条,十条封顶。

**误用二:提炼了,不注入(清单躺在 App 深处)。** 熟悉的死法,第三次强调:**没有注入通道的内容等于不存在**。修法:领悟清单进你的宪法区(Mem 篇)/晨间例程(Routinery 篇)/每日模板(Reflect 篇)——**用你已经搭好的任何一条注入通道,别为它新建系统**。

**误用三:注入时机与触发场景错位(prompt 对了,时机错了)。** 领悟清单每天早上看,但「把任务和自我评价绑定」这个坑,是在深夜面对未回复的消息时塌的——**早晨的注入到不了深夜的现场**。工程对应:静态 system prompt 覆盖不了运行时的特定情境,需要事件触发的动态注入。修法:给最重的两三条领悟配「现场触发器」——**在坑出现的物理现场放提示**(手机锁屏写一句、工位便签一句),让 prompt 在案发地点等着你。

**误用四:清单只进不出(prompt 通胀,第 N 次)。** 每期课程都往清单里加,一年后四十条——回到膨胀老路。**但这次的修法不同,因为领悟有一个独特的毕业机制:内化**——某条领悟已经变成你的默认反应(不假思索就先做两分钟版本了),它就该从清单退役,把位置让给还没长进骨头里的。**清单的理想状态是流动的十条:新领悟进,内化的出**——这恰恰是脚手架该有的样子(下一批文章的主题,先按下不表)。

## 边界

同构强度 B 级:CBT 对成人 ADHD 有随机对照试验证据,干预效果的维持期衰减在心理治疗研究中有据可查(增益随时间递减、需要强化练习),Inflow 的 App 化交付证据初步。声明:自助维持系统是治疗的延伸,不是替代——若领悟清单里反复出现同一条却始终内化不了,那可能正是需要和专业者深入工作的点;工具维持记忆,人才能完成改变。

## 今天就能试的 3 件事

1. **提炼你的领悟清单**:回想学过的所有课/书/治疗,写下仍然震动你的,十条以内。
2. **接入现有注入通道**:放进你的宪法区或晨间例程,今天完成。
3. **给最重的一条配现场触发器**:它出没的物理现场,放一句话的提示。

上课那一刻的清晰不属于你,能反复出现在眼前的那几行字才属于你——**知识的拥有权,不在学过,在还在场**。把最贵的领悟写短,放在路上,让未来的你天天撞见。
