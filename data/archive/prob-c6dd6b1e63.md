# 为什么用 Forest 治 ADHD 的情绪失调，和给 agent 套 会褪去的脚手架 是一回事？

> 情绪失调系列讲过羞耻(Focusmate 篇)——那是被别人定义的伤。这一篇讲它的孪生兄弟:**自我效能的塌方**。成年 ADHD 者的内部账本上,记满了「说好要做却没做」的坏账:办了没去的健身卡、买了没读的书、立了没守的 flag——每一笔都在给同一个信念充值:「我说的话不算数」。心理学叫自我效能感(Bandura):**「我能做到我打算做的事」这个信念,决定你敢启动什么**;而它的塌方是情绪失调的深层地基——很多暴躁和低落,底下是「反正我做不到」的绝望打底。Forest 那棵小树,在这个账本上做的事,比「防止玩手机」深远得多:**它是一台自我效能的微额充值机**——而这正是它作为脚手架的褪去路径所在。

先看充值机制。自我效能的最强来源,Bandura 说得明确:**成就体验(mastery experiences)——真实完成的、可归因于自己的成功**。问题是 ADHD 的日常里,「完成」的定义总是太大(写完报告、坚持健身三个月),大到很难达成,于是账本上只有坏账。Forest 把「完成」切到了二十五分钟的粒度:**种树→守约→树活了——一次微小但完整、可见、不可抵赖的「我说到做到」**。一天四棵树,就是四笔进账;一个月下来,森林是一份视觉化的资产负债表——**「我说的话算数」第一次有了证据**。工程对应:agent 的可靠性也是这么建立的——不是靠一次大任务的成功,是靠小额、高频、可验证的履约记录累积出的信任曲线。

再看褪去,这次的机制是心理学里最优雅的一种:**自我效能是自增强的**。信念涨了→敢启动更大的事→更大的完成→信念再涨——**一旦飞轮转起来,充值机就不再是必需品**:真实生活的成就(交付的项目、兑现的承诺)开始供应比小树更硬的证据。演化路径:初期,树是唯一可靠的进账来源(生活里的大额完成还太少);中期,树和真实成就并行,树的情绪权重下降;成熟期,**树变成纯粹的专注工具(资源锁,回到 function calling 篇的本职),它的「效能证据」职能已被真实履历接管**。判断读数:看到自己的森林,情绪是「我需要它证明我还行」还是「挺好,一个工具而已」——前者还在初期,后者已经毕业。

误用两种:

**误用一:枯树被计入旧账本(充值机反向扣款)。** 树枯了——设计意图是「即时违约信号」,但对自我效能塌方严重的人,**每棵枯树都被自动归档进「我果然做不到」的证据库**:工具的反馈层被用户的羞耻滤镜劫持了。判据:枯一棵树,你的反应是「重新种」还是「今天又完了」?后者说明滤镜在工作。修法:**从小到不可能失败的时长开始(十分钟),先让账本连续进账两周**——充值机要先建立信用,才能承受偶尔的扣款;另外显式练习枯树话术:「一棵枯树=一次数据,不是一次判决」。

**误用二:效能证据永远停留在树上(飞轮没有接入真实生活)。** 森林郁郁葱葱,但「说到做到」的信念从没迁移到树以外——**约了朋友照样放鸽子,立的 flag 照样倒**:因为迁移不会自动发生,需要显式的桥。修法:**把真实承诺「种进」系统——把「今晚给妈妈打电话」也当成一棵树来守**(用标签标记「真实承诺」类);让充值机的证据类型从「专注了二十五分钟」扩展到「兑现了一个对人的承诺」——**效能感的账本是分科目的,哪个科目需要信用,就往哪个科目充值**。

## 边界

同构强度 B 级:自我效能理论与成就体验的作用有扎实文献,ADHD 与低自我效能的关联有研究支撑,微承诺的行为设计有依据;Forest 本身无对照研究,「充值机」是概念化表述。声明:自我效能的塌方若已深到「不敢启动任何事」的程度,或伴随持续的无价值感,这可能是抑郁的信号而非动机问题——小树充得了日常的值,充不了临床级的空;那时请找专业者,这本身就是一次「说到做到」。

## 今天就能试的 3 件事

1. **降到不可能失败的粒度**:今天种三棵十分钟的树——先保证进账,再谈金额。
2. **练一次枯树话术**:「一次数据,不是一次判决」——枯了就说,说完重种。
3. **种一棵「真实承诺」树**:把一个对人的小承诺放进系统守约——让证据迁移开始。

自我效能塌方的人,缺的不是道理,是证据——**那棵二十五分钟的小树,是你能拿到的最小面额的「我说话算数」**。攒够了,飞轮会自己转;到那天,森林只是森林,而你已经是那个说到做到的人。
