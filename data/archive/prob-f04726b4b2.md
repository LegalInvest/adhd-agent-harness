# 为什么用 Mem 治 ADHD 的想理解自己的大脑，和给 agent 套 生成核心 + 缺失的执行层 是一回事？

> 探针系列的记忆站。373 篇用 human-in-the-loop 的读法拆过 Mem(写入/组织/读出三环节的验收协议),这一篇换仪器读法:**你使用外部记忆系统的行为模式,是对你自己记忆架构的一次扫描**——「我记性差」这句 ADHD 者最常见的自我描述,和「我不会写作」「我时间管理差」一样粗糙:记忆是多级流水线(编码→巩固→存储→检索→前瞻触发),各级坏法不同、补法迥异,而多数人从没分清过自己坏在哪级。Mem 这类「AI 自动组织」的笔记工具,恰好把外部记忆的各级都产品化了——**于是你和它每一级的互动质量,就是你内部对应级别的读数**;agent 工程师评估一个模型的记忆瓶颈,靠的正是这种方法:接上外部 memory 系统,看哪一级的外挂带来性能跳变——跳变最大的那级,就是原系统最弱的那级。

先立一个对自我理解最重要的区分:**ADHD 的「记性差」大部分不是存储坏,是入口和出口坏**。研究方向上,这个人群的长期记忆存储本身相对完好,薄弱的是两端:**编码端**——注意没在场,信息压根没写进去(「忘了」的真相常常是「从未记住」:钥匙放哪不是忘了,是放的时候注意在别处,根本没编码);**检索端**——存进去了,需要时调不出或想不起「可以去调」;外加一个 ADHD 重灾区:**前瞻记忆**——「记得在未来某时刻做某事」(吃药/回消息/赴约),它不是回忆过去,是让记忆在正确的时刻自动弹出,依赖的恰恰是最弱的那套触发机制。三级三种坏法,Mem 探针逐级测。

**测编码级:看「三行规则」的难度**。373 篇的规则(存东西必须附三行自己的话)在这里变成测试仪:如果你发现写这三行经常写不出——「我为什么存这个来着?」在存入后三十秒就已经模糊——**你的堵点在编码级**:信息在你注意的浅层滑过,没有获得深加工;这个读数的迁移意义重大:你的「忘」多发生在分心时刻(边聊天边放钥匙/边刷手机边听安排),施工方案不在笔记工具,在**编码时刻的注意仪式**(重要信息复述一遍再放手/钥匙有唯一的家/听到任务当场三行);**测检索级:看「命中率日志」的败因**。373 篇的日志(每次「我好像记过」的结果)在这里读诊断:败因若集中在「压根忘了可以查」——你的堵点是**检索触发**:外部记忆存量再大,「遇事先查自己」的反射没长出来,系统等于不存在;施工:把「查」绑进固定场景(写作前必搜/会议前必搜)——**触发靠仪式,不靠记性**(用记性去记「要查记忆」,是死循环);败因若集中在「查了没找到」——堵点在**索引方式**(你存的关键词和你回想时用的关键词不一致——这是 Mem 的 AI 语义检索该救的部分,救不了就换工具);**测前瞻级:看你往 Mem 里塞了多少「未来的事」**。如果你的笔记库里堆着大量「记得下周问 X」「月底前办 Y」——这是一个响亮的读数:**你在用回顾式工具承载前瞻式任务**,而这是注定失败的错配(笔记不会在下周自动弹出);前瞻记忆的正确假体从来不是笔记,是**带时间触发的系统**(闹钟/日历/提醒事项)——这一个换管道的动作,能消灭一类困扰你多年的「怎么又忘了」。

三级读完,「我记性差」被拆成了一张具体的架构图:编码弱的人修注意仪式,检索弱的人修触发反射,前瞻弱的人换触发管道——**而且你大概率会发现,自己的存储级根本没坏**:那些真正编码进去的东西(感兴趣的知识/十年前的尴尬瞬间)你记得比谁都牢——这个发现本身就是一次平反:你不是「脑子漏」,你是入口的守门员经常离岗、出口的门铃从来没装。边界:三级自测是功能画像,不是神经心理学测评——后者(专业的记忆评估)能给出规范化的分数与鉴别(排除其他原因的记忆问题),显著困扰值得做一次;把 Mem 当探针的观察期,别忘了 373 篇的咀嚼警告依然生效(外部记忆须有一部分穿过脑子);以及最后一条温柔的边界:**有些「忘」不需要修**——忘掉大部分无关紧要的信息是记忆系统的功能而非故障,施工清单只该包含真的在伤害你生活的那几类。

## 边界

同构强度 B 级:编码/检索/前瞻记忆的分级与 ADHD 的研究方向清晰,外挂 memory 定位瓶颈是 agent 工程实践,「三级探针」结构对应清晰;自测为功能线索,非神经心理测评。声明:Mem 未被证实为 ADHD 治疗手段;显著记忆困扰请专业评估鉴别。

## 今天就能试的 3 件事

1. **测编码**:今天存三条东西,每条写三行「为什么存」——写不出的比例,就是你编码级的读数。
2. **迁移前瞻任务**:把笔记里所有「未来要做」的条目,搬进带时间触发的日历/提醒——换管道,立竿见影。
3. **给钥匙安一个家**:挑一件高频丢失物,给它唯一的位置+放下时说一声——编码仪式,从一件东西开始。

「我记性差」是一句冤枉了存储层的粗糙判决——**真正的案情通常在入口和出口:编码时注意离岗,检索时门铃没装,前瞻任务走错了管道**。三级探针测清楚,记忆的架构图在手,修哪级、怎么修,第一次有了地址。
