# 为什么用 Focusmate 治 ADHD 的想理解自己的大脑，和给 agent 套 生成核心 + 缺失的执行层 是一回事？

> 探针系列到了最有人味的一站。370 篇拆过 Focusmate 的五种成分(承诺/在场/宣告/时间盒/汇报)并给了变体实验,目标是搞清「你在为哪味药付钱」;这一篇把同一组实验倒过来读:**你对哪味药反应最强,暴露的是你的动机系统的接线方式**——具体说,是一个对自我理解至关重要的问题:**你的执行系统,有多依赖「社会性电源」?**这个问题的答案,解释的远不止「要不要订 Focusmate」,它解释你为什么在公司是骨干、独自创业就瘫痪;为什么学生时代有考试还行、毕业后自由散养就崩盘;为什么疫情居家那两年是很多 ADHD 者的至暗时期。

理论底座依然可以铺得很实。ADHD 的动机研究方向有一个反复出现的模式:这个人群的困难高度**情境依赖**——同一个人在有即时后果、有他人在场、有新鲜感的情境里表现可以完全正常,一旦进入「后果遥远+无人见证+任务枯燥」的三无情境就系统性失能;换句话说,**ADHD 的执行系统不是坏了,是对「外部后果密度」的要求比常人高**——常人的执行系统自带内部电源(内化的目标感能驱动无观众的长跑),ADHD 的执行系统更像需要外接电源的设备,而**社会性在场恰恰是最便宜、最高能的外接电源之一**。对应到 agent 架构,这就是「生成核心+缺失的执行层」的动机版:LLM 本身没有「想完成任务」的内驱,它的一切方向感都来自外部结构(prompt/奖励信号/human-in-the-loop 的监督节点)——**没有人怪 LLM「不自律」,工程师只管把外部结构接对**;这一篇要做的,就是测出你需要接多粗的线。

测法:用 Focusmate 的成分变体,读三个刻度。**刻度一:社会电源的增益倍数**。对比三种条件下同类任务的启动率与产出:完全独自/弱在场(自习直播间当背景)/强在场(Focusmate 一对一,有目标宣告和结尾汇报)——三档的差距就是你的读数:如果强在场比独自高出数倍,你是**高社会依赖型**——这不是缺陷宣判,是配置说明:你的职业与工作形态选择里,「结构与他人的密度」应该是一等一的决策变量(远程自由职业对你是 hard 模式,团队工作、共享办公、固定搭子是你的基础设施而非可选项);差距不大的人,社会电源对你是锦上添花,你的瓶颈在别的层(回 393/394 篇找)。**刻度二:哪种「他人」有效?**——继续细分:陌生人(Focusmate)有效还是必须熟人?一对一有效还是群体也行?**只对熟人有效**的人,起作用的多半是关系性的问责(在乎具体的这个人怎么看你——385 篇讲过的承诺重量),你该投资的是固定搭子和每周真人 check-in;**陌生人也有效**的人,起作用的是更纯粹的「在场效应」,你的方案可以更便宜更即插即用(任何自习室/咖啡馆/直播间都是你的电源插座)。**刻度三:电源的衰减速度**。同一个搭子/同一种形式用一个月,效果衰减了多少?——衰减快的人,新鲜感是你电源配方的重要成分,方案要设计轮换(换搭子/换场地/换形式的节奏);衰减慢的人,恭喜,你找到的是可以长期依赖的稳定基建。

三个刻度读完,把结论升格成自我理解:**「我需要有人在场才能干活」这句话,从羞耻的自白,变成一条中性的硬件参数**——如同「这台设备需要 220V 外接电源」从来不是对设备的批评。这一步换栏(从品格栏到参数栏)的心理意义,做过的人都知道有多大:几十年的「为什么别人不用监督我就要」的自我审判,原来审的是一条接线方式。当然,参数确认之后责任也随之确认:**知道自己需要外接电源的人,把电源接好是自己的份内事**——瘫痪在没有电源的房间里等内驱降临,从此不再是意外,是配置失误。边界:情境依赖的失能若已覆盖「有电源」的场景(有人盯着也完全动不了)、或伴随显著情绪问题,那超出了接线问题的范畴,请让专业评估进场;社会性电源虽好,别让它变成社交耗竭的来源——内向的高社会依赖型是存在的(需要在场,又被在场消耗),这类人要精细调剂量:弱在场(直播间/图书馆)常是他们的甜点区。

## 边界

同构定位(本文未做正式 A/B/C 分级):动机的情境依赖性有 ADHD 研究方向,外部结构供给方向感是 agent 工程实践,「社会性外接电源」的结构对应清晰;三刻度为实践翻译,无对照研究。声明:Focusmate 未被证实为 ADHD 治疗手段,自测不构成诊断;全情境失能与情绪问题请专业评估。

## 今天就能试的 3 件事

1. **测增益倍数**:同类任务,独自一次、Focusmate 一次,记启动与产出——你的社会电源倍数,两天出数。
2. **测「他人」的型号**:约一次陌生人、约一次朋友,比效果——搞清你要的是在场还是关系。
3. **写下你的接线声明**:「我的执行系统需要 X 级别的外接电源」——贴出来,然后把这周的电源插好。

「有人看着才能干活」不是你的道德bug,**是你的硬件参数——而参数没有好坏,只有接对和没接对**。测倍数、认型号、看衰减:读懂自己的接线图,然后像工程师对待设备一样,把自己接进对的电源。
