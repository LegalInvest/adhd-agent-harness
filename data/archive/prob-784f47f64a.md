# 为什么用 Focusmate 治 ADHD 的不知哪些方法有用，和给 agent 套 human-in-the-loop 监督 是一回事？

> Focusmate 可能是 ADHD 工具箱里机制最「裸」的一个:约一个五十分钟的视频时段,和一个陌生人各自开摄像头干自己的活,开头互报目标,结尾互报结果——没有 AI、没有算法、没有花哨的界面,就是「有人看着」。它也是 ADHD 社区口碑最一致的工具之一:body doubling(身体同在)这个从社区实践里长出来的概念,Focusmate 把它做成了随叫随到的服务。但恰恰是这个「口碑一致」,掩盖了一个评估问题:**它对你有没有用、在哪类任务上有用、以及它到底是靠哪个成分起作用的——这三个问题,口碑一个都答不了**。有人用它写完了论文,有人订了会员从没约过第二次;有人做机械杂务时如虎添翼,一到深度写作就觉得镜头是干扰。「亲测有效」的社区证言,统计上叫幸存者偏差——**而拆开一个多成分干预、验证它对具体个体的具体作用,正是 agent 工程里 human-in-the-loop 监督的日常工作**。

先拆 Focusmate 的成分表,它至少同时投放了五种机制:**①承诺装置**——预约本身就是一个 commitment device:时间被具体的他人锁定,爽约有社会成本(这可能是对启动困难最有效的成分:「九点有人等我」比「九点我该开始」硬得多);**②社会在场**——被看见的状态下,刷手机的行为成本上升(哪怕对方根本没在看你);**③目标宣告**——开头那三十秒的「我这个时段要做 X」,强迫你把模糊的意图序列化成一句具体的话(这是被严重低估的成分:多少时段的失败,其实失败在根本说不出「要做什么」);④**时间盒**——五十分钟的硬边界,给没有形状的工作切出一个块;⑤**结尾汇报**——「做完了吗」的微型验收,给时段一个反馈闭环。五种成分,对不同人的有效剂量完全不同——**而你的订阅费买的是全家桶,你的评估任务是搞清楚自己在为哪几味药付钱**。

agent 工程对这种情况的处理方式,就是本篇标题说的 human-in-the-loop:**系统的每个组件是否在创造价值,由人设计对照、看数据、下判断**——翻译到 Focusmate,给出一个两层的监督协议。**第一层:总体有效性**(它到底帮不帮你)。指标别用「感觉」,用两个硬数:**预约时段的启动成功率**(约了的时段里,真正开工的比例——对照组是你没约 Focusmate 的同类时段的启动率)和**时段内的实际产出**(结尾汇报时能报出的交付物);跑两周,数字并排,总体判断就有了。**第二层:成分拆解**(哪味药在起作用)。做几个便宜的变体实验:**测承诺成分**——不开 Focusmate,只在群里和朋友约「九点各自开工,九点五十互报」(纯承诺+汇报,无视频)——如果效果不减,你需要的是承诺装置,免费方案够用;**测在场成分**——开一个不认识的自习直播间当背景(纯在场,无承诺)——有效的话,你要的是「环境里有人」,图书馆和咖啡馆也能供货;**测宣告成分**——独自工作前,把时段目标写一句话发给自己——如果这一句话就显著改善启动,你缺的是意图序列化的仪式,连人都不用。**变体测试的意义不是省订阅费(虽然可能省),是把「Focusmate 有效」精确化为「我需要 X 成分」——之后你的整个工具选择都会更准**。

两个 Focusmate 特有的观察点:**其一,任务类型分层特别重要**——社会在场对机械任务普遍增益,对高认知负荷任务则因人而异(有人被镜头轻微唤醒到恰好,有人被推过唤醒的峰值——耶克斯-多德森曲线的个体差异);数据按任务类型分开记,别混。**其二,盯住「预约率的衰减」**——工具最诚实的指标不是用时的体验,是三周后你还约不约;衰减常见的原因不是「没用了」,是预约这个动作本身也是执行功能任务(工具的元成本,微工具篇的老话题)——修法是批量预约(周日一次约好整周)或把预约绑进固定例程,先别急着退订。边界:body doubling 在 ADHD 社区有大量实践证言与初步的机制讨论,但严格的对照研究仍薄弱——这不妨碍你用,只提醒你**对自己也保持这个诚实:社区的热情证言不能替代你自己的两周数据**;线上陌生人同工不适合所有人(隐私/焦虑),不适就换成分等效的变体;显著的功能损害,正式评估永远是第一站。

## 边界

同构定位(本文未做正式 A/B/C 分级):body doubling 有社区实践与机制讨论方向但对照证据薄弱,组件拆解与对照验证是 human-in-the-loop 工程实践;成分实验为自测方案,非临床试验。声明:Focusmate 未被证实为 ADHD 治疗手段,本文不构成产品疗效背书;显著功能损害请专业评估。

## 今天就能试的 3 件事

1. **约一个明早的时段**,并记两个数:开工了没、结尾报出了什么——先拿到你自己的基线数据。
2. **本周做一个变体实验**:挑一天用「朋友群互报」替代视频时段——测测你付费买的到底是哪味成分。
3. **周日批量预约整周**——把「每次都要约」的元成本一次付清,别让预约本身成为新的启动难题。

「有人看着就能干活」不是你的耻辱,是一味真实的药——**但全家桶里五味药,哪味对你有效,证言给不了答案**。启动率、变体实验、按任务分层:自己当自己的验收官,订阅费才花在刀刃上。
