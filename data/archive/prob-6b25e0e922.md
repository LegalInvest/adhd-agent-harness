# 为什么用 Structured 治 ADHD 的想理解自己的大脑，和给 agent 套 生成核心 + 缺失的执行层 是一回事？

> 探针系列继续推进。Tiimo 篇(394)测的是时间的「感知」层,这一篇的 Structured 探测的是它的近邻但完全不同的一层:**一天的「决策」层**——你的大脑在「接下来做什么」这个每天重复几十次的微决策上,到底消耗了多少、卡在哪里。Structured 的产品形态(把一天排成自上而下的时间线积木,执行时永远高亮「现在这块」)本质上是把「日内决策」这个功能整个外包;而按探针系列的方法论,**一个功能被外包后你的系统发生了什么变化,就是该功能原本状态的读数**——agent 工程的对照实验:把 agent 的「下一步选择」从自由决策改成读取预排的计划,性能变化的方向和幅度,暴露的是这个 agent 的规划器质量。

先建立一个多数人没意识到的事实:**「接下来干嘛」是 ADHD 一天里最大的隐形成本项之一**。每个任务间隙都是一个开放决策点——而开放决策对 ADHD 的执行系统是三重税:**选项生成税**(把「所有该做的事」从记忆里调出来——工作记忆薄弱,调取残缺,于是永远感觉「好像忘了什么」)、**比较税**(哪件更急?哪件更值?——延迟折扣的偏差让「最响的」总是赢过「最重要的」)、**启动税**(决定之后还要点火);三重税叠加的结果,是任务间隙的著名黑洞:上一件事结束,下一件事开始之间,不知去向的二十分钟(刷手机/游荡/做了件不在任何清单上的事)——**一天六个间隙,就是两小时的碎屑**。Structured 把三重税里最贵的前两项一次性外包(选项和次序昨晚已定,执行时只读不选),于是它成了完美的探针:**上线两周,观察你的间隙黑洞发生了什么**。

三种典型读数,三种大脑画像。**读数一:黑洞显著收缩**——预排的时间线让你的一天顺滑得多,间隙从二十分钟缩到两分钟——**你的主要瓶颈确实在决策层**:选择的自由对你不是自由,是税;这个读数的自我理解价值在于翻案:你可能一直以为自己「懒散」「没有行动力」,数据显示你的行动力完好,**坏的是行动之间的路由器**——施工方向明确:把「预编译」推广到一切高频场景(前晚排日/固定例程/默认餐单/衣柜制服化——每消灭一个开放决策点,就退一笔税)。**读数二:排了也白排**——时间线躺在那里,你照旧漂移:这排除了决策层是主瓶颈,病灶在下游的启动层(点火问题,393 篇的故障 C)或更下层的状态层(384/395 篇)——**这个「阴性结果」同样值钱:它阻止你继续在「计划工具」这个品类里烧钱**;**读数三:排线本身成瘾,执行率反而下降**——你爱上了「布置一天」的掌控感,排线四十分钟,执行三十分钟——这个读数暴露的是另一种结构:**你对「元工作」的多巴胺响应高于「工作」**(规划的新奇感+完成感是即时的,执行的回报是延迟的——完美符合延迟折扣的偏好排序);自我理解的翻译:你不是「假努力」,你是被奖赏结构俘获了——修法是给元工作限额(排线十分钟封顶)并给执行加即时反馈(每完成一块打个勾的仪式感,别小看它)。

拿到读数后,还有一层更细的自我认识可以挖:**看你的时间线在哪一类间隙上最常断裂**。午饭后的第一个块?(转场坡度问题——大休息后的重启动是最陡的坡,给它配专用仪式)傍晚四点后的所有块?(状态曲线问题——你的下午电压不支持深度块,重排负载)被临时事件打断后的余下全天?(恢复问题——一个中断毁整天,你缺的是 386 篇的断点协议)——**断裂的分布模式,比断裂的总量更有诊断价值**:它把「我执行力差」细化成「我在 X 型间隙上会断」,而 X 型间隙是可以专项施工的。边界:决策疲劳与预编译的框架是功能视角,读数不构成诊断;如果「无法决定接下来干嘛」的瘫痪感严重且弥漫(连预排的线也完全推不动、伴随情绪低落),要考虑的就不只是执行层,抑郁等共病的可能请交给专业评估;预编译虽好,给一天保留一两个自由块——全预编译的日子对 ADHD 是另一种窒息,378 篇说过的:结构是拿来供电的,不是拿来判刑的。

## 边界

同构强度 B 级:决策疲劳、延迟折扣与工作记忆调取有 ADHD 研究方向,预排计划替代自由决策是工程对照实践,「间隙黑洞读决策层」的结构对应清晰;读数框架为实践翻译,无对照研究。声明:Structured 未被证实为 ADHD 治疗手段,自测不构成诊断;弥漫性瘫痪感与情绪低落请专业评估。

## 今天就能试的 3 件事

1. **先量黑洞**:今天数一数任务间隙的「失踪时间」——间隙数 × 平均分钟数,看看你一天漏掉多少。
2. **跑两周探针**:前晚排线,白天只读不选——黑洞收缩了、没变、还是排线成瘾?对号入座三种读数。
3. **找你的断裂模式**:时间线总在哪类间隙断?午后/傍晚/中断后?——把「执行力差」细化成一个可施工的坐标。

「接下来干嘛」这五个字,是 ADHD 一天里最贵的重复开销——**把它外包给一条预排的时间线,你的决策层就在数据里显了形**。黑洞收缩是路由器坏,排了白排是点火坏,排线成瘾是奖赏俘获:三种读数,三张施工图。
